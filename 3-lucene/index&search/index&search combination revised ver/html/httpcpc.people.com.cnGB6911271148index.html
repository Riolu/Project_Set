我们纪念李大钊同志，必须像他那样弘扬爱国主义精神，为中华民族的振兴不懈奋斗。今天，我们正在为把我国建设成为富强、民主、文明的社会主义现代化国家，实现中华民族的伟大复兴而努力奋斗。我们一定要以李大钊同志为榜样，以满腔的爱国热情，开拓前进，完成肩负的历史重任。
