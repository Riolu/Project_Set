沪上9所高校志愿者为比赛全力服务 “小冰花”绽放花滑世锦赛

2015-03-26 11:38:52    来源：上海文明网   

昨天，2015国际滑联世界花样滑冰锦标赛拉开帷幕，舞台中央的各国运动员是当之无愧的主角，而活跃在赛场内外的数百朵志愿者“小冰花”，为保障赛事的顺利举行，同样功不可没，是名副其实的无名英雄。

“奉献、友爱、互助、进步，追求卓越、全力以赴，为祖国争光、为城市添彩！”3月17日上午，志愿者上岗宣誓仪式在东方体育中心举行，数百名志愿者齐声宣誓，将以实际行动努力践行志愿精神。

上海外国语大学、华东政法大学、上海理工大学、第二工程技术大学、上海对外经贸大学等9所沪上高校参与志愿者招募。775名志愿者为自己取名“小冰花”，赛事期间将分布在12个部门的62种岗位上，协助竞赛组织、开幕式服务、贵宾接待、媒体服务、安全保障、医疗服务和观众服务等。

志愿者历来都是重大赛事的一道美丽风景线，无论是世博会的“小白菜”“小蓝莓”，还是世游赛的“小番茄”，志愿者们都以青春阳光的微笑和周到细致的服务，赢得了国内外宾客的赞誉，成为了解读中国、宣传上海的绝佳名片。
