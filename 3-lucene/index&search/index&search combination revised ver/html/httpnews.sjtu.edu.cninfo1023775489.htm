打造辐射全国大学生创业园区
[文汇报]上海交大零号湾与辽宁创业机构等签约合作[图]
[发布时间]: 2015年10月26日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 沈依灵

 

上海交大媒体与设计学院的“火源智能硬件”创业团队的乔哲（右）和储程在进行产品功能演示。 新华社发  

上海交通大学的众创空间零号湾全球创新创业集聚区昨天迎来了3家知名创投机构入驻，并与辽宁大学生就业创业服务基地、四川成都电子科技大学“一校一带”园区达成战略合作协议，未来将打造辐射全国的大学生创业园区。  

成立半年多的零号湾，已经吸引了63家创业企业入驻，创业企业的从业人数超过350人。昨天，随着近4万平方米的科技大楼正式启用，上海创业接力集团、晨晖创投等创投服务机构也相继入驻。零号湾园区和辽宁大学生就业创业服务基地、四川成都电子科技大学“一校一带”园区达成战略合作协议，三方将共享创业导师、创投导师资源，相互为所在地的园区推荐年轻校友入驻园区创业，同时将不定期沟通和交流园区建设和发展经验。  

零号湾整合了政府的政策资源、高校人才和创新资源、企业的资金资源、管理资源和市场需求资源。昨天，零号湾还迎来了包括上海市大学生科技创业基金会（EFG）、晨晖创投等多个知名孵化器在内的多家全国知名专业孵化器入驻办公，关注智慧医疗、智能硬件、智慧城市、互联网金融等领域的创业培育，为创业者们提供兼具专业性和综合性的孵化服务，实现从单一孵化平台向多元化创业生态体系的升级，打造多样性创业孵化器生态链。
