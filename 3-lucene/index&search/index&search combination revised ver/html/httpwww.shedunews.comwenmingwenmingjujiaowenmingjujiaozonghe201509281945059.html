嘉定承中华优秀传统文化 塑农村精神文明之魂

2015-09-28 09:54:19    来源：上海文明网   

近年来，嘉定立足本区深厚的文化积淀，以“教化之城，礼乐嘉定”为文明创建主题，牢牢把握培育和践行社会主义核心价值观，传承中华优秀传统文化，不断推动美丽乡村建设，农村精神文明建设取得了长足进步。

敦风厉俗，促进道德风尚养成

通过创设载体平台、大力培树先进典型、广泛开展志愿服务活动，推动道德风尚养成。一是抓好道德典型学习宣传。广泛开展“中国好人”、“嘉定好人·每月一星”、 “十佳乡贤”、“最美家庭”等宣传教育和道德实践活动，通过拍摄微电影、张贴公益广告、组织公众海选、编印好人事迹漫画等形式，引导群众践行文明理念，凝聚向上向善的强大正能量。二是拓展道德实践教育阵地。着力推动道德讲堂街镇层面的全覆盖。将百姓宣讲团、上海百老德育讲师团等各类教育资源有机融入到100余个道德讲堂中，使广大群众在唱、诵、悟中提升道德判断力和道德荣誉感。大力推进乡村学校少年宫建设，为农村孩子增长见识、培养兴趣、学习技艺搭建平台。三是广泛开展“邻里守望”志愿服务。目前，嘉定区从事邻里守望志愿服务志愿者逾8万人，志愿服务团队250多个，各街镇已实现“邻里守望”志愿服务团队全覆盖，服务项目达100多种，涉及家政服务、文体活动、心理疏导、医疗保健、法律服务、救助帮困等多个领域。在志愿服务制度化、常态化方面，积极推动志愿者通讯录在街镇、社区层面的公开化，打造“一镇一品”服务项目，形成了“e嘉人”微信公众号等一大批品牌项目，满足了群众的服务需求。

礼乐教化，传承乡土嘉美人文

嘉定区广泛组织群众性主题实践活动，通过创新阐发礼乐文明在新时代的丰富内涵，引导广大群众传承和弘扬优秀传统文化。一是开展市民文明礼仪培训。自2013年启动“嘉定区市民文明礼仪培训工程”以来，用三年时间，对区内市民广泛开展的礼仪知识教育，打造和谐有序的人文环境。至今已经编印下发《嘉定区市民礼仪读本》等五本近40万册培训教材，组织包括网络教学等在内的3000余场培训，累计参训市民近50万人次。二是推进中华经典诵读活动。按照学校、社区、社会“点、线、面”梯次推进的工作思路，目前全区已组建市民经典吟诵团队161个，初步形成了区—镇（街道）—村（居）三级吟诵网络。三是传承好家风好家训。整理注疏嘉定望族先贤的家风家训，并在市民中广泛开展“五个一”现代优秀家风家训征集活动。同时，扎实抓好家风家训的宣传、展示和推广工作，在全区26万户家庭中开展最美家训评选、家训评议、家训书写等系列活动，利用微博、微信及台报网等媒体渠道，广泛传扬好家风、弘扬新风尚。

以文化人，描绘乡村和美画卷

坚持文化惠民、乐民、育民，不断改善农村发展软环境，描绘美丽乡村新画卷。一是深入开展“我们的节日”活动。通过举办民俗文化展示、传统服饰展演、诗词歌赋征集等群众性活动，探索礼乐文明在弘扬家庭美德中的作用，推动“三个注重”建设工作日常化、具体化、形象化、生活化。二是完善公共文化设施网络。以创建国家级公共文化服务体系示范区为契机，完善区、街镇、村居三级公共文化设施网络体系。通过“百姓大舞台”、“百姓大展厅”等贴近群众的平台载体，实现公共文化服务与群众需求的有效对接。三是优化公共文化服务机制。以公共文化资源供给为抓手，加大政府投入，组织开展系列文化活动，不断满足广大外来务工人员精神文化需求，更好促进外来务工人员等新市民群体融入嘉定、安居乐业，在促进文化融合、增进价值认同、强化社会凝聚力方面发挥积极作用。
