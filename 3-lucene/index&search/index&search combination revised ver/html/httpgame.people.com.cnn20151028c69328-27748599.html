年画是我国古代一种寓意吉祥的形象。旧时候，每当春节来临，家家户户都把房院打扫得干干净净，在堂房、卧室、窗旁，门上以及灶前，院内的神龛上，贴上焕然一新的年画，既用以创造喜气洋洋的新年气氛，又借以祈求上天赐给幸福，消除灾祸与不幸。

年画的取材内容极为广泛，诸如历史故事、神话传奇、门神类、金童子、戏曲人物、世俗风情以及山水花鸟等。画中形象质朴、自然、简练单纯，很多画面都有情节性、装饰性、趣味性，色彩鲜艳强烈。

年画的表现形式随着时代的发展也在发生变化.......

招财童子是幸运符，手托宝盘或元宝、如意等，有求必应，十分灵验。有招财，开运，纳福，高中，缘起，必胜，平安之意。

基于美好的祝福和愿望，治图传媒推出招财童子《年画》系列，旨在对中国文化的传承和发展，为了新时期的年轻人了解和喜爱中国的传统文化，让年画艺术新发展，愿招财童子给你带来健康福运，让你的生活处处充满欢乐梦想！（朱传戈）
