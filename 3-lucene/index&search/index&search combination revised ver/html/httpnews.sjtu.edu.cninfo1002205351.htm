你所关心、关注的，就是我们所不断努力的
[群众路线特稿]交大机关党委推“机关服务评议系统”架起“沟通便民桥”[图]
[发布时间]: 2014年02月25日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 孙佳

　　群众利益无小事。在党的群众路线教育实践活动中，机关党委为促进机关改进作风，提升管理服务质量，推进“服务机关、效率机关、责任机关”建设，推出了“机关服务评议系统”。

　　“你所关心、关注的，就是我们所不断努力的”，“机关服务评议系统”上线伊始，就给自己牢牢地定好了位。一个小小的网络窗口，架起了服务师生的“沟通便民桥”。“桥”的一端连接着师生关切的学习、工作和生活；“桥”的另一端连接的是机关的工作形象和服务质量。

　　“遇到好人好事，找它发个表扬贴”、“到机关办事情，碰到不顺心的事，找它吐个槽”、“食堂饭菜需要换换口味，找它提个建议”……运行半年下来，“机关服务评议系统”在师生中慢慢传播开来，成了师生信赖的贴心朋友。

　　小窗口 大平台

　　2013年6月21日，机械动力工程学院丁国良教授觉得心里被“堵”了一下——两周前自己上报科研院的科研项目还没有审批下来。因为按照惯例，一般一周左右就能完成审批。

　　难道审批没有通过？是不是上报材料不够？是不是哪个环节出了问题？……

　　“应该没问题的呀！”丁国良教授思来想去，还是没想到问题究竟出在哪？

　　想起前一段时间机关党委推出的“机关服务评议系统”，抱着试试看的态度，丁国良教授决定借助这一网络平台挑一挑“刺”。

　　“2013年科研项目的审批效率比以往降低很多。过去一般为一个星期左右就可以拿到审批件，最近一年变成两个星期还经常完不成审批。”他在“机关服务评议系统”中直言不讳。

　　令他没有想到的是，当天科研院有关负责人就给他回复邮件，向他表示歉意，感谢他提出批评意见，并答应尽快了解实际情况作出回复。

　　6月24日，工作人员仔细查阅了2012年2月至2013年6月期间丁国良教授的14份《订立技术合同（协议）责任保证和审批单》原件，发现丁国良教授所反映的情况“基本属实”。当天下午4点，丁国良教授接到有关负责人的电话。电话中，该负责人再次向他表示歉意，并汇报了项目管理中心拟采取的改进措施。

　　“非常赞赏科研院能在如此短的时间内提出提高合同审批效率的多种方法，这些措施如能实行应该会有很好的效果！”丁国良教授对处理结果比较满意，连连说了两个“没想到”：“没想到事情这么快就得到解决，没想到这个服务评议系统还真灵！”

　　截至目前，“机关服务评议系统”已接收评议30多件，其中投诉、批评15件，相关部处都在第一时间与评议人联系，迅速妥善处理，重要的及时向分管校领导汇报，并及时向评议人答复反馈处理情况，基本上都得到了评议人的谅解和称赞。

　　小转变 大文章

　　“门难进、脸难看、人难找、事难办”，是群众对机关的一般印象。在党的群众路线教育实践活动中，如何抓住契机，第一时间反馈师生意见，提升机关服务质量？机关党委一直在思考这个问题。

　　“开门迎评”！让各机关部处的工作置于全员监督之中，敞开门，请人来评议。在充分论证基础上，经2013年4月22日第8次党务专题会研究，学校决定在主页开设“机关服务评议”窗口，以方便广大师生对各部处在服务意识、服务作风、服务能力、服务效能等方面的表现进行评议评价、提出意见建议。

　　推出“机关服务评议系统”是上海交大打造“效率机关、服务机关、责任机关”的一个举措。2013年5月15日，机关党委在学校主页发布开设“机关服务评议”网络窗口的公告。公告一发布，就赢得一片赞许。

　　为了保障“机关服务评议系统”的运行，学校出台了《“机关服务评议”网络窗口管理办法》。“机关服务评议”窗口链接放在学校主页“服务”栏内。日常的技术支撑和运行维护，由网络信息中心、宣传部网宣办具体负责。

　　评议人的意见和建议会进入专设信箱，由机关党委负责接收、查阅、梳理、分类。每日查阅一次，每周梳理一次，每月汇总一次，每半年组织研讨一次。

　　机关党委根据评议内容的分类情况及轻重缓急，分别提交相关部处、分管校领导进行调查研究、妥善处理。比较简单的服务作风方面的问题，由相关部处3个工作日内解决，并向评议人反馈、听取满意度意见；比较复杂的涉及多部门的管理服务问题，由分管校领导督办，一般10个工作日内解决，并向评议人反馈、听取满意度意见。

　　机关党委还负责搜集、整理、归档、保存每份评议的原始内容、相关部处的处理意见、反馈满意度等情况，每半年向党办、校办、纪委、组织部、人力资源处、文明办、信访办等相关部门汇报一次，作为机关人员聘任、干部晋升、部门考核及党建和精神文明评优奖先的重要依据之一。其中，严重违反校纪校规、触碰“工作底线”的机关人员，年终考核档次直接确定为“基本合格”或“不合格”；按照聘用合同约定，及时终止聘期或解除聘用合同。

　　“滴水见太阳，小转变里蕴含着大文章”，有老师在系统留言这样说道：“机关的事务和工作面广量多，一些细微的转变和细节的提升，往往给人带来一股清新和亲近，这关系到师生对学校工作的认同。”

　　小细节 大格局

　　常言道：细节决定成败。

    “简政放权、创新服务；信息公开、阳光服务；一门受理、统筹服务……”在党的群众路线教育实践活动中，与开设“机关服务评议系统”相呼应，机关党委还推出了10项服务承诺制度。这10项服务承诺，涉及到机关服务的每一个细节。

　　承诺制度贯彻落实情况如何？机关党委给出了一份“清单”：

　　——2013年下半年以来，学校推行机关结构调整和改革，通过合署、合并、划转，优化了职能配置，减少了部门职责交叉和分散，27个部处优化整合为26个；

　　——通过简政放权，精简审批环节，下放审批事项，增加服务项目，提高服务效率。人力资源处减少对教职工一个月内因私出国的审批等6项审批事项（环节），211/985工程办公室对于不能免除的审批项目增加“一条龙送审”服务；

　　——通过举办“部处长论坛”、举办机关青年管理骨干培训班、组织发动机关青年参与大学生军训工作和暑期社会实践活动，提高机关党员干部的服务能力，提升综合服务师生能力；

　　——通过部门内部协同，完善网络办公，实行一口受理、一门式办公，尽量解决服务窗口分散、办事流程繁琐、推诿扯皮问题。学指委组建“学生服务中心”，把学生处、团委、武装部、就业中心等窗口服务集中到一个楼面，让学生到一个地方就可以办成多个事项；出入境服务中心开通“网络办公系统”，所有审批都在网上实现，最大程度地减少师生到机关办事的奔波之苦。

　　……

　　从细节入手，每项承诺都有非常具体的规范性要求；一诺千金，每项承诺背后承载和支撑的是学校建设“服务机关、效率机关、责任机关”的大格局。
