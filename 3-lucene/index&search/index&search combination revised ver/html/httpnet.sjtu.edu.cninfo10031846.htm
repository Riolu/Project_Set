校园网“现金交费”已开通网上充值

2015-10-19  点击：[]

各位老师：      

 

    目前校园网“现金交费”已开通网上充值啦，只需登录我的数字交大---》我的校园网---》账单查询---》    

现金充值，即可通过支付平台直接缴费。      

 

   网上现金充值步骤如下：

 

1. 用户可以用jaccount账号登录交大门户：http://my.sjtu.edu.cn我的校园网络-账单查询-现金充值-交费后弹出上海交通大学缴费系统页面，用户可以选择微信和网银等付费-在该页面下方：是否修改缴费单位名称：选择否，按下一步继续操作直到充值完成；

 

 

2. 支付完成后请勿关闭支付页面，请耐心等待页面返回，以保证充值到账；

 

3. 充值完成后可在历史交费记录中查看交费结果。如若充值不成功，请您记下充值时间到网络信息中心查询处理；

 

4. 需要发票的用户在一周内到网络中心开取；

 

5. 提交交费后没有弹出上海交通大学缴费系统页面，可能插件在浏览器中被禁用了，只要设置允许弹出即可。

 

如果您在充值过程中遇到任何问题，可以直接拨打我们热线34206060。
