热门推荐

《求是》
求是杂志是中共中央的机关刊物，是党中央指导全党全国工作的重要思想舆论工具，是党在思想理论战线的重要阵地。
《红旗文稿》
　《红旗文稿》为求是杂志社主办的、面向国内发行的综合性政治理论半月刊，是中共中央机关刊《求是》杂志宣传事业的重要组成部分，是党的宣传思想理论战线的重要阵地。
 
《党建》
《党建》杂志由邓小平同志亲笔题写刊名，创刊于1988年，是中宣部主管的综合性党刊。《党建》杂志连续三届获全国期刊界最高荣誉――“国家期刊奖”。
《人民论坛》
　人民论坛杂志创刊于1992年，由人民日报社主管、主办，江泽民同志题写刊名，2006年1月1日由月刊改成双周刊。为“全国百种重点社科期刊”、“国家期刊奖百种重点期刊”。
《国家治理周刊》
　《国家治理》周刊为我国首家以治理为主题的主流期刊，是贯彻落实十八届三中全会精神的重要载体。
《人民论坛・学术前沿》
　《人民论坛・学术前沿》 《学术前沿》杂志2012年创刊，以“对重大现实和理论问题的学术分析”为定位，侧重为政策决策提供学理支撑，是中宣部、全国哲学社会科学规划办公室课题成果唯一指定发布期刊。
《思想政治工作研究》
　　由中共中央宣传部主管、中国思想政治工作研究会和中宣部思想政治工作研究所主办，旨在通过理论研究、调查研究推动思想政治工作创新实。
中国行政管理
《中国行政管理》杂志是由国务院办公厅主管，中国行政管理学会主办，是新中国成立后创办的第一本研究行政管理的专业刊物，是反映政府行政管理理论与实践的全国中文核心期刊。
《前线》
　　《前线》杂志是中共北京市委主管主办的党刊。创刊于1958年11月，由彭真同志题写刊名，邓拓同志任主编。
《国家行政学院学报》
　　《国家行政学院学报》是国家行政学院主办的大型综合性政府管理理论刊物，是我国各级政府公务员和政府管理理论教学研究人员必读的权威刊物。
《行政管理改革》
　　指导行政管理改革的理念和方向，推动行政管理改革深入发展；为政府管理改革提供咨询建议，为建立完善的中国特色社会主义行政管理体制提供智力支撑。
《中国发展观察》
　　《中国发展观察》是国务院发展研究中心主办的、以发展为主线，以经济为重点的综合性月刊。发展中心的专家群，是月刊的强力支持者和基本作者。 。
《理论前沿》
　　《理论前沿》是中共中央党校主办的综合性理论半月刊 。1996年4月创刊,胡锦涛同志亲自撰写《发刊词》,引起很大社会反响。
《当代中国史研究》
　　当代中国史研究是当代中国研究所主办、当代中国史研究杂志社编辑出版的专业学术刊物，入选中文社会科学核心期刊、中文社会科学引文索引来源期刊。
《政治经济学评论》
　　主办者为中国人民大学书报资料中心，主编单位为中国人民大学经济学院。在作为学术期刊正式出版之前曾由中国人民大学出版社连续出版了15 辑。
