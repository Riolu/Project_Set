第三届《汉字听写大会》开播 改版升级全新亮相

 

   中新网7月17日电 第三届《中国汉字听写大会》总决赛首场比赛将于今晚在中央电视台科教频道正式播出。普通观众能否全程跟着听写？旗鼓相当的“学霸”对手需要过招多少轮才会决出胜负？成人听写团有没有高手表现惊艳？这些答案都将一一揭晓。

 

  “让书写在古籍里的文字活起来”

 

  《中国汉字听写大会》系第21届上海电视节白玉兰奖最佳“综艺栏目”奖得主、2014央视收视领跑者。今年暑期，它改版升级，全新亮相。

 

  2015年节目录制之前，总导演关正文给参加节目的文化嘉宾们发了一封公开信，坦率地承认节目曾在考题设计上出现了一定的问题，一些词汇为难而难，减弱了文化传播的价值。

 

  “要使中华民族最基本的文化基因与当代文化相适应、与现代社会相协调，以人们喜闻乐见、具有广泛参与性的方式推广开来，把跨越时空、超越国度、富有永恒魅力、具有当代价值的文化精神弘扬起来”，习近平的讲话广为人知。但如何对古籍进行“基因检索”，真正“让书写在古籍里的文字活起来”却并非易事。

 

  为了准备考题，节目题库组的专家和老师们查阅了大量的经典古文，从中精选了两千多个字词备用。而每一个字词的背后，还有大量的释义、用典等资料整理。

 

  “大车无r”的“r”字是2015年的考点字。原文出自《论语》的一段话。子曰：“人而无信，不知其可也。大车无r，小车无，其何以行之哉？”这句话的意思是，孔子说：“人要是失去了信用或不讲信用，不知道他还可以做什么。就像大车、小车没有了车辕与轭相连接的木销子，它靠什么行走呢？”“r”的意思是牛车车辕与轭相连接的木销子。

 

  关正文认为，原文有趣，例句就会被激活；例句有趣，字词就会被激活。观众接收到的，就不再是生冷的字词，而是历经千年积淀的、鲜活的前人智慧，以及为了更好感知这种智慧我们需要克服的语词障碍。再难的字词，也因此有了道理，有了趣味，有了传播的价值。

 

  南京师范大学教授、百家讲坛主讲人郦波已经连续三届担任汉听大会的文化嘉宾。在他看来，汉听题库的改变也透出节目的价值观：不是来秀汉字，或是炫认识多少汉字的，而是通过汉字了解其背后的文化。“一字一文化，一字一价值，一字一智慧，一字一历史，节目就是要把汉字背后的东西呈现在大众面前，呼唤出其隐藏其中能引起大众共鸣的我们民族的DNA，在文明的长河中，找到曾经支撑我们整个民族文明向前走的核心、有价值的东西。”

 

  双人上场同题对抗 不同难度字词呈梯级分布

 

  往届汉听大会都是由某省选拔出某一所学校的5位选手代表本省出战，今年改为多校优胜者组队，力求全省实力最强的5名高手入围。从今年3月起，节目组在全国31个省、市、自治区开展巡回赛，建立了从学校到区县、再到地市以及省级的层层选拔体系与汉字听写体验活动。5月下旬，选拔全部结束，包括港澳台代表队以及国际联队在内的36支代表队全部产生。

 

  今年还采取了全新的竞赛方式。往届汉听都是单人轮流上台书写，有的选手尽管总体表现优秀，但如果“运气不好”，碰上一个难写的字被挑落马下，对手则可能“坐收渔利”。今年改为每次都是两位选手同时上场，书写同一题目，都写对或者都写错比赛继续，直到其中一人失败离场。这对竞赛的公平性和戏剧性也是一大提升。

 

  采用双人对抗赛制后，每一对新上场的选手的第一题都将从常用易错字开始，第二题为应知应会类字词，第三题开始则进入带有经典例句的高难度题型，点评嘉宾郦波、蒙曼、康震、张一清等“名嘴”负责为观众带来超越字词的文化内涵解读。这样一来，不同难度的字词将呈梯级分布，观众可以全程参与互动。

 

  在晋级程序上，今年亦有调整。总决赛第一轮依旧是36支代表队180人参加，共进行9场，每场4个队比赛，产生5位晋级选手。他们将是本场表现最佳的5名选手，可能来自不同队伍，因此以往某位“大神”以一己之力带队晋级的场面将不再出现。而优胜者晋级之后，不再以省队为单位，而是以个人名义继续参赛。45位选手外加3位鼓励港澳台代表队中表现最优秀的外卡选手晋级半决赛，半决赛每16人为一场，共进行3场，每场产生4位晋级选手，12位选手晋级总决赛冠军争夺战。他们再抽签分组，两两对决，最终产生本届汉听大会的前三甲。

 

  大人和孩子谁更会写字？

 

  在往届的汉听大会里，成人听写团一直是关注的焦点。2013年首届汉听大会组建的成人听写体验团，10位成人选手里只有3人书写正确简单的“癞蛤蟆”一词；2014年成人听写体验团扩容至100人，成员来自社会各个行业，结果成人的书写正确率依然令人汗颜，也因此引起全社会对民众普遍的汉语语词水平的反思与审视。

 

  今年的汉听大会，国内主流媒体组建现场媒体竞赛团，不再仅仅是一个书写率对错的统计数据，相互之间也要分出高下。中新网、新华网10余家媒体，通过招募选拔来自民间的汉字高手，每队10人组建自己的媒体竞赛团参赛，在“第四现场”竞技。

 

  彭敏是在朋友圈看到招募媒体竞赛团的消息后决定报名的，不过正式参赛之前，他还是有一些紧张与担心，因为对自己的实力并不是很有信心，直到拿了个人单场冠军。比赛中，他不止一次写对小选手以及其他团员都没写对的字。彭敏说，今年“第四现场”竞赛性更强，选手一旦写错，自己的分数与名次就有可能被赶超。为了参加汉字听写大会，他曾三次坐高铁往返北京与安阳之间。因为杂志社每周二、四、五都要上班，身为编辑的他必须在参加完节目录制后当天返京。

 

  周慧是武汉大学文学院的研二学生，她在当地的选拔赛中就拿到了第一名。参加第一场比赛前，周慧和同伴都很放松，只想体验一下。可比赛真正一开始，“第四现场”前面的大屏幕不停地刷新着每支队伍的成绩，数据图上的柱形统计数字你追我赶，大伙儿一下子就紧张了起来。比赛一结束，原本还想着出去逛街的大家都纷纷跑回酒店拿起字典认真复习。周慧说，之前总觉得自己认识的字很多，几轮下来才知道汉字如此博大精深。“现场对手们都很厉害，小朋友们更是令人敬佩。”

 

  据悉，此次节目的录制地是在甲骨文的发现地河南安阳，可谓“汉听”回到了汉字故乡。
