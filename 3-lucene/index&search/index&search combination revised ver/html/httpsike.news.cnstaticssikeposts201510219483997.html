扎克伯格：使命、用心、向前看——你会改变世界

思·锐享

为读者提供最有价值的观点

发表于  10/27 06:30   约7分钟

　　近日，Facebook创始人马克·扎克伯格在清华大学经管学院做了一场中文演讲，本文根据演讲内容整理。

　　清华是非常好的大学，这里的学生正在创造很多重要且富有创新性的东西；你们是未来科技、商业及其他行业的领导者。今天我想谈一谈“改变世界”这个话题。

　　很多人会告诉你怎么创立企业，怎么解决问题。今天我想关注一个不同的问题：不是“怎么去创立”，而是“为什么创立”。这就是使命的召唤。

　　今天我想告诉你们三个故事。第一个故事是关于“使命”。

　　2004年，我创立了Facebook。因为我觉得，在网上跟人进行沟通是很重要的一件事。那时候，互联网上有很多网站，你几乎可以在上面找到所有的东西：新闻、音乐、书、电影，还可以买东西。然而，对于我们生活中最重要的事物——人与人之间的联系，却并没有哪个网站为之提供服务。

　　请大家看看这个房间，你们看到了什么？不是桌子或椅子，你关注的是人。这正是人类的特点。每个人都想跟朋友和家人多联系。当我们彼此联系、互相分享时，我们和家人、朋友之间的关系会更加亲密，生活也会变得更加愉悦。

　　我们的企业之所以愈发强大，是因为我们和客户进行了更好的沟通；而我们的社会变得更强大，也是因为我们通过联系和互通有无而掌握了更多信息。

　　我创立Facebook的时候，并不仅仅是要创立一个公司。我希望借此解决一个非常重要的问题：把人们连接在一起。当我去了解中国的公司时，例如阿里巴巴和小米，我看到了相同的事情——你的使命感会让你更专注。

　　我的第二个故事是关于“用心” 。

　　如果你有了使命感，你甚至不需要有完整的计划，只管往前走吧！你只需要更加“用心”。我在哈佛大学读书的时候，每天晚上都和朋友边吃比萨边讨论未来。

　　我们推出Facebook第一版时，我记得我们非常高兴，因为我们的产品把同学们连接起来了。当时，我们想，总有一天会有人创造出连接世界的产品。有趣的是，我没想到我会是这个人。当时我只是一个大学生。

　　我觉得，像微软和谷歌这种大公司，他们有能力去开发这个产品。他们有好几千个工程师和上亿用户，他们应该去开发一个国际社交网络平台。但是，他们为什么没这样做？

　　我常常思考这个问题。我们当时只是大学生，我们没有计划，也没有资源。我们是怎么创造出这个拥有十五亿以上用户、世界上最大的互联网社区的呢？我们只是更“用心”。在我们前行路上的每一步，都会有人对我们的想法提出质疑。

　　我们遇到了很多困难，也做过很多次改变。

　　一开始，facebook只是一个面向美国学生的小产品。那时有人断言：“Facebook只是给学生用的，它永远不会成为具有影响力的产品。” 可是我们继续前行，终于，我们把Facebook成功推广给了所有的美国人。

　　后来，又有人这样说：“好，现在很多人都用上了Facebook，但他们很快就会不用了。”可是我们还是没有停止努力。如今人们仍在用facebook，因为人跟人之间的联系是很重要的事情。

　　但是还有人质疑：“可能他在美国会使用，但在其他国家便不用了。”我们依然没有放弃，后来我们成功地把facebook推广到了其他国家。

　　也有人说：“社交媒体永远赚不到钱。”我们却不曾停下脚步，事实证明我们创建了一个强大的企业。

　　但依然有人在否定我们：“用户是不可能在手机上使用Facebook的。”我们继续进行改进。今天，我们已经成立了以移动客户端为中心的公司。在每一次被质疑时，我们并不知道那些问题的答案。事实上没有人能知道。我们能坚持对产品进行完善，是因为我们“用心”。当时也有很多其他公司在开创社交媒体，但他们惧于这些质疑。

　　我们认为，对连接世界来说，社交媒体非常重要。虽然我们无法预知每个问题的答案，但我们依然相信，我们能够继续做出有助于人们畅通联系的事情来。我们没别的方法，只是会更加用心。一直以来，我们未曾放弃努力，如今已经有十五亿人在使用Facebook。不要因为需在前进过程中做出调整，就罢手放弃。

　　中国有一句话我觉得很好：“只要功夫深，铁杵磨成针”。只要坚持努力，你会改变世界。

　　我的第三个故事是关于“向前看”。

　　马云说过一句话我很喜欢：“和15年前比， 我们很大；但和15年后比，我们还是个婴儿。”为了重要的使命，你会去了解更多事物，你也会发现要做的事情变得更多。十年前，我们的目标是连接十亿人。因为之前并没有互联网企业能做到， 所以当时我们觉得这已经是一个很大的目标了。

　　当我们实现了这个目标，我们开始明白，十亿只是一个数字，我们真正的目标是连接世界上的每一个人。这就困难多了。全世界差不多有三分之二的人无法使用互联网，要想把他们连接起来，我们必须扩大整个互联网的覆盖范围。

　　要达成这个目标，我们需要解决很多问题。世界上有十亿多人居住在没有网络的地方，所以我们需要开发新的技术——例如卫星和飞机，把他们用互联网连接起来。

　　还有十多亿人没钱去使用互联网，所以我们需要降低互联网的消费成本。此外，约有二十亿人没有接触过电脑或互联网。对此，我们需要制定新的方案，把他们连接起来。

　　三年前，我们成立了Internet.Org，试图扩大互联网的覆盖范围。我跟董事会说：“我觉得我们要花十亿多美元。”他们问我：“这个东西怎么赚钱？”我告诉他们：“我不知道。”但是我知道，将人与人连接起来是我们的使命，这才是特别重要的一件事。

　　我们必须向前看，我们现在还不清楚整个计划的具体细节。但是如果我们去做对人们有益的事，在将来，我们也会成为获益者。这就是“向前看”的意思。每走一步，你都可以创造出新的东西。

　　以前你觉得不可能的事情，现在都成为了现实。即便现在你面临非常大的挑战，只要你努力，也都能克服。请一直向前看，要知道，中国的历史也是不断创新的历史。

　　中国向世界贡献了四大发明：造纸、印刷术、指南针和火药。对于创新来说，学习是最重要的。在清华，你们有很好的学习机会，这有助于你们开创美好的未来。

　　几年前，我的妻子Priscilla在北京的一个医院学习。她选择来北京，是因为她想要在中国跟优秀的老师学习。我的中文很糟糕，但我还是很喜欢学习中文。

　　只要你多学习，就能在生活中创新，也会在工作上有所创新。在你开始做之前，不要只问自己怎么做，要问自己：为什么做？

　　你应该相信你的使命，听从它的召唤去解决重要的问题。要用心，不要放弃，要一直向前看。你们可以成为全球的领导者，可以去做提升人们生活质量的事情，可以用互联网影响全世界。

　　（作者：扎克伯格，美国社交网站Facebook的创办人）
