“少数民族地区普通话普及度抽样研究”

开题会在新疆师范大学召开


教育部语信司 

 

　 4月10日，国家语委“十二五”科研规划2013年度重大项目“少数民族地区普通话普及度抽样研究”开题会在新疆师范大学召开。

 

　　课题组组长、新疆师范大学教授李志忠汇报了课题研究内容、研究重点和难点、研究思路和预期成果。来自教育部语言文字应用研究所、新疆民语委、新疆大学等单位的专家学者对课题开题进行了指导。

 

　　与会专家充分肯定了课题的重要意义，认为选取少数民族聚居或多民族杂居等典型地区，深入农村和边远地区，通过实地考察、入户调查等方式进行抽样研究，对我国现阶段少数民族地区普通话普及情况作出数据描述、分析，有利于深入了解少数民族地区普通话普及的成绩与经验、困难与问题，为国家相关政策的科学制定提供依据。与会专家指出，调研前期准备意义重大，应进一步加强内容设计，精确选点，合理抽样，确保调研过程的科学性与调研成果的有效性。

 

　　教育部语言文字信息管理司派员参加开题会，并对项目的过程、成果与经费管理等提出要求。
