把舞台搭在田间地头 邯郸启动农民才艺大赛(图)

来源：人民网-河北频道    2015年10月28日09:44


图为《欢乐乡村・百里挑一》2015年邯郸市农民才艺大赛・鸡泽海选现场表演。

人民网石家庄10月28日电（张梦琪）近日，邯郸市“欢乐乡村”・“百里挑一”才艺大赛活动在鸡泽县曹庄乡大言寨村正式启动。

此次农民才艺大赛按照“政府主导、群众主体、社会支持、市场运作”的理念，通过“百里挑一”才艺比赛，将选出一批有特长的文艺骨干，让文艺骨干、文艺爱好者变成文化带头人，带动更多的群众参与“欢乐乡村”，不断满足农民群众对文化生活的新期待、新需求。

“我们把舞台搭在田间地头，就是为了让更多的群众登上‘欢乐乡村’舞台，做幸福生活的主角。‘欢乐乡村’舞台由群众来演，由群众来唱。在欢歌载舞中，给农民群众带来欢乐、带来幸福，让更多群众享受生活的美好。”邯郸市委宣传部常务副部长潘璐说。

本次才艺大赛与文艺志愿服务及公益广告宣传相结合，利用邯郸市交警支队的屏幕宣传车，宣传13条公益广告和文明交通常识。
