电子邮件 / 常见问题

2010-11-06  点击：[]

问：电子邮件POP3、IMAP、SMTP服务器的地址是什么？
答：POP3服务器：mail.sjtu.edu.cn 
    IMAP服务器：mail.sjtu.edu.cn
    SMTP服务器：mail.sjtu.edu.cn

问：校园网电子邮件服务有何限制？
答：普通用户信箱大小限制教工为5G，学生为2G，发送邮件附件大小限制为50M（对方收件服务器可能有更严格的限制）

问：垃圾邮件太多，如何防范和处理？
答：交大电子邮件系统安装有反垃圾邮件系统的邮件网关，可以在邮件到达交大邮件服务器之前拦截大部分的垃圾邮件，现在默认已启用该功能。据统计，目前邮件网关已经帮您拦截了90%的垃圾邮件。同时，交大电子邮件系统有垃圾邮件分拣功能，可以协助你处理一部分垃圾邮件问题。防范垃圾邮件最主要的还是尽量不要将Email地址留在互联网陌生网站上。

问：我的电子信箱为何无法接收邮件？
答：请用帐号密码登陆 http://mail.sjtu.edu.cn ，检查您的信箱空间是否已超过限制。
如果因为网络故障原因，电子邮件无法正常投递，正常情况下发送方会收到“信件无法投递”的提示信件。

问：邮件服务器要求安全身份认证，该如何设置？
答：如果您使用的是outlook express，进入”工具”-->“帐户”-->“邮件”-->“属性”-->“服务器”，在”发送邮件服务器”-->“我的邮件服务器需要身份验证”前的复选框内打勾，按”确定”退出即可。  

问：我为什么无法发送电子邮件？
答：如果您使用的是客户端软件收发Email，请检查客户端设置是否正确，很有可能是发送邮件所需的身份验证的选项没有选上。
如果您使用的是Outlook Express，请按照上一问答中检查相应设置。 

问：别人老是收不到我发的邮件，怎么回事？
答：邮件不能到达，可能原因有下：
    交大邮件服务器到对方邮件服务器之间的网络状况不好；
    收信人地址有误；
    对方的邮件服务器拒绝。
一般来说，邮件不能送达对方，自己会收到退信，退信里会返回邮件不能送达的原因。如果你不能读懂退信内容，请把退信转发给 service@sjtu.edu.cn ，我们帮你分析一下具体原因。 

问：在outlook express中为什么无法打开附件？
答：如果您使用了outlook express 6.0,那么默认设置中病毒防护是打开的，所以您无法打开某些附件。
解决的办法为，打开outlook express,进入“工具”-->“选项”-->“安全”将“不允许保存或打开可能有病毒的附件”前面的复选框去掉。
