      • 【健康解码】巧用食物补气血

        经常能听到一个说法：女人容易气血不足，脸色发黄，暗淡无光泽。美丽是人人都想拥有的，可是如果不好好的保养和调理，美丽就会与您擦肩而过。平时生活的饮食调理是很重要的，我们身体的营养全部来源于食物的摄取，因此饮食的科学、合理、营养是不能忽视的。

      • 【健康解码】 高压高、低压低 脉压差大不用理？

        血压分为收缩压与舒张压，收缩压与舒张压之间的差称为“脉压差”。例如，收缩压为120毫米汞柱，舒张压为80毫米汞柱，脉压差则为40毫米汞柱。通常，人们在测量血压时往往只注意血压的高低，很少会关心脉压差的大小，其实脉压差大小在医学上具有重要的临床意义。

      • 【健康解码】气血虚的女人长什么样？

        气血虚是很多女子都会出现的一种状态，一般和贫血、月经过多、等伴随出现。中医认为气是人体生命活动的动力，血是人体生命活动的物质基础，气和血是构成和维持人体生命活动的两大基本物质，为身体提供能量。气血亏虚主要表现为体内阴阳失调、血脉运行不畅等。

      • 【健康解码·时间都去哪儿了】“不难受”的高血压 健康吗？

        很多高血压患者认为虽然测量着高，但没有明显的不适症状，应该是没事儿吧？这就导致了诊断治疗的不及时。平时身体看起来一点问题也没有，但是在某一个时间，高血压可能会突然无声无息地袭击了你，知道了这一点，你还会认为血压高而身体没症状，是健康的吗？
