打造生命共同体 共享幸福和谐校园

2009-12-09 14:39:34    来源：上海市长宁区初级职业技术学校   


一、项目背景

㈠ 立项原因

1998年，长宁初职因聋校设点布局调整，教育对象的改变使办学难度剧增，教职工队伍只留下了18个“老弱病残”，学校陷入了办学的低谷。于是，每一位教职员工都产生了强烈的生存危机，形成了与学校共甘苦、共命运的朴实情感。由此为起点，我们开始了长宁初职“生命共同体”的建设。

㈡ 工作基础

在学校“生命共同体”建设的进程中，长宁初职不断积淀学校文化，在一个个生动鲜活的学校叙事中，提炼和发展了学校“生命共同体”的观点——“学生、家长、教师、学校是命运的共同体、利益的共同体，也是学习的共同体、价值的共同体，更是幸福的共同体。”


二、工作思路

㈠ 指导思想

在科学发展观、和谐社会观等观念的基础上，我们提出了“生命共同体”理念。其核心概念是和谐。学校“生命共同体”建设要追求学生、家长、教师个体自身的和谐状态，要和谐处理好学生、家长、教师、学校之间的关系，要实现“幸福学生、阳光教师、和谐校园”的办学目标，塑造一大批幸福学生，凝聚一个个弱势家庭，建设一支不断超越自我的研究型阳光教师队伍，营造暖意融融的和谐校园氛围。

㈡ 基本思考

学校“生命共同体”具有五个层次，由低层向高层发展，依次是“命运共同体”、“利益共同体”、“学习共同体”、“价值共同体”、“幸福共同体”。“命运共同体”是最基本的共同体认知，只有当人们认识到大家命运相息的时候，才会将利益的共赢作为自己各自发展的一种共同追求，形成“利益共同体”。进而当人们为共同利益而努力的时候，才需要在共同愿景下，通过共同学习获得各自的成长，形成“学习共同体”。于是，每个人的独立价值开始彰显，这时候，人们不同的价值观需要进一步在沟通中相融，形成“价值共同体”。在得到他人的理解与认同之后，自我实现才能够无限接近最大化，“生命共同体”才会上升为最高境界——幸福相生，形成“幸福共同体”。综上所述，学校“生命共同体”建设直接关注学校及学校相关人群的生命需要，关注各生命体之间的命运相息、利益相系、学习相长、价值相融和幸福相生。我们运用马斯诺需求五层次理论，来进一步介绍学校“生命共同体”建设的理解。

 


三、基本做法

㈠ 支持式学生课程，让每个孩子都能行

在“生命共同体”建设中，以“每个孩子都是金子”的教育哲学思想为指导，我校自主开发了针对智障学生特点的初职教育支持式学生课程体系：通用基础课程+专业技术课程+岗位体验课程+人与社会课程+综合康复课程，编写了系列校本教材80余册，通过个别化的学习进步档案、快乐考试等载体实施，关注每一个学生的发展。以职业适应、社会适应为核心，注重学生的体验实践，强调充分利用各种教育资源，合理运用现代信息技术，创设有利于学生发展的生态环境。注重培养学生主动参与、依法办事、自信负责、敬业吃苦、合作共处、感恩惜福的职业情感与态度，使他们成为自食其力的劳动者。在办学过程中，我们共同感受命运相悉、利益相系，挖掘每个学生的潜在能力，放飞生命，让每个孩子都获得成功的体验。

㈡ 支持式教师课程，让校园和谐阳光

在“生命共同体”建设中，形成了学校教师发展观：“教师是好人，是阳光，是钻石：做不断自我超越的智慧型阳光教师”。支持式教师课程实施十分重视学校文化生态的作用，强调充分发挥多种学习载体的功能，采取综合性的课程实施策略，并时刻关注教师学习共同体的知识管理。开发了“两个层级、四类课程”，即职业化层级的“师德修炼课程”和“心理修炼课程”，专业化层级的“观念修炼课程”和“技术修炼课程”，通过主题性校本教研、网络教师学习论坛、教师职业导航等途径实施，探索了教师成长记录册、阳光教师奖章序列等评价方式，促进了学校教师群体更快更好地发展，整个团队铸就了积极向上、团结奋斗的精神，这种精神正不断渲染到每一个人的心灵，使整个校园都阳光起来。

㈢ 支持式家长课程，让家校合作无间

根据学校课程资源理念、服务家长理念、感化家长理念、引动家长理念、家校互动理念，结合智障学生及其家长心智特点，开发了适合于学校家庭教育指导的支持式课程内容，形成了良好的家校互动。每一位家长都是待开发的宝藏，学校创新了“家长工作室”、“教育支持者”、“家校你我他合同”等系列工作载体。每天下午，有家委会的二名家长到学校家长工作室办公，接待其他家长的来电、来访；定期参加“职场演练”，参与学校民主管理；督评日常教育教学，组织策划系列亲子活动；收集各类教育信息，出版“家长心声”小报，成为家校沟通交流的桥梁，增进了家校间的沟通了解，赢得了全体家长、师生的肯定和美誉。


四、主要成效

通过10年生命共同体的持续建设，历届学生职业资格证书考核通过率达到99.10%，轻度智障学生就业率100%，学校支持式学生课程研究获得了上海市第八届教育科研优秀成果一等奖。2007年学生艺术作品展成功举办，出版了《放飞生命，让每个孩子都能行》作品专辑，获得了广泛的社会反响，众多媒体竞相报导，国家领导人习近平同志还曾与我校学生小记者亲切交谈，给予鼓励。近六、七年来，我校的家长满意度始终保持在97-98%的高位，家长中有的被评为“上海市好家长”，有的走上了“国际特奥家长论坛”，学校也成为了全国家庭教育研究基地。教师中先后涌现了全国“十杰”教师、上海市园丁奖、上海市新长征突击手、上海市金爱心教师、长宁区师德“十佳”等众多优秀教师，教师团队获得了上海市学习型团队创建奖，支持式教师师德修炼课程获得上海市普教系统十佳师德项目，出版了《上海教育丛书》等16本专著、3本教育刊物专辑。学校多次受到教育部、市教委领导的肯定与表扬，先后获得全国青年文明号、全国模范职工小家、上海市文明单位、上海市生命教育试点校等90项全国市区集体称号。

10年来，每一个初职人没有停止追求一流工作业绩的脚步,存在于共同体中生命共生互长，学生、家长、教师、学校发展喜人，生命共同体的理念、做法已根植于每一个人的内心。


五、书记感言

作为一所承载着特殊使命的特殊教育学校，长宁初职全体教职员工始终孜孜不倦于为生活在社会最底层的弱势群体默默奉献。构建和谐校园的关键之一就是焕发师生内在的激情，让学校的理想与办学目标，具体体现在培养学生身上，体现在学校的和谐发展上。他们传承特质、结合境遇、和谐发展，构成了学校“生命共同体”理念的精神实质和动态模式。

“每个孩子都是金子”，“每一位教师都是好人、钻石”，“每一位家长都是待开发的宝藏”，以此来给予学生、家长、教师高期待和高认同。学校有愿景，才有发展方向。多年来，长宁初职校的“生命共同体”理念，结合实践境遇，不断发展，不断与周围环境达到一种和谐共生的历程。
特殊教育是反映社会文明的一面镜子，教育要让智障孩子有尊严并幸福的生活，这是社会公平的要求。在这里，他们体验着每位特殊学生点滴进步所带来的快乐，不断缔造出学生、家长、教师与学校生命共同成长的幸福!
