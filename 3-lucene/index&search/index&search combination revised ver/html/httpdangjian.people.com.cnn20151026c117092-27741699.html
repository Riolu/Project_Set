四川省广元市昭化区：让“严”字成为每一名干部的操守

2015年10月26日16:48   来源：人民网－中国共产党新闻网

自今年5月28日“三严三实”专题教育活动开展以来，广元市昭化区始终以严的作风、严的纪律要求干部，“严”字已成为每一名党员干部的操守。

活动中，该区已举办了党政主要领导干部读书班2期、昭化大讲坛5期，邀请了中国人民大学教授任剑涛等12名专家辅导“四个全面”，集中解读了《党章》、《纪律处分条例》等规矩纪律，举办27场县级领导党课、114场乡镇部门党委（党组）书记党课。

照镜子，从严从实查找问题

每次集中研讨不少于两天，每一名县级领导干部发言不少于30分钟，发言材料要经过严格的审核……这样一些“规矩”让每一名干部感受到了“严”。

该区在“三严三实”专题教育活动中，已围绕“严以修身”、“严以律己”、“严以用权”三个专题进行了研讨。在发言材料审核上，每组的召集人会集中预审各县级领导发言观点，从一开始确保了材料主题突出、观点鲜明、问题精准，再经过四层审核，对内容空洞笼统、照搬照抄、整改措施不明等问题及时进行纠正，杜绝了发言“千人一面”的现象。在研讨会上，区四大班子查找问题36个，县级领导查找175个，乡镇部门查找2675个，分别列出“不严不实”问题清单，建立了整改台账，统筹解决教育实践活动未整改问题、庸懒散浮拖问题、领导班子思想政治建设问题和“5+2”突出问题1927个，一大批涉及群众切身利益问题得到有效解决。

该区还整理出各级党员干部学习心得体会、理论文章和专题发言材料共60篇，印发“严以律己”学习研讨成果集，将优秀篇目上传到昭化门户网、党员e家和各类学习QQ群，方便党员干部学习，持续发挥专题学习研讨作用，为践行“三严三实”注入新的动力。

正反检视，敲响“不严不实”警钟

“三严三实”是焦裕禄精神最大的体现。为学有榜样、做有规范、赶有目标，7月18日至23日，昭化区举办了党性教育异地培训班，55位领导干部前往兰考、红旗渠等地进行党性锻炼，接受焦裕禄精神、红旗渠精神的洗礼和传承，增强了奋力推进宜业宜居幸福昭化建设的思想自觉、行动自觉。

学员们普遍反映，本次异地培训是一次洗涤思想、震撼心灵、增强党性的学习机会，是学习生涯中一段难忘的历程，达到了学有所获、观有所感、思有所悟、境有所升的学习效果。

异地培训以后，该区参训学员自发撰写心得体会60余篇。

榜样的力量是无穷的，后进的警示同样发人深思。在学习交流会上，区国土资源分局对“不严不实”问题深刻反思，让各级领导干部反躬自省、引以为戒。

下发学习读本2300余册、光盘1500余张，召开各类学习会120余场次，以周永康、薄熙来、令计划、苏荣等严重违纪违法案件及区内查处案件为反面教材，深入剖析反思，举办“读书思廉”活动9场，全区党员干部拒腐防变能力明显提高。

该区成立了3个督查小组，开展“效能亮剑”专项督查16次，发现并督促整改问题110余个。切实解决发生在群众身边的“四风”和腐败问题，年内初核各类违纪案件线索52件，党纪政纪立案41件，其中“四风”案件9起，优亲厚友案件3件，套取涉农资金案件5起，给予党纪处分5人。

联动开展，把每一次学习化为工作的动力

该区印发了关于联动开展“三严三实”专题教育的通知，明确区乡村各级党员干部学习目标任务，将“新常态”大讨论和专题学习研讨相结合，部署了科级单位每半月1个学习专题的任务，分别开展“严以修身”“严以律己”“严以用权”“谋事要实”“创业要实”“做人要实”专题学习研讨和问题查摆。到目前为止，全区95个科级单位都已结束了前三个专题的学习研讨任务。

“三严三实”专题教育活动的开展是为了更好地推动工作落实。该区联动推进“七大”行动，把推进工作贯穿专题教育始终，实施政治生态优化行动、精准扶贫攻坚行动等“七大行动”，谋划储备项目293个，规划投资608亿元，稳步推进中纺集团60万吨油脂加工及物流、昭化城区至广元快速通道等一批重大项目。大力扶持发展“4+2”产业，积极推行“4+3”股权量化分红产业发展模式和“2+3”业主带动贫困户产业发展模式。推广“智力入股、反包倒租、就地加工、电商营运”的“雷鸣”模式，开启大学生创业和偏远乡镇、移民库区现代农业发展新路径。全面推行“自力更生、美好家园”行动，集中整治美化了村社环境卫生。高标准举办首届“昭化造”商品博览会，推出一批优质的“昭化造”产品。

专题教育活动期间，共清理建区26年来党建方面制度112项，制定了“废改立”专项实施方案，废除制度24项，修改制度13项，确定新建制度72项，为形成良好政治生态和培育优良干部作风提供了坚强制度保证。

(王万新 姜德辉）
