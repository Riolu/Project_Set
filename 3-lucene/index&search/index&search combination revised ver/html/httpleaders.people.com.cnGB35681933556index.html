中国苏州高新区简介
　　苏州高新区于1990年11月开发建设。1992年11月被国务院批准为国家高新技术产业开发区，1997年被确定为首批向APEC成员开放的亚太科技工业园，1999年被国家环保总局认定为国内首家“ISO14000环境管理体系国家示范区”，2000年被外经贸部、科技部批准为国家高新技术产业开发区高新技术产品出口基地，2001年被批准建设国内首家国家级环保高新技术产业园，2003年被国务院批准成立出口加工区。
