关于2015年上半年“高雅艺术进校园”工作总结的通知

2015-09-09 16:03:11    来源：上海教育新闻网   

各高校宣传部、各区县教育局宣传科：

现将对2015年上半年“高雅艺术进校园”工作进行总结，请各单位将2015年上半年（2015年1月1日-2015年6月30日）开展的“高雅艺术进校园”活动的演出发票复印件、开展情况统计表（纸质版一份及电子版）、工作总结报告（纸质版三份及电子版）报送至上海大学生艺术实践基地育人联盟“高雅艺术进校园”活动办公室，以便及时进行汇总，予以决算。

请于2015年9月25日前，将材料纸质版快递至：上海市长宁路491弄36号410室。联系人：黄老师 苏老师；联系方式：52393087,62525555-400。电子版报送至：gyysjxy2015@163.com，信封封面和邮件主题请注明：2015高雅艺术进校园。

注：

1、开展情况统计表范例和发票复印件范例详见附件，电子版请在上海教育新闻网文明在线“高雅艺术”专栏下载。

2.发票复印件请注明所看剧目、演出院团和演出时间。

3.同一剧团不同演出剧目的票款请分别开具发票。

4.请务必填写联系人电话（包括手机和座机）及邮箱。



附件：1、发票范例

2、“高雅艺术进校园”活动开展情况统计表
