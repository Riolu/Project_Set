台北公共自行车为何那么火

2015年10月28日 07:18:14 来源： 福建日报
  • 评论0
  • 打印
  • 字大
  • 字小

　　赴台游客骑乘YouBike，悠游于台北的大街小巷，感受不一样的城市风情。

　　YouBike是台北公共租赁自行车的英文名，因为显眼的黄橙色，也被称为“小小黄”。由于租金实惠、骑乘方便，越来越多的台北民众、外地旅客选择YouBike，悠游于台北市的大街小巷，解决了“步行太远，搭车太近”的尴尬距离。

　　在台北摩托车盛行的当下，You Bike为何会从乏人问津走向人气爆棚，它的魅力何在？

　　门槛低：免押金低租金

　　陈先生在台北内湖区的一家电视台工作。每天上班时间，他都要中转两样交通工具，先是从家里出发，搭乘捷运去内湖捷运站，之后再去就近的YouBike站点骑自行车20分钟到电视台。“捷运加YouBike，比开车更快捷、低碳，何况20分钟的骑行正好可以健身一下，两全其美啦！”陈先生说。

　　“YouBike的最大魅力在于它的低门槛，注册方便，又能免押金、保证金。”陈先生告诉记者，民众只要到其官网注册下，填个手机号和悠游卡号（或信用卡），不用缴交押金、保证金，也无需带身份证之类。注册完毕之后，就可以刷下悠游卡轻松骑行。

　　不过，“小小黄”最吸引人的一点还是租金便宜。“前30分钟免费，30分钟之后每30分钟支付10元（新台币，下同，10元新台币约合2元人民币），假如借车时间超过4小时，那么每超过30分钟就需要加倍收费。”You Bike事业部发言人刘丽珠对记者说。

　　到台北旅游的杨小姐特地办了张悠游卡，骑YouBike感受台北街巷的独特风情，“踩上YouBike才知道，从双连捷运站骑到大稻埕最热闹的迪化街，原来只要10分钟，还能看到与平日不同的街景”。

　　设计巧：智能防盗科学设点

　　YouBike不光是租金要比捷运、公交廉价，更重要的是还必须配备好车，吸引市民来骑行。刘丽珠坦言，YouBike都是捷安特自行车，每辆造价1.3万元，而且车架要比一般淑女车粗，轮胎加厚，更采用内变速器，把故障率降低。这样不仅民众好骑好操控，也能应对YouBike周转率高的挑战。

　　这样的好车，又不用缴交押金、保证金，它会不会容易丢失？对此，刘丽珠告诉记者，台北YouBike的失窃率很低，仅为0.38%，远低于伦敦的1.25%、巴黎的20%。“这其中除了民众素质比较高外，还要归功于捷安特与宽丰工业合作开发出的‘防恶作剧锁’。”同时，每部YouBike都安装无线射频辨识系统，以利于追踪，防止被盗。

　　“公共自行车考虑得最多还是便利性，要让民众在几秒钟之内，用悠游卡轻松地借还车。市民可以用手机实时查询各个YouBike站点车辆的多寡，还可以查找最近的站点。”一位工作人员告诉记者，YouBike在设计上坚持以人为本，比如坐垫高低可调节，右扶手处带三级变速，座垫中间挖空不会积水等，非常人性化。

　　当然，科学的站点设置更是关键的一环。刘丽珠告诉记者，YouBike站点的设置，都要围绕每一个商圈、居住区、学校、车站、捷运站等进行科学设定，最大程度方便民众的转乘，实现公共交通运输工具的无缝衔接。

　　配套全：周到服务专门车道

　　周到的服务也是“小小黄”赚取高人气的法宝之一。

　　“车子故障率高、站上缺车，常常是使用者的主要抱怨。”刘丽珠说，为了防止热点站没车租，而冷点地方车满为患，YouBike就利用统计数据“预先配置”车辆，24小时不间断调度。比如配合学生下课时间，事先把车送到学校附近的站点；趁晚上预先清空住宅区附近的停车柱，方便民众回家停车；利用半夜空车较多的时候，尽快把车送到早上用车量大的站。

　　不只是让市民有车可骑，还要把车况维持在最佳水平。YouBike有关人士表示，每一辆车每两个星期至少检查一次。骑满一百次的车，也会被系统锁定在站上，等维护人员来做例行检修。像在人流最频繁的捷运公馆站，每天超过3万次借还车，调度车就要来来回回数十趟。而调度员、维护员、服务团队犹如练兵般地营运操练，加上信息系统的配合，使得“小小黄”们在各站点有序流动，让台北租车顺畅，人气爆棚。

　　台北市还特地为绿色出行的人们辟出专用的自行车道，让民众安心骑行。此外还设置了专用红绿灯，倡导人车分道。目前，台北市拥有自行车道约495公里。

　　为引导民众绿色出行，台北市有意减少机动车位，计划将站点从现在的196个增加至2016年底的300个，发挥其路网效应，让更多人骑着自行车上下班。（记者李向娟　文/图）
