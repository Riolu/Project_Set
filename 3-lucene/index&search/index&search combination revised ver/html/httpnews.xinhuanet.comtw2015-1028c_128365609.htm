台湾流行二手市场淘好货

2015年10月28日 07:20:06 来源： 海峡导报
  • 评论0
  • 打印
  • 字大
  • 字小

　　今年苹果新品首发，台湾没有挤上首拨名单，iPhone6s本月9日才在台开卖，尽管时间延迟了，可依旧受到民众热捧。排队买新机的队伍中，不乏一些“年年抢头香”的“果粉”。事实上，他们不是有钱到可以任意买买买，而是代表一种很潮的消费习惯：以旧换新，不时换新机用。而支撑这种消费观的，则是台湾社会二手市场的流行。

　　尤其像手机、汽车这样的消耗品，更是台湾二手市场的常客。车主、机主拥有一台新汽车、新手机时，会极其注重养护，目的就是为了有朝一日卖掉它时能有个好价钱。

　　导报记者在台湾知名的二手买卖网站“雅虎拍卖”上看到，这里每天都有大量二手市场交易，3C产品是其中大宗，其他如衣物、包包、挂饰等，也有不小的交易量。一个有意思的现象是，每逢9月份苹果新机发布前，前一代的产品都会在此网站上密集挂售，“九成新”、“八成新”的二手产品比比皆是，至少从展示照片来说，看起来都没有大的磨损。

　　一名在大陆求学的台生告诉导报记者，台湾比较注重环保，大家对二手物品接受度提高，她从小就经常逛跳蚤市场，这里的东西普遍都便宜很多，虽然是“废物利用”，但很多东西很新。从小，学校也会经常举办跳蚤市场等活动，比如学校开运动会的时候，就有很多摊位，同学们喜欢在这里淘宝。

　　除了这些生活用品外，这两三年来，台湾二手书网上市场也很夯，最主要原因是台湾出版量大，每年平均出版4万多本新书，新书淘汰快速，生命周期短，直接造成二手书市场活跃。此外，在台湾，阅读渐成娱乐活动，娱乐性的轻小说、大众文学容易被转手抛售，成二手书占比最多的书种。当然，由于经济不景气，台湾年轻人普遍买不起房，租屋族得靠不断出清旧书才能购买更多新书，这也助推二手书的活跃。
