    新华社社长蔡名照会见日本时事社社长西泽丰

    2015年10月22日 20:28:29 来源： 新华网

    10月22日，新华社社长蔡名照在北京会见日本时事社社长西泽丰。新华社记者兰红光摄

    新华网北京10月22日电 新华社社长蔡名照22日在北京会见了日本时事社社长西泽丰。

    蔡名照说，中日两国是近邻，媒体在两国关系中发挥着十分重要的作用。新华社与日本媒体有着长期交往与合作。新华社愿进一步加强与包括时事社在内的日本媒体的友好往来，交流借鉴发展经验，深化各领域合作，为中日友好传递信息、搭建桥梁。

    西泽丰对新华社的发展表示赞赏。他希望与新华社保持密切交往，加强合作，共同致力于中日友好关系的发展。
