新书旧事感触多

――《柳斌谈语文》读后

教育部语用所 张世平

今年的国家语委咨询会上，得到老领导、原国家教委柳斌副主任亲赠的语文出版社出版的新书《柳斌谈语文》。春节前接《语言文字报》编辑同志电话，知柳主任点名要我写一篇读后感，以参加报纸将开展的“我读《柳斌谈语文》”征文。展读文集，往事历历，内心感慨中寓温暖。

称柳主任为老领导，并非泛泛而言。三十年前（1985年）国家教育委员会成立时，我在原教育部成人教育司工作。在礼堂召开的大会上宣读名单介绍领导，一个名字引起了一阵嗡嗡的感叹――从副省长岗位转任过来、排名靠前的柳斌副主任，其时只有四十八岁，在班子里最年轻。我是国家教委的一员，柳主任自然是我的领导，然而，柳主任分管教育的重镇基础教育，并曾分管语言文字工作兼任国家语委主任，却不分管当时我所在的司局，年轻小干部没有直接接触领导的机会。印象深刻的是一次改善机关福利时听举例说，柳主任工资不高家庭负担重，午餐自己带，是头天吃剩的米饭炒白菜……

和柳主任有了具体的工作联系，是在2001年我任教育部（国家教委1998年改回原名）语言文字应用管理司副司长以后。这时，柳主任虽已卸任国家教委党组副书记、副主任职务，但担任了全国人大常委及教科文卫委员会委员，以及国家语委咨询委员会副主任，无论是从宣传贯彻《国家通用语言文字法》的角度，还是从国家语委及语用司的工作上说，我和老领导的接触反倒十分紧密了。本书中的讲话，相当部分“我在场”，第一时间聆听了指导和教诲；很多活动如国家语委对北京市的语言文字规范化达标评估、中华经典诵读、学生规范汉字书写大赛等，还是我所在司主办、我参与组织的。近年来，在有些场合，我还有幸能够追随柳主任发表些看法。因此，目光在篇章间流转穿行，是研读也是重温――而所涉及的内容涵盖的范围，正是我近十几年来工作的领域，一篇接一篇地读下来，深有会心，感慨良多，真有不能释卷之感！

我认为，柳主任以分管基础教育的领导身份而兼管国家语言文字工作，是语言文字之幸。教育和学校，历来是语言文字工作的基础阵地，柳主任以他工作的便利、特别是学养和识见的支撑夯实了这一基础。同时，柳主任在倡导素质教育时，特别强调语文教育和学生语言能力的重要性，从推进基础教育角度说，也是抓住了关键环节。柳主任任期内对语言文字工作的推动，是历史性的；就基础教育、素质教育而言，据我所知，更是如此。

读《柳斌谈语文》后，我有以下三点感想、感受或曰感慨：

第一、全书浸透了一个老共产党员、一个中华民族优秀儿男对祖国、对人民、对民族、对母语、对语言文字事业的拳拳深情――这一特点，既体现在柳主任对汉语“准确、鲜明、生动、简练，既适合发展人的逻辑思维能力，也适合发展人的形象思维能力” （《自序》）的阐述中，体现在“我们民族的语言是世界上最优秀的语言之一，一定要重视我们自己优秀的语言文化传统”（《热爱母语，重视作文》）的教诲与呼吁中，也体现在对不按规定使用普通话、规范汉字，不分场合乱用或滥用方言、繁体字、外语以及网络语言，胡乱简化汉字、不尊重中华文化传统等现象的批评与鞭挞中。

第二、柳主任从教育、特别是基础教育和素质教育角度，从国家教委、国家语委领导和国家总督学的立脚点上谈语言文字工作、谈语文教育，站得高、看得远、分析得透，我感到高屋建瓴、鞭辟入里这样的成语用在这里，是恰切的。这从《教育战线要重视语言文字工作》《汉字简化是历史的进步》《还有四亿人不能用普通话交流》等篇都可清晰体察。此外，柳主任还在他的位置上，为语言文字事业争取到了中央领导的指示和支持，并记录、保留了珍贵的史料，如《立此存照》，即讲述了江泽民同志1992年对语言文字工作三点指示的由来。

第三、领导人的文集，往往官话、套话多，工作班子起草的文稿多，读之有时代有工作却没作者――没有或少有领导者个人的思想观点和风格性格。然而，这本书不是这样。《柳斌谈语文》既是作为国家语委国家教委领导的柳斌同志的，也是、更是作为专家、学者、诗人的柳斌先生的，书中汇集的，不是官样文章和别人的观点，而是作者自己在既往岁月里提炼出的黄金、累积成的珍珠。这些真知灼见，或在广众中或在促膝时，源源“谈”出，谈的特点很突出，“谈”主自身的特色很鲜明。如2000年在全国小学语文识字教学交流研讨会上，柳主任在论及汉字教学方法的改革时，谈到要处理好知识与技能、理论与实践、工具性与文化性以及继承与创新诸种关系，就是如此，可谓是分析深入、论述精辟，非有深切体会不能言。这来源于柳主任师范大学中文系出身，和大学、中学语文教师，语文教研室主任，素质教育研究者与倡导者，语文教学与社会语言生活关注者和调查者，以及诗词书法爱好者这一系列身份。因此，柳斌同志关于“语文”的观点，是成系列的、发展着的，是爱憎分明、旗帜鲜明的。

以上所言，带有随感性质，难以概括本书的全部特点。它所包含的丰富内容、所反映的多彩时代，是需要我们细读深品的。

去年在西安参加中华经典诵写讲系列活动颁奖仪式时，曾偷闲随侍柳主任至古城墙下文化市场觅闲章。印章很便宜，二三十元，习字者常用的文辞触目皆是，但柳主任想要的“中国梦”“民富国强”却没有。摊贩喃喃说刻这些没人买，我在侧默然，深深地为七十八岁老人孜孜LL的情怀而感动。后来，我托西安友人专去定制了这两个章，同时自作主张加刻了两枚作为小礼物向老领导致敬，文辞是：教倡素质语推雅言；春风扬柳演武成文。

                                        （2015年2月24日正月初六）
