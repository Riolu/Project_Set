招聘遛狗员需大学学历　专家：若为装点门面不应提倡

2015年10月28日 07:37:30 来源： 中国新闻网
  • 评论0
  • 打印
  • 字大
  • 字小

　　日前，有企业以月薪3000元招聘大学学历的专职遛狗员引发了网友的热议，这到底是对人才的不尊重还是市场的另类需求？对此，有HR表示，这类招聘的出现体现了社会分工及市场需求；专家表示，如果是企业为了装点门面，则不应被提倡。

　　网友称是人才浪费 学生家长表示不体面

　　日前有媒体报道，沈阳有家企业招聘专职“遛狗员”，职位要求是大学学历，有爱心，做事认真，待遇是月薪3000元人民币，有社保，不必坐班。企业称老板养的几条狗没人照顾，招聘的另一深层目的是为老总寻找一位助理。但招聘启事发出一个月后仍无人应征，目前已被企业撤下。

　　中新网生活频道浏览了一些招聘类网站后发现，确实有招聘遛狗员或宠物助训员这类岗位，月薪在2000-4000元，学历和工作经验不限，其中，一家公司对招聘的宠物助训员的职责要求是每天遛狗，记录并观察宠物的状态。

　　这样“奇葩”的岗位招聘引发了网友们的热议。有网友认为这种招聘是对人才的不尊重，称“大学四年高等教育不是为了遛狗，企业这种招聘，是对人才的不尊重，会造成教育的浪费。”更有网友认为，这种招聘是对人才的侮辱和歧视，称父母含辛茹苦供自己读书，本科毕业怎么能屈才去做这类工作。

　　但也有网友认为，遛狗员这种岗位算不得人才歧视，是市场另类需求，本科生不一定干得好，仅仅望文生义便排斥这个岗位才是职业偏见，并称遛狗也是技术活，并不是人们想象中的那么低级，只要是自己喜欢的工作就是一份好工作。也有网友跃跃欲试，说虽然觉得岗位和学历无关，但因为自己最喜欢狗狗了，所以这份工作明显很适合自己，大呼此岗位对上了自己的职业胃口。

　　那么，即将毕业的大学生对招聘大学学历的“遛狗员”是怎么看待的呢？小李是某大学中文系一名大四学生，目前正面临四处找工作的时期，他在接受中新网生活频道采访时表示，自己不会去应聘这类岗位，觉得自己不适合，并直言大学毕业了如果当遛狗员的话，可能会被人笑话；在医学院学医的大学生小卢也表示，将来想从事对口的工作，肯定不会去做遛狗员这类工作。

　　一位大一新生的母亲罗女士也认为，遛狗应该用不上大学学历，并称女儿本科毕业后如果从事遛狗员这类工作，自己会觉得不太体面。

　　专家：如果为噱头 不应提倡

　　那么，遛狗员这类岗位需要大学学历到底是人才浪费还是企业对人才的另类需求？北京一家公司人力资源部招聘主管安先生表示，公司招聘遛狗员，尽管有学历要求，但也是可以理解的，这体现的是社会分工及市场需求。但如果企业老总招聘遛狗员仅仅出于一时的爱好，且又要求本科学历，那这可能就是叶公好龙、有钱任性了。

　　该主管同时表示，很多网友认为这类岗位需要本科学历是对人才的浪费，大学毕业生及家长也不愿意从事这类岗位，应该是由于这类岗位目前在社会上并不普遍，很多毕业生也认为这类工作并不需要本科学历，希望自己能从事与学历和专业更对口的工作。

　　一位曾在宠物美容店从事过宠物美容工作的美容师则认为，遛狗员这个工作其实并不低端，目前国外就有遛狗员，待遇还并不低，在一些犬业发达的国家，有牵犬师这样的职业。国内目前就很多人从事跟宠物相关的职业，例如美容护理人员、宠物医生、宠物营养师、宠物培训师、托管人员等。

　　对于本科生如果当遛狗员是否是人才浪费的问题，该宠物美容师表示，其实跟宠物相关的职业并非人们想象的那样轻松简单，需要从业人员具备一定的手艺和耐心，更需要从业者具备能同宠物打交道的天赋。所以，遛狗员并不是低端和无聊的代名词，本科毕业生想要做好，应该也需要一定的专业知识和能力。

　　中国劳动学会常务理事苏海南在接受中新网生活频道采访时表示，对于这样的招聘要求，背后可能存在两种内涵：如果是企业老板希望遛狗员这样的家庭服务人员素质和文化素养高一些，那么这样的发自内心的招聘需求就不应被当做坏事，经济越发达，经济富足的人越多，生活服务业里像遛狗员这样的新工种开始出现，这样的招聘需求不应被反对。

　　另一方面，如果这样的招聘仅仅是一种噱头，是企业老板为了显示身份、装点门面的话，苏海南表示，这样的招聘要求就不值得提倡。
