七千元买来红木茶台 用了不到一个月就掉漆

原标题：七千元买来“红木”茶台 用了不到一个月就掉漆

　　 鲁先生花七千元钱在光彩大市场买到一套“红木”茶台，用了不到一个月，发现茶台桌面边缘掉漆变色。他与商家协商退货，但商家表示只同意修，不同意退货。

　　在鲁先生经营的茶叶店内，放着一套“红木”茶台。妻子陈女士说，茶台是花7000元从光彩大市场“华夏一品”专卖店，8月20日买来的、有售货单。

　　“9月12日那天我坐在桌边，突然发现桌边露出一块白色，心想这是怎么了，就用手抠了抠，发现桌面掉漆，里面的木头是白色的，根本不是红木。”陈女士说。

　　记者看到，茶桌呈红色、桌面带有檀木木纹，掉漆部分内部白色，具体什么材质不好判断，掉漆部分周围的漆也是一抠就掉。“当时商家说，这是红木的正品，没想到能是这样”，陈女士说。发现茶台掉漆后，她又检查了一下其他细节，发现有的地方漆都没喷匀。茶台无任何标识证明生产厂家及所属品牌，也没有出厂合格证。

　　发现问题后，陈女士和丈夫马上找商家沟通，希望退货。但是商家对他们“态度很不好，拒绝沟通”。卖家李女士称消费者提出的退货要求很无理，“可以维修，但是不能退货”。

　　陈女士订货合同上注明，因商品质量出现问题的，按《省家具协会三包规定》。记者查询山东省家具产品“三包”责任暂行规定了解到，销售者应向消费者提供家具应当出具有效的发票和三包凭证。家具自交货之日起30日内发生严重质量问题，消费者可选择退货、换货或修理。家具自交货之日起60日内有以上所列情况者，可选择换货或者修理。

　　目前，鲁先生已经向光彩大市场和岱岳区工商局反映了情况，光彩大市场负责人邵经理称她了解情况后再答复。岱岳区工商局负责人表示，已经将情况转到工商所处理。

(责编：孙红丽、唐k)
