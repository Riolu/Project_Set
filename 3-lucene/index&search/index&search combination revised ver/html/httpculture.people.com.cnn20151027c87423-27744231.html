著名剧作家、词作家阎肃（人民网 陈苑 摄）

人民网北京10月27日电  （陈苑）今早微信微博被著名剧作家、词作家阎肃去世的消息刷屏，人民文化刚刚独家连线阎肃家人，证实阎肃目前正因病住院休养，去世消息为假新闻。9月11日，阎肃曾做客人民文化“回望文艺工作座谈会一周年・文艺名家话精神故乡”系列访谈，回顾一年来文艺界涌现出的新气象，分享心中的“精神故乡”。

【名家名片】

阎肃，1930年5月9日出生于河北保定，重庆大学毕业，剧作家、词作家。

1953年加入中国共产党，历任西南军区文工团分队长，空军歌剧团编导组组长，空军歌舞剧团创作员，中国剧协第三、四届理事。

创作的歌词《我爱祖国的蓝天》、《下四川》1964年获第三届中国人民解放军文艺会演创作优秀奖。歌剧《江姐》1977年获第四届中国人民解放军文艺会演创作奖。与人合作的歌剧《忆娘》、京剧《红灯照》1979年获中华人民共和国建国三十周年献礼演出创作一等奖。1986年加入中国作家协会。中国作家协会会员，中国剧协副主席，中国音协委员。
