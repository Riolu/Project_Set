零号湾科技大楼启用仪式隆重举行[图]
[发布时间]: 2015年10月26日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 孙佳

由上海交大、闵行区政府和上海地产集团三方合作共建的新业态众创空间——“零号湾全球创新创业集聚区”日前迈出重要一步，零号湾核心区域主体建筑——近4万平米的科技大楼正式投入使用。 

10月25日，大楼启用暨首批机构入驻仪式隆重举行。上海市人大常委会副主任、上海交通大学党委书记姜斯宪、闵行区区委书记赵奇、上海地产（集团）公司董事长冯经明等出席活动。启用仪式由江川街道党工委书记刘琼主持。 

仪式上，上海交大副校长吴旦、上海地产（集团）公司副总裁蔡顺明向首期入驻零号湾的合作机构中国电信、中国银行、交通银行、上海银行代表赠送纪念品。 

上海交大党委副书记朱健、电子科技大学党委副书记申小蓉、辽宁省大学生就业局副局长王宪明分别代表零号湾、成都电子科技大学“一校一带”园区、辽宁省大学生创业教育实训基地签署战略合作备忘录。根据协议，零号湾将与另外两家单位共享创业、创投导师资源，相互推荐校友入驻园区创业，交流研讨建设发展经验。 

闵行区区委副书记、区长赵祝平与首批入驻园区的三家孵化器机构负责人张德旺、肖文彬、陆华，共同为三家孵化器机构揭牌。三家机构分别是接力空间零号湾、晨晖创投@零号湾，以及iWORK+咖啡创业空间。  

电子科技大学党委副书记申小蓉代表战略合作方发言，上海市大学生科技创业基金会秘书长、上海创业接力科技金融集团董事长张德旺代表孵化器机构发言，他们都表达了推动零号湾成长发展的美好意愿。 

在欢快的音乐声和热烈的掌声中，姜斯宪、赵奇、冯经明共同为科技大楼揭幕。  

赵奇最后代表零号湾三方共建主体讲话。他表示，零号湾是一个集聚了创新创业梦想的启航平台，具有超强的感染力，他从隆重而热烈的启用仪式和代表们的发言中，感受到了大家对创新创业事业的热爱、热诚和热情。他与大家分享了三点体会，他认为，能从“零到一”就能从“一到十”。作为上海南部科创中心的一个非常重要的组成部分，零号湾是立足闵行、服务上海、辐射全国并进而影响全球的起航点。从今年4月零号湾名字的诞生，到今天科技大楼正式启用，标志着我们实现了“零”的突破，并在“一”的基础上向着更高的目标迈进。  

合而为一就能无往而不胜。零号湾可谓汇集了天时地利人和。零号湾整合了政府的政策资源、高校的人才和创新资源、企业的资金、管理和市场需求资源，各方资源和力量在这里汇聚融合，形成“化学反应”，将推动创新创业从量变到质变。  

始终如一就能守住初心。创新创业需要感性和热情，更需要理性和恒心，他希望大家在纷繁复杂的市场浪潮中，坚守梦想不忘初心。他表示闵行区对创新创业的支持将长期“不变”，一是扶持创新创业者的决心不变，二是支持零号湾的初衷不变，三是服务全市、服务高校的理念不变。他表示，闵行区委区政府将成为零号湾最大的后台服务器，通过不断完善公共产品和服务，最终把零号湾建成业界一致公认，国内外具有广泛影响力的创新创业示范区。通过市区校企的全面合作，创新机制整合资源形成合力，不断发掘高校的智力，激活市场的活力和社会的创造力，实现创新驱动转型发展。他最后祝愿创客团队能在这里遇见各自的天使，实现创客梦，祝愿零号湾能茁壮成长，实现中国的硅谷梦，祝愿闵行实现新发展，共筑中国梦。  

仪式结束后，姜斯宪、赵奇、冯经明、赵祝平等领导和嘉宾还参观了新大楼，并与已进入大楼工作的孵化器和创业团队进行交流。  

出席活动的还有闵行区政府、上海地产（集团）公司，上海交大相关单位和部门的负责人；园区战略合作伙伴单位代表；第一财经投资管理有限公司总经理陆天旗，“饿了么”创始人张旭豪，大米科技创始人杨健，创业岛创始人郭鑫等科创行业的领军人物和专业服务机构代表；江川区域党建联席会成员单位的领导；以及合作服务机构和入驻机构的代表。  

从2015年4月共建三方签署合作备忘录以来，零号湾正以跨越式步伐成长壮大。短短半年，入驻园区创业团队已达63支，创业者人数超过350人，大有“近者悦而尽才，远者望风而慕”的态势。科技大楼作为里程碑式的成果，将进一步发挥优势资源的集聚效应，完善三类主要职能体系的“黄金组合”：一是以较大优势学科为特征的专业孵化器，二是优势互补的创新创业孵化器主体，三是围绕创新创业的金融服务、融资投资、法律服务等支撑体系。多方优势资源的协同将共同为零号湾培育最佳创业生态体系。
