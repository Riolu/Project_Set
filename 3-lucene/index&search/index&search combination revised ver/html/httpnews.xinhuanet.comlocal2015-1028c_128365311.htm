“环卫工捡瓜子皮”刷爆朋友圈 引发网友反思(图)

环卫女工王明琴用手将瓜子皮一个一个抠起。

　　几天来，微信朋友圈里一张照片引发广大网友的关注。照片中，一位环卫女工蹲在马路边，捡拾着路面上的瓜子皮。路上一闪而过的车流和女工专心致志的神情形成鲜明对比——它所传递出的正能量正感动着张家口这座城市。

　　一抹橘黄

　　触动路人心灵

　　尽管事情已经过去十天，但回忆起当时的情景，拍摄人李雅娟依然感慨颇多。

　　10月16日下午3时，张家口市桥东区荒山绿化管理处工作人员李雅娟开车外出办事，在建国路阳光锦园小区门口等人时，一名蹲在地上的环卫女工引起了她的注意。这位身穿橙色工作服、头戴红色帽子的环卫女工，正在用右手将嵌在路面上的瓜子皮一个一个抠起。

　　“当时我觉得非常震撼，心里像被什么揪了一下，就拿出手机拍了下来。”李雅娟说。当晚下班回家后，李雅娟将照片发到了微信朋友圈。

　　一张照片

　　引来满城赞誉

　　让李雅娟没有想到的是，她的这张照片引起了网友的共鸣。“这是今年最让我们感动的照片！”“真没想到，在我们享受城市美好环境的时候，有人如此默默付出！”“向清洁工致敬！”“让我们行动起来，为清洁工做些力所能及的事”……照片发出短短几小时，李雅娟的朋友圈几乎被各种评论刷爆。

一把扫帚伴随王明琴整整六年。

　　随后，“环卫工捡拾瓜子皮”的新闻照片迅速在新浪河北、河北新闻网、张家口新闻网等网站传播转载，影响力逐步扩大。

　　网友中华鲲在自己的博客上转发并评论道：“看到这张图片，我第一时间被震撼了。心中充溢着满满的感动。环卫工捡起的不仅是瓜子皮，更是我们对这份工作的敬意和对自身行为的反思。责任多一些，理解多一些，这个城市将更美好。”

　　网友小四说：“平凡的举动却折射出震撼人心的力量。清洁工蹲下，却赢得了我们对这个岗位的敬意，也呼唤着城市文明能迈出更大的一步。”

　　一份工作

　　平凡而又伟大

　　深秋的张家口落叶缤纷。10月26日下午，记者在建国路上见到了照片中的主人公王明琴，她正在和同伴们一起清扫落叶。一阵风刮过，树叶打着旋飘下来，落在她的红色帽子上，也落在她脚下刚刚清理干净的地面上。她再一次挥动扫帚，扫起柏油路上落下的枯叶。“每年这个时候最忙，叶子总也扫不完。”王明琴很平静地对我们说。

　　今年44岁的王明琴，在桥东区环卫处清扫二大队已经做了6年的清洁工，专门负责建国路一带的清扫工作。“虽然累点儿，但这活我干得挺高兴。”说起工作，王明琴露出几分骄傲。

　　王明琴回忆，16日下午，她清扫至阳光锦园小区门口时，看到路上有一堆瓜子皮，用扫帚清扫后，还有不少瓜子皮嵌在道路缝隙里扫不起来。“别的地方都挺干净，就这儿有一堆瓜子皮，看着特别扭，我就蹲下用手捡。”她淡淡地说。

　　几天来，她的照片经过微信转发，被越来越多的人关注，偶尔也会被路人认出。面对记者，王明琴显得有些局促。“也没啥，这种事儿我们常遇到。我只是干了自己该干的工作，我们队里每个人都是这样干的，很多人都比我干得好。只要我们环卫工人的工作能得到大家的理解和尊重，心里就觉得挺温暖。”王明琴羞涩地笑起来，笑容显得格外美丽。
