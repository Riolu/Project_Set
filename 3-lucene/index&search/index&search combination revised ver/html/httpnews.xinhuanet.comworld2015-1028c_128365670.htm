伊朗希望加入金砖国家新开发银行

2015年10月28日 07:28:28 来源： 人民日报
  • 评论0
  • 打印
  • 字大
  • 字小

　　德黑兰消息：据伊朗塔斯尼姆通讯社26日报道，伊朗财经部副部长哈扎伊表示，伊朗希望加入金砖国家新开发银行。

　　报道称，哈扎伊在伊朗与巴西的一场经济会谈中表示，伊朗有意向加入金砖国家新开发银行，以拓展与该银行成员国之间的合作。分析人士指出，伊朗与伊核问题六国（美国、英国、法国、俄罗斯、中国和德国）今年7月达成的伊核问题全面协议于本月18日生效，西方对伊经济制裁有望解除，伊朗希望通过加入金砖国家新开发银行，为本国发展筹集更多资金。

　　2014年7月15日，中国、巴西、俄罗斯、印度和南非在巴西福塔莱萨签署协议，决定建立金砖国家新开发银行，初始资本为1000亿美元，总部设在上海。今年7月21日，金砖国家新开发银行在上海正式开业。（记者杨迅）
