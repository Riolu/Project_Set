双十一值得入手的性价比手机盘点
 
2015年10月28日 06:10:32 | 责任编辑: 杨帆 | 来源： 手机中国   

　　随着十一月份的临近，一年一度的购物狂欢节也即将到来。阿里方面也邀请了著名导演冯小刚打造了“天猫2015双11狂欢夜”，让这个火热的双十一变得更加不寻常。

　　作为一个主打购物的节日，“双十一”吸引了一大批的消费者。现在的智能手机已经成为现代人必不可少的工具之一，消费者的需求也推动了市场的繁荣。作为第一梯队的手机产品，各大手机厂商也摩拳擦掌，双十一的竞争会更为惨烈。

　　接下来，小编本着为大家谋福利的原则，精选了市面上较为热门的高性价比机型，喜欢的朋友不妨多加关注，打算双十一购机的朋友也可以多加了解一下。

　　OPPO R7s

　　10月20日OPPO在京举行了新机发布会，其全新一代“万人迷”新机OPPO R7s正式发布。作为OPPO R7的全新升级版，OPPO R7s延续了OPPO一贯的高颜值基因，整机采用金属一体机身。另外OPPO R7s配置了更大的3070mAh大电池，VOOC闪充的注入，也让续航时间大大增加。

　　OPPO R7s的VOOC闪充

　　OPPO R7s配置3070mAh大电池和VOOC闪充，充电5分钟即可通话2小时。通过系统省电精灵和屏幕省电优化，也让OPPO R7s的续航时间大大增加。VOOC闪充是OPPO自主研发的技术，选用全套定制的硬件设备，从电源适配器、充电线、接口到电池，全端式5重防护，保障充电更为安全快速。

　　OPPO R7s

　　OPPO R7s（移动版）选用5.5英寸1080P AMOLED屏幕，有着2.2mm超窄边框，显示效细腻饱满。OPPO R7s内置一颗联发科MT6752八核处理器，辅以超大的4GB RAM+32GB ROM，大型后台应用皆可迅速切换。OPPO R7s搭载1300万像素后置镜头+800万像素前置镜头，主镜头为F/2.2光圈、搭配PDAF相位对焦可实现最快0.1秒对焦，让美好瞬间皆可轻松捕捉。
