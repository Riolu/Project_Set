阿富汗发生强烈地震

时间:10月26日下午 地点:阿富汗兴都库什地区

专题摘要: 阿富汗东北部26日下午发生强烈地震，巴基斯坦、印度、乌兹别克斯坦等邻国均有强烈震感。据中国地震台网测定，此次发生在阿富汗兴都库什地区的强震震级为7.8级地震，震源深度210公里。
