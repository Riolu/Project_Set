"标题党"要当心玩火自焚
2015年10月15日 09:46

3个女人和105个男人的故事》《豪门浪子啊,却为真爱遁入空门》……看到这两个标题,你能想到它们指的分别是《水浒传》和《红楼梦》么?然而,让我们颇感无奈的是,这种典型的“标题党”做派,如今在网上无处不在,防不胜防。有不少网站编辑都喜欢在标题中加入色情、凶杀、阴谋、犯罪、性暗示的元素,以为这样就能获取更多的点击量。

有人认为,从信息传播的角度来看,“标题党”从媒体诞生之初就有。但不得不说的是,似乎从来没有哪个时期的信息传播,像现在这样热衷于“标题党”。不仅是娱乐、体育、社会新闻,连严肃的政治、军事新闻,都能看到“标题党”的身影。最近,有网友对比了某新闻网站与其他网站转载同一则新闻时所取的标题,就引发了热议。

通常来说,“标题党”惯用的手法,包括捕风捉影、断章取义、剪辑放大、移花接木等。在一些网络编辑看来,网友都喜欢那些低俗、刺激的内容,对正襟危坐的严肃讨论不感兴趣。于是,哪怕夸张、失真、歪曲事实,他们也要将内容尽可能地与性、犯罪、八卦、心灵鸡汤扯上关系。有些网站,很多标题都在刻意迎合部分公众对官员、富人、腐败、房价等话题的情绪。

不得不承认,“标题党”在提高新闻点击率上确实有一定效果——同样一则新闻,如果炮制一个惊悚猎奇的标题,点击率确实有可能会增加。究其因,浅阅读是网络时代的主流,网友很难认真、完全地阅读一篇长文章,如果没有一个能吸引眼球的标题,有些文章估计没人会点开看。在“点击就是经济”的网络媒体环境中,有些网站将点击量与考核挂钩,也助长了“标题党”乱象。

“标题党”片面追求耸人听闻的标题,无视标题与文章、标题与事实的关系,既有违媒体的基本操守,也是不可持续的。如今很多人将“标题党”称为“新闻怪胎”,就是因为这种操作手法完全不符合新闻规范,与“客观”“真实”的原则背道而驰。长此以往,将破坏媒体行业的总体生态。

当罔顾真相、背离事实被一些媒体奉为圭臬,难以想象其如何能持续健康发展下去。毕竟,读者不是傻子,上过一回两回当后,“再回首只有恨意”。一些媒体即使现在能靠“标题党”增加点击量和阅读量,到最后只会被网友们抛弃。

跳出媒介批评的视角,“标题党”利用网友猎奇心理而舍弃社会责任的行为,也是缺乏道德良知的。没有底线地“语不惊人死不休”,不仅可能会激化社会矛盾、撕裂社会共识,还会传递一种扭曲的价值观,造成社会认知混乱等多种恶果。比如,在“石狮灭门惨案”事件中,当事人通过“标题党”的手法,将死老鼠编造成耸人听闻的命案,就完全混淆了娱乐与严肃新闻的区别,最终被拘留也是罪有应得。

以网络为代表的媒体发展,赋予了每个人更大的表达自由,但这并不意味着就可以毫无底线地胡编滥造。“标题党”盛行,在侵犯公众知情权的同时,也在透支媒体公信力。从媒体发展和社会进步来说,这种行为无异于饮鸩止渴。

当然,要扭转“标题党”乱象,既有赖于公众增强判断力、不要被忽悠,相关部门的规范,也有赖于媒体从业人员加强行业自律。不管是记者还是编辑,都应该规范新闻操作,别玩“标题党”那一套。
