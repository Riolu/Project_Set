沪今年将开243所“爱心暑托班” 惠及近2万名小学生

2015-06-11 11:16:49    来源：上海文明网   

作为市政府年度实事项目之一，今年暑假，团市委、市教委、市妇联等将在全市开办243所小学生“爱心暑托班”，预计将惠及近2万名小学生。
　　“爱心暑托班”共两期，每期约三周，分别于7月和8月上旬开出。办班地点在各区县社区文化活动中心、社区学校或青年中心等，每班招生30至40人。原则上暑托班每人每期收费600元，包括学生午餐费、保险费等，具体学费由各办班点根据实际情况适当调整。
　　“与去年首次开办相比，今年暑托班课程更丰富了。”团市委副书记刘刚介绍，243所暑托班全部采用标准化课程，团市委向社会组织购买了安全教育和传统文化两类课程。安全教育类课程传授小学生应急自护常识，进行突发事件逃生拓展训练；传统文化类课程包括“百善孝为先”、“最美古诗词”、“乐理课程”等。今年，上海市农商银行、电力系统等企业组织加入，一个网点对应一个暑托班，针对小学生开设“小小金融家”、“用电安全与常识”等理财和安全培训。
　　根据上海新高考改革方案，高中生综合素质评价将成为高校招生录取评价方式之一，社会实践属于综合素质评价的一大方面。作为面向社会的公益服务项目，“爱心暑托班”首次向本市高一学生发布志愿者岗位招募，据初步统计，可提供总计10万学时的实践岗位，这是目前最大规模的高中生社会实践认证项目。高中生志愿者正常完成服务后，有关部门对志愿者名单和服务时长进行公示，无异议后录入“博雅网”，记入上海市高中生综合素质评价信息管理系统。
　　“爱心暑托班”分布情况，家长和学生可通过“上海学联”、“青春上海”、“上海12355”等微信公众号查看，只要输入所在区县名，便可查看详情，或登录团市委官网查看。报名时间为6月15日9时至6月17日16时，报名地点可通过以上微信公众号查询。根据就近原则，凭报名者的电子学生证和监护人身份证，到报名点现场报名。“上海学联”微信公众号还将开放高中生志愿者报名平台，报名时间为6月10日18时至6月12日17时。 

每人只能参加一期

对于孩子们而言，上学时他们都期盼着寒暑假，而对家长来说，这两个月却令人发愁。上班后孩子往哪儿送？安全问题如何保障？如何让孩子不虚度这两个月……团市委去年首次联合相关部门，在全市范围开设“爱心暑托班”，甚至出现了“一位难求” 的现象。据悉，今年上海市小学生爱心暑托班将在7至8月期间继续开展，在全市范围内建设243个小学生爱心暑托班，覆盖大部分街道、乡镇、社区，利用学校教室、青少年活动中心、社区文化活动中心、社区学校等场地作为办班点。据悉，今年的“爱心暑托班”共开展6周，每三周为1期，共2期。6月15日上午9点至6月17日下午4点为报名时间，根据就近就便原则，凭报名者的学籍证明和监护人身份证到报名点现场报名即可，建议收费标准与去年相同，依旧为600元/人/期。市民可以拨打统一服务热线12355咨询具体信息。值得注意的是，考虑到扩大项目的受益面，此次报名仅限选择参加一期。
　　此外，为适应高考综合改革中对于建立高中生综合素质评价体系的要求，今年的暑托班将尝试接收少量高中生志愿者，高中生志愿者与大学生志愿者以一定的比例分配到办班点开展服务工作，高中生志愿者的服务时间将计入综合素质测评中的实践学时中。
