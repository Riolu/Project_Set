共同经营好上海网络编辑的大家庭
2009年9月29日 15:36

　　踏上工作岗位后，重新又到充满书香的复旦校园，接受互联网新闻与信息业务的培训，很幸福。我不知道大家有没有这样的感觉，作为一个网络编辑，我们的工作就是面对互联网上海量的信息，每天，我们的大脑都被塞得满满的，很少有时间去回头看看已经发生的事，也很少有时间去想想将来可能发生的事。对我来说，这一个月的培训，是一个充电的机会，也是一个让我总结和思考的机会。 　

　　在这一个月里，我们学习了系统的新闻传播学理论和马克思主义新闻思想；我们回顾了互联网发展的历史，分析了现状，也展望了未来；老师们为我们强调了坚持舆论导向的重要性，也传授了网络评论的写作、网络新闻标题的制作等编辑实务，总的说来，我们的收获是满满的。  

　　因为收获，更要感谢为我们上课的所有老师。他们将自己的研究成果毫无保留地奉献出来，不仅从理论上丰富了我们的头脑；还要花费时间，寻找最新、最典型、最生动的事例来加深我们的理解。像昨天为我们上课的张志安老师，为了更好的地举例说明网络编辑对突发新闻的处理，不惜在一堂课里让自己“自杀”了七八次。在这里，我要特别感谢网络处的张晓宇处长和刘华宾老师，对于我们第九期的学员来说，参加这一次培训，我们放弃的是这一个月的休息日，而对他们来说，放弃的却是这一年还有今后不知多少个日子的休息。  

　　同时，我也要感谢和我同一小组的同学，在小组讨论中，他们积极奉献智慧，发表见解，大家并不成熟的观点在讨论中碰撞，互相启发。希望这种交流能够一直维系下去，大家有空都能到培训班的论坛上“坐坐”，我们可以谈谈业务，也可以说说心事，让我们的培训班成为上海网络编辑的一个大家庭。

　　

　　对我们所有学员来说，今天的结业并不是终点，而是一个新的起点。我祝愿在座的每一位都能学有所成，并能在今后的工作中学以致用。最后，我祝愿我们的培训班能越办越好，也祝愿大家在下个月的认证考试中都能顺利过关。

　                                                           东方网 姚莉莉
