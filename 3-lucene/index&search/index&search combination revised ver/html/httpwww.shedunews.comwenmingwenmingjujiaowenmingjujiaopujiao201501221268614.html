沪中小学生寒假作业主打“亲情牌”画张亲戚称谓图

2015-01-22 10:28:14    来源：上海文明网   

本周，沪上各中小学进入期末考试周。随着寒假的临近，不少学校都在酝酿布置有趣的寒假作业。制作“家风家训”电子书、画一张亲戚称谓图、身边的安全妙招大搜集……记者采访发现，今年的寒假作业主打“安全牌”和“亲情牌”。
　　安全，成为今年多数学校寒假作业的关键词。一师附小就让小朋友在寒假中搜集安全小妙招，包括出行安全、过节安全和消防安全等。
　　记者还发现，沪上学校布置的寒假作业还充满了传统“年味”。卢湾第一中心小学的寒假作业别具一格，请孩子们制作家风家训的电子书。“通过征集每个家庭的家风家训，再由‘小百花人才库’的小画家手绘成四格漫画，制作成电子书。”校长吴蓉瑾告诉记者，用出自孩子自己家庭的风尚感染和教育他们，或许更能深入人心。
　　春节期间少不了拜年，如今的独生子女拜年时总是遭遇尴尬事:不知亲戚咋称呼。面对这一情况，北蔡中学布置的寒假作用很“实用”——请学生画一张亲戚称谓图，把家中的长辈亲戚的称谓理理顺。校长赵之浩表示，中国人由姻亲而产生的亲戚关系比较庞杂，其派生出的各种称谓也是中华传统文化的一大特征，“亲戚关系称谓”的教育和普及作为一种中华文明礼仪教育，学校也希望通过这项作业帮助学生将民俗称谓弄准确并代代传承下去。
