微软校园软件操作系统批量许可自带的系统密钥

2014-04-01  点击：[]

批量激活对应的KMS客户端密钥

下方列出的密钥是通用卷许可密钥(GLVK). 这些密钥预制在企业版本的Windows系统安装软件中。允许使用这些密钥的用户电脑通过与KMS激活服务器配合，激活并获得一段时间（一般是半年）的使用许可。

一般情况下用户不需要输入这些密钥，只有当误操作删除了本地储存的密钥时，才需要使用这些密钥。

单独使用这些密钥无法激活Windows，必须与KMS服务器配合。

数据来源: http://technet.microsoft.com/en-us/library/jj612867.aspx
