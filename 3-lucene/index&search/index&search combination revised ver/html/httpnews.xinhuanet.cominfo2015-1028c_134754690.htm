淘汰的手机如何变废为宝？

2015年10月28日 06:12:49 来源： 人民邮电报
  • 评论0
  • 打印
  • 字大
  • 字小

　　当前，智能手机已经成为人们生活的必需品，更新换代也非常快。但由于回收和无害化措施没跟上，这些废旧手机绝大多数闲置在家。有的废旧手机落到不法分子手中，手机内的电话号码、短信等信息被他们利用从事诈骗活动。那么，淘汰后的旧手机如何处理才好呢？其实，与其把它贱卖了，还不如把它变成宝，让它继续发挥余热。

　　一是可变身3G、4G无线网卡+无线路由。目前好多手机都支持网络共享热点功能，支持USB共享模式和无线共享模式，其功能与我们日常使用的无线网卡和3G无线路由类似，而目前比较好的无线网卡价格都在三五百元左右。

　　二是可作为家庭安防摄像头使用。可以将它放在家中的某个角落或窗台上，作为家庭安防监控或婴儿监视器使用。目前市面上有大量的相关应用可以实现这个目的，如iCamSpy、Presence和Manything都是非常强大的App，可以帮你保护财产安全或远程监控家中的情况。

　　三是可变身行车记录仪、手机导航。手机的拍照效果和性能要比行车记录仪好很多。目前，很多智能手机都是支持导航的，如果你的车上没有导航的话，快用旧手机做一个车载导航吧。

　　四是可变身相机显示屏。像索尼DSC-QX100等一些镜头机从外面看就像是一枚镜头，其实挂上一部手机就可以拿去拍照，跟小巧的微单一样，成像效果还不输于手机。拥有LED闪光灯的手机，关键时候可以做应急手电筒，拍照补个光都不错。

　　五是可作为气压计使用。最新的iPhone内置了气压传感器，旧手机同样可以通过安装应用程序来查阅当地的天气和气候状况。包括iOS平台的Barometer和Android 的Altimeter都可以实现这样的功能。
