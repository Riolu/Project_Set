“你诚我信・大家谈”――专家畅谈网络诚信
2015年10月20日 09:57

10月19日，由国家网信办网络社会工作局指导，新华网和中国网信网联合推出的“你诚我信·大家谈”系列访谈正式与网民见面。

“你诚我信·大家谈”系列访谈聚焦网络诚信这一热点话题，邀请业内多位知名专家，从法律、征信、媒体、文化等维度进行探讨，旨在引导网站和网民依法办网、诚信用网，抵制和防范网络违规失信行为，加强网络诚信理念宣传，不断推进网络诚信建设制度化。

专家们从自身研究领域出发各抒己见，共同探讨网络诚信建设制度化、网络社会信用体系、互联网行业信用标准化制定等问题，大力弘扬网络诚信、抵制网络失信，进一步打造诚实、自律、守信、互信的网络环境。讨论的话题既有对网络谣言、网络诈骗、网络侵权等现实问题的反思，也有对落实网络实名制、完善网络信用管理法律法规、互联网领域征信建设等的政策建议，以及对网站和网民共同坚守网络诚信的呼吁。

首期访谈邀请中国政法大学传播法研究中心副主任朱巍，通过现实生活中的案例，从法律角度深入浅出地讲解“大数据时代该怎样保护我们的隐私”，阐释“网络安宁权”“被遗忘权”等与网络生活息息相关的法律术语，为网民提供应对网络诚信建设中新问题、新挑战的策略。

“你诚我信·大家谈”系列访谈专题视频将在中国网信网上集中展现，具体链接为：http://www.cac.gov.cn/ncwx/1.htm。
