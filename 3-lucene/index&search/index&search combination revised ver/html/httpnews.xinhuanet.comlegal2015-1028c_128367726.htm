美女驾车被火车拦腰撞击身亡
 
2015年10月28日 13:22:02 | 责任编辑: 泓翔 | 来源： 中国日报网   

　　10月28日报道，台湾高雄前晚发生严重车祸，上月才在台湾拿到车牌的22岁香港女学生黄咏贤驾驶私家车，带爱犬外出洗澡，经过一个道路与铁路交接的平交道时，黄女开车冲过已放下的栏杆，但车子停在路轨上，她疑因受惊未及反应，被一列台铁列车撞上并推前200米。黄女与爱犬被困变形的私家车内，消防员花个多小时才将她救出，送院证实不治，车上的小狗也当场死亡。
