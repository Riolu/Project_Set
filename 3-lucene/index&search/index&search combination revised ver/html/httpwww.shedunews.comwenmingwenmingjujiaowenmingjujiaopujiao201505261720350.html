黄浦运用新媒体推进未成年人社会主义核心价值观教育

2015-05-26 10:24:59    来源：上海文明网   

近年来，黄浦区充分发挥新媒体优势，利用网络社区、手机APP、微信公众号等方式，拓展了未成年人社会主义核心价值观教育的渠道， 2014年6月改版上线的黄浦区未成年人网站——“零零岛”，截止2015年3月底，注册用户已达到了4万3千余人，点击量达60万次，受到了全区未成年人和家长的欢迎，取得了良好的社会反响。

以兴趣为引导，增加教育的吸引力

以寓教于乐作为理念，将青少年喜闻乐见的微博、微信、贴吧、论坛、商城、迷你游戏等传播形式与社会主义核心价值观内容有机融合，使思想道德教育避免简单枯燥的说教灌输。在“零零岛”网站中，采用内容生动、界面友好、易于交互“广播站”，通过“零零社”、“城市学校少年宫”、“学生社区实践指导站”三个公众账号，每日发布相关资讯，未成年人用户可以在自己的朋友圈内转发、点评和自己生活相关的信息；用FLASH动画，为未成年人提供了生动活泼的“社会主义核心价值观”解读，新颖的形式，激发了未成年人吸收正能量的兴趣。

线上线下共建，扩大教育的参与面

通过与新媒体结合，各类道德实践活动形成了“线下线上”良性互动格局。在暑期学生社会实践活动中，学生们将参观学习爱国主义教育基地的感受，以感言、照片、图画等形式，通过“文文明明幸福行”小微博进行分享，学生们通过每天数百条原创贴子和跟贴，在互动交流中，激发了他们的美好体验。“美德少年”“文文明明幸福小使者”等评比活动的信息，提高了学生们参与活动的主动性。去年“优秀童谣征集活动”中，通过“零零岛”在线上宣传、展示优秀作品；在线下，与各个街道的学生社区实践指导站合作，组织创作交流，提升了活动的知晓度，扩大了活动的参与面，网上征集的优秀作品，还获得了市级评选活动的一等奖。

学生参与管理，提升了教育的效果

黄浦区根据新媒体参与性强的特点，积极探索体验式教育与社会主义价值观结合，提升思想教育的效果。黄浦区让未成年人担任网络社区或者线上活动的管理员，使未成年人自己创造交流内容、自己参与管理、实现自我教育的“小世界”，在同龄人群体中学会表达、倾听与沟通，实现个体成长与群体共同发展。在“零零岛”网络社区的“爱心正能量”、“国学大讲堂”等主题论坛中的管理员，都有未成年人担任，不仅使论坛更加活跃，更加有吸引力，同时，虚拟空间的体验，进一步激发了广大未成年人现实生活的责任心和荣誉感，在主动参与的过程中,潜移默化地接受了社会主义核心价值观的熏陶。
