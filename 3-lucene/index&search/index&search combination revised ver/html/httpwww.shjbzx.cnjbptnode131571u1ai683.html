黑客"入侵"汽车引关注 车企亟需重视互联网安全问题
2015年10月23日 11:03

新华网北京１０月２１日电（记者华晔迪）不久前，两名黑客在美国做了一场试验：利用互联网技术侵入一辆行驶中的切诺基吉普车电子系统，远程操控加速、制动系统、电台和雨刷器等装置，甚至让汽车冲进路旁沟里。这一试验引发全球汽车企业对智能汽车安全问题的关注。

这两名黑客查理·米勒和克里斯·瓦拉塞克２１日现身北京举办的国际前瞻信息安全会议，并向观众演示远程破解汽车电子系统。前述试验经媒体报道后，直接导致切诺基吉普车生产商、知名汽车企业美国菲亚特克莱斯勒汽车公司紧急宣布，在美国召回１４０万辆轿车和卡车，以防黑客通过互联网远程控制操控车辆。

“在万物互联时代，安全已经突破软件、内容、服务等界限，甚至可能会危及人的生命。”３６０公司创始人、总裁齐向东在接受记者专访时说，车联网的快速发展，不断将汽车推向智能化，汽车已经变成轮子上的计算机，汽车云服务、车内通信系统等领域都将面临安全问题的挑战。

中国是否也有汽车已面临同样的安全问题？３６０公司网络安全攻防实验室刘健皓透露，今年他们破解了一辆中国品牌的汽车，他们发现在手机ＡＰＰ一端，这辆汽车存在安全漏洞，因为该车有一个功能是在无人状态下，车主可以通过手机ＡＰＰ操纵车辆倒车入位。

记者注意到，无论是去年被黑客破解车辆系统漏洞的特斯拉、还是今年被破的切诺基吉普车，抑或是中国国产品牌汽车，共同点之一是在车联网系统上实现了高度的智能化。通过互联网，汽车可以将相关数据传回工厂，用户也可通过ＡＰＰ反向控制车辆。

“尽管中国国内尚未发生类似黑客攻击导致事故的事件，但试验表明通过攻破电子系统控制汽车已成为可能，车企不敢掉以轻心，毕竟谁也承担不起这类风险。”一位汽车行业人士说。

前述汽车行业人士也担心，类似安全问题凸显，有可能导致车企在汽车电子系统运用方面采取保守策略，减缓智能汽车推向商用的进程，影响车联网发展。事实上，一些车企在发现系统漏洞后，已经选择通过关闭云服务，以降低遭受黑客攻击风险。

齐向东透露，包括３６０公司在内的多家企业、机构和车企正在筹备成立中国车联网安全联盟，旨在集合全社会的网络安全能力，跨行业进行技术交流和合作，推动产业和技术创新，共同面对和解决汽车的信息系统安全问题。
