改变上网习惯 保护个人隐私
2015年10月28日 11:08

个人信息了然于心

林钧跃提醒大家，要知道什么是你个人的信用信息，比如你独有的、与隐私相关、牵扯到生活的关键信息。

针对这类信息，要明确交易对方是谁，清楚做什么事的时候才能给对方。因为要使用对方的服务，对方需要了解你的情况，所以他有权知道你的信息，但并不是全部。

收快递之后涂掉地址

针对个人隐私地址，比如私人信件、快递的地址，用完之后一定要涂掉。定期对电脑、手机做漏洞检查。可以购买安全手机，保护你的通话信息。

改变网上操作行为保护隐私

在网上操作时，可以嵌入反侦查的方法，改变网上的操作行为，在定点的时间误导侦查方，反侦查方法依据具体人员层次和希望保护的程度而定。
