冯树勇：中国田径里约之战非常困难　洋帅成功率三四成

2015年10月28日 07:29:21 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

　　新华网北京１０月２７日体育专电（记者周欣）２７日，在首届全国竞技体育科学化训练培训班上，中国田径队总教练冯树勇作为第一位主讲人畅谈了他对于竞技运动训练比赛若干问题的认识，但是提起在今年田径世锦赛上创造佳绩的中国田径队能否在里约奥运会再现辉煌时，他表示“难度很大”。

　　中国田径队在８月北京举行的世界锦标赛上收获１金７银１铜的出色战绩，让国人看到了中国田径崛起的信号。

　　“明年里约奥运会，对于‘中国田径队能否再现北京世锦赛的成绩，或者更上一层楼’，我的预测是难度很大。首先是世锦赛的级别和奥运会完全不一样。我们运动员在比赛强度、赛制以及赛场气氛方面有着天壤之别。其次，我们在鸟巢比赛享有主场优势，衣食住行、后勤、医疗、保障等都享受很多的便利。而去里约，光飞行就是２０多小时的路程，还有１１个小时的时差，再加上语言、外部环境等方面的差异，都是影响运动员发挥的重要因素。第三，客观来说，中国田径在世锦赛上有不少偶然和运气的成分，”冯树勇表示。

　　冯树勇举例说，２６岁的巩立姣在女子铅球比赛中以７厘米的微弱差距负于老对手德国的施瓦尼茨，但这是在世界最好的女子铅球选手因伤缺席的结果，否则她只能收获铜牌。女子标枪中，中国选手吕会会最终获得第二，但实际上有四五个比她实力强的对手本次没有发挥水平，此外还有男子４Ｘ１００米接力队的银牌突破，是在美国队犯规的情况下取得的。

　　“在里约，不要指望其他外国选手全部发挥失常，只有我们发挥极致。现在离里约奥运会只有９个月的时间，中国田径大幅度提高水平，也许只有个别项目可以做到，因此里约之战，不会那么容易。”

　　对于在今年世锦赛大放异彩的中国田径队，有些人简单地归功于外教和外训的功劳。冯树勇表示，聘请外教和外训的目的是为了增强自信，外教在中国的成功率其实只有三、四成。

　　“首先，取得一定成绩的中国运动员并不完全是外教带出来，包括苏炳添和中国田径接力队，都是本土教练调教的结果。其次，外教给中国带来了先进的思路和理念，但需要根据我们运动员的身体条件和环境来进行调整和变通，照搬过去的成功经验很难成功。我们聘请的外教中，也就只有百分之三十到四十的成功率。”

　　冯树勇指出，中国田径除了采取“请进来”外教，还不定期组织重点运动员“走出去”出国外训。外教给中国带来了很多先进的技术和训练理念，但他们同样需要重新调整和适应。

　　“外教大都有辉煌的履历，但他们此前执教的是身体素质好的黑人选手，或者高水平运动员，跟我们队员的身体条件不一样。像鲍威尔的教练兰道尔·亨廷顿来执教苏炳添，他的训练一定要有变化。他能否接地气，能否适应我们对手的条件来进行调整很关键。不是任何一个教练按照刘翔的训练计划，就能培养出第二个刘翔，”冯树勇表示。

　　“聪明的外教会花很长一段时间来测试、了解自己的队员，然后改变训练上的一些东西。”比如，女飞人韦永丽的外教兰迪就根据她的身高、腿长、手长，为她制订了起跑策略。而竞走队米拉诺在心理调节方面就很有一套。而中国短跑接力队的成功，也与外教在一些关键环节的点拨有一定的关系。

　　“‘请进来走出去’的最大收获是，我们的运动员更自信、更放松，他们会说英语、敢于交流，心理上更加强大，这和过去有天壤之别的变化，”冯树勇说。
