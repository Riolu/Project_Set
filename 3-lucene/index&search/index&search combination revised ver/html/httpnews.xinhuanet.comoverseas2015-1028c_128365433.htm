港媒：王健林财富年增170亿美元 相当于冰岛GDP

2015年10月28日 07:09:19 来源： 参考消息网
  • 评论0
  • 打印
  • 字大
  • 字小

　　港媒称，房地产和娱乐大亨王健林登上2015年福布斯中国内地富豪榜榜首。王健林的个人财富猛增近170亿美元，增加额与冰岛的国内生产总值（GDP）相当。

　　据香港《南华早报》网站10月26日报道称，尽管经济增长放缓，但中国内地富豪榜前100名的财富总和从2014年的3760亿美元增至4500亿美元，增幅达20%。

　　报道称，大连万达集团董事长王健林的个人财富从132亿美元跃升至300亿美元。这主要是万达集团旗下的商业地产公司2014年12月在香港上市和电影院线2015年1月在深圳上市所致。2014年冰岛GDP为169.3亿美元。

　　阿里巴巴集团董事局主席马云居第二位，其个人财富达到218亿美元。

　　福布斯上海分社社长拉塞尔·弗兰纳里说，尽管中国经济正在放缓，但一年来许多中国富豪的财富仍有大幅增加。

　　报道称，弗兰纳里说：“这主要归因于内地股市上涨和互联网等行业新出现的商业机遇。”

　　在富豪榜前十名中，有六位来自互联网行业。除马云外，这几位包括腾讯董事长马化腾、智能手机制造商小米公司创始人雷军、搜索引擎百度董事长李彦宏、京东商城创始人刘强东和在线游戏与新闻门户网站网易创始人丁磊。

　　报道称，据估计，饮料制造商杭州娃哈哈集团董事长宗庆后的个人财富从106亿美元缩水至73亿美元，排名从第六位跌至第11位。
