责任编辑：文艳琼2015.10.28

VOL.797

为了面子500万打水漂了

　　同学聚会，我让老婆想个办法说点事情给我撑撑面子，老婆满口答应，刚走进门就开始追着我打：“500万又打水漂了……”同学们听完都一边劝架一边羡慕地看着我，老婆骂了一会儿接了一句：“让你不听我的，双色球不买我说的号……”

悟空要学云计算

　　菩提老祖将悟空唤至身前：“ 你已学会长生不老术和七十二变，今日为师欲传授你新的法术。” 悟空道：“是何法术？” 菩提老祖道：“看到这天上的云彩了吗？这边有七朵云彩，那边有五朵云彩，一共有几朵？” 悟空答：“十二朵。” 菩提老祖道：“嗯，我要教你的就是云计算。”

饿晕了

　　刚刚叫了几个朋友一起吃饭， 其中有一个，昨天通宵在网吧打网游，一直打到今天下午4点多。等菜的时候，这个家伙脑袋“砰”就摆在桌上，都不嫌疼。我们推他，不醒，刚好服务员妹子在旁边，这下可把那妹子慌的， 一边跑一边喊：“经理经理经理！5号桌有人饿晕了！”

表白被拒绝之后

　　其实表白被拒了完全不用怕难堪，被拒绝了可以通过马上转移话题来化解尴尬气氛。比如：“我其实喜欢你很久了。” “但我不喜……” “嗯，那你觉得明年全球的经济走势会对我国经济产生什么影 响？”

女人是水做的

　　我：“都说女人是水做的，你怎么这么暴力！”女友：“我也是水做的呀！嘿嘿，开水。”
