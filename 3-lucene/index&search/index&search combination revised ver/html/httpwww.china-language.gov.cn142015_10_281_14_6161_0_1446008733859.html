教育部语用司姚喜双一行赴中国传媒大学调研

来源：语用司
　　

    10月16日，语用司司长姚喜双、宣教处处长容宏一行赴中国传媒大学就“第17届齐越朗诵艺术节暨全国大学生朗诵大会”的筹备工作进行调研。中国传媒大学播音主持艺术学院院长鲁景超、党总支书记郑维林以及活动筹备组成员等参加了调研座谈会。

　　今年是中国人民抗日战争暨世界反法西斯战争胜利70周年，第17届齐越朗诵艺术节以纪念抗战胜利为主题展开活动。座谈会上，中国传媒大学播音主持艺术学院分别从朗诵节专项经费的使用情况、活动进展情况、入选决赛的环节设置与节目安排、嘉宾邀请、活动宣传及播出平台等方面进行了汇报。

　　姚喜双司长肯定了中国传媒大学“齐越朗诵节暨全国大学生朗诵大会”这一活动品牌对普通话的推广与提高、经典文化的弘扬传承所做出的重要贡献。他指出活动要深入贯彻落实习近平总书记在文艺座谈会上讲话的重要精神，要在安全、宣传、资源建设与利用等方面，做到战略性思考、系统性谋划、创造性操作。大型活动的安全问题是保底，预案要做到细致、全面；宣传方面要突出爱国主义主题、扩展信息发布渠道、强化社会主义核心价值观；资源建设方面则要擅于经营、拓展，把资源服务于整个社会特别是西部建设,加强与港澳台地区的联系。

　　座谈会后，姚喜双司长一行参观了播音主持艺术学院的资料展示平台和正在筹建中的中华经典诵读资源库，并为资源库的搭建工作提出了意见和建议。（中国传媒大学供稿）
