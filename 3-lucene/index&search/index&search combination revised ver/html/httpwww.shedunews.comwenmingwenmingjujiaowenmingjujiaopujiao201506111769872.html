歌声嘹亮，唱响爱国情怀 未成年人“童心向党”

2015-06-11 11:35:11    来源：上海文明网   

为深入开展“我的中国梦”主题教育实践活动，在广大未成年人中进一步弘扬中华传统文化、传承民族精神，培养坚定热诚的爱国主义情感，徐汇区文明办、教育局在全区未成年人中全面组织开展“童心向党”歌咏活动。各学校积极响应，在全体学生中宣传发动，利用班会队会、文艺表演、拓展活动、红色旅游等一切机会，组织开展歌咏活动，引导学生以生动活泼的表演形式唱响歌颂党、歌颂伟大祖国、歌颂中国特色社会主义的优秀歌曲。全区8万余名未成年人热情昂扬、精神振奋，争先恐后地投入到热烈蓬勃的歌咏活动中，在优美的旋律和嘹亮的歌声中，表达出自己对祖国母亲的深厚感情。

徐汇区“童心向党”歌咏活动，经层层宣传发动，兴起了自下而上积极推进的热潮，学生热情参与、音乐老师精心指导、班主任老师支持鼓励、德育教导积极引导、校长关心支持，使歌咏活动在每所学校蓬勃开展，全区蔚然成风，在广大学生中形成了“人人会唱歌、班班有歌声、校校有团队”的热烈氛围。 

近日，徐汇区未成年人“童心向党”歌咏展演在徐汇区向阳小学隆重举行，脍炙人口的经典爱国主义歌曲和与时俱进的核心价值观歌曲，在舞台上久久回荡，把徐汇区未成年人“童心向党”歌咏活动推向了新的高潮。
