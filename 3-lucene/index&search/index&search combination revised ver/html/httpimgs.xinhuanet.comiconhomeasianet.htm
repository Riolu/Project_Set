了解海外企业商情，中国企业走向世界的信息发布网络－－亚洲网

    总部设在澳大利亚的亚洲网是由13个国家的通讯社联合组成的企业信息发布和传输网络，向亚太地区和全世界播发各类企业动态和商品的信息。亚洲网成立于1995年，其强大的网络系统至今在亚太地区已涵盖了34个国家的5000多家传统和电子媒体，并在国际媒体领域的知名度正逐年上升。
    新华通讯社作为亚洲网的创始者之一，与其他五个国家性通讯社--澳大利亚联合通讯社、日本共同社、马来西亚国家通讯社、韩国联合通讯社和印尼安塔拉通讯社--共同组成董事会。
    亚洲网随时接收来自世界各地企业、非营利机构、政府机构的商业信息，并将接收到的商业信息，根据客户的要求，发至有关成员通讯社，由成员通讯社译成该国文字，向当地媒体播发。
    亚洲网的信息内容主要涉及欧洲、北美和亚太地区各著名跨国公司的业务动态、新技术和产品、技术转让、企业的机构变动、投资动态等。
    亚洲网拥有一个全球性的销售网络，并得到在北美和欧洲占主导地位的权威信息发布公司的鼎力相助。
    作为亚洲网的成员，新华社愿意协助更多的中国企业走向亚洲，打入欧美。对公司企业来说，亚洲网以其无可比拟的强大传播功能，可为公司企业树立形象、产品推介、招商引资发挥其它广告无法企及的作用。
