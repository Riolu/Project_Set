　　新华网北京１０月２７日电 让全面小康照亮我们的梦想

　　新华社评论员

　　金秋的阳光普照大地，令人感到温暖而充实。这是收获的季节，也是规划未来的时刻。

　　正在北京召开的党的十八届五中全会，研究关于制定“十三五”规划的建议，为中国未来５年乃至更长远的发展谋篇布局，举国关注，举世瞩目。在网上，在街巷，在工厂，在乡村，在全球各个角落，人们以各种方式关注这次重要会议。“下个五年值得期待”“‘十三五’规划与每个人密切相关”“冲刺２０２０”……人们的感言中，道出了共同的心声——全面建成小康，让中国梦更加精彩，让我们的生活更加美好。

　　“日月忽其不淹兮，春与秋其代序。”在历史长河中，五年不过是忽忽一瞬，但无论对行进在复兴之路上的中国，还是对我们每一个人来说，即将结束的“十二五”，都有着非比寻常的意义。回首这五年特别是党的十八大以来，国家的发展有目共睹，老百姓的日子也越来越有滋味。

　　从广大群众“有活干、有钱挣”，到越来越鼓的腰包；从越织越密的社保网，到拔地而起的保障房；从山区孩子手里热腾腾的营养午餐，到高考、户籍、司法等领域促进公平正义的改革措施纷纷出台……民生改善成为人们最为深切的感受，中国梦呈现出厚重的民本底色。“人民对美好生活的向往，就是我们的奋斗目标。”习近平总书记掷地有声的宣示，化为指引发展的价值导向，激荡在梦想纷纷绽放的时代洪流之中。

　　梦想的可贵，在于它不仅是指引前行的灯塔，更是砥砺奋进的动力。抚今追昔、畅想未来，不断攀升的幸福指数为梦想注入新的内容，对美好生活的新期待加高着梦想的海拔。从吃得饱到吃得健康，从安居乐业到天蓝、地绿、水净，从“楼上楼下电灯电话”的物质憧憬，到“世界那么大，我想去看看”的精神期盼，亿万个不断生长的梦想与时偕行，激发出源源不断的奋斗热情，成为中国发展进步的力量之源。

　　难走的路是上坡路，登顶前的冲刺最关键。在改革开放３７年后的今天，人们对生活的期待值持续上升，对发展质量和水平的要求也在水涨船高。从现在到全面建成小康社会，未来５年是实现第一个百年奋斗目标的决定性阶段。新旧问题交织，发展的瓶颈、成长的烦恼与转型的阵痛相互叠加，梦想之路不会一帆风顺，需要继续付出艰辛努力。

　　问题是时代的声音，也可以成为成就梦想的契机。“十三五”规划，将聚焦如期全面建成小康社会这个既定目标作出全面部署，这不仅意味着经济社会发展的全面进步，也意味着为人的全面发展拓展更大空间。眺望２０２０，无论是稳增长、转方式、调结构、促改革等宏观课题，还是扶贫开发、棚户区改造等民生工程，或者是破解教育、医疗、养老等方面的难题，都在诠释以人民为中心的发展导向，都在为１３亿人构建更加坚实、更加公平、更加开放的梦想舞台。从百姓视角来观察，全面建成小康，就是不断增强人民群众的获得感、满足感、幸福感，不断为普通人创造实现梦想的机会。

　　人心是最大的政治，梦想是力量的源泉。改革开放以来，我国经历了７个五年规划，这３５年不平凡历程，见证了中华民族奋起直追、实现复兴的坚定信念，也激荡着一代代中国人“要的是更加美好的生活”的梦想心声。时至今日，谋发展、促改革，建设更富强的国家、过上更幸福日子，依然是人心所向、共识所在。未来征程上，牢牢抓住“增进人民福祉、促进人的全面发展”的出发点和落脚点，用好民生“指南针”，让发展更有温度、幸福更有质感，我们就能回答时代课题、回应人民关切，汇聚起１３亿人同心共筑中国梦的磅礴力量。

　　“全面建成小康社会，是我们奋斗目标的第一步，也是关键一步。”小康，这个中国人不倦追求的梦想，历经千年的期盼、百年的奋斗，距离我们从未如此之近。让我们携手前行，在“十三五”的接力奋斗中，开创国家发展、民族复兴、人民幸福的新图景，成就我们这一代人的光荣与梦想。
