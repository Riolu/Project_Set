新能源车政策盘点：17地享双重补贴

2015年10月28日 12:26:11 来源： 法制晚报
  • 评论0
  • 打印
  • 字大
  • 字小

　　今年以来，国家与地方政府已经出台了相关的补贴标准和政策，全国已有17个省份公布了地方财政的补贴政策。

　　今年上半年，国家部委新出台了近10项新能源汽车鼓励支持政策，包括减免新能源车船购置税、开放电动乘用车准入等。

　　国家补贴 最新标准最高达19万

　　早在2013年9月，国家相关部门出台了《关于继续开展新能源汽车推广应用工作的通知》，其中明确了在2013年-2015年，对消费者购买新能源汽车继续给予补贴。

　　最近两年来，国家针对新能源汽车已经出台了相关的补贴标准和政策，同时地方政府为了配合国家新能源汽车的战略也有针对性地推出了地方的新能源汽车政策，一系列优惠让很多消费者开始关注并考虑购买新能源车。

　　国家对于进入新能源车目录的纯电动汽车、插电式混合动力汽车和燃料电池汽车给予购车补贴。

　　新能源汽车的补贴采用的是逐年递减的政策，今年年初最新的新能源补贴金额在3.325万-19.0万元间，其中电动汽车和插电式(含增程式)混合动力汽车的补贴为3.325万-5.7万。

　　地方跟进 17省份发布补贴标准

　　地方的新能源汽车推广政策，一般应有两大政策：推广实施方案和补贴申领办法，或者推广实施方案应包括补贴办法。除了中央的政策频出台外，各地方城市结合自身实际，纷纷落实推广实施方案。

　　据《法制晚报》记者统计，目前，全国已有30个省份发布了关于新能源汽车产业发展和推广应用的实施意见。

　　在已发布推广实施方案的地区中，共有17个省份或城市发布了地方财政补贴标准，分别是北京、天津、重庆、上海、山西、陕西、甘肃、山东、贵州、福建、浙江、湖北、湖南、江苏、安徽、广东以及海南等地区。以上地区将享受来自中央和地方政府的双重补贴。

　　补贴比例 10地按国家标准1:1给予

　　从补贴比例来看，上海、甘肃、陕西、山西、海南、贵州和广东等7地，均根据各地方自身实际推广实施方案，其余10个地区均按照国家补贴标准1:1给予补贴。

　　以广东为例，广东省将按照国家购车补助标准的一定比例和实际推广应用数量对各市分类给予购车综合补助。

　　按照城市人均财政收入水平，各城市被分成四类地区，广州市为一类地区。 广州与惠州新能源汽车补贴不退坡，即按照2013年国家新能源补贴标准，对购买新能源汽车的消费者进行1∶1的配套补贴。

　　据公开资料显示，地方财政补贴上，地方财政补贴的金额最高的为山东和江苏两地，补贴金额可达到6万。同时，还免征车辆购置税。

　　其中江苏的补贴标准较为特殊，纯电动乘用车按轴承的不同进行补贴，大于2.45米每辆补贴2.4万元、小于2.45米每辆补贴1.8万元、小于2.2米每辆补贴1万元。纯电动专用车的补贴标准则维持不变，按电池容量每千瓦时补贴800元，每辆最高不超过6万元。
