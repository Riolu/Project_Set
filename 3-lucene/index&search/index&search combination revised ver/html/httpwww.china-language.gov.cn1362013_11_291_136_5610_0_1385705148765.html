国台办:两岸合编《子海珍本编》 实现古籍共享


2013年11月27日 10:58　来源：中国新闻网

 

    中新网11月27日电 国务院台办发言人范丽青27日在例行新闻发布会上介绍，两岸合作编纂的《子海珍本编》的第一辑即将出版，大陆卷的发行仪式将于11月30日在山东大学举行，台湾卷发行仪式将于12月12日在台湾举行。

 

　　范丽青回复称，两岸合作编纂的《子海珍本编》的第一辑即将要出版，这是两岸合作编纂“子海工程”的首批重大成果，实现了两岸古籍典藏共享的学界夙愿，是两岸共同弘扬中华优秀传统文化的成功实践。

 

　　范丽青解释，“子海”是“子书渊海”的简称，“子海工程”就是将古籍中子部精华典籍加以整理，对传统子学进行研究，是推动中华文化走向世界的大型文化建设系统工程。

 

　　范丽青透露，这部《子海珍本编》将分十辑出版，最终将收录5000种子部古籍。《子海珍本编》(第一辑)大陆卷的发行仪式将会于11月30日在山东大学举行，台湾卷发行仪式将于12月12日在台湾举行。
