首届世界互联网工业大会将在青岛举行
2015年9月25日 15:52

在“中国制造2025”和“互联网+”的行动浪潮中，我国的工业转型升级之路到底该怎么走？10月15日至16日将在青岛举行的首届世界互联网工业大会，力求给出路径和答案。

本届大会由工业和信息化部、中国工程院指导，青岛市政府和经济日报社联合主办，以“互联网时代工业变革之路”为主题，深入探讨互联网时代新一轮工业革命的转型路径和价值取向。

大会将围绕业界普遍关注的智能制造发展、业态模式创新、产业跨界融合等重大命题开展对话交流，推动各方进一步凝聚共识、深化合作、共享机遇、共创未来。

青岛市委常委、副市长王广正在首届世界互联网工业大会发布会上表示，世界互联网工业大会旨在为学习借鉴国际前沿理念和先进模式、推动“中国制造2025”和“互联网+”行动等重大战略的实施搭建国际产业交流与产能合作高端平台，促进全球互联网与工业各领域融合创新发展。
