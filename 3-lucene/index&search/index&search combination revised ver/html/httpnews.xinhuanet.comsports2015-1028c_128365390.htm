曝足协不受理恒大申诉:不根据单方要求来定裁判

2015年10月28日 07:07:41 来源： 网易体育
  • 评论0

  • 字小

　据《北京青年报》消息，因被鲁能打进一粒疑似越位球，广州恒大提前一轮卫冕中超的愿望破灭。恒大、鲁能比赛当值助理裁判穆宇欣对于此粒进球有效的认定或将一个赛季以来整体表现平稳的本土裁判带入争议的漩涡。虽然中国足协今明两天内将对此粒入球进行专家评议，虽然足协依规则不会受理恒大俱乐部有关此次判罚的申诉，更不会应任何一家俱乐部要求在关键战中换外籍裁判组，但联赛末轮国安与恒大搏命战外籍与本土主裁判如何取舍，令中国足协左右为难。

　　裁判评议专家组今明断案

　　前天恒大、鲁能比赛第94分钟，也就是补时倒数第2分钟，鲁能队韩鹏接到王永珀的横传球后，打空门得手，凭借此球鲁能以2比2逼平恒大队，也打破了主队提前一轮实现中超5连冠的愿望。赛后，恒大俱乐部及球队主帅斯科拉里均表达了对“鲁能队此粒进球越位在先”的质疑。而赛后恒大球员黄博文还通过微博发出了进球的立体成像图，外界也对此粒进球展开热议，绝大多数人都认为这是一粒明显的越位进球。一位前国际裁判向北京青年报记者表示，“从目前图像资料显示情况来看，这是明显的越位球。”尽管如此，中国足协鉴于此次事件重大，还是决定慎重评断此次判罚。据了解，裁判办5人专家评议小组将在今、明两天内，通过邮件形式对此次判罚进行评议。

　　足协不受理恒大申诉

　　对于此次争议判罚，恒大俱乐部坚持认定为“误判”，恒大俱乐部赛后决定就此次判罚向中国足协提出申诉，同时要求中国足协为中超最后一轮国安主场与恒大的比赛安排一整组外籍裁判，恒大俱乐部申诉书的内容称，“助理裁判员在视野并未受阻的情况下居然判罚进球有效。恒大俱乐部在此对于助理裁判员在比赛中的严重错判正式提出严正的抗议和申诉。因为错判，使本场比赛的结果发生了改变，并极有可能对整个赛季的最终结果产生非常重大的影响，使中超联赛的公正性受到严重的挑战，我们表示强烈的愤慨和不满。我们再次建议，为了使比赛更加公平、公正，希望争冠相关的所有比赛场次的主裁判及助理裁判均使用外籍裁判。”

　　中国足协相关人士昨天向北青报记者表示，按照有关规定，俱乐部不可以就裁判判罚问题提出申诉，中国足协不会受理任何类似申诉。此前，中国足协曾经按照相同规定，先后驳回了来自贵州人和与辽宁宏运两俱乐部有关裁判问题的申诉。该人士同时确认，中国足协也不会根据某一家俱乐部单方的要求来确定比赛的裁判员，即使中国足协聘用外籍裁判或调换本土裁判，也都与俱乐部自身的诉求无关。据了解，中国足协为确保中超收官阶段的裁判工作有序，已就最后两轮中超邀请部分外籍裁判执法部分关键场次作出预案，但外籍助理裁判本不在邀请之列。一般来说，从决定起用洋哨到落实其来华执法至少需要一周时间，因此从时间和程序来说，邀请洋助理裁判执法国安、恒大之战的难度相当高。就在上周，原定执法国安客战泰达比赛的马来西亚裁判苏赫兹就因行程问题无缘来华执法。（记者 肖赧）
