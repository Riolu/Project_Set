上海市志愿者协会第二届理事会第四次（扩大）会议召开

2015-09-28 11:01:18    来源：上海文明网   

为进一步提升上海市志愿者协会统筹协调全市志愿服务工作的能力，推动上海志愿服务工作逐步走向制度化、规范化、社会化，近日，上海市志愿者协会第二届理事会第四次（扩大）会议在余德耀美术馆三楼会议室召开。中共上海市委宣传部副部长、市文明办主任潘敏主持会议并作重要讲话；中国志愿服务联合会副秘书长段桂青，市文明办副主任、市志愿者协会常务副会长姜鸣等领导出席会议。市志愿者协会第二届理事会理事、监事，以及各区（县）、委（办）志愿服务工作负责人出席会议。 

会议审议通过了关于潘敏同志担任上海市志愿者协会第二届理事会会长，增补杜国巍同志为上海市志愿者协会理事、副秘书长的议案；通报了部分理事、监事因职务变动自然更替的情况；审议通过了关于修改《上海市志愿者协会章程》的议案。  

市文明办副主任、市志愿者协会常务副会长姜鸣向大会作了协会2015年工作情况和下阶段工作安排的报告。目前，全市共有注册志愿者180多万，市级志愿者服务基地102家，市志愿者协会直属总队92支，形成了助老、助残、助医、助学、交通、环保、科普、文明劝导等20类共五万多个志愿服务项目。今年以来，协会围绕中心、服务大局，着力从志愿服务制度化建设、品牌性活动、组织体系建设等方面入手，为推动上海志愿服务事业的新发展做了大量的工作。 

会议还进行了四项签约仪式，市文明办副主任、市志愿者协会常务副会长姜鸣同志代表上海市志愿者协会，分别与市社区服务中心主任王涛签订上海市志愿服务一网一线合作备忘录；与中国志愿服务联合会副秘书长段桂青就对接志愿云信息系统签订协议；与蚂蚁金融服务集团公共服务事业部总监钟华斌就开设支付宝“上海志愿者”服务窗签订意向书；与市民信箱信息服务有限公司副总经理陈黄海就开设“市民云+上海志愿者”服务窗签订协议。 

中共上海市委宣传部副部长、市文明办主任、市志愿者协会会长潘敏就近期的志愿服务工作作了重要讲话。他指出，志愿服务工作要着力培育志愿文化，深化志愿服务的价值内涵，要在充分把握上海发展实际和市民群众需求的基础上，建设与上海国际文化大都市相匹配的志愿文化；要着力加强法制建设，提升志愿服务制度化水平；要着力加强组织体系建设，推动志愿服务纵深发展，充分发挥协会的统筹协调作用，注重整合利用各类资源，拓展志愿服务事业发展的广阔空间；要着力完善志愿服务平台，更好地服务群众需求，充分发挥好社区志愿服务中心和志愿服务信息平台的枢纽作用，注重资源整合，培育品牌项目。 

据悉，上海市志愿者协会为加强信息化建设，与中国志愿者联合会“志愿云”对接，今后上海注册志愿者将拥有全国统一志愿者证编号，并实现志愿服务时间全国转移接续；上海市民和在上海的游客通过支付宝城市服务“上海志愿者”和“市民云+上海志愿者”服务窗，随时随地利用手机客户端，实现加入志愿者队伍、查询志愿服务时间和查找志愿服务项目等功能；开通“上海志愿服务热线”，为全市注册志愿者和志愿服务组织提供有关上海志愿服务政策及相关活动等内容的来电咨询服务。
