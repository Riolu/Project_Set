交通部官员咋瞒报配偶百万基金？

              花玉喜

2015年2月，交通运输部党组拟提拔一批局级干部。组织部门对照中组部反馈查询结果；对考察对象个人有关事项报告进行核实。发现一考查对象未申报其配偶持有的167万元基金。其配偶未报告基金持有数超过规定数额。交通运输部党组取消其领导职务考核对象资格。对其批评教育，责令补报有关事项并进行通报（2015年10月26日中纪委网站）。

官员考察对象为什么要瞒报配偶持有的167万元基金呢？显然，交通运输部该官员夫妻双方都是公职人员。如果有一方不是公职人员，也就不存在所谓的瞒报行为了！交通运输部这位官员为什么要瞒报偶持有的167万元基金呢？显然，瞒报总有瞒报的原因，瞒报总有多少不足为外人道的理由！

交通运输部官员瞒报配偶持有的167万元基金，或许是因为他们家庭存在一定金额的非法贪腐收入。如果该官员据实申报，有可能暴露腐败问题被查处。如此，该官员自然不得不瞒报，瞒报总比暴露腐败问题风险小一些！

交通运输部该官员瞒报配偶持有的100多万元基金，是情非得已不得不瞒报。因为这100多万元基金是非正规渠道的不合法收入，你让该官员怎么能不瞒报？因为这笔资金中有违规经商的收入，更有家中红白喜丧事收受的大量礼金！既然是大量非正规渠道的不合法收入，你让这位官员又怎么不瞒报？

交通运输部该官员为什么要瞒报配偶持有的100多万元基金呢？是因为官员总有种种过多的额外收入，导致与家庭实际收入不相符！或许，该官员的柴米油盐酱醋茶都有人奉送，烟酒、服饰也都有人奉送。有多少吃不了用不了的物品还要卖钱。如此官员收入怎么能没有出入？如此你让官员又怎么能不瞒报？

官员不如实报告个人有关事项，实质是一种严重的欺骗组织行为。官员瞒报家庭实际收入更是对党、对组织的一种不忠诚行为。向上级组织报告家庭实际收入，也是一个党员干部应尽的义务。

2015年7月，交通运输部通报，在部机关干部选任工作中，就有1名局级、12名处级领导职务考察对象瞒报个人有关事项。现实中哪一级组织，哪一部门不存在官员瞒报个人事项现象？由此可见现实中官员对个人有关事项报告隐瞒现象多么严重？官员瞒报现象非常普遍，已经成为官员组织纪律松弛的一大忧患！

交通运输部党组拟提拔官员有未如实报告个人有关事项现象，仅被取消其考察对象资格，并给予批评教育，责令补报！显然，这样的取消考察资格、给予批评教育、责令补报还不够！还应该弄清该官员家庭瞒报的金额是怎么来的？背后存在多少违纪违规行为？是否牵扯多少经济犯罪？

新修订的《中国共产党纪律处分条例》第七章“对违反组织纪律行为的处分”内容，增加了不按照有关规定或者工作要求向组织请示报告重大问题，不如实报告个人有关事项，篡改、伪造个人档案资料等违纪条款。并对党员违反组织纪律行为，作出明确处分规定。坚持从严治党，就必须注重依规依纪治党，切实加强党内监督。因此，对党员干部瞒报现象，还需要有震慑力的处理力度！不然，多少党员干部的瞒报现象也会屡禁不止！不然，严肃党纪很难传递到位！
