关于上海交通大学第四届英文网站评选活动的通知

时间：2015年06月10日 09:16 发布人： 浏览次数：[]

[责任编辑]李敏妍 [信息提供]党宣 

 

　　全体师生、教职医护员工： 

 

　　上海交通大学英文网站建设与管理是学校国际化战略的核心任务之一，也是学校国际形象塑造与国际影响力提升的重要举措。在学校第十次党代会报告中，推进高层次国际化办学，构建国际合作交流新格局与塑造学校良好的社会形象和国际形象成为学校未来发展的重要建设任务。 

 

　　推进校院两级英文网站建设是本年度学校重点工作之一。第四届上海交通大学英文网站评选活动本着“以评促建”的指导思想，已于5月启动，现进入公开展示和网络投票环节。全校师生、教职医护员工可登录评选活动专题网，参考《评选标准》进行网上投票（《评选标准》详见附件）。 

 

　　感谢您的参与！ 

 

　　投票说明： 

 

　　1．此次评选参评单位共三十九家（学院、直属单位），展示类网站四家（不参评）； 

 

　　2．投票者需通过jaccount登录进行投票，每个登陆账号可进行三次投票，每次须推选二十家单位； 

 

　　3．网络评选时间：2015年6月10日-6月20日。 

 

　　活动时间：2015年6月10日-6月20日 

 

　　投票网址：202.120.33.37/15web/english.php 

 

　　上海交通大学党委宣传部（新闻中心）
　　二〇一五年六月十日
