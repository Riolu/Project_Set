英媒称安倍在中亚揽生意很费力：中国价格更低

2015年10月28日 07:05:58 来源： 参考消息网
  • 评论0
  • 打印
  • 字大
  • 字小

　　英媒称，日本首相安倍对中亚的旋风访问，意在为日本企业招揽生意和抗衡中国在该地区日益增强的影响力。

　　据英国《金融时报》网站10月26日报道称，安倍在土库曼斯坦被授予荣誉博士学位，在乌兹别克斯坦为一家新歌剧院开张剪彩，随后在10月26日抵达吉尔吉斯斯坦首都比什凯克。安倍此访是9年来日本领导人首次到访苏联解体后的中亚。

　　设在阿拉木图的风险评估公司总裁、哈萨克斯坦政治学家多瑟姆·萨特帕耶夫说：“他们担心未来中国将是在这里的主要力量。”

　　安倍把向快速发展的亚洲市场出口基础设施作为其经济战略的中心，可是他达成协议很费力，因为中国愿意提供无附加条件的资金和最低价。

　　报道称，印尼最近决定与中国合建高铁，这令日本行业一度黯然神伤。英国决定用中国技术建造一座核电站，这令核电行业惊骇，担心这将是中国追求全球市场的参照电站。

　　奋力应对商品价格下跌和俄罗斯经济衰退的中亚国家政府很可能对日本兴趣增加的前景表示欢迎。蒙古国议会议长10月22日对安倍说，他把日本视作蒙古国的“第三个邻国”。

　　但中亚商界人士和分析人士深深质疑日本能否提供一个有意义的制衡物来对付中国。中国在过去15年向自然资源和基础设施投入大量资金，目前是该地区最大的贸易伙伴。

　　报道称，东京拒绝加入今夏启动、由北京主导、资本达500亿美元的亚洲基础设施投资银行。

　　虽然长期以来日本是该地区发展援助的主要捐助国，但经济联系相对有限。日本国际石油开发株式会社（INPEX）持有陷入麻烦的哈萨克斯坦大型卡沙甘油田股份，还有几家日本公司在石油和铀业赢得建造处理厂的合同。

　　格拉斯哥大学中亚问题专家卢卡·安切斯基说：“我非常怀疑日本和中亚之间有开展合作的可能性。”他认为，地理障碍和该地区糟糕的投机环境将阻碍安倍的努力。

　　报道称，一名曾在日本工作过的哈萨克斯坦高管把东京的中亚政策描述为“一团乱麻”。他提到建造哈萨克斯坦首座核电站的竞争，称日本企业参加了几个不同的竞标活动。

　　他说：“这就是为什么中国更加成功。它要直截了当得多。他们推出一家由拥有主权担保的中国银行资助的中国竞标者。”

　　报道称，安倍此访带着大约50家日本银行、贸易公司和大学，还宣布价值数十亿美元的合作项目。不过了解这些协议的人士称，它们仍远不是具有约束力的合同。

　　安倍与土库曼斯坦总统别尔德穆哈梅多夫签署了几笔据说价值180亿美元的项目合作协议，不过并未公开细节。

　　报道称，日本企业计划在巨大的加尔基内什天然气田附近建厂，但不拥有这些工厂。这可能包括一座造价80亿美元的天然气处理厂、一座造价40美元的用天然气作原料的化工厂和一处造价20亿美元的生产化肥用尿素的设施。这些日企包括三菱和伊藤忠商事公司等贸易公司以及日挥公司和千代田公司等工程公司。

　　在乌兹别克斯坦，安倍宣布价值85亿美元的合作项目，包括一项三菱重工业公司和三菱商事公司耗资10亿美元建造一座化肥厂的合同。

　　每当国会休会时，安倍就疯狂出访，带着促进日本经济和抵消中国日益增强影响力的双重目的，不过对中亚的访问相对靠后。

　　报道称，日本内阁官房长官菅义伟说：“这将发展我们与不同国家的关系。日本帮助了这些国家的建设，我们希望再接再厉。”（编译/王海昉）
