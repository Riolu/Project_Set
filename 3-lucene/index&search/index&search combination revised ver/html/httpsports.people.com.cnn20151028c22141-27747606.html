联赛杯-张伯伦沃尔科特伤退 阿森纳爆冷0-3出局

来源：人民网-体育频道  2015年10月28日08:05


沃尔科特受伤离场

人民网北京10月28日电（杨磊） 北京时间10月28日凌晨，2015/16赛季英格兰联赛杯第4轮继续进行，阿森纳客场0比3不敌谢周三，张伯伦和沃尔科特先后伤退，华莱士、若奥和哈奇森连入3球。

阿森纳开场不久即遭遇打击，张伯伦受伤被沃尔科特换下。枪手第19分钟再遭打击，替换出场沃尔科特也因伤离场，本纳塞替补出场。谢周三第27分钟凭借全场首次射门取得领先，普迪尔左路突破回传，罗斯-华莱士禁区边缘内射入左下死角。

罗斯-华莱士禁区弧顶任意球直接射门稍稍偏出右侧立柱。赫兰面对3人防守强突禁区左侧的射门被切赫扑出近角。谢周三第40分钟扩大比分，罗斯-华莱士左路战术角球传中，若奥小禁区前头球破门，2-0。

下半场，谢周三第51分钟继续扩大优势，罗斯-华莱士任意球开出，汤姆-李斯禁区右侧凌空垫传，哈奇森近距离打入空门，3-0。阿森纳第70分钟险些破门，弗拉米尼角球混战中外围斜传，默特萨克小禁区左侧头球攻门被威尔德史密斯神勇救出。

阿森纳出场阵容(4-2-3-1)：33-切赫；2-德布希，21-钱伯斯，4-默特萨克，3-吉布斯；20-弗拉米尼，47-卡马拉(60',37-贝里克)；15-张伯伦(5',14-沃尔科特)(19',36-本纳塞)，28-坎贝尔，45-伊沃比；12-吉鲁
