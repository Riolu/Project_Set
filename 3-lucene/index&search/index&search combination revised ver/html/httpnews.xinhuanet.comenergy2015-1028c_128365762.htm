中英能源合作为全球应对气候变化树立典范

2015年10月28日 07:45:00 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

　　新华网北京１０月２７日（记者王海清）习近平主席访问英国期间，相关方就共同修建和运营欣克利角Ｃ核电站在伦敦达成战略投资协议，中方企业联合体将持有该核电站３３．５％的股份。该项目是中英愈加密切的能源合作的一个缩影，显示了近年来中英能源合作不断深化、内容更加多元、更加重视清洁能源的趋势，为两国合作应对气候变化提供了新的契机。

　　历史性旗舰工程

　　欣克利角Ｃ核电站选址英格兰西南部，估计项目第一期建造成本达１８０亿英镑（约合２８０亿美元），预计２０２５年建成完工投入运营。该核电站将是英国单体容量最大的发电站，也是２０多年来英国首个新建核电站。

　　习近平表示，欣克利角Ｃ核电站项目是中英合作的“旗舰项目”。英国首相卡梅伦则称赞，该项目具有历史性意义。

　　这是中国企业首次在欧洲参与民用核电站建设，中英两国都将从中受益：一方面，中企将在英国能源市场扮演更重要角色，并有机会向世界其他国家出口核技术和设备；另一方面，英国现有核电站日渐老化，中方投资将为英国核电系统注入活力，并创造更多就业机会。

　　法国电力集团是欣克利角Ｃ核电站项目主要投资方。集团英国子公司总裁文森特·里瓦兹说，该项目投资为英国有史以来获得的最大外国投资，核电站工程建设将创造２．５万个就业岗位。虽然不是所有人都赞成建设欣克利角Ｃ核电站，但这不是一座核电站那么简单，它将产生深远的积极影响。

　　多元化能源合作

　　备受关注的欣克利角Ｃ核电站只是中英能源合作的“冰山一角”。在全球积极控制污染排放和追求低碳发展的背景下，中英两国能源合作越来越重视清洁能源和可再生能源。近年来，双方签署了巨额天然气商业合同，并在太阳能、风电、潮汐能发电领域展开合作。

　　习主席出访英国期间，中英双方签署的能源大单除欣克利角Ｃ核电站项目投资协议外，还有中国华电集团与英国石油公司签署的《天然气长期贸易协议》。

　　今年６月，中国港湾工程公司中标英国斯旺西湾潮汐发电项目，合同金额３亿英镑（约合４．６亿美元）。作为全球首个潮汐泻湖发电项目，该电站投入运营后每日发电量将能满足大约１２万户家庭电力需求。

　　２０１４年１２月，中国广核集团从法国电力集团手中收购英国三个陆上风电场８０％的股份。

　　２０１４年６月，中国国务院总理李克强访英期间，中海油与英国石油签署了２００亿美元的天然气供应合同。按照该合同，英国石油将在从２０１９年起的２０年内每年为中海油供应１５０万吨液化天然气。

　　２０１４年４月，中航国际正式向英国运营方交割由其承建的１２兆瓦地面光伏电站。该电站已于当年３月正式并网发电，在夏天每日所发电量足够２５００户家庭使用。

　　位于伦敦的国际知名律师事务所品诚梅森最近发布的一份研究报告说，更多中国资金将投资英国基础设施项目，而能源领域将成为最大受益者，尤其是核电、风电和太阳能。

　　共创“绿色”未来

　　中英两国能源合作大踏步向前发展的同时，两国领导人很清楚地看到了气候变化带来的共同挑战。

　　访英期间，习主席在不少场合高度评价两国在追求绿色发展方面取得的成就，同时呼吁双方在气候变化方面加强政策协调，展开务实合作。

　　中英两国签署的《中英清洁能源伙伴关系谅解备忘录》将为两国气候变化合作带来新机遇。《中英关于构建面向２１世纪全球全面战略伙伴关系的联合宣言》也将气候变化列为中英合作的一个重要领域。

　　可以肯定的是，中英有强烈的经贸合作诉求和政治意愿为两国开创更为绿色的未来，这也对全世界有益。（完）
