文武双全 25万内四驱功能强大的SUV推荐

　　如果按照四驱功能来划分的话，目前的SUV基本可分为城市及硬派两种。就市场占有率来看，更注重公路属性的城市SUV无疑更获得消费者青睐。或许你不禁会想，作为家用、通勤的工具，如果能配上一定的四驱功能岂不两全其美？完美倒说不上，不过今天推荐的四款车确实是普通城市SUV行列中，四驱能力不俗的几个。

　　● 东风日产 新奇骏

　　官方指导售价：18.18~26.78万

　　眼看2015年又将接近尾声，在刚刚过去的上半年中东风日产新奇骏的销量同比增幅高达87.3%，销量于所有合资紧凑SUV中排名第二。卖的好自然有其道理，除了这一车型消费者所看中的省油、经济、大空间外，奇骏的四驱功能在所有同级城市SUV中也是较为突出的，对它来说无疑是锦上添花了。

　　-它的四驱功能如何？

　　奇骏采用的是一套多片离合器式的智能全模式四驱系统，也就是我们常说的适时四驱，这种四驱系统在20万级别的城市SUV比较常见。其核心部件是一套由电磁控制的多片离合器式的中央差速器，即通过放松和压紧来分配前后轴的动力。前后轴均为开放式差速器，并带有电子辅助制动功能。

　　奇骏四驱系统设置了三种驱动模式，驱动旋钮位于换挡杆后方，具有2WD(两驱模式)、Auto(自动模式)、LOCK(四驱锁止模式)三种模式。

　　在两驱模式下，中央差速器断开，车辆为前驱状态，也是日常使用最多的模式；在Auto自动模式下，主要还是以前驱为主，不过车辆会根据车辆的路况(如前轮打滑、加速起步等)，适时介入四驱，这个模式主要用于湿滑或坑洼等；而在Lock模式，中央差速器锁紧，前后轴的动力分配固定在50:50，多用于较为复杂的越野路面。

　　- 外观/内饰：温馨居家风

　　风格转型无疑是新奇骏此次换代的重点目的之一，新一代奇骏采用了家族式SUV风格，从远处观察有些楼兰与新途乐的影子，先不说它像谁多一些，但重点是，这一款奇骏向型男转变确实成功了，包括笔者在内的大多数同行认可度都挺高的。

　　新奇骏的内饰采用了与天籁相似度很高的家族式设计，咋们先不讨论时不时尚或者好不好看，最重要的一点就是，全新的内饰布局让奇骏看起来更显档次，整体的氛围感要更好，营造出更好的亲和力。

　　- 空间：座椅舒适度高

　　奇骏的座椅可以说是从天籁身上移植过来的，不同的是坐姿保持了SUV该有的高度，其中高配车型驾驶员是带有8向电动座椅调节的，但略微不足的是副驾驶位依然是全手动控制。

　　新奇骏前排的空间，头部很宽裕，而且横向空间很好，所以能轻易制造出开阔感。后排空间那也是同级领先水准的，180cm体验者头部能有四指空间，腿部两拳以上的余量，椅背角度不再像老款那样可调，但是倾斜度和包裹性很合理，比老款坐着舒服多了。

　　- 动力：输出平顺 保养便宜

　　新奇骏上分别采用了2.0L和2.5L两款发动机组合，而2.0L发动机还采用了缸内直喷技术，最大功率110kW（150PS）/6000rpm，最大扭矩200N·m/4400rpm，采用了缸内直喷技术，动力上领先同级别2.0L发动机。两款发动机均采用了可模拟7个挡位的CVT变速箱。

　　日产的CVT变速箱相信不少网友都有所了解，经过多次技术更新之后，最新一代CVT变速箱在初段响应时更加灵敏，给人以一种冲劲十足的感觉，加速十分的平顺，感觉。

　　根据我们的保养手册，新奇骏的小保养仅需要299元一次，性价比可谓是非常的高。而根据4S店采集回来的数据计算，东风日产新奇骏10万公里保养总费用为：9448元（CVT）/9482元（6MT），同级车型中有极大的优势。

　　车型点评：对于其它竞品，奇骏的优势就在于平时可供你舒舒服服开，通勤、买菜、出游无所不能，而真正遇到复杂路况时又能够给你足够信心与应对，再加上不高的保养成本和舒适的车厢设计，这车不火那也是说不过去的。
