从自然中寻找极简主义中的温暖感

  • 评论0
  • 打印

  • 字小
时尚观点

    10月27日，ROSEMOO容子木2016春夏女装成衣系列在北京798艺术工厂发布，此次系列的主题为“万物之寂”，灵感来源于日本传统美学——侘寂。

    如果这是一篇命题作文，那么这场秀是可以拿高分的，全黑色背景，幻灯在如纱般的白色幕布上打出云朵、远山等水墨画般的图案，一束白色灯光打出T台位置。如流水般清滢、干净的音乐响起，模特从白色薄纱中翩然走出，颇有几分独立于武林之外的仙气和禅意。

现代都市越来越繁华、喧嚣，因此一系列主张“东方文化传承与自然生活美学”追求的品牌应运而生。这些品牌在设计上有诸多雷同：线条宽松、简约，以沉稳内敛的自然色系为主基调，糅合亲肤的天然棉麻衣料。

“最初带给我灵感启发的是主题面料，当我看到一些棉麻面料的时候，我会想到极简主义中的wabi-sabi体现出的一种温暖感和瞬间感，并不是想现代主义中所表现出的冰冷感。”容子木设计师李铂楠在接受新华网采访时说。

通过后处理的工艺使其面料材质呈现质朴的肌理感，像水中的波纹、山上的岩石，变现出一种很有力量的极简主义。帽子和衣服是成套搭配的，通过现代主义的折叠来整体的提升模特的气质感，同时跟服装中的剪裁相呼应。

似乎只要你提到东方、禅意、自然这些词汇，脑海中总会出现一个布衣棉裙、素颜直发的女人形象。如何在一众看似很像的品牌中做到不一样，想必今年容子木也考虑到这个问题了，因此，很难得的在黑白灰色中间，看到了鲜艳的红色出现，依然是大气、利落的剪裁，却用针织和丝绸来表现，层次分明，成为整场秀的亮点。（luluorange)
