近期逾90家公司高管增持 上市公司稳步回购股份

2015年10月28日 07:31:50 来源： 中国证券报
  • 评论0
  • 打印
  • 字大
  • 字小

    根据相关数据统计，进入10月份后，实施股份回购情况的上市公司有12家，共回购股份总量约为7970万股，回购金额总计约为12.12亿元。而这12家公司中主板公司占了6家。不过，目前完成的回购数量与当时的承诺仍有很大距离，未来一段时间预计相关上市公司回购仍会进行甚至加大回购力度。

    而从上市公司的高管增持股份情况看，近一个月有92家公司的高管净增持，增持总量约4500万股，部分公司大股东及高管增数量较大，已经完成了当时承诺的增持量。

    上市公司稳步回购股份

    根据Wind数据统计，10月1日至10月27日，国元证券、青岛海尔、伊利股份等12家公司发布公告，披露了前期实施股份回购情况。国元证券表示，截至2015年9月30日，公司累计回购股份464.6704万股，占公司总股本的比例的0.2366%，最高成交价为16.10元/股，最低成交价为15.12元/股，支付总金额约为7348.31万元（含交易费用）。青岛海尔公告称，截至2015年9月30日，公司回购股份数量为3774779股，占公司总股本的比例为0.062%，成交最高价为9.80元/股，成交的最低价为9.02元/股，支付总金额为3563.8260万元（含印花税、佣金等交易费用）。

    数据显示，截至9月底，这12家公司共回购股份总量约为7970万股，回购金额总计约为12.12亿元。而根据这些公司之前的公告，其回购上限总金额约为160亿元，二者相差近140亿元。

    从各家公司回购情况看，回购数量差别比较大。伊利股份回购超过3700万股，而回购量小的仅55万多股。金额差别也很大，高的达到5.88亿元，少的不到30万美元。

    公告显示，这12家公司所回购的股份将用于注销、或用于员工持股等。从实施的效果看，总体上对稳定股市价格起到一定作用，也增强了投资者信心。

    部分公司由于种种原因，终止了回购计划。盛和资源公告称，随着市场由剧烈异常波动逐步趋向常态化波动发展，加之公司最近正在进行经营业务范围内的整合、收购事项，公司股票已于2015年8月10日正式进入重大资产重组停牌阶段，股份回购事项已经不适用于公司的战略发展，决定终止回购事项。菲利华则称，根据中国证监会《上市公司回购社会公众股份管理办法（试行）》要求，同时考虑到公司将在股东大会批准后实施限制性股票激励计划，公司认为不宜在近期实施回购股份事项。

    部分公司是由于客观原因而终止了回购计划，如重组而停牌，回购难以实施。部分公司回购股份进程较慢，这或许与公司的资金状况相关，也或与当时公司设定的回购价格相关。

    92家公司高管积极增持

    根据东方财富的统计数据，最近一个月两市共有92家上市公司高管净增持公司股票，总量约4500万股。其中，增持最大的前十家公司高管增持量约3000万股，耗资超过4亿元。以岭药业和南风股份的高管净增持量都在500万股上，增持所用金额均在9000万元以上。

    以岭药业董事副总经理9月30日增持3000万股，南风股份实际控制人之一、公司董事杨子江9月30日增持496万股。根据公告，杨子江于2015年9月30日通过银河汇通60号定向资产管理计划在二级市场增持公司股票496.2173万股，占公司股份总数的0.974%，增持金额为9120.4739万元。公告同时表示，其增持是基于对公司未来发展前景的信心以及对公司价值的认可。

    上市公司高管增持股票，反映了高管对自家公司发展前景的信心，也向广大投资者传递了强有力的信号，未来或者具有向上的空间。

    从行业分布看，高管增持股份的公司没有明显的行业特性。但从增持股份数量前50家公司看，来自主板的有13家，来自创业板的有15家，其他基本来自中小板。

    部分公司大股东完成增持

    上市公司大股东也在积极增持股份，动手早且完成承诺也较早的，这在大型国企中表现比较明显。中国建筑的控股股东中国建筑工程总公司通过上海证券交易所交易系统增持公司股份2350万股。早在7月份宝钢股份大股东宝钢集团通过上海证券交易所交易系统增持了公司股份144万股，占公司总股本的0.0087%。

    此外，以岭药业控股股东以岭医药集团近日公告，完成了增持金额不低于3000万元达到3014万元，增持187.64万股。公司同时公告，其他高管人员也完成了当时的增持承诺，合计增持数量超过800万股。

    控股股东及公司高管对公司的经营和发展期情况远比普通投资者更为熟悉，他们的增持无疑具有极大的号召力，特别是以百万甚至以千万计的大手笔，对稳定投资信心和公司股价起到了重要作用。
