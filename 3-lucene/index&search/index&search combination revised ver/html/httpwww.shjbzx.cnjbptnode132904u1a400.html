培训班学员感言:带着问题来 带着收获回
2009年9月29日 15:35

　　我是一个相对特殊的学员，因为我是在网上看到这次培训的通知，自己掏了腰包，请了年假来上课的。

　　我来自于一个专业做钢铁市场资讯的网站；有一天，我在电梯里遇到我们的经理，我问他：“我们的网站属于什么性质？”他说：“我们属于行业门户吧。”我觉得他的语气不大肯定；第二天，我又在电梯里遇到我们的主管，我问他：“我们的网站属于行业门户么？”他说：“算不上吧，我们只是普通的商业网站。”对于他们的不同解答，我很迷茫：公司的老大们姑且不能明确自己的定位，那作为一个刚从学校毕业一年，事业刚刚起步的小小网站编辑，我的位置又在哪里？

　　于是，带着三个问题，回到我的母校回到教室来找答案。我的三个问题是：一、作为网站编辑，所需的知识基础有哪些？二、商业网站在转载其他网站新闻时，需要注意些什么？三、其他网站的编辑们都在做些什么？在这个行当，我是不是有职业发展的空间。

　　在这里，我要感谢市政府新闻办的领导们和复旦大学新闻学院的教授、讲师们给我们带来这么好的课程，也感谢刘华宾和应健老师孜孜不倦的陪读，我不仅得到了大部分的答案，还有了额外的收获：我对于网络编辑从理论到实践有了全面的了解和认知，并且得以机会和在座那么多来自知名网站的同行们交流，这对我日后的工作和生活都是非常受用的。

　　此外，我也给这次培训提一些建议：随着网络融入商业，商业网站越来越多，商务用户越来越多，我希望可以开展专门针对商业网站编辑的培训课程，因为我们这些网站、这批人更需要规范和培养。

　　其他建议：

　　一、类似于小组讨论题的那些问题，希望舆情处理的课中讲解些详细的解决思路。

　　二、黄旦老师在课上播的视频（2003年新闻奖颁奖典礼）让我很有感触，虽然培训时间紧，但是如果能给半天，以这些影音资料为教学内容，不仅丰富形式，调动兴趣，其实收获也不小的。

　　理论类课程的内容和授课顺序我觉得安排得很好，而应用类课程从顺序上而言，并没有什么逻辑上的规律，换句话说，就是理论课程感觉成一个体系，而应用课程相对分散和不成体系，虽然时间有限，应用方面的内容很多，但是也需要有循序渐进的感觉。

　　                              学员  张璐
