东华“小甜橙”服务春运 自编广场舞

2015-02-04 13:23:30    来源：上海教育新闻网    作者：段然

“我是你的小呀小甜橙，怎么奉献都不嫌多；勤劳的工作温暖你的心窝，点亮我春运的火……”一年一度的春运即将启幕，来自东华大学的35名“小甜橙”春运志愿者也将正式上岗，前往铁路上海站进行春运志愿服务。

传递欢乐 自编广场舞《小甜橙》

为了能给春运旅客带来欢乐和轻松，东华“小甜橙”在紧张的培训之余特别编排了一套广场舞。这套名为《小甜橙》的广场舞改编自网络神曲《小苹果》，舞步简单却充满欢乐，歌词朴实却道出了 “小甜橙”们的心声。平日里“小甜橙”们一有空就加紧排练，一方面为志愿服务锻炼体力，另一方面也希望在为旅客带去欢乐的同时亦能展示出东华“小甜橙”们的青春活力。据悉，这套广场舞将于2月4日铁路上海站南广场首演。

东华 “小甜橙”连续三年服务春运，去年凭优异表现荣获“全国铁路2014年春运青年志愿者先进集体”称号。在这个被称作38年来最长的寒假里，东华“小甜橙”放弃宝贵的假期，精心筹划，秣马厉兵， “橙”心“橙”意备战春运志愿服务。尽管今年春运志愿服务周期只有半个月，但早在2014年底东华大学就启动了 “小甜橙” 春运志愿者招募活动。今年的东华“小甜橙”志愿者招募异常火爆，除了东华大学理学院阳光服务队的基础班底外，其他学院的同学报名也相当踊跃，其中还包括不少硕士研究生。同学们投身春运志愿服务的热情高涨，无奈名额有限，最后通过两轮面试才敲定名单。

《甜橙指南》在手 出行不愁

今年东华“小甜橙”主要服务于铁路上海站安检处、服务台、扶梯口、军人老幼候车室、广场岗亭等九个岗位。这些岗位人流密集，前来咨询的旅客数量庞大。

为了让旅客更方便快捷地获得信息，结合以往经验，东华“小甜橙”们编印了一本《东华甜橙指南》并将在铁路上海站免费发放。这本三折页的小册子里汇总了旅客们最迫切想了解的信息，其中包括铁路上海站车站地图、车站各部分功能介绍、车站周边地形图、12306网站网址、手机客户端二维码、12306使用简介等实用内容。手握这本《东华甜橙指南》，旅客们可以按图索骥快速找到问题的解决方法，少了一丝麻烦，添了一丝从容，春运返乡之路也会顺畅许多。

慈善贴纸传递正能量

志愿服务，热心助人，东华“小甜橙”的爱心之举并不满足于此。今年春运志愿服务期间，他们还将发起“甜橙慈善贴纸”接力活动，将互助友爱的志愿精神传递开来，身体力行践行社会主义核心价值观。

“甜橙慈善贴纸”上印有东华“小甜橙”专属标志以及一句慈善箴言，在受助乘客允许的前提下，“小甜橙”们会在乘客的行李箱上贴上一张“甜橙慈善贴纸”。受助乘客如果在之后帮助了其他乘客，他们可以将这张贴纸进行转赠。你帮我，我帮他，大家帮大家， “小甜橙”们希望通过这种方式吸引更多的人参与到春运志愿服务中来。获得贴纸的旅客还可将转赠的过程拍照连同个人信息和详细地址发送给东华大学理学院阳光服务队的微信公共平台（微信号：DHU-lxy-ygfwd2014）， “小甜橙”们随后会送上一份新年礼物。此外，往年深受旅客欢迎的“东华小甜橙春运志愿者”新浪微博以及微信公共平台今年也将继续为第一时间为旅客提供线上春运志愿服务。
