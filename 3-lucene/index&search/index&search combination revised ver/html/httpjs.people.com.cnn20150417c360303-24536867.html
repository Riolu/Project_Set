为创新再添动力 上海交大苏北研究院落户淮安

2015年04月17日11:01
打印    字号： 

　　昨日上午，上海交通大学与淮安市正式签署共建上海交大苏北研究院协议，拉开了双方合作共赢的序幕。上海交通大学校长、中国科学院院士张杰，市委书记姚晓东出席签约仪式并分别致辞表示热烈祝贺。上海交通大学副校长吴旦，副市长唐道伦代表合作双方签署协议。上海交通大学原党委常委李建强，淮安经济技术开发区党工委书记周毅等参加签约仪式。

　　姚晓东对上海交大和各位专家长期以来给予淮安发展的关心支持表示衷心感谢。他说，此次成功签约是淮安对接高端资源、推动创新驱动发展的一次重要机遇。淮安主要经济指标增幅连续九年位居江苏省前列，综合实力不断迈上新的台阶，是一个充满生机与活力的地方。我们在推进创新驱动发展、引进高端资源过程中，十分注重与当地产业、企业的紧密结合。上海交大苏北研究院依托上海交大高端资源，为淮安的发展注入了强劲动力，为淮安建设苏北重要中心城市提供了强力功能支撑。市委市政府将为苏北研究院提供热情细致的服务，相信苏北研究院会成为上海交大所有研究院中发展最快、最好的研究院，不仅在淮安科学跨越发展中发挥重要作用，而且在苏北地区产生重要影响。

　　张杰愉快地回忆起令他“耳目一新”的淮安考察之行。他说，淮安是周恩来总理家乡，是一个令人尊敬的地方。淮安全市上下满怀创新驱动发展的激情，对未来更是充满美好的憧憬。上海交大是一所将促进民族振兴和社会发展作为自身责任和历史使命的高等院校。在与淮安的合作中，上海交大将把在其他地方设立研究院的成功经验运用到苏北研究院中来，与淮安紧密携起手来，实现共同的发展愿望，为地方发展提供有力的科技支撑。

　　为进一步深化上海交大与淮安市人民政府战略合作关系，加快重大科技成果转化和高新技术产业化步伐，增强协同创新能力，推进淮安创新型试点城市建设，双方经过多次交流沟通，决定共建上海交通大学苏北研究院。双方将共同推进协同创新，以发展高科技项目应用研发、以科技植入提升淮安企业科技实力、吸引科技成果在淮转化、实现产业化为目标，充分利用淮安的产业、环境等优势与上海交大的科技、人才优势，通过“政、产、学、研”结合，积极开展创新研发与科技成果孵化、转化、产业化，在发展新兴产业、高新技术及其产业化的进程中求得互利共赢。加强资源的整合，利用上海交大的优势，拓展国际合作与交流，致力于人才培养等合作空间。(杭卫东 杨尚)
