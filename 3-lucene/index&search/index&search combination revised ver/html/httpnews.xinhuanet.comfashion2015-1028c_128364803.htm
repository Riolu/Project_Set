特约评论：看着简单 想就复杂

深度观点

   2016春夏中国国际时装周 中法艾蒙时尚学院毕业生作品发布

    其实一向是非常喜欢在现场欣赏服装设计专业毕业生作品的，原因无他，只是因为年轻人的无所畏惧、无所顾忌，从而能够带来不被市场与消费群禁锢的创意，即便有很多显得幼稚。梅赛德斯-奔驰中国国际时装周（2016春夏系列）第三天上午的中法艾蒙时尚学院2015届毕业作品发布会，自然也能够满足这样的期许。毫无疑问，这场发布的配饰是很引人注目的，从头饰、花朵到项链到鞋子，充分体现了什么叫做矜持的夸张。而更令人印象深刻的，却是由本次发布会的主题“无界”拓展开来的服装设计所展示出的新生代设计师的审美观和价值取向。毫不隐瞒地说，对于本场发布所展现出的文化倾向，个人是抱以批判态度的，虽然众多的服装主题表达中呈现出了关乎宗教、种族、环保等的元素，但性别特征的隐喻尤为突出，同性关系、两性关系的表达令人咋舌。作为应届毕业生，他们的作品应当能够直观反映90后及00后消费群体在时尚文化上的价值取向，难道这一两代年轻人的人生观已经能够包容如斯广大？这样的价值观也将是未来时尚的主流？抱以怀疑！

2016春夏中国国际时装周 Giada Curti高定时装发布

    Giada Curti高定时装发布会，相对来说很多人并没有在意，但就服装而言，这是一场非常值得细细品味的时装秀。特别突出的是在几乎每个模特肩部塑造的大型绢花，让所有的创意都有了主线。这个来自意大利罗马的品牌，设计师完全继承了该地区传统的刺绣、钉珠和蕾丝技艺并将之发扬。典型西式结构的领口造型和腰节处理，令人着迷；男装的一些元素被运用在女装设计中，礼服的搭配上添加了职业的气氛，显得现代感十足。而值得学习的是，设计师对于衣身各部位比例的把握和色彩的搭配技巧，非常彰显人体美。

   2016春夏中国国际时装周 容子木·李铂楠时装发布

    现在京城的服装圈里，ROSEMOO容子木已经成为一个传奇，优异的市场业绩，并没有令走向大众化的庸俗。容子木·李铂楠时装发布会再一次展示了毕业于北京服装学院的这位年轻设计师对于美学、文化、自然、时空等沉静的理解与诠释。自然、朴素、雅致、宁静、简约等已经成为李铂楠的标签，大量运用传统手工和天然面料，通过衣身与细节结构上的变化结合对材料的再创造，体现出东方哲学美的内敛。比较令人吃惊的是，在本场发布的作品中出现了饱和度极高的大红色，在大比例自然色系中跳跃而出，不知道是否意味着接下来，这个品牌的风格将有所转变。除却服装，本场发布对于灯光的使用造成观众欣赏的困惑，却是败笔。

   2016春夏中国国际时装周 中纺设计中心·李石勇时装发布

    中纺设计中心•李石勇时装发布会所展示的服装从材质上明显分为两个部分，丝绸印花系列和棉麻系列，力图传递出同样主题的中国传统文化语言。但事实上，两种材质由造型、配色、细节、工艺等方面传递出的设计信息告诉人们，这是两个时代的东西。丝绸系列中大量中国的传统服装结构、配色方式和纹样处理，令人恍若置身古代，算是古色古香的现代演绎罢。而棉麻系列虽然在整体造型上采用了中国传统的宽衣博带，但在结构设计中却大量运用了典型西式的设计语言，褶皱层叠随处可见，远看颇有些魏晋古风，近观却是古罗马、拜占庭和维多利亚式的入侵。东西方服饰文化如此结合，能够完美融合吗？颠覆和创新应该如何进行？超现实主义吗？

   2016春夏中国国际时装周 意大利新秀设计师作品发布

    在意大利新秀设计师作品发布会上，一直试图从外国设计师作品中寻找到一如国内新锐设计们热衷的各种设计手法或元素，但很遗憾，收获不多。可能本次ITA -意大利对外贸易委员会隆重推出的9位青年新秀设计师并不是很“国际化”的？和下午罗马的设计师作品不同，年轻设计师们所展示的服装作品甚或连刺绣和钉珠都没有出现，他们把注意力更多投向了由面料本身和面料间配伍形成的服装风格上，注重产品的简约造型和丰富可搭配性，以及大工业生产的便利性。是的，在这场发布中，体会到最深刻的就是国内外新锐设计师对于设计成本控制的区别，要承认意大利人更加实际。

   2016春夏中国国际时装周 AOLISHA·魏新坤婚纱礼服发布

    婚纱这种舶来品，在被引入国内后，其相关着装文化已经发生了相当大得变化，或许应该说是本土化的改变。这种改变一方面丰富了国内婚庆礼仪的服饰文化内容，一方面令时装增添了新的门类，算是好事。AAOLISHA·魏新坤婚纱礼服发布会展现的婚纱设计或更加丰富。婚纱设计大抵是有着色彩、工艺、材料和造型等的限制，其突破也尽在此范围。魏新坤本次的婚纱设计，大量新型蕾丝材质的使用及与丝绸面料的复合，珠片刺绣配合网纱结合新工艺，实现了更空灵的柔美表达，值得称赞。

2016春夏中国国际时装周 纺大惟尚·孙菊香时装发布

    纺大惟尚·孙菊香时装发布会是丰富多元的，有极为中式的体现，也有极度西化的表达，古希腊裙、旗袍的结构有，超大廓形与紧身小黑裙也有，好在整体色彩同意在素雅的品相，没有完全跳跃出现。其实就单品而言，不少服装作品是值得称赞的，领口、腰身、袖子等上的细节处理很值得一观，相关服饰配搭的也颇为精致，由真丝及丝毛面料制作出的品质感亦很高。而遗憾在于中式优雅与西式品位的艰难整合，东西方美学的融合并不那么简单。

    （作者 田占国 中国服装协会评论员 中国国际时装周最佳时装评论员 ）
