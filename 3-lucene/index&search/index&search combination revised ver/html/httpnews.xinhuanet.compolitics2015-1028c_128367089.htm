名人再“躺枪”该担忧什么

2015年10月28日 09:51:55 来源： 解放日报
  • 评论0
  • 打印
  • 字大
  • 字小

　　10月27日，某女高音歌唱家发微博称：“一位快乐的老者、一位为大家带来无数优秀作品的大家——阎肃老师，因病，今日于空军总医院去世。”经多家媒体“跟进”发布，该消息在网上得到快速传播。然而，该微博随后即发表郑重声明：刚刚所转消息不实，严重失误，向老人家及家属严重道歉。

　　阎肃是我国著名的文学家、剧作家、词作家。上世纪80年代后，他多次参与策划春节晚会和一些大型晚会，如公安部晚会、总政全国双拥晚会、“3·15”晚会等，此后还多次担任中央电视台青年歌手电视大奖赛、《星光大道》等的评委工作。这则假消息被媒体迅速传播，固然与阎肃是一位德高望重的公众颇为熟悉的艺术家有关，但也再次反映出当下媒体的浮躁。

　　“名人躺枪”的事件如今已经并不鲜见。例如，之前就曾爆出新加坡总理公署网站文告的截图，称李光耀已经逝世，然后全球媒体纷纷转载，最终却证实是一条假新闻。再联想到，此前被讹传的“成龙因心脏病突发在美国去世”、“姜文因心梗去世”、“金庸因中脑炎合并胼胝体积水救治无效身亡”等，阎肃老师此次遭遇的假新闻事件不过是再一个翻版而已。

　　需要指出的是，此次误传最初源于名人微博，这也提醒我们，除了组织化媒体之外，自媒体在发布消息时，也应该更为谨慎。既然是“媒体”，就应当牢记真实是新闻的生命。尤其对于有影响力的公众人物来说，一定要对手中的“麦克风”保持敬畏，不能因为不熟悉而太过随意，更不能忽略消息的真实性，否则难免会因为“误传”而造成“误伤”。

　　由于时效的关系，纸媒这次并未“中招”，但是这并不意味着可以安枕无忧。应该说，近些年关于媒体患上了“浮躁病”的声音很多，其中既有媒体的自省，也有公众的吐槽。一些严肃媒体或著名大媒体集团在“秒杀时代”也难逃“浮躁病”的传染，不少媒体甚至世界知名媒体都曾经裹入一些传播假新闻的全球事件中。

　　不管是每天推出十几个小时新闻产品的电视台，每天推出十几万字的纸媒，还是每天要几乎24小时滚动播出的新媒体，抑或是广大的自媒体，要想不出现任何一个错别字、一个标点符号的误用、一丁点新闻事实的出入，确实很难。但是，在抢新闻之前做必要的核实，应该不算是太高的要求吧？（众奕）
