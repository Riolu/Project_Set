网络工作者的迫切希望
2009年9月29日 15:09

　　举报中心的各位工作人员：
　　
　　您们好，我是一个普通的网络工作者，由于工作性质的原因，我平时有很多在网上浏览信息的时间，这两年来的上网给我的感触也很大，因为现在大部分的网站（包括一些大型门户信息网）上都有好多色情、淫秽网站的图文链接，其内容真是不堪入目，更有甚者通过聊天工具发不不法信息，比如说QQ天天都会收到一些色情网站，邪教颓文的网站，另外在通过出差到各地走访和网友聊天的同事，更加痛恨垃圾网站对社会、对大家的危害，也同时感到了我们这个部门的责任重大，因为根据的上网经历、到网吧观察和一些资料来一看我国有60%的网民都是青少年，这一人群是我们的国家的栋梁啊，可他们现在正在网络世界里做些什么呢？--关注青少年！！！现在网上的垃圾信息太多，很容易浏览，而他们的自制能力又很差，这一点我真是深感担忧！！
　　
　　我虽然只是一个普通的网络工作这也没能荣幸加入我们伟大的党，可作为一个网络工作者，我感觉：网络确实是一个人们方便，快捷交流的平台，它给我们的生活带来了很大的改观，但同时我也希望这个美好的空间能得到净化，在给人民方便的同时，更能给人民带来精神上升华；作为一个普通的中国人，一个普通的中国青年人，我更加热烈的希望我们的国家更加强大！！！
　　
　　在这里我想说一声“举报中心的各位工作人员您们辛苦了，感谢您们对网络空间净化所做的一切，同时也希望国人自强！！！年轻人更该自强！！！
　　
　　最后，再次的感谢你们辛苦的工作！你们是网络的卫士！
　　
　　祝你们全部人员身体健康！
　　
                                                 　一个普通的网络工作者
　　
　                                                       2005年10月28日
