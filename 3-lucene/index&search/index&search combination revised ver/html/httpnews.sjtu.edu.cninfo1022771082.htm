校图书馆获首届全国高校图书馆阅读推广案例大赛一等奖[图]
[发布时间]: 2015年10月22日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 周衡 思源

2015年10月16日，首届全国高校图书馆阅读推广案例大赛总决赛在华中师范大学科学会堂一楼报告厅举行。大赛由教育部高等学校图书馆情报工作指导委员会读者服务创新与推广工作组主办，上海交通大学图书馆、上海财经大学图书馆、华中师范大学图书馆共同承办。经过初赛的层层筛选，共有38个阅读推广案例入围总决赛。中共湖北省委外宣办副主任谢双林、湖北省教育厅高教处正处级调研员张锦枫、华中师范大学党委副书记黄晓玫、湖北省高校图工委秘书长王新才、教育部高校图工委副主任陈进等领导及全国180余所高校图书馆代表出席了开幕式及比赛。经过紧张的角逐与评选，交大图书馆申报的案例“鲜悦：以人为书，分享智慧”喜获一等奖，“揽月之星，阅读之巅”获得优秀奖。   

开幕式上，谢双林在致辞中对本次比赛给予了高度肯定。他指出本次大赛是深入贯彻落实党中央国务院关于深化全民阅读战略部署的一项重要举措，是在以实际行动践行社会主义核心价值观。他希望全国各高校图书馆能够借助本次大赛的平台，培养大学生良好的阅读习惯，不断提高全民阅读的自觉性和积极性，为努力实现中国梦、全面协调推进“四个全面”战略布局做出更大的贡献。   

陈进指出，阅读推广工作有很多方面，对于大学图书馆馆员而言，如何设计形式多样的活动，营造书香氛围，引导大学师生更多更有效地接触优秀作品，在很大程度上考验了图书馆员的智慧和服务创新的能力。举办这次大赛的目的就是要推广阅读文化，弘扬大学精神，进一步检阅和展示高校图书馆在阅读推广中的作用，交流阅读推广经验。   

总决赛上，参赛的案例形式不拘一格，有传统的读书会、读书沙龙、阅读知识竞赛、阅读训练营，也有“互联网＋”背景下的经典名著在线游戏、一站到底名著阅读、新媒体阅读推广、微拍电子书等。来自全国高校图书馆的9位评审专家与近100名大众评委共同现场打分。经过激烈角逐，四川大学图书馆、清华大学图书馆、上海交通大学图书馆、北京大学图书馆、武汉大学图书馆、天津财经大学等6个案例获得一等奖，其中由校图书馆人文拓展团队申报的案例“鲜悦：以人为书，分享智慧”以总分第三的成绩获得一等奖，工学部电子电气学科团队申报的案例“揽月之星，阅读之巅”获得优秀奖。此外，决赛还评出二等奖12项、三等奖17项；香港大学图书馆、香港理工大学图书馆、澳门大学图书馆的参赛案例获得特别奖。   

据悉，首届全国高校图书馆阅读推广案例大赛于2014年10月启动，2015年上半年，全国六个分赛区及港澳台地区组织开展了分赛区的初赛工作，共有456个案例参赛。其中，38个案例入选决赛，并与118个优秀案例（含单项奖案例）参加了决赛现场的海报展示。
