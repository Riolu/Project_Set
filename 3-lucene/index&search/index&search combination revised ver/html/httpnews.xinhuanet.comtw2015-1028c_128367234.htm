国台办新任发言人首秀 盘点历任发言人

2015年10月28日 10:31:34 来源： 新华台湾
  • 评论0
  • 打印
  • 字大
  • 字小

　　“今天的发布会由我来主持”，国务院台办10月28日上午举行例行新闻发布会，新任发言人安峰山首度登台亮相。

　　国台办自2000年开始举办新闻发布会，迄今共有张铭清、李维一、杨毅、范丽青、马晓光、安峰山等六位发言人。

　　国台办新闻局副局长安峰山发言人首秀。

　　马晓光，2014年1月任国台办新闻发言人。马晓光长年从事对台研究综合与两岸协商谈判工作，原任国台办综合局局长、海峡两岸关系协会秘书长，2013年底转任国台办新闻局局长。

　　范丽青，2007年11月任国台办新闻发言人，她也是国台办首位女新闻发言人。原为新华社资深记者，2006年7月调任国台办新闻局副局长。

　　杨毅，2007年1月成为国台办新闻发言人。台湾媒体给杨毅的评价是“反应机智”，因为他擅长以“四两拨千斤”的答法化解媒体尖锐的问题。

　　李维一，2002年3月任国台办新闻发言人，后赴澳门中联办任职。

　　国台办首位新闻发言人张铭清，离任后担任海协会副会长、厦门大学新闻传播学院院长。
