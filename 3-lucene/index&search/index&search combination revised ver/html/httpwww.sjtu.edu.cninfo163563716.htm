上海交通大学图书馆120周年纪念文集征稿通知

时间：2015年05月06日 16:13 发布人： 浏览次数：[]

[责任编辑]李敏妍 [信息提供]图书馆 

 

　　2016年是上海交通大学建校120周年，也是图书馆建馆120周年。在120年的岁月变迁中，上海交通大学图书馆从最初一个小型藏书室，发展成为今天拥有丰富资源和先进设施、提供一流服务的现代化大学图书馆。无论是徐汇校区的老图书馆与包兆龙图书馆，还是闵行校区的包玉刚图书馆与新图书馆，都是交大人流连忘返的精神家园与心灵栖息地。一届又一届的交大学子在图书馆里汲取知识的营养，畅游知识的海洋，在图书馆的陪伴下度过了最为美好的青春时光，成长为国家与社会的精英和栋梁。多少已经毕业的交大学子留下“最难舍图书馆”的离别赠言！无论是莘莘学子，还是教职员工，图书馆承载了一代又一代交大人的情感寄托与奋斗历程。 

 

　　值此上海交通大学120周年校庆来临之际，为了挖掘交大人与图书馆一路走来值得铭记的故事，启迪后学更多了解和利用交大图书馆，上海交通大学图书馆计划于2016年出版第一本纪念性文集《圕.梦想之帆》（备注：“圕”（tuān）为“图书馆”的简写，是上海交通大学图书馆曾任馆长，我国著名图书馆学家杜定友先生于1924年创造，被图书馆界广泛认知和流传）。现开始面向校内外进行征稿，欢迎广大交大校友、在校师生及馆员讲述您与交大图书馆之间的故事，踊跃投稿。 

 

　　一、征文主题 

 

　　本次征文以“我与交大图书馆”为主题，可以从不同角度讲述您与交大图书馆的点滴故事，如您对交大图书馆的印象、您与交大图书馆的交集、您在交大图书馆的收获与感悟等。 

 

　　二、稿件要求 

 

　　1. 字数不限，建议采用Word文档格式（*.doc）。 

 

　　2. 文体不限，如诗歌、散文、议论文、随笔、杂谈等。 

 

　　3. 内容不限，可灵活多变，但必须积极向上，情感真挚。 

 

　　4. 可提供相关照片与图片等资料，并附简短文字说明。 

 

　　5. 需要附上作者简介，应至少包括：作者姓名、性别、出生年、在校时间、身份（校友、在校老师或特聘教授、在校学生、馆员等）。 

 

　　6. 必须为原创作品，严禁剽窃和抄袭行为。 

 

　　三、稿件著作权 

 

　　1. 投稿人保证其向本馆所投作品是其本人或与他人合作创作的成果，或对所投作品拥有合法的著作权，无第三人对其作品提出可成立之权利主张。 

 

　　2. 投稿人保证其作品不含有违反宪法、法律及损害社会公共利益的内容。 

 

　　3. 投稿人授予本馆享有文章的内部使用权。 

 

　　4. 投稿人保证向本馆所投之稿件，尚未在任何刊物上发表。 

 

　　四、投稿流程 

 

　　通过投稿邮箱投稿，所有稿件将由上海交通大学图书馆组织审核，通过评审的文章将被收录纪念文集，公开出版。同时，其他优秀稿件也将陆续刊载在网站、交大校报、《思源知讯》等媒体上。 

 

　　五、时间截点 

 

　　截稿日期：2015年9月1日
