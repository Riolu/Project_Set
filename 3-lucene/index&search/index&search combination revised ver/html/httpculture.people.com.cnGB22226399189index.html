话题1：

当代中国文艺的使命

主持人：李小萌

  • 阿来：“上世纪80年代以来，我们对那种极‘左’的或者某种教条的意识形态可能对文艺所造成的破坏性，警惕性都非常高。但在市场经济时代，一部分文化产品加入到产业化、商业化的过程当中，我们对于消费主义可能给文艺所造成的损害这种警惕性不够。从文学的生产环节来讲，不管是创作者，还是出版商，肯定面临着考验。”[视频]
  • 周大新：“其实无数前辈都讲过，他们当年都是潜心写作，奉献出好作品，让人们记得了自己的名字。作家主要靠作品来影响人的精神世界，因此，努力写出无愧于这个时代的作品，就是参与了民族精神的铸造。”[视频]
  • 王树增：“在我心里，我对‘精品’两字非常敬畏。把它看成是一直在追寻的彼岸，就像追求真理一样，只能逐渐地接近它。精品，是很难企及的标准，就因为标准高，所以才需要大家去追求。”[视频]
话题2：

人民文艺的力量源泉

主持人：许博

  • 叶辛：“我是知识青年上山下乡的一代人，最初上山下乡时看到的中国农村，和我十年以后再理解的中国农村不是一回事。一个作家，只有进入生活的形态之下，提起笔来再写作，才能真实反映那个时代的生活，才能写出好作品。”[视频]
  • 六小龄童：“根据吴承恩先生的名著拍摄的《西游记》，在中央台地方台已经播出了三千多遍，30多年来60亿人次观看，也就是说每一个中国人平均看了三次。在明年，也就是猴年，我将推出一部电影，让世界观众关注中华传统文化，关注西游文化。”[视频]
  • 何向阳：“作家如何去捕捉人物心中的奥秘？没有其他奥秘，就是深入生活。‘深入生活、扎根人民’八个字好像很朴素，内涵却非常丰富。”[视频]
  • 却却：“我做了十多年的民国史了，做了十多年的口述史，我的作品素材从哪里来，我为什么会变成今天这样的风格，全部基于此。我现在作品大部分的台词，基本上都来自于那个时代的人，会尽量还原他们的所思、所想、所说。”[视频]
话题3：

创作无愧于民族和时代的文艺精品

主持人：蔡国庆

  • 范迪安：“《清明上河图》能够被今天很多老百姓所喜爱，就是因为它是经典，它是精品。我呼吁，艺术家应该以创作精品为目标。如果今天没有精品意识，就不能给后世留下当代的经典。”[视频]
  • 张纪中：“我在研读中国古典名著中，越发觉得中华文化的灿烂和其在世界上的价值，可用‘浩瀚’形容中国文化的博大精神。古人有一个信念，就是为了我的族人和部落，不惜牺牲所有东西，黄帝就是这样，为了中华部落的统一做了艰苦努力。中华民族能够坚持到今天，与这种信念密不可分。” [视频]
  • 茅善玉：“精神家园在哪里？那就是舞台。虽然人生短暂，但艺术家在舞台上塑造了许许多多的角色和人物，大大丰富了他们的人生经历。”[视频]

名家话“精神故乡”

著名京剧表演艺术家尚长荣做客人民网，畅聊座谈会后的感悟，他感叹，习总书记的重要讲话“就像是一颗定心丸，一场及时雨”，“我们真正感受到了来自党和政府的重视，感到腰杆硬了”。
著名评书表演艺术家刘兰芳做客人民网，畅聊座谈会后的感悟。一年来，她深入全国各地表演，用忙碌的脚步践行着习近平总书记的重要讲话精神，她由衷感叹，“我们的传统文化越到基层越受欢迎，越有观众”。
天津歌舞团一级演员关牧村做客人民网，畅谈畅聊座谈会后的感悟。关牧村说，她是从车工成长为一名歌唱家的，这些年来一直不忘人民群众，“无论什么时候，树高千尺不忘根”。
上海京剧院一级演员史依弘做客人民网，畅聊座谈会后的感悟与践行，分享“精神故乡”与人生信仰。史依弘至今牢记习总书记对广大文艺工作者提出的“坚守艺术理想，不断提高学养、涵养、修养”的期望与要求，并努力在不断地学习与创新中先将自己“变成一件艺术品”，再让观众去欣赏。
中国文联副主席、中国音协副主席徐沛东做客人民网，畅聊座谈会后音乐界的新气象，并解读了他的新老代表作。文艺创作要扎根人民、扎根生活，这也是徐沛东一贯的创作道路。徐沛东笑称，他的作品有很强的泥土性，他自己也非常反对不接地气的创作观。
表演艺术家六小龄童做客人民网，畅聊座谈会后的感悟与践行，分享“精神故乡”与人生信仰，并呼吁大家尊重民族文化，不要“恶搞”经典。
中国舞蹈家协会副主席冯双白做客人民网，畅聊座谈会后的感悟，以及新作《永恒的刀郎》的创作经历，他表示，“文艺创作要深入生活，常怀敬畏之心。”
中国舞蹈家协会副主席陈爱莲做客人民网，畅谈了在过去的一年里身边发生的诸多变化。作为新中国第一代舞蹈家，陈爱莲感谢党和国家多年的培养和爱护，在舞台上跳了60多年，从下基层，下农村到走上世界的舞台，她深深地感受到自己精神的归宿就是舞蹈。
广州军区战士杂技团演员吴正丹做客人民网，回忆起去年参加文艺座谈会的经历，吴正丹自称因为自己太年轻。在她看来，艺术无止境，即使成名，艺术造诣上还要接受更大的挑战，文化理论修养上也有必要再回校园加强。
叶小纲的作品擅长将中西音乐形式结合，用西方人听得懂的音乐语言诉说中国的故事，让观众深受感动，成为“走出去”的先锋。在音乐里，是什么打破了中西文化界限？如何让外国人听得懂我们民族的声音？叶小纲分享感悟。
“生活是艺术创作的源泉，这是亘古以来的真理。”著名导演翟俊杰今天作客人民网文化频道，以“主旋律电影如何讲好中国故事”为题，与网友在线交流。翟俊杰从17岁开始当兵，曾经上过战场，所以在拍摄战争电影时，他严格要求还原细节。
著名编剧何冀平做客人民网，畅谈编剧人生，并与网友在线交流。“其实做编剧最主要的是作品要能够搬上舞台或者荧幕，因为剧本写得再好没有排成作品也是纸上谈兵。我经常说要在文字中注入生命，把它变成永久的形象。”

众人说文艺新期待

百姓热聊对文艺的新期待
街头采访： 老百姓对文艺有哪些期待？一名群众说：“我还是希望能够有更多的结构精巧的影视剧作品出现，比如美剧《绝命毒师》。路遥先生的《平凡的世界》，根据《平凡世界》改编的，前段时间非常火，这是非常棒的，我希望能够有更多这样的剧集。”[详细]
  • "评书大师"刘兰芳老书新说
    刘兰芳： 老书新说，旧书新评。我们要适应年轻人的需求，让他们理解和传播我们的传统艺术，这样才能让传统的民间艺术焕发青春。我想说好书，说好英雄人物，把他们的精神与品德贯穿在节目里。[详细]
  • "老戏骨"尚长荣分享座右铭
    尚长荣： 习总书记的讲话，像一阵和煦的春风，吹进了广大戏曲工作者的心中，我们真正感受到了来自党和政府的重视，感到腰杆硬了。习总书记的讲话，就像是一颗定心丸，一场及时雨。[详细]
