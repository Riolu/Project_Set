“发现双创之星”走进上海　创客聚力美丽中国梦

2015年10月28日 12:49:29 来源： 央视网

“发现双创之星”活动创客说环节在上海科技馆举办。

　　上海，中国金融中心，国家经济晴雨表，中国新一轮改革排头兵。在这样一座历史积淀丰厚、经济活力充沛的城市里，创新创业是从不会缺席的主题。

　　10月27日，“发现双创之星”大型系列活动走进上海。活动分主论坛和分论坛，分别在上海创业者公共实训基地和在上海科技馆盛大举行。现场，来自15个中央部委和上海市、杨浦区政府部门的相关领导发布和解读了最新的“双创”政策，并同十多位创客面对面交流，答疑解惑。另有多位创客上台分享了自己的创新创业历程。

　　中国网络电视台对活动进行了全程的视频直播和实时图文报道。

　　上海市副市长周波在活动现场表示，加快建设具有全球影响力的科技创新中心是新形势下国家对上海的新要求和新定位，同时也是上海城市发展的必然需求。这个时候，“发现双创之星”走进上海大型主题活动既是广大创业者的一次盛会，也是上海推进创新创业工作中的一件盛事。

　　科技部政策法规与监督司司长贺德方表示，在社会各界的共同努力下，“大众创业万众创新”重在走进一个崭新的时代，伴随着中国经济的升级，已经成为创新中国、智慧经济的重要标识。

成功创客分享经验。

　　杨浦区政府作为主办方之一，对本次活动给与高度重视。杨浦区区委副书记、区长谢坚钢在致词中介绍，杨浦区要打造上海科创中心的重要承载区，要建成万众创新的示范区。

　　上海市发展和改革委、科技创业企业中心等相关领导现场发布了多项进一步推进“双创”工作的政策措施，包括将孵化器评定权和科技成果处置权交还市场、创新创业人才落户上海等新政。

　　下午的“创业者说暨创新创业颁奖典礼”上，多位创客走上讲台分享了自己的双创经历。创客们纷纷表示只要大家一起努力，美丽中国梦想就一定能实现。

　　知名创客、投资人陶闯教授也受邀上台演讲，为创客们现场说法。他号召全社会鼓励创业，打击投机。不以成败论英雄，要享受创业的过程，欣赏创业的激情。

　

“发现双创之星”走进上海颁奖现场。

　　来自上海的数十家创业企业分别获得“最具投资价值”、“最具投资人气”、“最具创业精神”等多项大奖。

　　“发现双创之星”走进上海活动吸引了大量创客以及有志于创新创业的人士，上下午两个论坛，会场座无虚席，不少观众甚至站着听完全程。

　　“发现双创之星”大型系列活动走进上海，由发展改革委、教育部、科技部、工业和信息化部、财政部、人力资源和社会保障部等15个中央部委联合指导，中国政府网、中国网络电视台、上海市杨浦区政府联合主办，证券时报协办。

　　该系列活动是国务院政府职能转变中首次以活动的形式做政策解读，走进寻常百姓家，以后还将常态化。5个月来，活动从北京开始，已走过天津、安徽、山东、江苏，上海等六个城市，受到了国务院领导的肯定和表扬，并要求要采取多种形式把“大众创业 万众创新”推向深入，继续做好后续工作。（记者 王甲铸 李文亮 孔华 唐诗颖）
