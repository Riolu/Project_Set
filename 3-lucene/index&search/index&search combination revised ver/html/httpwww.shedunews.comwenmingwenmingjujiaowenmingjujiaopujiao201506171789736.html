沪爱心暑托班招募志愿者 上万高中生同时在线报名

2015-06-17 16:25:25    来源：上海文明网   

团市委今年依托“爱心暑托班”平台，尝试从大学生志愿者岗位中，拿出2962个志愿者岗位向高一学生开放。6月10日18时，报名系统正式开放，即刻就有上万人同时在线报名，3个多小时内，大多数岗位被“一抢而光”。业内人士提醒，社会实践、志愿服务的初衷在于引导学生接触社会，家长应根据孩子的兴趣爱好选择实践岗位，不要为完成学时背离初衷。
　　期末考未结束就寻实践岗位
　　今年公布的《上海市普通高中学生综合素质评价实施办法（试行）》，将学生参与军训、农村社会实践、志愿服务（公益劳动）、社会文化活动、社会考察（调查）等社会实践活动情况纳入综合素质评价体系，并作为重要的记录内容。期末考试还未结束，一些高一学生的家长已在为孩子寻找暑期社会实践岗位。
　　“爱心暑托班”志愿者名额被家长“秒抢”，原因可能有二：一来这是市政府实事项目，社会影响大；二来部分家长和学生可能觉得，这是一次完成社会实践学时的好机会。据悉，参加“爱心暑托班”志愿服务项目，可以计20至40个学时。
　　记者调查发现，有少数高中生家长抱着这样的心态：暑期参加社会实践时间集中，可以一次性多完成点学时，避免等到高二高三时，再花时间用于社会实践，可以集中精力学习。
　　暑托班带孩子并非人人胜任
　　“在暑托班带孩子，其实也是门技术活儿，并不是每个人都能胜任的。”往年参与过“爱心暑托班”的志愿者表示。记者从“上海市学生联合会”官方微信了解到，“爱心暑托班”志愿者不仅需要耐心、爱心和责任心，还要善于表达和沟通。此外，看护孩子更需要注意安全，志愿者不仅辛苦，压力也大。团市委呼吁，高一学生报名需要理性。
　　此次“爱心暑托班”志愿者的报名按需求量的150%设置上限，目前已报名人数是实际需求人数的1.5倍。报名后由各区县团委实施选拔，确保让适合的高中生参与。今年7月，经选拔后入选的高一学生志愿者将经过培训，分批上岗。
　　参加社会实践不能只为学时
　　其实早在2010年，市教委有关文件已明确要求，高中学生须完成60学时的志愿者服务。
　　“鼓励学生参加社会实践，主要是为了引导大家在参与社会、服务社会的过程中，增强社会责任感，提升创新精神和实践能力，不应简单地理解成为完成学时而进行社会实践。”市教委德育处有关人士说，社会实践目前已列入课程计划，学校会有计划地组织学生完成军训、农村社会实践、志愿服务（公益劳动）等社会实践活动。
　　目前首批认定的市、区县两级学生社会实践基地有1000多家，可提供学生实践岗位6万多个。家长无须“抢岗位”，也不用担心“一岗难求”。
