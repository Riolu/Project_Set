关于警惕“黑宽带”陷阱的公告
2015年1月9日 09:31

    近期，不少市民反映遭遇“黑宽带”低价陷阱，不仅网速慢，而且还经常断网，对正常工作和生活带来负面影响，甚至经济损失。为此，上海市通信管理局提提醒广大市民安装宽带时，千万不要被超低价迷惑，一定要选择我局认可的正规接入企业，防止上当受骗。目前上海只有12家企业能够为家庭提供宽带接入服务（附后）。
    对于那些违法提供“黑宽带”的企业，我局和有关部门将进行严厉查处，欢迎广大市民踊跃举报。
    一、中国电信股份有限公司上海分公司；
    二、中国联合网络通信有限公司上海分公司；
    三、中国铁通集团有限公司上海分公司；
    四、东方有线网络有限公司；
    五、上海长城宽带网络服务有限公司；
    六、上海电信住宅宽频网络有限公司；
    七、上海科技网络通信有限公司；
    八、上海聚友宽频网络投资有限公司；
    九、上海国通网络有限公司；
   十、上海市南电信服务中心有限公司；
