第九届全国语言文字应用学术研讨会通知（第2号）

 

第九届全国语言文字应用学术研讨会筹备工作基本就绪，将于2015年10月16至18日举行（9月中旬发正式通知）。现奉上第2号通知，就有关事项通知如下：

一、论文材料：

尚未提交论文提要者请于2015年8月31日前用电子邮件或传真寄达，以便专家审查合格后发出正式通知。

二、会议时间：2015年10月16日至18日

三、会议地点：湖北师范学院（交通指南详见正式通知）

四、日程安排：10月16日报到，10月17至18日会议。

五、需交纳费用：

①会务费：600元/人（学生300元/人）。

②住宿费：自理。

六、提前到达或推迟离会的代表，请自行解决食宿问题。

七、邮寄地址：100010 北京朝内南小街51号 中国应用语言学会（筹）秘书处。

联系人：刘子琦。

联系电话：（010）65592968、（010）65592909，（010）65130341（传真）。

电子邮箱：zhgyyyyxh@sina.com

 

中国应用语言学会（筹）

教育部语言文字应用研究所
