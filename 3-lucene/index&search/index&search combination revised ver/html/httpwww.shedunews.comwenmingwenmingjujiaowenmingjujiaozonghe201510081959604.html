“在上海爱上海”沪上新媒体专题报道展现上海之美

2015-10-08 15:09:08    来源：上海文明网   

你心目中的上海是什么？国庆节当天，上海观察用12张图片生动诠释了这个话题，从怀旧的永久牌自行车、上海牌手表，到日新月异的地铁网络、陆家嘴天际线，张张图片引起广大网友共鸣。
　　这组报道是沪上主流新媒体“在上海，爱上海”专题报道的一个缩影。国庆黄金周期间，包括上海发布、文汇报新媒体、新民晚报新民网、看看新闻网、澎湃新闻网等官方发布号和十余家主流新媒体开展集团军作战，在各家网站、客户端以及微博、微信等全媒体平台上，重磅推出专题报道，让网友足不出户就可以全方位感受上海之美。
　　“在上海，爱上海”的内容涵盖“上海好人”、“上海美景”、“上海历史”、“上海美食”等若干主题，各家新媒体立足定位，用网民喜闻乐见的图片、视频、微电影等形式，挖掘出上海之美的点点滴滴，传播了积极向上的正能量。这些报道最后又在“上海发布”平台上集中推送，这也是沪上主流新媒体矩阵首次集体亮相。
　　上海之美在旅游景点里，在新老地标里，也在浓浓的城市文化里。在旅游休闲热潮中，东方网为网友支招新的旅行方式——循着上海路名的历史去旅游；“伴公汀”公众号则带着网友“漫步”17个区县，发现上海不一样的美；东方广播“话匣子”公众号“穿越”到数十年前，细数橡皮筋、勾脚跳、斗鸡等那些让人难忘的弄堂游戏。
　　全媒体表达是这次主题报道的一大特色，展现了上海主流媒体在媒体融合转型方面的新探索、新成绩。SMG看看新闻网用一组组精彩视频，带人们品尝弄堂里的葱油饼，走近在上海打拼的创业团队，感受上海生生不息的城市精神；　在文汇报新媒体的视频中，画家用水墨画的方式栩栩如生地讲述了上海6000年悠久历史；　新民网则带着网友从空中看上海，航拍了拥有空中花园、壮观瀑布的松江天马山深坑酒店，高达632米的上海中心大厦，以及水系面积占40％的青浦青西郊野公园等上海三个新地标。
　　“在上海，爱上海”主题报道在互联网舆论场引起了巨大反响，仅上海发布平台相关微信直接转发量达到50万次；新浪微博相关话题的阅读量达到227万，评论上千条。
　　据了解，今后“在上海，爱上海”有望成为沪上主流新媒体的常设栏目，将举办更多线下活动，加强与网民互动。
