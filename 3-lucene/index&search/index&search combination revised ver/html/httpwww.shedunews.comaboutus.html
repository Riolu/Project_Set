上海教育新闻网：了解上海权威教育新闻资讯的第一选择

【发展定位】

上海教育新闻网（www.shedunews.com）由上海教育报刊总社主办，于2010年2月26日正式开通，是上海教育系统的主流网络媒体。上海教育新闻网以"发布权威资讯，引导教育舆论，提供教育服务，推动素质教育，优化网络生态"为己任，是上海权威教育新闻资讯发布和优质教育服务资源分享的"主平台、主出口、主门户"，是教育管理者、教育工作者、学生、家长和所有关心教育的社会公众获取资讯、交流心得、分享经验的新媒体。

【资源优势】

上海教育新闻网得到上海市教卫党委、上海市教委的高度重视和大力支持，拥有上海教育报刊总社（三报八刊）强大的教育资源优势，不仅研发了网站、微博、手机报、校园视频推送等多个新媒体平台，还推出了《上海教育手机报》、"东方校园视线"、"教育名片"等多个深受欢迎的新媒体产品。

【网站特色】

资讯内容权威实用
      以主流教育新闻为旗帜，以优质教育服务为支撑，以数据库建设为目标，及时传递上海教育领域的权威教育资讯，大力弘扬上海教育系统的良好精神风貌，并为广大受众提供实用贴心的教育服务。

线上活动丰富多彩
      紧扣重要时点和节点，利用互联网平台，举办了各种丰富多彩的素质教育活动，"红色寻根""文化寻根"千字文微博文征集、"诚信文化校园行"等活动吸引了上海数百万青少年参加，在社会上引起强烈反响，对推进未成年人思想道德建设、丰富校园文化建设、加强校园精神文明建设起到积极作用。

手机媒体精准传送
      根据受众需求度身定制，以上海教育系统高端人群为主要读者的《上海教育新闻手机报》（专业版），短小精悍、选编独到，受到上海市教卫党委、上海市教委和各高校、各区县等各级领导的好评。

校园视线深受欢迎
      "东方校园视线"首批在黄浦区试点，以"分众"模式，全力打造一个以新闻、科普、益智等为主打内容的互动新媒体，已成为很多青少年喜闻乐见的"校园视频窗"。"东方校园视线"实行学校、区县和新闻网三级管理权限，由用户积极参与内容共建。

校园视线深受欢迎
      "东方校园视线"首批在黄浦区试点，以"分众"模式，全力打造一个以新闻、科普、益智等为主打内容的互动新媒体，已成为很多青少年喜闻乐见的"校园视频窗"。"东方校园视线"实行学校、区县和新闻网三级管理权限，由用户积极参与内容共建。

【重点项目】

教育名片
      上海特色、优质学校资源的集中展示平台，以新兴媒体手段凝聚各校优质教育资源，为各校展现办学特色、为公众了解学校提供一站式服务，是上海各所学校在互联网上的一张教育名片。

招生考试
      还在为孩子中高考烦心吗？如何每天得到第一手招考资讯呢？上海教育新闻网招考频道网罗全上海最新最权威的中高考信息，全面展现每年最新的政策资讯、学校的招生动态、考前心理调试、饮食保健等实用资讯。用户轻点鼠标，招生资讯尽在掌握。

阳光体育
      与上海市学生阳光体育大联赛组委会共建"上海市学生阳光体育大联赛"专题网页，多方面、多形式、立体化深入报道联赛的内容，凸显上海市各区县学校在增强学生体质、促进学生素质全面发展方面的举措、经验，展现师生快乐运动的风采。

视频拍摄
      上海教育新闻网拥有一支高质量的视频专业队伍。除了拍摄制作自己的视频节目外，还承接包括上海市学生阳光体育大联赛在内的多项专业视频拍摄制作工作。同时，还为很多基层学校提供了视频宣传片拍摄制作等服务。

【技术服务】

上海教育新闻网有成熟的信息技术服务团队，在不断创新开发网站新功能的同时，为两委、高校、区县及中小幼等基层单位提供了稳定可靠的技术服务。两年来承担了包括上海教卫党建网、上海教育文明在线、阳光体育大联赛、头脑奥林匹克(简称OM)等网站的技术与内容托管服务。

网站同时提供专业调查、评选、投票等互动应用服务，先后为"上海教育年度新闻人物评选""大学生年度人物评选""大学生思想状况调查""中小学生网上控烟知识问答""中职生发展状况调查""诚信文化校园行调查"等大型网上互动评选调查项目提供专业技术服务。

【获得荣誉】

2010年，获得上海世博会公众参与馆优秀合作网站荣誉称号。
       2011年，在上海世博会网络传播评选中，荣获最佳报道优秀奖，受到国务院新闻办公室表彰，上海教育新闻网也是所有获奖者中唯一一家教育类网站。
       2011年，在由上海市互联网舆论宣传领导小组办公室、上海市精神文明办主办的上海市第五届优秀网站评选活动中，上海教育新闻网荣获"优秀网站"荣誉称号，跻身"百佳"行列。
