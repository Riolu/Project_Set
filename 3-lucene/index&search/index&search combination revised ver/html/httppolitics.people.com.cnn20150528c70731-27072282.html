上海市大学生文化创意创业基地在东华大学成立

原标题：上海市大学生文化创意创业基地在东华大学成立

　　中新网上海5月28日电(记者 许婧)“上海市大学生文化创意创业基地”28日在东华大学揭牌。

　　大学生创业企业往往起步于小微，时尚创意产业“小微企业居多”的特性，常常使得大学生创意创业团队“依托学校和政府支持进行组团发展”成为最佳选择。

　　当天揭牌成立的基地，计划以东华大学大学生创业孵化楼作为载体，汇集上海文化创意创业学生，建设上海文化创意创业集聚区。

　　来自上海交大、同济大学、华东理工大学等上海18所有关文化创意高校及临港集团、上海寅嘉创业投资管理有限公司、苏河汇、启迪之星(上海)等企业成为同期成立的“上海市大学生文化创意创业联盟”首批会员。据悉，今后还将继续吸纳其他相关高校、企业和社会组织加入“文创联盟”，为上海文化创意创业提供创新人才和智力支撑。

　　东华大学创新创业中心主任宋丽贞介绍说，上海市大学生文化创意创业基地之所以在东华大学成立，是基于东华大学在创意设计领域的学科建设、人才培养、产学研合作、文化辐射等特色和成果而建立。除了有独特的创新创业生态文化，2014年，东华大学毕业生创业人数居上海高校之首，毕业生创业形成了鲜明的科技创业、创意创业特色。

　　东华大学校长蒋昌俊教授表示，前不久东华成立了上海高校首个众创空间，此次又在东华成立上海市大学生文化创意创业基地，既是东华多年来创新创业教育优良传统的有效继承，也是东华适应社会发展新常态，加大为国家和上海城市发展培养和输送创新人才力度的集中表现。

　　蒋昌俊说，未来，东华大学将依托以“纺织”为一体，“材料”和“设计”为两翼的学科特色新格局，打造“政产学研用”协同创新体系，构建多元化人才培养模式，积极建设上海市大学生文化创意创业基地等平台辐射更多高校和地区，为服务上海科创中心建设作出贡献。

(来源:中国新闻网)
