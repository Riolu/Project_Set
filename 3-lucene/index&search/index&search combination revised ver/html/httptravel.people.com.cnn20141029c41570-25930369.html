探访那些曾经奢华私密的顶级会所（组图）

    人民网北京10月29日电 （记者张希、徐婵）近日，中共中央办公厅、国务院办公厅转发了10部门《关于严禁在历史建筑、公园等公共资源中设立私人会所的暂行规定》。今年11月1日起，属于私人会所性质的项目立项、经营许可、工商登记等事项，主管部门将不予办理。那些曾经占据过各大媒体头条，以“奢华”、“顶级”、“私密”闻名的各地会所，现在都变成什么样了呢？

   【北京・乙十六御膳堂】

    地理位置：北京市北海公园

北海公园内御膳堂乙十六大门紧锁

    整顿前：奢华消费

    在停业整顿以前，御膳堂雅间的最低消费标准是3000元，一间装修豪华，彰显皇家气派的大房间，最低消费标准竟高达1.5万元。菜单上的一款“宫廷盛宴”，每位1888元起。

    整改后：团购亲民

    乙十六餐厅改变了将普通顾客拒之门外的会所性质，并且调低价格，在最近开始用团购招揽生意。团购所推出的套餐人均消费均未超过200元。其中最贵的菜品是“多宝鱼”，价格也就198元。鱼翅、燕窝等昂贵菜并未出现在菜单中。

    【北京・健一公馆】

    地理位置：北京红领巾公园

    整改前：吃饭人均800元 可提供挡车牌服务

    以前，在这里预订20人包间需16000元(人均800元)，且额外收取15%的服务费，可为公款消费提供用于报销的整额发票，如果是接待领导，健一公馆还提供遮挡车牌等服务。

    整改后：转变经营方式

    2014年2月，“健一公馆”名称变更为“北京健一迎祥会议服务中心”（简称健一中心），转型为以会议和客房为主、餐饮为辅的经营方式；面向百姓开放，调整菜品价格，推出38元/位和58元/位的商务套餐，无最低消费、包间费和服务费；对于宴会服务，不设最低消费，不设套餐，依客户需求量身定做；为方便大众消费者入店进餐，门口将不再设立保安。

 

 

    【北京・“问月楼”饭店】

    地理位置：国家4A级景区北京紫竹院公园内

几位顾客在整改后的“问月楼”享用38元简餐和饺子。新华社记者 李文 摄

    整改前：售卖超千元菜品

    以前这里一般的菜都在百元以上，还有超过千元的菜品。

    整改后：推低价菜特价菜

    2014年1月，“问月楼”饭店拆除了影响公园景观的木质围栏，撤下了“游人止步”、“禁止入内”等标志牌，饭店调低收费标准，推出低价菜、特价菜，全面向市民开放，调价后的菜品价格多数在百元内。

 

    【河南郑州・梦溪园餐厅】

    地理位置：郑州紫荆山公园

梦溪园原来一天只开放四个小时。

    整改前：只开放4个小时

    在整改前，该园的开放时间为上午9点到11点，下午3点到5点，全天只开放4个小时，在中午吃饭时间和晚上变身私人饭堂。

    整改后：全天开放 取消非文化服务内容

    2013年7月31日，梦溪园调整开放时间，将原来的每天4小时改为全天免费开放，取消一切非文化服务的内容，撤除所有与园内无关的用具和设备，清除一切带有经营性质的项目和活动。

 

   【宁夏银川・农格海宝高尔夫练习场】

    地理位置：银川民族北街海宝公园

  

2014年6月24日，

宁夏银川市海宝公园的农格海宝高尔夫练习场（整改前）新华社记者 王鹏 摄 图片来源：新华网

2014年7月18日，练习场设施被拆除后 （图片来源：宁夏日报记者 宗时风 摄）

    曝光前：高尔夫球场藏身公园

    该高尔夫练习场被曝光前，占地近40亩，设24个标准打位，广告宣传为银川市区唯一一家真草高尔夫练习场，会所内还配备餐饮等场所。

    曝光后：练习场设施拆除 场地用于市民休闲

    2014年7月15日，宁夏国土资源执法监察局约谈农格海宝高尔夫球练习场负责人，企业书面保证不再经营。在国土部门的监督下，目前该练习场已经拆除了训练设施、围栏和宣传广告，将场地还原用于市民休闲娱乐。

 

    【四川成都・食画花园餐厅（二环路内侧）】

    地理位置：浣花溪金林半岛桥下游左岸城市绿地小游园

   

食画花园餐厅拆除以前（图据网络）

食画花园餐厅拆除以后

    整改前：装潢奢华 一瓶茅台卖到近3千元

　　该餐厅以前主要经营中餐，菜价从几十元到数百元不等，如一份莲白38元、一道法国鹅肝凉菜188元、一份黑龙雪花498元、一瓶飞天茅台售价为2988元。一张结账单显示，4人用餐价格在1600元左右，包间要收取服务费。

    整改后：废墟一片

　　2014年6月，现场不见餐厅，只见蓝色的打围栏，站在金林半岛门口能看到河边被拆除一空的现场。除了两台孤零零的推土机外，河边再不见任何餐厅建筑的踪影。由于刚刚下过雨，空荡的河边只有一片泥泞。

 

    【江苏南京・琵琶湖公馆】

    地理位置：南京紫金山景区琵琶湖公园

整改后改名为琵琶湖酒店。

    整改前：吃饭消费才能进

    整改前，公馆设有10余处大小不等的包间，就餐环境相当好，游客除非在这里吃饭，否则只能远看不能游玩。据当时服务人员曾介绍，在这里就餐，稍微好点，一人普遍消费在200元左右。

    整改后：推平价消费

　  2014年10月媒体记者探访时，琵琶湖酒店推平价消费，提供简餐服务，不设最低消费，还在网络上搞起团购，原先200元/人甚至更高的消费标准，现在直接降了一半。

 

    【广西柳州・巨型豪华会所】

    地理位置：广西柳州市跃进路公共绿地

图为拆违现场。中新社发 雷彬 摄

    整改前：占用公共绿化用地逾万平方米

    这座“威尼斯国际会所”建筑面积曾达1.17万平方米，装修豪华。耗资4000多万元人民币、占用公共绿化用地逾万平方米，它并没有合法的规划建设手续，属于违法建筑。

    整改后：依法强制拆除

    2014年3月13日，广西柳州市组织城管等相关部门近百人，对该市耗资4000多万元人民币、占用公共绿化用地逾万平方米的巨型豪华会所进行依法强制拆除。

 

    【浙江宁波・日湖亚洲会所】

     地理位置：宁波日湖公园

拆除前。（图片来源：人民图片）

拆除后。（图片来源：人民图片）

 

    整改前： 宁波少数人的“后花园”

    以“遁”公园、“傍”古建，侵占公共场所，成为少数人“后花园”的宁波日湖亚洲会于2006年开始营业 ，一直被人号称后现代主义新锐，整体设计、布置独特，讲究细节、追求品质、尽现奢华典雅，共有4个楼层，总营业面积达万余平方米，投入数千万巨资精心打造了57间KTV包厢和浴场。

    整改后：一片废墟

    2014年3月4日，该市组织特警、城管等工作人员，冒雨进行了强制拆除。

 

   【浙江杭州：西湖会】

    地理位置：杭州西湖景区

西湖会转型为“开心茶馆”。中新社发 谢盼盼 摄

    整改前：接待众多政要人物

    该会所坐落于杭州西湖边曲院风荷公园内，在整改前，餐标一般在500到800元一位。这里曾经接待过众多政要人物，甚至承担了部分国宾馆的功能，接待过的名人更是不计其数，还举办过诸如Dior、SK-II、保时捷等国际大牌的发布会。

    整改后：弘扬茶文化

    2014年1月27日，西湖会转型为“开心茶馆”，以弘扬茶文化的新形象出现在民众眼前，它成为继“西湖会所”关停后的首家转型场所。今后茶馆的龙井茶最低价格18元一杯，多数菜肴价格在30元以下，未来还将推出“免费学茶艺”、“茶艺表演”等活动。
