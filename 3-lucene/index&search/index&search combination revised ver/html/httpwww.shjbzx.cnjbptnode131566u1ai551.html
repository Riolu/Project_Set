中美警方发起多国多地区“天使行动”
2013年10月23日 09:45

　　由中美两国警方发起，英国、法国、德国、意大利、西班牙、俄罗斯、瑞典、澳大利亚、新西兰、泰国、马来西亚、日本、韩国、加拿大、阿根廷以及香港、澳门、台湾警方共同参与的打击儿童淫秽色情网站联合执法行动（代号“天使行动”），历经半年的侦查取得圆满成功，一举摧毁儿童淫秽色情网站4个，抓获包括网站开办者、版主、重点发帖会员等犯罪嫌疑人250余名。这次“天使行动”，是中美警方继2011年联合打击“阳光娱乐”淫秽色情网站联盟后，联手打击网络犯罪的又一成功范例，是中国公安机关组织的规模最大、参与国家和地区最多的一次联合执法行动，不仅沉重打击了网络色情违法犯罪活动，而且对促进国际执法合作、打击跨国犯罪具有积极的借鉴意义。中国警方的务实高效合作，特别是为“天使行动”付出的艰苦努力，受到了日前参加“天使行动”总结通报会的多国多地区警方的高度评价。

　　依法严厉打击网络儿童淫秽色情犯罪，切实保护未成人权益，世界各国警方有着相同的共识。2012年9月，美国联邦调查局向中国公安部通报一服务器位于美国、名为“拯救天使”的儿童淫秽色情网站的线索，后又通报了“我爱幼幼”等类似网站线索。公安部迅速行动，立即部署北京、江苏、安徽等地公安机关开展侦查。经侦查，“拯救天使”网站注册会员超过100万人，下设“正太幼男资源专版”等9个专版、“正太写真原创区”等44个二级分区，内容涉及大量儿童淫秽色情信息。网站权限设置严密，分为“管理员”、“版主”、“功勋元老”、“高级VIP用户”、“会员”等多个级别，并通过收取会员费牟利。经扩线侦查，专案民警又发现犯罪嫌疑人开办的另一个百万会员级的儿童淫秽色情网站“呦吧论坛”。经对网站违法犯罪线索梳理，版主、重点发帖会员等涉案人员分布数十个国家和地区。

　　中央政治局委员、中央政法委书记孟建柱，国务委员、公安部部长郭声琨同志对此高度重视，听取有关情况汇报，要求加强执法合作，共同打击网络色情犯罪，坚决保护儿童身心健康。经公安部和美国联邦调查局协商，双方决定联合侦办“拯救天使”系列儿童淫秽色情网站案。2013年4月17日，中国公安部和美国联邦调查局共同邀请德国、澳大利亚、新西兰等15个国家执法机构代表在北京召开案件协调会，统一开展代号为“天使行动”的联合执法行动。公安部及时将梳理排查的线索通报涉及国家警务联络官，同时将相关线索移送香港、澳门、台湾执法部门。公安部成立由部领导挂帅的专案指挥部，实行统一指挥，部署相关省市公安机关开展专案侦查。5月下旬，分别在江苏、浙江抓获“拯救天使”和“呦吧论坛”网站开办者刘某晶（男，32岁，江苏徐州人）和服务器空间提供商杨某（男，28岁，浙江台州人），“我爱幼幼”网站开办者刘某华（男，30岁，香港籍）也于近日被抓获归案。8月中旬，公安部部署全国30个省市公安机关集中开展收网行动，一举抓获包括网站版主、重点发帖会员在内的犯罪嫌疑人180余名。经审查，犯罪嫌疑人刘某晶、刘某华对租用美国服务器、开办儿童淫秽色情网站的犯罪事实供认不讳，其他犯罪嫌疑人也交代了大量搜集淫秽色情图片、视频等信息并上传网站的犯罪事实。香港、澳门、台湾警方共抓获违法犯罪嫌疑人60余人。

　　在“天使行动”中，中国公安机关通过大量细致的分析排查和证据认定工作，查明了全部涉案违法犯罪人员的分布情况，并及时通报相关国家和地区，为中美乃至多国多地区联合执法行动提供了情报线索和证据保障。公安部通过警务合作渠道与参与国家保持24小时热线联系，先后举办多国联合执法通报会、交换证据、案情磋商、双边和多边工作会晤等75次，来往函电500余件，并多次协调安排与香港、澳门、台湾警方的工作交流。美国联邦调查局获取了淫秽色情网站会员数据，及时关闭了违法网站，并根据中方提供的线索已逮捕2名犯罪嫌疑人。其他相关国家警方根据公安部通报的线索证据均已启动调查工作，德国已抓获1名犯罪嫌疑人并锁定另2名犯罪嫌疑人，澳大利亚已锁定2名犯罪嫌疑人，新西兰对1名犯罪嫌疑人正在深入调查，境外发布儿童淫秽色情信息会员等违法犯罪人员的调查仍在进一步工作中。

　　公安部有关负责人表示，联合执法行动是国际执法合作进入深度合作阶段的体现，是对新的合作理念、合作模式和合作机制的全面实战检验。“天使行动”取得圆满成功，得益于中美警方及其他国家和地区警方的精诚合作、形成合力，得益于全体参战人员高度负责的精神和艰苦细致的工作，对进一步推动中国与有关国家和地区执法部门的务实合作，进一步完善联合执法工作机制具有重要的意义。中国警方将继续加强与世界各国的执法合作，严厉打击网络淫秽色情、诈骗、黑客攻击破坏等各种违法犯罪活动，切实维护网络安全，保障公民的合法权益。
