省煤炭工业厅、省地方税务局联合发布公告

山西省实施煤炭资源税减征政策

2015年10月28日09:37    来源：人民网-山西频道    手机看新闻

人民网太原10月28日电（张婷婷）为落实国家有关煤炭资源税优惠政策有关事项，近日，山西省煤炭工业厅和省地方税务局联合发布公告，将对部分煤矿实施煤炭资源减征的优惠政策，其中，对衰竭期煤矿开采的煤炭，资源税减征30%。对充填开采置换出来的煤炭，资源税减征50%。

据了解，衰竭期煤矿是指剩余可采储量下降到原设计可采储量的20%（含）以下，或者剩余服务年限不超过5年的煤矿，以煤炭企业下属的单个煤矿为单位确定。无法查找原设计可采储量的，衰竭期以剩余服务年限为准。剩余服务年限将由剩余可采储量、最近一次核准或核定的生产能力、储量备用系数计算确定。充填开采是随着回采工作面的推进，向采空区或离层带等空间充填矸石、粉煤灰、建筑废料以及专用充填材料的煤炭开采技术，主要包括矸石等固体材料充填、膏体材料充填、高水材料充填、注浆充填以及采用充填方式实施的保水开采等。采用充填方式实施的保水开采是指为保护地表水体、开采煤层上覆岩层水体和下伏岩层水体，采用充填方式减少水体损害的煤炭开采方法，主要包括采空区充填、上覆岩层离层带注浆充填、下伏岩层注浆充填加固等。

山西省境内符合国家规定的煤矿均可申请确认煤炭资源税减税项目。其中：大同煤矿集团、山西焦煤集团、阳泉煤业集团、潞安矿业集团、晋城无烟煤集团和山西正华集团公司所属煤矿由集团公司审核后，上报省煤炭工业厅审查确认。其他煤炭企业所属煤矿，上报资源税纳税所在地市级煤炭行业管理部门审查确认。

按照规定，煤炭行业管理部门若发现衰竭期或者充填开采煤矿与申报情况不符、生产情况发生变化、不符合减税条件或者对计税依据产生影响的，将于15个工作日内通知同级税务部门。税务部门在征管中发现纳税人减税项目存在上述情形的，将暂停执行优惠政策，并于15个工作日内上报煤炭行业管理部门予以确认。与此同时，山西省主管税务机关，将对可采储量、实际生产能力、已享受减税优惠数量等及时监控比对，若发现纳税人违规享受减税优惠政策，将按税收征管法有关规定进行处理。
