教务处关于组织2014-2015学年春季学期课程重修免修考试(补考)的通知 
2015-07-17   内容编辑：王嘉滢   点击数：7076 收藏本页   推荐阅读 0  纠错  

各院系：

2014-2015学年春季学期课程的重修免修考试现开放学生报名，报名时间为2015年7月20日-7月22日下午17:00。

一、重修免修考试科目：

由教务处根据2014-2015学年春季学期课程学生考核成绩分布情况统计。但属于以下情况的不得申请参加重修免修考试，学生必须参加课程重修。

1、2014-2015学年春季学期不及格但不为2014-2015学年春季学期本专业计划内的课程；

2、各学期开设的通选课、任选课、不安排重修免修考试的课程及学校规定必须重修的课程；

3、擅自缺考或者在考试中有舞弊行为的课程，期末考试成绩为0分的课程；

4、重修仍不及格的课程，包括由于培养计划变更修读的用于可申请重修替代的课程；

5、从未修读课程（即没有选课及成绩记录的课程）。

6、夏季学期不及格的课程。

二、报名资格与报名手续的办理：

1、符合报名条件的学生在规定时间内登陆教学信息服务网（electsys.sjtu.edu.cn）一专选课―补考报名，选择参加重修免修考试的课程；

2、缓考课程考试安排与本次重修免修考合并进行。2014-2015学年春季学期缓考申请成功的同学必须参加本次重修免修考试，系统已自动设置为报名成功，请上学期缓考申请成功的同学在规定时间内登陆查看并确认考试科目；缓考课程考试只有一次机会；

3、报名操作办法：点击“补考报名”，是否参加补考一栏显示为“是”即报名成功。反之点击“取消补考报名”；

4、报名系统关闭后不再受理，视为放弃本次重修免修考试，且报名资格不顺延至以后学期。报名时间为2015年7月20日-7月22日下午17:00，逾期不再受理任何报名及取消报名事宜。请于报名系统关闭前再次确认报名状态是否无误。

三、成绩与学分认定：

本次参加考试的课程通过的成绩记录登记60，成绩单中显示“P”，视为课程通过，但不列入各类积点与积分计算；不通过及缺考的成绩记录登记0，成绩单中显示“F”，列入各类积点与积分计算。本次考试成绩不覆盖2014-2015学年春季学期的不及格成绩及“缓考”标记。

四、考试时间：

查询考试安排的时间预计在夏季学期第4周周五（具体时间教务处主页会另行发布通知）；考试时间安排在秋季学期开学前1周周四（9.10）、周五（9.11）和秋季学期第1周周二（9.15）共三天。

五、其他：

1、成功报名重修免修考试的学生必须按考试安排的时间地点参加考试，缺考按零分计。重修免修考试报名成功的课程2015-2016学年秋季学期不得重修，必须在以后学期自行择期重修；教务处将于夏季学期第三周周一（7.20）在教务处主页公布2015-2016学年秋季学期开设的重修班信息，请及时关注，并在重修免修考试报名系统关闭前慎重选择，重修免修考试报名系统关闭后将不再受理任何报名及取消报名事宜。

2、报名时如未显示重修免修考试科目，请先核查是否满足考试条件。如遇到问题请联系学院教务办或注册与学务中心（新行政楼B108房间）。

3、离校学生如无法自主报名，请在规定时间内email至helenwjy@sjtu.edu.cn（请写明学号、姓名、课程名称、课程代码等信息，且课程须符合重修免修考试条件，不符合条件的课程一律不受理）；或在报名时间内至新行政楼B108办理，逾期均不再受理。

六、本次重修免修考试不收取费用。

请学院协助督促符合条件并有意参加本次考试的学生按时报名。

                                                                   教务处

2015年7月17日
