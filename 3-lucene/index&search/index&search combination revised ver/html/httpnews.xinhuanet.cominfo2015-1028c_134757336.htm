苹果6s印度遇冷 当地人都在用哪些手机？
 
2015年10月28日 09:05:07 | 责任编辑: 杨帆 | 来源： 环球网   

　　印度作为一个超级人口大国，其经济近年来也进入了一个高速发展的时期，各大手机厂商自然不会放过这个庞大的市场，都在暗中争夺其市场份额。近日有印度媒体报道，iPhone6s在印度遇冷，销量骤降。那么，印度人们到底都在用着什么手机呢?

　　三星在印度称霸

　　根据IDC最新公布的数据，三星在印度市场占到了23%的份额。如今的印度智能手机市场还处于一个起步阶段，智能机普及率只有10%左右，而三星在印度市场主打的普遍是中低端的手机，这一定程度上迎合了印度人们的喜好，用比较低的价格买一个国际性的大牌在他们眼中显然更为值得。
