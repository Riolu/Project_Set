新电子邮件系统快速使用指南

2011-12-28  点击：[]

一、基本信息

容量：教工5G，学生2G

单封邮件发送附件大小：50M

 

二、      新老邮件的过渡问题（Webmail用户）

1、 邮件：1月4日起，新邮件系统启用，原邮件系统的邮件已导入，新的邮件将全部接收在新系统。同时，旧的邮件系统继续保留1月4日以前的邮件，仅供查询，不再接收新的邮件，也不能发送邮件。如果需要查询老的邮件，可以用jaccount登录http://oldmail.sjtu.edu.cn 查询，旧邮件系统2012年3月31日停用。

2、 原邮件的文件夹：由系统自动导入到新邮件系统，如果要删除某个文件夹，可以右键点击文件夹，选择“删除”即可。

3、 地址簿：由系统自动导入原有的地址簿，但分组会丢失，用户需要自行重新分组。登陆后点击“首选项”，选择“地址簿”。

5、 邮件过滤：需要用户在新的邮件系统中自行重新设置。登陆后点击“首选项”，选择“过滤器”。

 

三、 outlook、forxmail等客户端用户使用

对于使用客户端的用户，本次邮件升级增加了安全的POP3和SMTP加密连接，强烈建议用户设置加密连接方式收发邮件，提高数据传输的安全性。设置服务器地址也是mail.sjtu.edu.cn，具体设置方式请参考如何在邮件客户端设置安全的加密连接：outlook设置 foxmail设置 Outlook Express设置，其他客户端请参考上述类似设置。

注意：由于新旧系统更换，您的邮件客户端会再次在新系统上把邮件收一遍，因此你的邮件客户端可能会重复收一次信（仅会收一次），下次再收信就正常了。

 

四、      主要新功能和新特点

1、增强了电子邮件系统的稳定性和扩展性。

2、增强了对移动终端的支持。

3、增强了对垃圾邮件的过滤功能。您需要根据您实际使用情况，设置适当的评分机制即可，点击“首选项”，选择“邮箱”，右侧“垃圾邮件选项”里选择过滤级别。

4、支持HTTPS、POP3S、SMTPS、IMAP4S加密方式，提高用户数据传输的安全性。

 

更多功能和使用帮助，请访问邮件系统 http://mail.sjtu.edu.cn 登陆后点击“帮助”。
