烤箱加热可以延长锂电池的寿命

2015年10月28日 06:10:11 来源： 环球网
  • 评论0
  • 打印

　　近日，来自加州理工大学的研究人员详细探讨了如何通过更高的温度来解决这些树突，并且可能会延长电池的寿命，一起来看看吧。首先，我们来说说正常工作的电 池内部的情况。当电池产生电流的时候，电子从阳极流出来，通过电池外部的电路流入阴极。阳极通常由锂等导电的金属构成，其内部的一些原子会在失去电子(用 于产生电流)之后变成离子，并通过被称为电解质的导电液体介质移动到阴极。

　　而给电池充电则是一个翻转的过程，离子返回到阳极上。但是当它们这样做的时候，离子并不能均匀地附着在阳极上。相反，这些离子会形成一个微观的坡度，经过多个充电周期之后就会最终成为长长的分支。

　　当这些树突和阴极接触时就会造成短路。这时候，电流将会通过树突流动而不经过外部的电路，电池也会因此失去效用而“死亡”。

　　电流流过树突的时候也会使后者的温度升高，再加上电解质通常都是一种易燃物，因此，树突可能会因为温度过高而出现燃烧的状况。另外，即使树突没有造成电池短路，它们也会从阳极上脱落，在电解质中漂浮。这时候，阳极损失了材料，电池也就不可能存储跟原来一样多的电量。

　　这项研究成果已经登上了美国《化学物理学》杂志的封面。来自加州理工大学的科学家 Asghar Aryanfar 说：“树突非常危险，并且会减小可充电电池的容量。”他表示，虽然研究人员研究的是最有效的锂电池，但是成果是可以广泛应用的，因为树突问题在可充电电池上是非常普遍的。

　　研究人员对长出树突的锂电池进行了测试，对它们进行了长达几天时间的加热。他们发现，55 摄氏度的高温能够有效缩短树突的长度多达 36%。

　　为了弄清楚究竟是什么导致了这一收缩现象，研究人员使用计算机模拟了热量对单个锂原子的影响。模拟结果显示，温度升高使得锂原子进行了“洗牌”一般的运动，这种运动足够使得树突结构被推翻。

　　Asghar Aryanfar 表示，通过量化需要多少能量才能改变树突的结构，研究人员可以更好的了解它们的结构特征。而这一研究结果表明，让你的电池重新焕发活力，可能只需要一些额外的热量。
