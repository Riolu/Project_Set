经济增速放缓，就业如何？9个月，全年就业目标完成

2015年10月28日 07:48:09 来源： 人民日报
  • 评论0
  • 打印

  • 字小

　　经济增速放缓，就业表现如何？人力资源和社会保障部新闻发言人李忠10月27日在人社部新闻发布会上介绍说，今年前三季度，就业形势总体稳定。1—9月，全国城镇新增就业1066万人，城镇失业人员再就业435万人，就业困难人员实现就业129万人。三季度末全国城镇登记失业率为4.05%，保持在较低水平。

　　今年四季度及明年就业形势将保持稳定

　　李忠说，就业平稳主要表现在三方面。一是城镇新增就业已提前完成全年目标任务，失业率仍然保持较低水平。1—9月全国城镇新增就业人员1066万人，已经超额完成了全年1000万人的目标任务。登记失业率4.05%，也低于4.5%的全年调控目标。二是市场供求基本平衡。据人社部对100个城市的公共就业服务机构人力资源市场供求数据调查显示，三季度市场的求人倍率是1.09，表明市场岗位需求略大于求职人数。三是高校毕业生、农村转移劳动力和城镇就业困难人员等重点群体的就业工作平稳推进。

　　针对“企业出现大规模裁员潮”的传闻，李忠予以了否定。他介绍说，在经济下行和结构调整中确实有一些企业出现岗位流失和减员现象，但总的来看，涉及范围不大、人员分流平稳，未发现普遍性的裁员潮。根据人社部对全国3万多家企业监测的情况看，在走势上，三季度以来监测企业岗位流失率低于上半年岗位流失情况；从行业上来看，岗位流失主要集中在煤炭、钢铁、化工等行业，多是化解产能过剩或环境治理的重点行业。

　　“在调研中我们也了解到，经过政府帮扶和企业履行社会责任，裁减人员通过转岗培训实现再就业或者自主创业的方式，基本上都得到了妥善安置，就业形势保持了总体稳定。”李忠说。此外，从县乡村三级反馈情况看，目前外出务工农民没有因为企业关闭而出现大规模返乡，这也从一个侧面佐证了当前就业形势总体稳定。

　　据介绍，下一步人社部将继续推动离校未就业毕业生就业促进计划和大学生创业引领计划，组织开展第三届部分大中城市联合招聘、高校毕业生就业服务周等专项活动，帮助高校毕业生尽快实现就业。

　　李忠预测，今年四季度及明年就业形势还将保持稳定。“作出这个判断是基于国家已经出台了一系列稳增长、促改革的措施，这些措施落实力度正在加大。”
