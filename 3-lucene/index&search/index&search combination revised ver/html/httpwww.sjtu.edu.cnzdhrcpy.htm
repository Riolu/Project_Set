本科生培养
　　学校共设本科专业66个，涵盖经济学、法学、文学、理学、工学、农学、医学、管理学等8个学科门类。经过不断的总结提炼，学校本科人才的培养目标是：实施与通识教育相融合的宽口径专业教育，使学生的知识学习和能力培养植根于丰厚的人文社会科学和自然科学的沃土，使学生具有志存高远的品性、身心和谐的人格、追求卓越的意志和海纳百川的胸怀，成为德、智、体、美全面发展，知识、能力、素质协调统一，具有宽厚、复合、开放、创新特征的高水平、高素质、国际化人才。
研究生培养

　　上海交通大学从1931年起培养研究生，是国内最早开始招收研究生的大学之一。目前，已建立了哲学、经济学、法学、教育学、文学、历史学、理学、工学、农学、医学、军事学、管理学等全部12个学位门类的研究生培养体系，研究生在校规模达到2万多人。我校稳步推进研究生培养机制改革各项举施，博士研究生的生活待遇得到明显提高；积极申报教育部和上海市研究生教育创新计划，注重培养学生的创新精神与实践能力；积极推进研究生教育国际化进程，开展国家公派出国留学研究生项目、博士生国外访学计划、海外高层次合作办学等项目。近年来，我校研究生教育质量稳步提高，为学校建设综合性、研究型、国际化的一流大学提供了有力保障。
博士后流动站
　　上海交通大学博士后流动站现有控制科学与工程、船舶与海洋工程、机械工程、材料科学与工程、动力工程及工程热物理、力学、电子科学与技术、信息与通信工程、计算机科学与技术、电气工程、生物医学工程、基础医学、口腔医学、管理科学与工程、物理学、环境科学与工程、核科学与技术、数学、仪器科学与技术、生物学、工商管理、外国语言文学、临床医学、化学、药学、园艺学、光学工程、土木工程、法学、应用经济学、软件工程、生态学博士后流动站共32个，涵盖理、工、医、管、法、经、农、文等8个学科门类，所有的32个流动站覆盖我校一级学科博士点的89%。
留学生培养

　　自2000年，学校成立国际教育学院以来，交大留学生教育发展迅速，来自韩、日、美、德、法、英、澳大利亚等100多个国家和地区的留学生不断增长；攻读本、硕、博的学位生比例大幅增加，留学生层次逐年提高。据2012年统计，全校各类留学生达到5915名；学位生2087名，其中本科生1668名，硕士研究生344名，博士研究生75名。与2011年相比，留学生本科生增长24%，留学生研究生则增长92%。就留学生所读学科而言，文学占64%，工学占10%，管理学占9%，医学占6%，经济学占6%，其他占5%。

　　此外，为响应国家推动汉语国际化发展战略，交大自2005年起先后在马来西亚吉隆坡、澳大利亚新南威尔士大学、美国UCLA、美国普度大学和德国海德堡大学等合作建立五所海外孔子学院。
成人教育

　　上海交通大学的成人教育源于“五四”运动后的民众教育和工人夜校。成人高等教育起始于1956年。成人教育学院成立于1992年。目前具有专任的师资队伍，其中博士学位占50%，具有海外学习经历的占65%。共有工、医、农、文、法、经济和管理学科等50多个应用型专业。学院积极发展高层次的继续教育、职业培训和外籍人士的职业技能培训。
网络教育
　　上海交大网络教育学院成立于2000年7月，是全国现代远程教育协作组副组长单位和上海现代远程教育协作组组长单位。网络教育学院在外延上走“上水平，做贡献”的发展道路，在内涵建设上集中精力设计并实践一个基于下一代互联网的新一代E－learning的模型。学院以保证教学质量为前提，运用现代远程教育技术，形成了“教育无处不在，内容时刻更新”的混合型教学模式。近三年，学院项目获国家级教学成果奖一等奖、国家科技进步二等奖、上海市科学技术进步一等奖等殊荣。
技术学院
　　技术学院负责交大技术应用性本科人才的培养，学院设在交大七宝校区。学院现设有电子工程、计算机、经济管理、外语及交通运输共五个系。有专职教师39名，正副教授27名；博士学位及在读博士学位教师9名。
　　学院重视学生的全面发展，成绩显著。近三年来有多名学生在国家核心期刊上发表论文十多篇。各专业设置紧密结合当代科技前沿和企业需求，毕业生均有中级以上岗位资格证书或技能证书。毕业生就业形势良好，多次荣获学校就业先进单位；2007届344名毕业生就业率达95.1%。
