中国依法严惩网络攻击破坏犯罪 态度坚决 措施有力
2015年9月25日 15:25

编者按 网络攻击具有匿名性和跨国性特点，这使网络安全成为复杂的全球性问题。中国有关网络安全公司不久前发布报告显示，中国长期以来遭受来自境外的大规模网络攻击，国家安全和利益面临严峻威胁。但网络不是法外之地，同样要讲法治。中国政府是网络安全的坚定维护者，坚决反对并依法打击任何形式的网络攻击行为，主张各国本着相互尊重和相互信任的原则，共同构建和平、安全、开放、合作的网络空间。

当前，我国公共安全形势总体向好。但是，公共安全领域的隐患仍然存在，网络安全问题尤其引人关注。

网络社会非“法外之地”

日前，为充分发挥人民法院审判职能作用，切实维护公共安全，最高法院发布了《关于充分发挥审判职能作用切实维护公共安全的若干意见》。《意见》提出，依法惩治网络攻击破坏犯罪和利用网络实施的各类犯罪，切实维护网络秩序，净化网络空间。

据2015年中国网民权益保护调查显示，九成网民认为隐私权是重要的网民权益，七成网民遭受过网络身份、网络活动信息的泄露。据中国互联网协会12321举报中心统计，今年上半年接到的105件举报中，八成涉及网络安全，有的甚至是网络犯罪，包括隐私的窃取、流氓行为、诱骗欺诈等。

网络空间是现实社会的延伸，网络秩序是公共秩序的有机组成部分。网络社会不是“法外之地”，不管什么人，在我国境内实施网络攻击和网络商业窃密都是违反国家法律的，都应受到法律的追究。

据最高法研究室负责人介绍，近年来，针对信息网络实施的攻击破坏和利用信息网络实施的各类违法犯罪活动日益突出，危害十分严重。对此，要依法打击非法侵入、破坏计算机信息系统以及制作、销售、使用“伪基站”设备等犯罪活动，从严惩治针对基础信息网络、重要行业和领域的重要信息系统、军事网络、重要政务网络、用户数量众多的商业网络的攻击破坏活动，从严惩治利用攻击破坏非法获取国家秘密、商业秘密、公民个人信息等犯罪活动，有效保障信息网络自身安全。

“依法打击网络违法犯罪、促进完善网络空间管理，是人民法院的重大政治责任、法律责任和社会责任。”最高法研究室负责人指出，要坚决依法打击网上造谣、传谣行为，惩治利用网络实施的盗窃、诈骗、敲诈勒索、寻衅滋事、贩卖毒品、传播淫秽信息等犯罪，维护网络社会安全，促进网络秩序的进一步规范，为广大人民群众提供一个健康、有序、清朗的网络环境。

近年来，北京市东城、海淀、石景山等区检察院，均成立了网络犯罪案件专业化处室或办案组，一批典型案件被以“团队作战”方式成功办理。辽宁省大连市检察院网络犯罪检察监督处自2012年成立以来已办理此类案件49件90人。

今年以来，检察机关重视维护网络安全，依法惩治利用网络实施的有组织制造传播谣言、诈骗、赌博、传播淫秽信息等犯罪，净化网络空间。

没有网络安全就没有国家安全。今年先后召开的全国检察机关第五次公诉工作会议、“互联网+检察工作”座谈会上，面临网络安全的新形势，充分履行检察职能，坚决维护网络信息传播秩序和国家安全、社会稳定成为会议热点。在7月召开的大检察官研讨班上，最高检有关负责人强调，要深入研究网络犯罪的新特点,积极参与“清网行动”，坚决惩治网络攻击、网络入侵、网络窃密、散布违法有害信息等犯罪，坚决惩治利用网络敲诈勒索、诈骗、赌博、制造传播谣言、传播淫秽信息、窃取公民个人信息、侵犯个人隐私、侵犯知识产权等犯罪，维护国家网络空间主权、安全和发展利益。

打管结合联动“净网”

据统计，近年来，中国的执法部门与70余个国家和地区开展了1000余次网络犯罪领域的执法协作，先后侦办了网络诈骗等一批跨国网络犯罪案件。同时，在网络犯罪的情报信息交流、侦查取证、培训等方面，中国与多国开展合作,为共同打击网络犯罪提供了重要的支撑和保障。

近年来，传统违法犯罪加速向网上发展蔓延，网络对违法犯罪的聚焦放大作用更加明显。互联网的便捷性、虚拟性、跨界性给打击犯罪带来新挑战，加剧了治安问题的复杂性。

2015年3月18日，江苏省徐州市某药业有限公司负责人报案称，其公司的网站服务器被非法入侵，入侵者恶意篡改公司网站文件，在其网站页面添加大量赌博网站的内容，对其公司的形象及信誉造成恶劣影响。徐州市公安局网安支队迅速对涉案服务器进行勘验。通过对木马程序进行分析，民警发现了控制木马的嫌疑人刘某。经扩线侦查，民警又发现向刘某出售网络控制权限和木马的嫌疑人牛某。近期，办案民警分别抓获刘某、牛某等7名犯罪嫌疑人，一举捣毁了这一涉嫌非法入侵服务器篡改网站内容的犯罪团伙。据统计，该团伙涉嫌入侵企事业单位网站2000多家。

据了解，在公安部统一指挥下，截至目前，各地公安机关已侦办网络犯罪案件7400余起，抓获犯罪嫌疑人1.5万余人。
