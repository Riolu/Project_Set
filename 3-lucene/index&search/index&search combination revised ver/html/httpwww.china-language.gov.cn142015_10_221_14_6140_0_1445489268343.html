语用所领导到部教育信息管理中心就推进信息化建设学习汇报

 

为落实教育部副部长、国家语委主任杜占元同志指示精神，推动语言文字信息化工作，并结合教育部语用所巡视整改工作安排，2015年9月29日，语用所所长张世平，副所长魏晖、吕同舟，所长助理王晖到教育部教育信息管理中心进行专题调研。

语用所领导班子成员对教育信息管理中心在“小金教”项目上给予我所的大力支持表示感谢，并介绍了语用所承办或开办的中国语言文字网、语用所网和几个业务网站的运营情况，还就拟建设的“国家通用语言文字公共服务平台建设”项目和信息安全问题进行了咨询。

教育部教育信息管理中心主任展涛、副主任曾德华同志介绍了教育部电子政务系统建设、国家教育信息管理服务平台建设情况。他们认为语用所面向国家语委、应用语言学科提供学术服务，面向社会提供公共服务，工作很有成效，充分利用信息化技术提升服务品质，把语言文字研究成果由信息化服务体系呈现出来，意义深远，前景广阔，教育信息管理中心愿意为此提供必要的项目支持。

双方对合作的领域、内容和工作模式交换了意见，还对各自单位事业改革情况进行了探讨，双方表示将进一步加强交流，相互支持，努力推动有关项目的合作。
