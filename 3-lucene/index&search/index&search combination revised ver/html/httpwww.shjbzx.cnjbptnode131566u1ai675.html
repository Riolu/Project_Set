中国驻英大使接受英国广播公司采访谈网络安全等问题
2015年10月19日 11:02

中国国家主席习近平将于10月19日至23日对英国进行国事访问。中国驻英国大使刘晓明在访问前接受英国广播公司(BBC)采访时表示，中方到英国来是为了双赢合作，如果双方要成为伙伴，相互之间应有一个基本的信任。

刘晓明在BBC强档品牌节目“新闻之夜”演播室接受直播专访，回答有关中国经济、网络安全、中英经贸投资合作和人权等问题。

关于中英两国在建设核电站方面的合作，刘晓明说，英国希望中国来英投资建设核电站，是因为英国需要中国的资金和技术，中国拥有先进的核电技术，中国的核电站数量比许多国家都要多。

谈到英国媒体对核电站安全性的质疑，刘晓明说，中国将严格遵循国际标准，我们到英国来是为了双赢的合作，不是为了控制英国的核电站。如果双方要成为伙伴，相互之间应有一个基本的信任。如果任由这种负面言论发展，中国企业可能就不敢来英国投资了。如果这样，那么就和财政大臣奥斯本所说的“英国是对中国投资最开放的西方国家”背道而驰了。

谈到网络安全问题，刘晓明说，不久前，习近平主席对美国进行国事访问的时候，中美两国就共同打击网络犯罪达成共识。中国的任何政府部门都不会参与网络间谍活动，中国政府也不会支持任何这样的行为。这是中国政府的承诺。

关于人权问题，刘晓明说，中英两国的社会制度、历史、文化有很大差异，双方处于不同的发展阶段，我们之间有不同，这很正常。但我们仍然可以合作，我们可以求同存异。
