    意大利米兰世博会简介
        会期与主题：2015年意大利米兰世界博览会是继2010年上海世博会后经国际展览局正式批准的新一届综合性世博会，将于2015年5月1日至10月31日在意大利米兰市举办，为期184天。届时参加国家和国际组织将达到150多个，参观游客数千万人次以上。
        本届世博会主题为“滋养地球――生命的能源”，旨在探寻为全球提供充足、优质、健康和可持续发展的食品保障方式，寻找合理利用资源、保护环境、滋养人类、反哺地球的有效途径，促进人与自然和谐均衡发展。

中国馆标志
中国馆标志用线与面勾勒出大自然的轮廓和曲线：蓝色代表天空，绿色代表田野，金色代表粮食，红色代表人和生命，水墨纹理代表中国传统文化。图形透叠寓意“世间万物和谐共生、天人合一”的传统思想。将中国传统书法和绘画元素融入现代图形设计，强化中国传统文化特色。
