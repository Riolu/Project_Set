云南罗平出现壮美云海

“枫景”醉人

新疆哈密胡杨“醉”美金秋

金秋十月，新疆著名胡杨观赏地哈密伊吾县淖毛湖的胡杨迎来了最佳观赏时节。40多万亩胡杨沐浴秋色，泛黄染金，在戈壁滩上演绎着独具风韵的生命传奇。

云绕雪山别样美

金秋额济纳 醉美胡杨林

古隘雄关

中国的古关隘，规模宏大，建筑雄奇。战时可作为边塞之防守，平时可装点山川之壮美。历代戍边武将巡逻至此抒发豪情，文人骚客游历至此即兴吟咏，留下很多诗词歌赋。
