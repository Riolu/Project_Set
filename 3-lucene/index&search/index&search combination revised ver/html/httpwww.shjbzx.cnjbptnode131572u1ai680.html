全国互联网违法和不良信息举报受理量9月份首次突破三百万件

2015年9月，全国各地网信办举报部门、各网站采取有力措施，积极推进网络举报工作，共受理网民举报333.1万件,较上月增长34.4%，举报受理量首次突破三百万件。经审核，有效举报177.9万件。截至9月底，全国共处置网民有效举报155.9万件，处置率达87.7%，较上月增长近五个百分点。

综合分析全国举报情况，淫秽色情类有害信息举报仍较为突出，达109.5万件，占比61.6%；政治类有害信息举报占比14.6%；侵犯网民权益类有害信息举报占比4.3%；诈骗类有害信息举报占比3.5%；暴恐类有害信息举报占比3.2%；网络敲诈和有偿删帖类有害信息举报占比1.9%；赌博类有害信息举报占比1.7%；其他有害信息举报占比9.2%。其中，利用微信、微博贩卖淫秽色情资源，虚假票务诈骗、航班取消退款诈骗、网络购物诈骗等违法行为受到网民集中举报和谴责。

截至9月底，全国已处置网民有效举报155.9万件，处置率达87.7%；通过各类渠道向网民反馈处置结果154.2万件，回复率达86.7%。其中，中国互联网违法和不良信息举报中心直接受理、处置网民有效举报31051件；各地网信办举报部门共接受网民有效举报56762件，处置率达87.8%；中央重点新闻网站共接受有效举报626件，处置率达99.2%；全国各省、市、自治区属地重点网站（包括主要商业网站）接受有效举报169万件，处置147.7万件，处置率达87.4%。

主要商业网站接受有效举报167.1万件，处置145.9万件，处置率达87.3%，较上月增长5个百分点。其中，腾讯受理举报142.6万件（微信56.6万件、QQ81.7万件、网站3.2万件、微博1.1万件）；新浪受理公众举报12.3万件（新浪微博10万件），阿里巴巴受理公众举报7.2万件，百度受理公众举报4.7万件，凯迪网络受理公众举报875件，天涯社区受理公众举报481件，优酷网受理公众举报370件，搜狐网受理公众举报285件，凤凰网受理公众举报124件。

根据公众举报，有不法分子利用微博、微信贩卖淫秽色情资源，情况突出。经核查，不法分子注册多个微博账号，通过头像设置、资料简介、私信等内容宣传、推广出售色情视频资源的微信账号，引诱网民关注，通过组建微信群，不定期发布淫秽色情视频短片，诱引群内人员购买，并游说购买者邀请好友进群，为其推广。由此形成的淫秽色情有害信息传播链条隐蔽，传播速度更快、影响人群更广，影响恶劣。举报中心已将网民举报的“女郎lolita”等24个非法微博账号、“sslpc88334”等22个非法微信账号转交相关部门依法处置。

9月末，临近节假日出行高峰，各类虚假票务诈骗、航班取消退款诈骗、网络购物诈骗等违法活动猖獗，受到网民集中举报。不法分子开设特价机票网站，利用远低于市场价的“机票”诱骗网民上当；冒充航空公司发送航班取消通知，向网民骗取“退票费”；通过各种途径获取消费者购物信息，冒充网站客服，谎称由于网络升级、网站故障等需要办理退款，套取受害者银行账户密码，盗取资金。年底前，各类电商促销活动将连续开展，各类网络诈骗还将伺机而动，举报中心提醒广大网民注意防范。

截至9月底，举报中心今年已发放举报奖励857万元。举报中心欢迎广大网民积极举报互联网违法和不良信息。
