国足世预赛名单：黄博文张稀哲入选 郜林蒿俊闵无缘

2015年10月27日 20:09:13 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

　　新华网北京１０月２７日体育专电（记者公兵）２７日，中国国家男子足球队公布了参加１１月世预赛两场比赛的２４人备选名单，黄博文、张稀哲重新回归，郜林、蒿俊闵无缘入选。

    国足主帅佩兰表示：“我们即将迎来２０１５年最后一次集训，现在中超联赛还有一轮比赛，后面还有足协杯决赛，再加上恒大参加的亚冠主客场决赛，使得我们的备战遇到了一些困难。我们在挑选球员时必须考虑这些因素。”

　　佩兰说：“我们将于１１月５日在长沙集中。恒大参加亚冠决赛客场比赛（７日）的国脚要在８日才会回到国内，这期国家队里有８个恒大球员。考虑到最后一轮中超联赛和亚冠比赛中可能会有球员受伤，所以这次集训名单是２４人，这也是防患于未然。”

　　对于１１月１２日主场对不丹队和１７日客场对中国香港队的比赛，佩兰表示，目标只有胜利，“联赛已经接近尾声，球员们很疲劳，特别是精神层面会非常疲劳，所以他们都非常了不起，希望他们能够坚持打好今年最后几场比赛。比如恒大，希望他们打好亚冠比赛的同时也在国家队保持积极的精神状态。等我们客场打完中国香港队后，他们可以立刻返回只有一步之遥的广州准备亚冠决赛次回合（２１日），希望他们赢得亚冠冠军。”

　　包括客场对卡塔尔队的比赛在内，国足一直未能很好地解决进攻问题，佩兰对此表示：“我们会强化个人与整体水平的提升。这一期我们也招入了一些新生力量，希望能够对进攻有所改变。”

　　佩兰所说的新生力量包括此前呼声很高的黄博文、张稀哲，他说：“黄博文和张稀哲在联赛中表现优异，所以得以参加本次集训。”

　　佩兰招入黄博文、张稀哲是否迫于舆论压力？他表示：“我们与卡塔尔的客场比赛不尽如人意，但我对球员的精神面貌、态度非常满意。但后面比赛的目标就是要赢球，我们这次招入攻击型的前场球员的目的就是要提高攻击效率。”

　　“之前对卡塔尔队的比赛和接下来的两场比赛对手类型不一样。打卡塔尔可能我们需要在中场加强硬度，但这次为了增强进攻我们需要的球员也是不一样的。我们本期集训需要的就是中前场攻击型球员，他们（张稀哲和黄博文）的回来会增强前场进攻、增强得分能力，他们的能力一直都是配得上入选国家队的，所以他们的入选是实至名归的，”佩兰说。

　　对于郜林的又一次落选，佩兰说：“我一直认为郜林是中国最优秀的本土前锋之一，他在上半赛季表现非常好，但如何一直维持较高的状态也是我们考量球员的因素，队员们的身体状态、精神面貌我们都会有所考虑，国家队的大门一直对所有优秀球员开放，包括郜林。”

　　附国家队２４人名单：

　　广州恒大：郑智、曾诚、邹正、于汉超、张琳芃、梅方、郑龙、黄博文；

　　江苏舜天：任航、吴曦、吉翔、孙可；

　　上海上港：颜骏凌、武磊、蔡慧康、于海；

　　山东鲁能：王大雷、王永珀、杨旭；

　　北京国安：于大宝、张稀哲；

　　河南建业：毕津浩；

　　广州富力：姜至鹏；

　　西班牙巴列卡诺：张呈栋。
