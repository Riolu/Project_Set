走进武汉园博会 西安园：“大雁塔”夜间自发光

西安园的入口，记者看到一面铁艺雕花景墙，黄褐色的背景下，一峰峰硕壮的骆驼高昂着头一个接一个地在丝绸之路地图上前行。西安园的设计人员说，西安作为丝绸之路的起点，是丝绸之路经济带建设的先锋，因此西安园的展园主题紧紧围绕着丝绸之路而展开。[详细]

探秘武汉园博会:《阿凡达》特效团队制作4D大片

《穿越长江》是为园博会长江文明馆4D体验厅量身订制的一部4D影片，片长10分钟。长江文明馆是一座以长江为主线的跨区域的博物馆，但又不局限于传统博物馆以实物展示为主，该体验厅就是用年轻人喜闻乐见的形式，介绍长江人文风景以及对长江沿岸城市未来模样的幻想。 记者昨从武汉市园林和林业局获悉，为制作这部堪称史上“最梦幻”的长江题材影片，剧组总导演Chuck将带队赴长江三峡，寻找一位在三峡地区拍摄记录30年三峡的摄影爱好者，以求影片能真实还原长江原貌。记者在市林业局大楼找到《穿越长江》剧组时，Chuck正在向中美工作人员讲述影片梗概，“夕阳下，远处的滕王阁像一幅水墨画。这时，天空中飞来一只可爱的江豚，它就是‘楚楚’，一只凤凰伴随在它左右。‘楚楚’一直往前飞，飞过滕王阁，顺着长江飞到黄鹤楼，再飞到岳阳楼……”[详细]

揭秘园博园看不见的“世界”

园博园北入口，拾级而上，登顶后眼前豁然开朗，15米高的“荆山”已雏形初显，山上绿意盎然，鸟语花香，很难想象这座“荆山”是在金口垃圾场原址上修复而成。 运行16年的金口垃圾场，地下已积攒大约500多万立方的生活垃圾。这些垃圾是如何在短短3年内，全部实现无害化处理？如今“荆山”绿树掩盖下的地块，曾经经历了怎样的修复和变化？昨日，记者探访揭秘。[详细]
