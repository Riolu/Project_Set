重拳出击 为少年儿童健康成长营造良好环境
2015年7月6日 16:39

7月6日，全国“扫黄打非”办公室通报了“护苗2015”专项行动第一批案件查处情况。这批通报的8起案件中，5起为网站登载淫秽色情漫画案，3起为侵权及非法少儿出版物案，是“扫黄打非”护苗行动取得的专项整治成果。

全国“扫黄打非”办公室有关负责人表示，扫除非法、有害少儿出版物，为少年儿童健康成长营造良好环境，是“扫黄打非”的重要工作任务。自今年3月开始，全国“扫黄打非”办公室组织相关部门深入开展“扫黄打非·护苗2015”专项行动，集中时间、集中力量，严厉打击制售非法、有害少儿出版物违法活动。今年以来，全国已查缴各类非法、有害少儿出版物近18万件，查办了一批违法违规案件，惩处了一批违法犯罪分子，有效净化了少儿出版物市场环境。下一步，全国“扫黄打非”办公室将进一步加大整治力度，积极发动群众举报制售传播非法、有害少儿出版物及信息行为；加大执法力度，立足挖窝点、打团伙、破网络，确定重要案件，实施重点打击；对专项行动中查处的大要案进行挂牌督办，实现从立案到审判的全程督促，确保案件查处的法律效果和社会效果。 这批案件查处情况如下：

1. 陕西“猥琐叔小漫画”网站登载淫秽色情漫画案。据群众举报，自2015年以来，“猥琐叔小漫画”网站（域名：www.weisuoshu.com）大量登载淫秽色情漫画。5月28日，西安市雁塔区“扫黄打非”办公室协调当地公安部门开展调查工作，并于当日将犯罪嫌疑人齐某抓获。经查，齐某于2014年10月注册www.weisuoshu.com域名，2015年4月开始从互联网上下载《姐姐妄想日记之销魂体验》、《少女初次体验之公园寝取》等淫秽漫画，上传到“猥琐叔小漫画”网站进行传播。5月30日，西安市公安局雁塔分局大雁塔派出所依法将齐某刑事拘留，6月24日，对齐某以涉嫌传播淫秽物品罪提请雁塔区人民检察院批准逮捕。目前，案件在进一步侦办中。

2. 北京“动漫之家”网站登载淫秽色情漫画案。据群众举报，自2015年以来，“动漫之家”网站（域名：www.dmzj.com）大量登载淫秽色情漫画。6月4日，北京市文化市场行政执法总队对该网站进行执法检查。经查，该网站系北京凌飞天下软件有限公司主办，在网站“日漫”频道登载《霸王爱人》、《艳遇公馆》等47部淫秽色情漫画作品。6月23日，北京市文化市场行政执法总队责令该公司停止登载漫画作品，并对该公司作出罚款10万元的行政处罚。

3. 北京“BOX盒子游戏”网站登载淫秽色情漫画案。据群众举报，自2015年以来，“BOX盒子游戏”网站（域名：www.boxuu.com）大量登载淫秽色情漫画。6月5日，北京市文化市场行政执法总队对该网站进行执法检查。经查，该网站系欢动（北京）科技有限公司主办，该公司在网站“盒子动漫”频道登载《黑色幽默》、《九泉之岛》等49部淫秽色情漫画作品。6月23日，北京市文化市场行政执法总队责令该公司停止登载违法的漫画作品，并对该公司作出停业整顿90天和罚款3万元的行政处罚。

4. 浙江“游侠漫画网”网站登载淫秽色情漫画案。据群众举报，自2015年以来，“游侠漫画网”网站（域名：www.158c.com）大量登载淫秽色情漫画。6月1日，浙江省新闻出版广电局对该网站立案调查，指示杭州市文化广电新闻出版局调查取证。经查，该网站系杭州凤侠网络科技有限公司主办，在网站上登载了《监狱学园》、《鬼灯家的大姐姐+妹》等52部含有淫秽色情、凶杀暴力的漫画作品。6月23日，浙江省新闻出版广电局对该公司作出取缔游侠漫画网和罚款4万元的行政处罚。

5. 四川“52天漫画网”网站登载淫秽色情漫画案。据群众举报，自2015年以来，“52天漫画网”网站（域名：www.52tian.net）大量登载淫秽色情漫画。6月8日，成都网安支队对该网站进行执法检查。经查，该网站系成都五二天科技有限公司主办，在网站上登载了《弟弟的饲养方式让我来告诉你》、《静临同人》等13部淫秽色情漫画作品。6月10日，成都市公安局直属分局对该公司下达责令限期改正通知书，责令该网站暂时关停整改。

6. 北京东方利源印刷厂盗印《阿衰全集》案。2015年5月26日，北京市文化执法、公安等部门联合行动，在通州区北京利源印刷厂内查获少儿图书《阿衰全集》10600册，码洋近100万元。经初步核实，该书涉嫌盗版。目前，涉案人刘某等3人已被公安机关依法刑事拘留，案件正在进一步侦办中。

7. 北京宏泰印刷有限公司盗印少儿图书案。2015年5月12日，北京市文化执法、公安等部门联合行动，在通州区北京宏泰印刷有限公司的印刷车间，现场查扣杨红樱作品《笑猫日记》系列丛书20种34700册、码洋52万元，《三毛全集》系列图书11种57690册、码洋140万元。经初步认定，上述图书涉嫌盗版。目前，犯罪嫌疑人冯某某等2人已被公安机关依法刑事拘留，案件正在进一步侦办中。

8. 浙江宁波制售盗版少儿图书案。2015年4月29日，浙江省宁波市“扫黄打非”办公室协调文化执法、公安等部门捣毁一制售侵权盗版少儿出版物的窝点，当场查获《数学A》、《公文数学教室——教材D》等各类侵权盗版出版物2万余册，查缴涉案的复印机9台、电脑4台、打孔机2台、切纸机1台，犯罪嫌疑人施某某、杨某某夫妇被抓获。目前，案件正在进一步侦办中。
