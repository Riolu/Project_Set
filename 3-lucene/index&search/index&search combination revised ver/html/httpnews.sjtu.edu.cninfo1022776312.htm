上海交大举办2015首届大学生城市治理案例挑战赛[图]
[发布时间]: 2015年10月27日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 沈崴奕 孙佳

10月23日至24日，首届大学生城市治理案例挑战大赛在上海交通大学徐汇校区拉开帷幕。作为国内第一个关于城市治理方面的大学生案例挑战赛，本次大赛由上海交通大学主办，国际与公共事务学院承办，来自香港大学、香港中文大学、台湾大学、台湾政治大学、澳门大学、北京大学、中国人民大学及上海交通大学等两岸四地8所知名学府的12支代表队参赛，百余名优秀学子齐聚交大。  

大赛拉开帷幕

在23日上午的大赛开幕式中，上海交通大学学指委秘书长林立涛、国际与公共事务学院院长钟杨和党委书记曹友谊、台湾政治大学冷则刚教授等来自两岸四地的领导嘉宾出席开幕式，国际与公共事务学院副院长章晓懿主持开幕式。 

林立涛在致辞中代表上海交通大学对各位同学的到来表示热烈欢迎。他希望同学们能够珍惜宝贵机会，多学习、多交流，并祝愿所有同学能够在比赛中发挥自己的水平，以出色的表现为校争光。 

钟杨强调，同学们在汲取知识和掌握技能的同时还需要有强烈的社会责任感，期待同学们学以致用，融会贯通，进一步提升社会责任感，并祝大赛圆满成功。 

曹友谊指出，大赛旨在培养学生实践能力，增进学生对国情的深入了解，提升学生对城市发展问题微观剖析的能力；也是推动两岸高校学生学术交流，促进相互了解、共同进步的一次有益尝试。  

台湾大学潘欣欣博士作为参赛高校代表发言。香港大学蔡皓麟、北京大学邹瑞阳和上海交大张嫣婷代表参赛学生发言。 

大赛评审组组长樊博、台湾政治大学冷则刚以及上海交大港澳台办公室曹伟为各代表队授旗。 

焦点对话论坛

在23日下午的“焦点对话论坛”上，大会主办方邀请中国人民大学孙柏瑛教授、复旦大学唐亚林教授和上海交通大学陈映芳教授做主旨演讲，同参会的两岸优秀学子分享在城市研究方面的成果及心得。 

孙柏瑛教授以“城市治理的灵魂:我们应遵从怎样的逻辑”为演讲主题深入浅出地分析了当前中国城市化进程所面临的四大陷阱，即经营性陷阱、分割的陷阱、同质化陷阱和封闭性陷阱，并提出应采取区域一体化、协同治理、城乡统筹等应对措施。  

唐亚林教授从“当代中国国家建设的新命题：大都市治理模式的创新”的主题出发，具体剖析了当代中国大都市治理面临的深刻挑战并指出大都市治理的核心理念在于“繁荣，安全，自治，宜居，和谐”。  

陈映芳教授作“研究者与研究行为——以城市治理研究为例”的主题演讲，她从研究者角色谈起，帮助同学理解思考城市的价值，陈教授指出研究者应了解自身扮演的角色，研究者的城市观不能伪中立。 

在互动环节中，多位同学分别就“城市研究的切入视角”、“老城区改造的现实矛盾”、“城市治理的政治渠道”以及“城市农民工的社会融入”等问题进行提问和探讨。各位专家学者耐心解答并指出城市治理的议题纷繁复杂，城市快速发展中所展现出来的弊端也层出不穷，当代青年要勇于探索、勇于发声。  

案例成果展示

大赛于前期下发城市治理案例摘要，大赛开幕期间邀请专家做案例全文讲解，帮助参赛学生掌握赛制安排和案例核心内容。10月24日，大赛进入决赛成果展示阶段和讨论质询环节。各代表队运用公共管理的专业知识和公共政策的模型框架对城市治理的案例进行解读、探讨和汇报。 

华中科技大学张毅教授、山东大学王佃利教授、上海市教委规划处处长黄海洋以及上海市静安区民主宗教事务办公室副主任冒瑾辉、上海交通大学樊博教授、上海交通大学王郁教授、上海交通大学郑晓华博士等专家学者出席大赛成果展示和讨论质询环节并担任评委。 

大赛特别设置质询讨论环节以促进各高校代表队的相互交流和思维碰撞。所有参赛团队随机分为六个小组，每个小组内的两支团队就对方团队案例文本未及之处提出疑问。大赛评委评委每桌循环走动听取各小组辩论过程，考察每支团队的表现，并予以适当的专业指导。 

大赛共评出团队冠军、亚军、季军各1支队伍，个人优秀表现奖4名。中国人民大学二队，北京大学一队以及上海交通大学二队分获团队冠亚季军，个人优秀表现奖荣归香港大学一队梁卿、台湾政治大学周家鸿、台湾大学陈则宇、香港中文大学何镁洛四名参赛选手。  

本次大赛围绕城市治理的主题，以现实案例分析解读，论坛前沿观点碰撞，实地参观考察等形式多样、内容丰富的比赛设置，为来自港澳台和内地的8所高校的同学们搭建学术交流、思维碰撞的平台。首届大学生城市案例挑战大赛的成功举办，不仅注重学生学术热情激发，增进学生实践能力培养，还以此为契机培育当代大学生对社会建设的责任意识。
