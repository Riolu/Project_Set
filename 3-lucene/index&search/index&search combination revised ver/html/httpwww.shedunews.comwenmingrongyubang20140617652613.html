【领奖通知】网络微小说大赛和网页设计展示活动

2014-06-17 11:23:23    来源：上海教育新闻网   

本网讯 在上海市教卫工作党委宣传处、市教卫工作党委系统文明办的高度重视和大力支持下，由上海教育新闻网承办的“文明上网共建和谐——我的中国梦”主题系列活动，即网络微小说大赛和网页设计展示活动顺利落下帷幕。

本次活动共收到来自全市17个区县、25所高校的参赛作品7600多篇，经过初评、终评等环节，有150余篇网络微小说和网页设计作品在比赛中获奖。

颁奖会已于6月14日举行，请未领奖的获奖者按以下要求进行领奖。

 

领奖地点：上海市长宁路491弄36号7楼

联系电话：62525555-783（俞老师）   62525555-810（刘老师）

领奖时段：周一至周五上午9:30至下午4:30

截止时间：2014年7月5日


 

【附件】获奖名单：

（1）网络微小说大赛获奖名单.swf

（2）网页设计展示活动获奖名单.swf

（3）网络人气奖-微博转发人气奖-最具网络价值奖-优秀指导奖.swf

（4）优秀组织奖获奖名单.swf
