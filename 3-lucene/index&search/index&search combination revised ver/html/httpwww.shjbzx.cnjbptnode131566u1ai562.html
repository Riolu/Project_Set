上海最大传销案宣判：主犯曹建华被判13年
2014年1月6日 11:20

来源：IT时报

　　短短6年时间，上海家帝豪电子商务有限公司披着电商的外衣，以“消费养老”的名义，飞速发展代理商6万余人，收取参与传销活动人员缴纳的传销资金高达10.9亿元。

　　去年12月24日，这起上海有史以来涉案人数最多、涉案金额最大的传销系列案在黄浦、虹口、宝山、闸北4家法院同时宣判，家帝豪创始人曹建华犯组织、领导传销活动罪，被判处有期徒刑13年，并处罚金1000万元。另有10名家帝豪主要负责人、部门负责人以及68名代理商也被判获刑。

　　家帝豪公司及相关人员终于受到了法律的制裁。从2011年开始，《IT时报》连续披露家帝豪借“消费养老”名义进行传销，见证了家帝豪走向灭亡的始末。

　　曹建华和他的家帝豪

　　曹建华在企业界并不是默默无名之辈，上世纪80年代先后创办了“霞飞”、“奥利斯”等知名化妆品品牌。

　　2004年，曹建华在上海注册成立了上海家帝美容保健用品有限公司，期间推出了超天美容宝口服液等产品。之后，这家公司几易其名，2007年更名为上海家帝电子商务有限公司，在2010年6月又更名为上海家帝豪电子商务有限公司，曹建华和其儿子分别占有公司90%和10%的股权。

　　2007年起，曹建华通过旗下一个名为“我爱我买”电子商务购物平台，以推销网站销售商品为名，要求参加者购买5000元或者1万元的商品套餐获得加盟资格，并按照推荐发展加盟的先后顺序组成上下线关系，直接或者间接以发展人员的数量作为返利依据，不断发展他人参与此活动。

　　在2009年11月，上海市工商局原卢湾分局以非法传销对家帝公司罚款80万元，但这次被罚并未让曹建华停下来。2010年，曹建华推出“消费养老”项目，以消费积分、消费养老计划、联盟商家、期权计划等概念来吸引人，先后以家帝公司和家帝豪公司名义继续在全国范围内发展代理商。实际上，这次的传销模式与2009年之前基本相同。

　　本报深度揭露家帝豪骗局

　　2011年7月，接到读者的反映，本报开始关注到家帝豪“消费养老计划”。

　　调查后发现，“消费养老模式”疑点重重：其网站上销售的诸多保健品没有“准生证”，“消费养老计划由中国保护消费者基金会信誉保障”并不属实；没有保险资质。而且本报调查后发现，其发展代理商的方式，直接或间接以发展人员的数量作为返利的依据，不断发展他人参与传销活动。

　　2012年，家帝豪公司又玩出新花样，疯狂招募“联盟商家”，参加者花10万元购买5台POS机，然后就能“躺在床上赚钱”，而且还能获赠期权，等家帝豪上市后就能大赚特赚。另外还推出了新型代理商模式，同样涉嫌“传销”。

　　在2012年3月份，记者多次暗访家帝豪公司，并参加了其举办的培训课程。家帝豪培训课极其狂热，上课前，“老师”会带领两三百人的听课者一起高呼“家帝豪好”，连呼多遍，再加上投资10万年赚115万元的金钱诱惑，不少人纷纷掏钱，尤其以中老年人居多。

　　在多次前往家帝豪暗访后，对方注意到了记者。2012年4月，记者以朋友的身份陪同一名不想再作为加盟商的投资客去家帝豪公司索讨款项。在这过程中，对方人员对记者拍照，并狠狠地说“我知道你是记者，我知道你是谁”。之后，110警察赶到现场，但是家帝豪人员的态度仍十分嚣张，称要将记者的照片在网上公布，让家帝豪“十万代理商”找记者算账……

　　家帝豪轰然倒塌

　　家帝豪蒙骗了不少人，但它的“商业神话”并没有持续多久，就在记者调查的同时，2012年4月，家帝豪东窗事发。

　　家帝豪山西的总代理在临汾、运城、太原等地招募代理商被当地警方盯上，警方将此称为“集资诈骗、组织领导传销案”。记者当时采访获悉，此案是由临汾公安局的一位副局长挂帅，成立了专案组调查，曹建华本人在2012年4月12日被临汾警方抓捕。

　　与此同时，临汾公安、工商部门赶赴上海，会同上海警方，对家帝豪公司进行了搜查，接到受害者投诉的上海警方也对此进行了立案调查。上海警方侦查后，将材料移交至检方。不过在2012年底，检方将此案退回警方，要求补充侦查。在2013年初，警方经过补充侦查后再将此案移送至黄浦区检察院。而曹建华在临汾关了半年后，被移交至上海警方，同时家帝豪也被临汾工商局罚款750余万。

　　2013年5月，上海法院对家帝豪案进行开庭审理，在去年12月份一审宣判。检方披露，家帝豪的收益实质上来源于加盟者缴纳的费用，而非正常的商品经营利润。而家帝豪收取的10.9亿资金大多用于曹建华等高管的个人花用、分成及代理商的返利。
