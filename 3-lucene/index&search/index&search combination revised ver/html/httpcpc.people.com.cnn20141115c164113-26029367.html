“习大大”加油网络留言板开通上线 多渠道接受网民留言

2014年11月15日09:22   来源：人民网－中国共产党新闻网

点击进入“习大大加油”网络留言板

中国共产党新闻网北京11月15日电　(李源)今天是新一届中央领导集体执政两周年的日子。十八大之后短短两年，以习近平同志为总书记的党中央统筹国内国际两个大局，统筹伟大事业伟大工程，用中国梦凝聚中国力量，以作风建设提振全党精气神。励精图治的改革举措，奋发进取的务实行动，开创了党和事业发展的新局面，开拓了治国理政的新境界，形成了全党全社会的新风气。

对习近平总书记，民间早已悄然称之为“习大大”。这个称呼从国内流传到海外，从民间流传到官方。在“习大大”履新两周年之际，人民网・中国共产党新闻网策划推出“习大大加油”大型网络留言板，网友可以通过强国论坛、人民微博、七一社区、手机人民网等多个入口参与留言，为“习大大”加油打气！表达期待！

・作为中国深化改革的“新设计师”，习大大掌舵，创造出一段段辉煌，为百姓所赞颂。得民心者得天下！听百姓心声的转播台，解决百姓疾苦的手术台，惩治腐败的审判台，传扬福音的扬声器。
