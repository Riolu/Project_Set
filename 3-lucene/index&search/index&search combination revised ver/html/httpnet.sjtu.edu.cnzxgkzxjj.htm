网络信息中心概况

       网络信息中心其前身上海交通大学计算中心，创建于1973年，是我国高校中建立最早、规模最大的计算中心之一。随着学校规模的扩大，于1987年在闵行校区设立了分中心。鉴于学校计算机网络建设和信息化建设的整体考虑，于1996年正式改建为网络信息中心。2004年7月，学校成立信息化办公室后，更名为网络中心。2006年12月，信息化办公室下设的信息中心并入网络中心，再次更名为网络信息中心。

       作为学校“数字大学”和信息化建设的核心部门，网络信息中心主要承担三方面工作：

       第一、负责中国教育和科研计算机网络(Cernet)华东南地区网、上海教育与科研计算机网(Shernet)和上海交通大学校园网(SJTUnet)的建设工作，是三个网络的运行和管理中心。

       第二、负责学校各类信息系统建设，以及管理校内单位代码和信息标准的制定。

       第三、负责学校高性能计算中心建设、运营及管理，面向学科与科研提供高性能计算服务。

       2009年3月，网络信息中心迎来一次发展的新机遇，位于闵行校区的图书信息楼竣工，建筑总面积4.4万平方米，网络信息中心部分建筑面积近9千平方米，楼内汇集了校园网、数据中心、托管、高性能计算等设备机房，设置了功能完备的运行监控中心和舒适的用户服务环境，为管理、开发人员提供了良好的工作环境，成为学校数字大学建设管理、运行服务、技术开发和研究试验的新基地。

      通过近年来211工程、985工程的建设实施，网络信息中心逐步建立了一支高水平的信息化技术团队，在包括下一代互联网关键技术、网络安全体系建设、基于SaaS的云计算技术、高性能计算等方向开展了深入研究并取得了佳绩，为学校信息化建设稳居国内高校前列发挥了重要作用。
