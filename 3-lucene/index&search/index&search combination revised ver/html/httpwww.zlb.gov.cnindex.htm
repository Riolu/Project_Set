“世界旅游经济论坛·澳门2015”开幕
      10月12日上午，“世界旅游经济论坛·澳门2015”开幕典礼在澳门威尼斯人会展中心举行。澳门中联办副主任姚坚与澳门特区行政长官崔世安、柬埔寨首相洪森及夫人文拉妮、全国政协副主席兼世界旅游经济论坛大会主席何厚铧等共同出席。 [详细]
中国银行澳门分行向澳门本地慈善捐款逾千万澳门元
      10月11日上午，中国银行澳门分行成功举办“根植澳门65载慈善行·中银与您同迈步”活动。澳门中联办副主任姚坚与澳门特区政府行政法务司司长陈海帆、外交部驻澳公署副特派员潘云东、澳门特区政府社会工作局局长容光耀、体育发展局代局长潘永权等共同出席。 [详细]
孙达会见国家旅游局局长李金早一行
      1日下午，澳门中联办副主任孙达在新竹苑会见来澳出席世界旅游经济论坛的国家旅游局局长李金早一行。孙副主任介绍了近期澳门经济社会发展的相关情况，并表示希望国家旅游局继续发挥“促进澳门世界旅游休闲中心建设联合工作委员会”机制的作用，进一步支持澳门世界旅游休闲中心建设，促进澳门经济适度多元发展。 [详细]

客观看待“一国两制”澳门实践
