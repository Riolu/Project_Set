培训班学员感言:突出理论 增加实务
2009年9月29日 15:35

　　尽管事先组长关照说希望能多提些意见，但是我还是想先说，自己在这次培训中受益匪浅。主要有以下三点：

　　首先是能够一次性听到这么多优秀的老师和学者的讲课是这次培训令我感到最兴奋的事，出乎我的预想之外。我今年刚从上海外国语大学新闻传播学院毕业，大学期间就陆续接触到很多复旦新闻学院老师的著作。一直希望有机会能来听，如果没有参加这次的培训，我想我也不可能有机会坐在复旦大学的校园内面对面地聆听这么多老师的授课了。一些复旦的同学告诉我，他们作为本科生很少有机会听到像李良荣等老师的讲课。所以能有这样的一次培训机会真的感到很幸运。

　　其次，这次培训的课程也十分多元化。既有偏理论性的课程，也有指导实际编辑运用的课程，既有宏观的国情市情舆情的介绍，也有微观的诸如广告、网络编辑等内容。让我们从多领域，多角度，多层次地对网络新闻与信息业务有更全面和深刻的了解。

　　从我的个人的角度而言，这次培训所传达的一些知识与我的工作职能非常贴近。比如东方网采编中心高国营总监讲解的网络编辑业务专题，对我们从事新闻编辑也好，传播信息、论坛发帖也好，有着非常直接的指导作用。

　　由于不少学员除了新闻编辑外，还负责网络的综合运营，论坛的管理等工作，我想在今后的培训课程中，可增加网络综合管理的培训，丰富培训的实务，让学员在实务方面多进行沟通交流，并有综合性的提高。

　　参加培训的第三点受益就是，能够借此机会与来自不同网站的同僚进行沟通，增进相互了解，相互学习。在沟通过程中，我也又在网上找到了很多好的网站。

　　另外，这次班级还分成小组进行互动讨论。我认为是个很好的形式。可以在今后继续深化完善。

　　还有一点建议。培训中，每堂课的时间都比较长，学员容易分散精力，错过精辟的讲解。老师也显得辛苦。可考虑增加并固定课堂中间休息的时间，提高培训的效率。

　　感谢我们小组的成员和组长邓叶峰，他们在讨论中都提供了很好的帮助。

                            新民网  徐靖
