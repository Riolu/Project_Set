微软校园软件正版化 / 常见问题

2010-12-03  点击：[]

问：教师帐号，但不能登录“下载列表”和“快速指南”，或者其他相关帐号的问题。
答：请致电：34206060-8303。  

问：IE6下载超过2G的文件有问题？
答：该问题已经解决。需要注意的是，IE6下载超过2G的文件时，显示的下载进度条和完成量不正常。
微软的解释：http://support.microsoft.com/kb/298618
建议使用IE7或者Firefox来浏览和下载。  

问：下载的ISO文件刻盘后，没法启动？
答：可能是如下的问题引起的，请参照检查：
1、BIOS中没有设置成光驱启动优先；
2、刻录的时候，没有使用镜像刻录的方式，而是采用文件刻录；
3、光驱有问题；
»应该选择镜像刻录方式，以nero为例
 
问：安装时找不到SATA硬盘。
答：1、如果你手头有SATA驱动，那么在系统安装提示“按F6加载第三方驱动”时，按下“F6”键，插入驱动软盘安装SATA驱动即可。
   2、如果没有SATA驱动，那么需要在BIOS设置中，将SATA硬盘设置成“兼容模式”（由于不同的BIOS有着不同的表述，我们没办法在这里详细描述，请谅解）。系统安装完成后，如果你又安装了SATA的驱动，那么也需要在BIOS中修改相应的设置，否则系统会崩溃。
