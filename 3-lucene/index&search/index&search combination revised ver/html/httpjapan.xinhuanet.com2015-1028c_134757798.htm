从头到脚 看智能穿戴如何将你全副武装

    新华网日本频道综合：如今，运动手环、智能手表等各类穿戴式产品已开始将我们从头到脚全副武装，智能body的时代已悄然来临。

    【头部】：

    1.智能头盔

    由以色列一家科技创业公司开发的LifeBEAM智能自行车头盔受到骑行爱好者们的追捧。戴上它，可以监测骑行者们的生命体征、心率、血流量、卡路里等指标。

    2.防身智能发卡

    发卡也能变智能？是秘密武器还是刑侦大片？日前，有国外厂商开发了一款女性专用的智能防身发卡， 戴上它，你就拥有了一个含监控摄像头和麦克风的秘密防身武器。

    这款发卡包含了蓝牙音频、视频和地理定位应用程序，可与智能手机同步。当佩戴者遇到麻烦时，First Sign的智能发卡也会自动提醒有关部门。
