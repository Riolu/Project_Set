前9月反腐成绩单:8省2.9万余人受处分 吉林最多

2015年10月28日11:07  来源：人民网  手机看新闻  字号

人民网北京10月28日电 进入十月，各地纪委陆续晒出了今年前三季度纪检监察机关案件查处情况。据不完全统计，目前至少已有吉林、贵州、广东、辽宁、甘肃、广西、北京、青海等8个省份公布了前9月反腐成绩单，共给予党纪政纪处分29389人，其中吉林处分6794人居首，同比增长近6成。

8省2.9万余人受党政纪处分 吉林最多同比增58.3%

据公开报道梳理，已公布前9个月纪检监察机关案件查处情况的8个省份，共有29389人受到党纪政纪处分。其中吉林处分人数最多，吉林各级纪检监察机关9个月立案7676件，同比增长73.4%；给予党纪政纪处分6794人，同比增长58.3%，且所立案件中，涉及了地厅级干部26件，同比增长116.7%。

其次是贵州和广东。两地今年前三季度均有超5000人受到党纪政纪处分。贵州9个月共立案6760件，党政纪处分6056人，移送司法机关579人，涉案金额5.78亿元；广东共立案9801件，同比增长32.5%，广东所立案件中涉及省管干部149人。

“拍蝇”力度不减 对“小官贪腐”问题零容忍

8省份9个月查处近3万人，彰显了自中纪委而下各级纪检监察机关“拍蝇”力度不减的成效。今年以来，各地纪委继续加大对群众身边不正之风和腐败问题的查处力度，对“小官贪腐”问题零容忍。

据通报，广东今年已排查出农村基层党员干部违纪违法线索11902条，查处农村基层案件6263件，增长41.1%。北京今年已查处“小官贪腐”329人。吉林前9月共查处发生在基层和群众身边的腐败问题7346件，同比增长70.88%。

北京去年中央巡视组在北京巡视期间，提出了“小官贪腐”的问题，今年初，北京市委常委、市纪委书记叶青纯表示，今后要将惩治“小官贪腐”作为长期工作来抓。贵州省纪委有关负责人近日也表示，各级纪检监察机关要持续强化对扶贫开发和各项惠民资金的监督，持续保持严查快处基层干部身上和群众身边腐败问题的高压态势。据悉，目前，多地纪委安排专项监察等对发生在基层和群众身边的腐败问题加以严查；多地采取公开通告等方式，敦促存在违纪违法问题的基层党员“主动投案”，“拍蝇”工作正紧锣密鼓继续中。
