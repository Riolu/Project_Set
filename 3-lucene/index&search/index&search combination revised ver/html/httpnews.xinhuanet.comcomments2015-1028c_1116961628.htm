9元机票是让利还是噱头？

2015年10月28日 10:29:41 来源： 长沙晚报
  • 评论0
  • 打印
  • 字大
  • 字小

    随着“东北最美高铁”的开通，长春到延吉乘坐高铁只需两个多小时，而且车隔差不多就半个点，这让“吃货”的天堂——延边一下子火了。“东北最美高铁”的热乎劲还没散去，南航吉林分公司又对外发出消息——延吉、长春开通“空中巴士”，机票价格仅260元。紧接着，九元航空宣布，将于10月29日正式执飞长春到长白山航线，最低票价仅为9元。航空公司在这时开通“空中巴士”和9元票价的航班，一场“空铁大战”难道即将上演？（10月27日《新文化报》）

    花9块钱就能坐飞机，这不是开玩笑吧？不是。那是不是脑筋急转弯，坐的玩具飞机？也不是，是真人坐真飞机，而且是从长春飞到长白山，路程数百公里。人们不禁要问，如此几近免费的价格，到底是真的向旅客让利，还是一个吸引眼球的噱头？

    说纯粹是噱头，恐不确切。据悉，该航线每个航班放10张以上9元票，这与时下一些楼盘“××元起”而起价实际仅有一套的促销来比，要厚道得多。但若说是实实在在的让利，恐怕也难让多数公众认同。毕竟这9元的价格低得太任性、太离谱，已远远低于成本价，显然已不是正常、理性的优惠促销。因此，9元票价的航班，实属介于噱头与让利之间的一种恶性竞争。由于这种竞争没有可持续性，虽有小部分消费者能从中获得一时之痛快，但从长远来说，并不那么乐观，甚至可能反受其害。

    低价旅游团的恶性竞争就是一个摆在眼前的例子。这种恶性竞争发展到极致，就出现了“零团费”旅行，游客不需交纳任何车费、住宿费、导游费、管理费，甚至不用交门票费，就能组团旅游。殊不知世上没有免费的午餐，低团费、“零团费”不过是一个引君入瓮的诱饵，其背后是引诱甚至逼迫游客购物。日前在香港，就发生了一起“零团费”带来的惨案。一名黑龙江游客因不满强迫购物而被殴打致死。昨日又有媒体报道，江西南昌的邓女士一行参加“零团费”港澳游，花了近12万元购买珠宝玉器等商品，而拿回南昌一检测竟然只值几百元。类似的乱象不胜枚举。

    在市场上，资本的本性就是逐利，即便是换了马甲，变了花样，这样的本性是不会改变的。既如此，资本如果突然无缘无故地做起赔本买卖，那就是一种反常行为，消费者别高兴得太早。就“零团费”和9元机票来说，旅行社与航空公司或许会称，这种离奇低价是有原因的，是一种营销策略，是应对竞争的结果。可是，竞争的终极目的，绝不是让自己亏本，而恰恰相反是为了赢利，为何要使此“苦肉计”？若是为了扩大客源，其方式方法有很多，比如提高自身服务质量、提供个性化服务等，为何却仅盯着价格战且将其推向极致？这背后的目的实在令人费思量。

    作为消费者，我们盼望着各类市场主体能不断展开白热化的竞争，以期获得最理想的服务、质量与价格。然而我们也深知，正常的、理性的价格竞争，应是循序渐进而不是一下子将底裤脱个精光，更不应该自己做亏本买卖。无论是高铁还是航空，我们既不希望看到价格垄断，铁板一块，死水一潭，也不希望看到一方故意亏血本这样的“苦肉计”。因为，鹬与蚌的“相争”游戏，死了谁都不再精彩。（袁云才）
