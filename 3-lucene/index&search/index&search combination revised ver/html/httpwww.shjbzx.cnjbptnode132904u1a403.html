建议与回复
2009年9月29日 15:36

　　市政府新闻办和复旦新闻学院开设的这个互联网新闻与信息业务培训班，使我们从业者既能学到新闻理论知识，又能听到网站工作实务讲解，而且它还为我们同行提供了一个交流的平台，对我们的网站日常工作很有用。如舆情分析、专题分析、危机公关、整合营销等课程，都是能立刻学以致用到工作中的。

　　不过因为参加培训的学员并不都是做网络新闻的，而且大多不是新闻科班出身，四个周末的课程内容密集，消化起来还是有难度的，但是课程都很经典和精彩，所以如果能增加新闻传播理论知识的课时，整个周期上能再适当延长一个周末或许会更好。

　　另外对于市政府新闻办还有几点建议：

　　1、培训课中除了基础的新闻理论、互联网发展趋势等课程要讲，因为很多学员不是做社会新闻的，如分类信息网站等，所以最好有加强从业者的政治敏感度的课程。

　　2、利用网络资源上开通一个网站信息即时交流平台，如，甲方网站正需要什么资源，乙方网站正在推出什么活动，丙方网站正在招聘什么职位，然后有兴趣合作的其他网站可以跟帖或站内消息直接联系，或者干脆建立个站内学员可以浏览的业内简历库。

　　3、因为现在非常多的网站并不属于政府、事业编制内，而这一批从业人员除了有考资格证书的需求外，也有继续深造读统考在职研究生的需求，希望新闻办和新闻学院也可以为培训班同学们提供这样的机会。

　　培训工作小组回复：

　　1、马克思主义新闻观是整个培训的基础和灵魂，我们每期都聘请复旦大学新闻学院的资深专家讲解，力争讲活讲透讲出新意，贴近工作实际，决不照本宣科人云亦云。

　　2、这个建议很好。网络交流平台一直在建设和完善中，也希望众多学员献计献策，共同把我们自己的园地经营好，我们已经在平台上开通了招聘频道，其他内容我们和会逐步实现。

　　3、办这个培训的目的有几个，一是从体制建设上，为互联网新闻与信息服务建立行业从业标准和准入制度，二是切实提高人员素质，三树立和巩固这个公共信息行业自己的专业主义追求。您所提的办在职研究生班的建议我们其实已经有所考虑，在探索、完善目前培训项目的基础上，我们会适时推出更好更深的培训。行业培训决需要有梯度推进，不能是一锤子买卖。
