四季度车市反弹在望 或现U型增长

2015年10月28日 08:51:34 来源： 北京青年报
  • 评论0
  • 打印
  • 字大
  • 字小

　　进入四季度以来，不管是产能，还是销量，中国汽车市场似乎走出了阴霾。有迹象表明，在上月底小排量汽车购置税的政策刺激下，加上适度宽松的货币环境，四季度车市有望触底反弹，中国车市全年有望实现7%的增长。

　　政策刺激下的北京车市

　　尽管在9月单月，北京新车销售3.67万辆，同比负增长14.45%，增幅低于全国16.55个百分点；环比负增长8.93%，增幅低于全国30.53个百分点，但是从季度数据来看，三季度新车销量整体有所回升，累计同比降幅从去年的-15.01%收窄到-6.62%。今年1-9月份，北京累计新车销售35.21万辆，同比去年39.53万辆，下滑10.93%，增幅低于全国11.23个百分点。分析认为，北京车市销售下滑的主因仍然是限购限行以及外地来京购车人员的减少。

　　进入四季度第一个月份10月，虽然来北辰亚运村汽车交易市场（简称亚市）的外埠进京旅游并购车人数绝对量有所减少，但仍然占车市客流的40%左右，基本保持着原有传统特征，是京城车市重要的购车群体。亚市十一期间销售数据显示：1.6L及以下进口车同比增长1倍，2.1L-2.5L排量车型同比增长了20%，3.1L以上排量进口车同比增长33%。国庆七天长假交易量累计同比增长15.83%，节日后期环比前期增长4.7%。

　　亚市副总经理颜景辉表示，受购置税减半这一利好政策的提振，从十一节后亚市的数据看，车市人气有所增加，销量同比正增长。

　　车企对四季度车市有信心

　　中国汽车工业协会的统计也印证了这一说法。9月单月，全国汽车产销量比上月较快增长，产量同比降幅趋缓，销量呈小幅增长。9月共销售汽车202.48万辆，自1月、3月后年内销量再次超过200万辆，环比增长21.60%，同比增长2.08%，结束几个月以来的下降趋势。其中乘用车销售175.12万辆，环比增长23.41%，同比增长3.26%。

　　在这一消息的提振下，全国汽车生产量也有所增加，9月份汽车生产189.43万辆，环比增长20.63%，其中乘用车生产了162.07万辆，环比增长20.75%。这一数据表明，企业对于四季度车市充满信心。

　　从9月份的乘用车销售数据看，除了SUV市场高达59.5%增长之外，微客、轿车和MPV仍然处于下滑态势，下滑幅度分别为36.4%、11.3%、5%。就在9月底的29日晚间，国务院有关支持小排量汽车发展的措施出台，从2015年10月1日到2016年12月31日，对购买1.6升及以下排量的乘用车实施减半征收车辆购置税的优惠政策。目前，车市64%的车型符合优惠条件，加上节能惠民补贴，部分车型最高可享逾万元政策福利，这一政策对于微客以及小排量轿车刺激作用可想而知。国家信息中心信息资源开发部刘明表示，参照2009-2011的车市走向，此次利好政策将使车市从L型增长转为U型增长。

　　车企要为销售终端解困

　　从全国乘联会发布的数据看，9月份全国广义乘用车产量达到163.3万辆，和8月份比较增长了20.3%。而销量数据显示，9月汽车销量为170.8万辆，同比增长了2.5%，环比增长18.3%。也就是说，经过整个夏季的产能调整，车企也寄希望于通过四季度的产能冲刺，实现全年产销的更大突破。从车型结构来看，9月份轿车和MPV销量环比分别增长了18.8%和10.9%，车市回暖迹象明显。

　　从车企的批发销量来看，9月乘用车批发量达到175.9万辆，同比和环比分别有3.7%和23%的增长，因此终端经销商的销售压力在四季度会进一步扩大，这也将促使车企在年底推出更为实惠的购车优惠政策，加上购置税减半政策，车市利好在四季度会不间断，消费者买车完全可以选择在这个年底时节。据进口车联席会相关统计，9月份仅进口车平均优惠超过9 万，优惠幅度达到14.2%，达到历史新高，预计年底之前这一趋势将持续。

　　全国乘用车市场信息联席会秘书长崔东树认为，9月车市零售企稳回升到4.8%，好于预期。9月份是三季度末，厂家季度末的销量冲刺意愿较强，购置税减半政策刺激车市，激发了厂商的市场信心。　(记者 何登峰)
