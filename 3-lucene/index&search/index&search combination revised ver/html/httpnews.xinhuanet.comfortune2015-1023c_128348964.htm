证金系组团“包场”多家公司股东榜

2015年10月23日 07:27:21 来源： 上海证券报
  • 评论0
  • 打印
  • 字大
  • 字小

　　随着上市公司三季报披露进入密集期，在市场上举足轻重的“证金系”的持股情况进一步浮出水面：此前较为罕见的证金系账户组团“包场”上市公司股东榜的情况开始集中出现，而除创业板、中小板的中小市值公司外，大蓝筹中也出现了证金系账户高比例持股的现象。

　　在今日披露三季报的上市公司中，华北制药、良信电器、长盈精密、中国联通等公司的股东榜变化尤其显著，证金系账户几乎组团“包场”这些公司的股东榜单，且其账户组合计持股比例也均在上市公司总股本的10%以上。这些都是在此前披露的三季报中较为罕见的。

　　具体来看，长盈精密流通股东榜由证金系账户组占据八席，合计持股比例高达14.54%；华北制药截至三季度末的股东榜中，“证金-汇金”账户组囊括了除第一、第二大股东外的十一席，合计持股比例13.74%；良信电器的流通股东榜则由证金系账户组占据九席，合计持有良信电器总股本的13.52%；大蓝筹中国联通的股东榜上，除大股东外，证金系账户组占据十二席，合计持股比例高达10.69%。

　　此外，友阿股份股东榜上，证金系账户组占据八席，合计持股9.29%。而王子新材、贵州茅台、广汇能源、阳泉煤业、上海凯宝、金杯电工、信隆实业、沧州明珠、珠江啤酒等多家上市公司股东榜也有证金系账户现身。

　　在股东榜被证金系账户组“包场”的上市公司中，有的已在前期A股市场大跌时通过龙虎榜数据透露端倪：如良信电器7月8日龙虎榜显示，中信证券北京金融大街营业部和望京营业部当天分别买入8.53亿元和8063万元，合计占到当日该股总成交额的94%；而中信证券北京总部营业部也在7月6日起就开始大笔买入华北制药，登上龙虎榜。

　　此前，部分上市公司主动公告披露证金系持股，曾引发市场对证金公司投资偏好的猜测，如今随着三季报披露的深入，可以清晰地看到，证金系虽然“雨露均沾”，建仓的上市公司数量较多，但投入重金的还是中国联通、贵州茅台之类的大蓝筹、白马股以及业绩稳定的创业板优质公司。

　　如证金系持股比例达11.7%的创业板上市公司长盈精密，三季度业绩就比较亮眼。据披露，今年1至9月，公司实现营业收入27.43亿元，与上年同期相比增长84.92%，实现归属于上市公司所有者的净利润为3.22亿元，与上年同期相比增长了62.87%。细查股东榜，证金系合计持股比例高达14.54%；其中，汇金公司持股2576.43万股，为第二大股东；七只中证金融资管计划分别持股794.24万股，同列第四大流通股东。（赵晓琳）
