网络文艺，当如春日清风
2015年10月19日 11:07

习近平总书记刚刚发表的《在文艺工作座谈会上的讲话》中指出，要适应形势发展，抓好网络文艺创作生产，加强正面引导力度。中央政治局前不久审议通过《关于繁荣发展社会主义文艺的意见》，也特地强调了“大力发展网络文艺”，引起广泛关注。

从痞子蔡的《第一次亲密接触》，到《甄执》《花千骨》等网络文学被改编成影视作品；从《东北人都是活雷锋》，到《两只蝴蝶》《小苹果》等网络歌曲走红……伴随中国网民规模达到6.68亿新高，网络文艺迎来了自己的“黄金时代”。在十几年的时间里，网络文艺以其海量的作品规模、广泛的受众群体、持续的舆论影响，形成巨大的文化力量。

网络文艺顺应了文艺大众化的趋势，激活了大众的文艺潜能，造就了“人人都是创作者”的新格局，形成了网络文艺的鲜明特点：门槛低，易发表，易传播，群体大，市场化。但是也暴露出一些问题，如原创剽窃、盗版严重，作品中炫富、拜金大行其道，暴力、色情渲染屡禁不止等等。可以说，那些将低俗等同于通俗、将欲望等同于希望、将单纯感官娱乐等同于精神快乐的网络作品，是文艺百花园中的毒株败草。

对此，我们必须有清醒的认识。文艺是民族精神的火炬，是时代前进的号角。不能让网络文艺成为唯市场而动、哗一时之宠的文艺“商品”，不要让网络文艺成为唯点击率马首是瞻的廉价玩物，而要不断提升网络文艺的质量，为整个中国文艺的繁荣贡献力量，为实现中国梦提供健康精美的精神食粮。

提升网络文艺的质量，不能仅靠某个部门单一的力量，也不可能一蹴而就。既需要探索建立符合网络文艺发展规律的管理、创新、发展机制，为繁荣网络文艺创造理想的环境；又需要所有网络文艺创作者，尤其是那些思想精英、文艺精英加入，并牢固坚持以人民为中心的创作导向，为历史存正气，为世人弘美德，聚焦中国梦的时代主题，唱响爱国主义主旋律，传承和弘扬中华优秀传统文化；还需要所有网民一起打造文明的网络文艺生态，不管是阅读网络作品、观看网络影视，还是点赞转发文艺作品、创作经营网络作品，我们都需要传递真善美，传递向上向善的价值观，彰显信仰之美、崇高之美，让网络文艺的活力之花盛情绽放。

每个时代都有每个时代的精神。举精神旗帜、立精神支柱、建精神家园，是当代中国文艺的崇高使命，网络文艺则是不可或缺的重要支撑。唯有扫除颓废萎靡之风，网络文艺才能像春天里的清风一样，温润心灵、陶冶人生，才能在思想上、艺术上取得成功，又能在市场上受到欢迎。
