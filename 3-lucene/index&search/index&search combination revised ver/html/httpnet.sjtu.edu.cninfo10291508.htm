黑莓连接学校无线网络

黑莓型号：8520
信息如下：

首先要安装“上海交通大学网络信息中心数字根证书”
到http://wlan.sjtu.edu.cn上下载，首页上有下载链接。

解开后，选择其中的sjturootpda.cer，复制到手机存储卡的某个目录
如下图：


选中证书文件，按“确认键”，会弹出安装证书的对话框，按部就班操作即可。
没抓图，要删了证书再重装一遍，偷懒了。

安装完成后，再选中证书文件，按“确认键”，会显示证书信息。如下图

 

开始配置无线链接。进入“管理连接”


选择“设置WLAN网络”，进入，选择“手动添加网络”，如下图

键入SSID，如下图：


注意，上海交通大学统一认证的新无线网络名称为：SJTU


选择“安全类型”为：PEAP
输入“用户名”和“用户密码”
在“CA证书”中，选择：SJTU.EDU.CN CA Root Certificate
在“内部链接安全性”中，选择：EAP-MS-CHAR v2
其他的不用管，如下图

 


选择“保存”，会询问是否保存为配置文件。

连接后，就会看到在“中国移动”边上，出现我们保存的“配置文件名”(下图红色划线所示)。

完成！
