    新华社社长蔡名照会见墨西哥通讯社社长拉莫斯

    2015年09月23日 09:48:18 来源： 新华网

    9月22日，新华社社长蔡名照在北京会见墨西哥通讯社社长亚历杭德罗·拉莫斯。 新华社发（和静远 摄）

    新华网北京9月23日电 新华社社长蔡名照22日在北京会见了墨西哥通讯社社长亚历杭德罗·拉莫斯。

    蔡名照说，中墨虽相距遥远，但两国友谊源远流长。新闻媒体肩负着增进两国之间友好交往的重要使命，中墨媒体合作应与两国关系发展趋势相适应。新华社高度重视对墨西哥及拉美地区报道，愿同墨通社一道，在过去30多年密切合作的基础上，进一步加强各领域友好合作，相互学习借鉴新闻业务发展经验，为促进两国人民之间的相互了解和友好交流作出贡献。

    拉莫斯表示，很高兴来新华社访问，新华社在新媒体领域积极有为给人留下深刻印象。墨通社多年来始终同新华社保持密切合作，愿进一步加强在传播国家形象、应对新技术挑战等方面的合作，为增进墨中友好作出新贡献。
