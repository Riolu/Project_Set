长春8万名公办教师录入“影像识别管理系统”严查“乱补课”
 
2015年10月28日 05:31:49 | 责任编辑: 冯文雅 | 来源： 新华网   

　　 10月27日，工作人员在通过“中小学教师影像识别管理系统”进行查询。自2015年7月份开始，长春市教育局针对公办中小学校和在职教师“乱办班、乱补课”现象开展了专项治理行动，并将全市2000多所中小学校的近8万名公办教师照片录入“长春市中小学教师影像识别管理系统”数据库，便于督查人员辨识各类办学机构中任教教师的身份。目前，长春市教育局共接到“乱办班、乱补课”现象举报300余件，解散“黑班级”100余个，查处违纪教师47名。 新华社记者 许畅 摄
