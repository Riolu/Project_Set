    卡宾先生：过去的经历为今天的创作带来帮助

    2015年10月26日 17:03:31 来源： 新华网
    分享到：

        10月25日，中国国师时装在京举行期间，设计师杨紫明(卡宾先生)接受新华时尚专访。

        曾经是知名拳击手、赛车手的卡宾先生杨紫明如今已经是享有盛名的服装设计师。在接受新华网采访时，杨先生表示这种跨界转型的最重要的是保持一种对生活的激情和对未知事物的一种新鲜感和好奇心。他在采访中说道：“也可能是我过去有很多很多的经历给今天的我、今天的创作带来了很多的帮助。比如说在我的品牌的DNA里面你会看到有勇敢、激情、快乐、自由等等，这一切都是我个人的性格。”他认为在创作的道路上虽然有很多煎熬，甚至是迷茫，但是当他能解开这些、有个新的作品的时候他会觉得非常开心，他认为这种种享受能抵掉一切的艰辛。
