人民日报社社长杨振武（右）会见英国驻华大使吴百纳。（摄影：何萌）

    人民网北京6月24日电（记者 周卓斌）今天上午，人民日报社社长杨振武在报社会见了英国新任驻华大使吴百纳。双方就中英关系及进一步加强两国媒体交流等共同关心的话题交换了意见。人民日报社副总编辑阎晓明、秘书长王一彪陪同会见。
