新华网对话罗杰斯: 论道中国金融市场碰撞智慧火花

10月27日下午，由新华网、纽约金融学院主办，新华网河南分公司承办，国泰宝景基金协办的“2015新华网高端金融峰会暨对话罗杰斯——论道中国金融市场”活动在郑州举行，被誉为“华尔街神话”的国际投资大师罗杰斯莅临现场并发表主题演讲，与专家学者、企业家面对面对话，把脉中国经济现状，论道中国金融市场，分享了他对投资、生活以及家庭的独到见解。

金秋十月，华尔街投资大师罗杰斯要来郑州了，参加由新华网、纽约金融学院联合主办的“2015新华网高端金融峰会暨《对话罗杰斯——论道中国金融市场》”活动。届时，众多投资者将与投资大师面对面，倾听他对于中国金融市场的精彩论述。

嘉宾介绍

许思涛

许思涛现任德勤中国首席经济学家及合伙人。他曾先后任职于多家业界领先机构。曾驻新加坡在标准普尔集团所属 的 MMS 国际担任新兴亚洲经济学家，负责对韩国、印尼、泰国的经济研究(1995-1996)。之后驻香港，在渣打银行任东北亚地区经济学家(1996-2000)， 后来又在法国兴业银行担任首席亚洲经济学家(2000-2002)。许先生曾出任中国工商银行(亚洲)经济主 管(2003-2004) ，负责中国最大银行的海外旗舰的宏观经济研究。

王福重

中央财经大学教授，中国著名经济学家。 主要研究领域为经济学和金融学基础理论，在一流学术期刊发表论文100余篇。主要著作有《纯粹经济学》《金融的解释》《公平中国》《写给中国人的经济学》《人人都爱经济学》等。主持多项国家和省部级科研课题。

精彩观点

罗杰斯：中国股市还没有泡沫，但有形成泡沫的危险

对于公众关心的中国股市是否存在泡沫的话题，罗杰斯也分享了自己的判断，“现在还没有泡沫，但是有泡沫的可能。”但同时他也提出应尽量避免出现单边的直线上升，否则就会形成泡沫。“我希望整个市场能够冷静一段时间，有一个慢牛的行情，这样牛市的行情会更加长久。”

投资大师吉姆·罗杰斯：与新华网已是老朋友了

正在福州参加2015年首届互联网金融大会暨新华网思客会的国际著名投资家和金融学教授吉姆 罗杰斯莅临新华网福建分公司参观访问，并与新华网董事长、总裁田舒斌就互联网金融、新华网的发展等话题进行亲切交流。其间，罗杰斯谈到，他与新华网已是老朋友了。他说，其实他从30多年前来中国就知道新华社，了解新华社的影响力，当然那时候还没有新华网。现在新华网也名扬海内外，希望可以越办越好。

投资大师吉姆·罗杰斯上海解密“投资那些事”

他表示，从1993年到2015年，他给予中国很大的关注，并在中国市场进行了很多大笔投资。他希望，中国和美国的资本市场能够在未来进行更为深度的结合。 　　谈到他在中国资本市场投资如何进行行业选择时，罗杰斯说他在中国A股市场的主要投资集中在铁路、医药、房地产、旅游和农业等领域。

吉姆·罗杰斯:等房价降下来才会考虑到中国投资房地产

罗杰斯先生还认为，在投资中国方面，他表示将会非常关注中国政府的宏观调控。在他看来，中国的农业、铁路、能源等方面是比较有前景的投资行业。同时还对贵金属、股票、证券、美元等金融投资业内一些自己的见解和投资的方向。
