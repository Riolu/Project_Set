    是谁见不得南海风平浪静

    2015年10月28日 12:50:45 来源： 解放军报

    　　10月27日，美军“拉森”号导弹驱逐舰打着所谓“航行自由”旗号驶入中国南沙群岛有关岛礁近岸水域。美方这一炫耀武力的行动，引起国际舆论一片哗然，更引发中国民众强烈愤慨。中国海军依法对美舰进行了跟踪、监视和警告。

    　　中国对南沙群岛及其附近海域拥有无可争辩的主权，这有着充足的历史和法理依据。中国在自己的领土上开展建设，是主权范围内的事情，合情、合理、合法。中方已多次申明，南海岛礁建设是出于和平目的，不会对各国依国际法在南海享有的航行和飞越自由造成任何影响。近日，中国南沙华阳和赤瓜两座灯塔的建成，可为航经该水域的各国船舶提供航路指引、安全信息、应急救助等公益服务，大大改善了南海水域通航环境。同时，中国是海上通道安全最大的利益相关方，海上通道的畅通与安全，最符合中国的利益。美国无理指责中国破坏“航行自由”，罔顾事实，罔顾中方的严正表态，实属充满主观臆测的霸权行径。

    　　近年来，南海局势保持了总体稳定，世界多国均从中受益。尤其是近段时间，中国和东盟各国在“南海行为准则”等领域的协商取得积极进展。然而，南海的稳定俨然成为美方的一块“心病”。美方不久前刚刚表态要在南海等问题上“开展良性互动”，随即又派出军舰，放着宽阔的国际航道不走，专门绕道进入中方驻守岛礁近岸水域炫耀武力，进行挑衅，对中国国家安全特别是岛礁设施和驻守人员安全构成严重威胁。我们不禁要问，美方这种滥用“航行自由”、意图制造南海地区军事化的胁迫行为，究竟是为了维护地区稳定，还是为了破坏地区稳定？究竟是谁，见不得南海风平浪静？

    　　美方炫耀武力的行径，再次折射出其根深蒂固的冷战思维。从阿富汗到伊拉克，从利比亚到叙利亚，铁一般的事实证明，美国一再轻言动武、一再轻启战事，只会把当地原本稳定的局势搅乱，让地区直接相关国家承受最沉痛的伤害。近年来，面对中国的快速发展，美国一些政客表现出某种心理失衡，丧失了处理对华关系时应有的理性和持重。在南海问题上，美方一再声称要为地区盟友承担所谓“义务”，然而其行为非但没能缓和局势，在盟友面前博取“面子”，反而进一步恶化了地区安全环境，损害地区国家利益，暴露出其无理、霸道和蛮横。

    　　南海诸岛自古以来就是中国领土，这是老祖宗留下的。任何人要侵犯中国的主权和相关权益，中国人民决不会答应，中国军队决不会答应。中国军队维护领土主权和海洋权益的决心坚定不移，我们将采取一切必要措施维护自身安全。

    　　在全球化进程日益深化的今天，国际社会已然成为你中有我、我中有你的命运共同体，中美两国概莫能外。美方此番挑衅行为，罔顾国际大势，破坏两国友好的民意基础。作为世界上最大的发展中国家和发达国家，中美关系稳定与否将深刻影响国际和地区局势走向。奉劝美国以两国关系大局和地区和平稳定为重，恪守承诺、纠正错误，不要进一步侵害地区国家利益，扮演南海和平的“搅局者”。（钧保言）
