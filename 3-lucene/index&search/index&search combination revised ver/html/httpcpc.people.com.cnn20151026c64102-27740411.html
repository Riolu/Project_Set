李鸿忠：强化管党治党责任 从严管好一把手

 

2015年10月26日11:03   来源：湖北日报

原标题：强化管党治党责任 从严管好一把手 时刻保持清醒头脑 严守党规党纪底线

23日，省委书记李鸿忠主持召开省委常委会会议，传达中央有关通报精神，研究深入落实全面从严治党要求、加强对全省各级领导干部特别是“一把手”的教育、监督和管理；学习新修订的《中国共产党廉洁自律准则》、《中国共产党纪律处分条例》；传达学习中央近期召开的第五届全国道德模范座谈会、党委（党组）意识形态工作责任制座谈会、繁荣发展社会主义文艺推进会精神，研究我省贯彻落实的意见。

会议认为，从严治党，关键要从严治吏。从严治吏，关键要加强对领导干部特别是“一把手”的监督和管理。“一把手”在一个领导班子和一个地方工作中处于核心地位，起着关键作用，负有全面责任，对政治生态具有源头性、导向性、引领性作用。“一把手”如果自身过硬，在“清廉为官、事业有为”上以身作则、率先垂范，在履行管党治党政治责任、党风廉政建设主体责任上始终保持清醒头脑，勇于担当、敢抓敢管，就能够引领构建风清气正、凝心聚力、共谋发展的良好政治生态。相反，“一把手”如果自律不严，不讲政治纪律、政治规矩，甚至滥用权力，违法乱纪，自身问题严重，就必然带坏一批干部，败坏一方风气，形成系统性塌方式腐败；抑或“一把手”仅仅做到自律自清，但不能认真履行管党治党的责任，不愿得罪人、只当“老好人”，也会导致“劣币驱良币”现象，形成政治生态的“破窗效应”，助长不良风气扩散蔓延。因此，落实全面从严治党要求，必须扭住党委主体责任这个“牛鼻子”，抓住“一把手”这个重点和关键。

会议指出，要以学习贯彻中央通报精神为契机，进一步加强对全省各级领导干部特别是“一把手”的警示教育，使其正确认识自己，时刻保持自警自省，真正把理想信念宗旨这个核心价值观作为“高线”、坚决守住党的纪律这条“底线”。要进一步强化党委（党组）“一把手”管党治党、抓党风廉政建设的“第一责任”，在继续保持反腐倡廉高压态势的同时，坚持把纪律挺在前面，加大监督执纪问责力度，倒逼“一把手”既严格自律、管好自己，又敢抓敢管、善抓善管，切实建好班子、管好干部、带好队伍。要进一步加强领导干部队伍建设，把选优配强“一把手”作为重中之重，着力选拔政治强、能力优、作风实、风气正、敢担当的干部担任“一把手”，为落实全面从严治党、推进“四个全面”战略布局实施提供坚强组织保证。

会议认为，中央印发新修订的《中国共产党廉洁自律准则》、《中国共产党纪律处分条例》，是党内政治生活中的一件大事，是党中央在新的历史条件下加强党风廉政建设和党的纪律建设的重大举措，对于推进全面从严治党、永葆党的先进性和纯洁性，具有十分重大而深远的意义。全省各级党组织和广大共产党员要认真学习这两个党内法规，坚决贯彻执行。

各级党委要担当和落实好全面从严治党的主体责任，切实抓好《准则》和《条例》学习宣传、贯彻落实，把党规党纪刻印在全省广大党员特别是党员领导干部的心上，做到内化于心、外化于行。各级领导干部特别是主要负责同志要发挥表率作用，以更高更严的要求，带头维护党规党纪的严肃性和权威性，敢于担当、敢于较真、敢于斗争，保证把纪律和各项党内法规执行到位。各级纪委要坚持把纪律挺在前面，全面履行监督执纪问责职责，加大查处违反《准则》和《条例》行为的力度，进一步探索建立不敢腐、不能腐、不想腐的有效机制。

会议强调，中央近期分别召开第五届全国道德模范座谈会、党委（党组）意识形态工作责任制座谈会、繁荣发展社会主义文艺推进会精神，充分体现了对深入践行社会主义核心价值观、加强社会主义思想道德建设，抓好意识形态工作和社会主义文艺繁荣发展的高度重视。全省各级党委（党组）和相关部门要结合工作实际，认真抓好会议精神的学习贯彻。特别要进一步加大对第五届全国道德模范的宣传力度，以多种形式，大张旗鼓、深入广泛宣传他们的先进事迹、高尚品质和崇高精神，建立健全褒扬激励制度，树立“好人有好报”的价值导向，营造崇德向善的良好社会环境和浓厚氛围；同时要加大对歪风邪气和失德行为的谴责抨击力度，适时开设“道德法庭”，抑浊扬清、惩恶扬善，努力促进良好社会风气形成。要进一步强化各级党委（党组）抓意识形态工作的主体责任、“一把手”的第一责任和分管领导的直接责任，狠抓责任制落实，严格责任追究，倒逼全省各级党委（党组）特别是主要负责同志勇于担当，敢抓善管，牢牢掌握意识形态工作的主导权。要深入学习贯彻习近平总书记在文艺工作座谈会上的重要讲话精神，认真贯彻落实党中央关于繁荣发展社会主义文艺的意见，进一步提高思想认识，强化文化担当，重点抓好我省文艺队伍建设、文艺院团改革发展、文艺精品创作，巩固我省文艺繁荣发展的大好局面。

会议还研究了其他事项。（记者黄俊华）
