习近平三天三提中美网络合作 互联网巨头点赞
2015年9月28日 12:41

中国国家主席习近平美国当地时间23日会见出席中美互联网论坛代表时表示，中美都是网络大国，双方拥有重要共同利益和合作空间，应就网络问题开展建设性对话，打造中美合作的亮点。

这是他三天来第三次强调在网络问题上应开展对话与合作。一天前，习近平在西雅图市发表演讲时强调，中国是网络安全的坚定维护者，愿同美国建立共同打击网络犯罪高级别联合对话机制。

启程赴美访问前，北京时间9月22日，习近平在接受《华尔街日报》书面采访时亦表示，中国互联网蓬勃发展，为各国企业提供了广阔的发展机遇和市场空间。中方欢迎外国企业在华发展，尊重和保护外国企业合法利益。中美双方在网络安全上有共同关切，愿同美方加强合作。

中外互联网巨头认为，习近平就互联网治理密集发声，传递出对外开放的坚定决心。

“我一直在读他(习近平)的著作，我知道他今天所讲的与此前的思想都是一致的，那就是中国会坚定不移地对外开放。”Facebook(脸谱)创始人扎克伯格表示，习近平的讲话“展现了一贯的立场和态度，也给美国的互联网企业以信心”。

在亚马逊总裁贝索斯看来，习近平强调要开放市场，推动中美双方企业平等进入“体现了大国的风范”。他表示，现在的关键是双方要拿出实际行动，推动这些原则和理念成为现实。

实际上，近年来，中国在信息网络技术产品、应用服务等相关领域已经进一步开放。中国已是美国信息技术网络产品的最大用户，也是美国高科技网络公司的主要出口市场。

习近平的讲话亦被视为向美国递出了加强沟通与合作的“橄榄枝”。

目前，中美网络关系正处于全面磨合期，在网络空间主权、网络自由等一些基本原则上达成了共识，但在网络攻击、网络监听等部分领域还存在分歧。

值得注意的是，近年来“中国网络间谍”、“中国黑客威胁”等言论在美国频频被热炒，甚至一度传出“美国欲对中国实施网络报复”的消息。

在此情况下，中美两国政府、企业和其他利益相关方以理智务实的态度加强沟通与合作，无疑符合双方共同利益。

阿里巴巴董事局主席马云注意到习近平强调了沟通的重要性。“中美之间需要信任和尊重，而沟通正是实现目标的路径。”

360公司董事长周鸿t表示，如果没有合作机制解决网络安全问题，中美两国都将成为受害者。“希望借此机会在中美两国互联网公司之间建立一定的合作机制，这是非常好的契机。”

英特尔CEO科再奇亦表示，中美双方不应该互相指责，而应积极开展合作，“这样会让事情朝好的方向发展”。
