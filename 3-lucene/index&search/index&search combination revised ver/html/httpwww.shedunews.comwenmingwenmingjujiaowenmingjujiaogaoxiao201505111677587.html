90后大学生在线调查：母亲节最好礼物是“陪伴”

2015-05-11 10:36:43    来源：上海文明网   

昨天是西方的母亲节。在人人网发起的在线调查中，2452名90后大学生表达了自己的想法。65.1%的受访者称知道母亲节在每年5月的第二个星期日。同时，知道自己母亲生日的比例更高，达75.2%。
　　调查显示：50.8%的90后称自己从上大学开始就想到要报答母亲养育之恩；从高中阶段就懂得报答母亲的有29.5%；自称还没意识到要报答母亲的则占19.7%。至于母亲节送礼的问题，表示会送的占23.2%。受访者普遍认为，给母亲最好的礼物其实是更多的陪伴和关心，每天会给妈妈打电话的占38.5%，一周给妈妈打一次电话的占20.4%。
　　此外，受访者自我评价显示，超过一半的90后认为现在年轻人比较孝敬父母；认为当下年轻人不孝敬的占10.7%。
