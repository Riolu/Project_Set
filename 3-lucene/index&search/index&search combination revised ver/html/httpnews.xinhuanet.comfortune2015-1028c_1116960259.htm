互联网缩招裁员？BAT均否认

2015年10月28日 09:09:57 来源： 解放日报
  • 评论0
  • 打印

　　业内人士认为盲目扩张之后，互联网行业需调整人才结构

　　近期，全球互联网企业开启缩招减员潮的消息不绝于耳，不仅国内互联网三巨头百度、阿里巴巴和腾讯（简称BAT）被传缩减校园招聘或社会招聘人数，英特尔、微软、联想、凤凰网、推特等境内外互联网或科技公司都传出裁员的消息。

　　这些消息是否属实？记者向BAT等互联网公司求证得到的答案是“否认减招”，不过这些公司也强调，要招聘真正有价值的人。业内人士分析，在“互联网+”热潮中，确实存在企业盲目扩张、资本一味求大的现象，只是互联网行业的健康发展不可能依靠员工规模、融资额等数字，需要更加稳健的人才结构。

　　BAT校招不缩量但有新要求

　　就在外界认为BAT将减少招聘时，百度CEO李彦宏10月26日晚上亮相复旦大学，为今年校招站台。他承认，将暂停一段时间的社会招聘，“因为之前招的人有点多，现在想稍微消化一下，校招的员工明年才入职，不会有影响。”

　　阿里巴巴和腾讯也都表示对招聘有新要求。阿里巴巴校招团队负责人曲洋介绍说，今年各岗位将执行更加严格的择优录取标准，目前总共发放1407份校招录用意向书，加上正在进行的实习生转正及专场面试，还将陆续发出数百份录用通知。

　　腾讯则表示，从人数看，今年社招和校招的数量不减反增，其中社招目前大约有1000个岗位仍在公开招聘，比去年增加40%，校招总共发出2000份录用通知书，与原计划持平。

　　互联网人才结构头重脚轻

　　业内人士认为，此前资本蜂拥进入互联网行业，导致互联网行业出现过度扩张和人才结构性失衡的问题。

　　一方面，原本一些“小而美”的互联网公司异常膨胀。记者在一家新兴互联网公司看到，公司还处在“烧钱”阶段，但员工数量在年初就已超过3000人，目前更是突破一万人。就连该公司的员工也悄悄抱怨公司扩张太快，原本高效的运转模式已不见踪影，一件小事要向多层领导汇报。

　　另一方面，互联网公司为招揽人才给出的“高帽子”，让行业缺乏能做实事的人才。知名职业社交网站领英发布《中国互联网人才库报告》显示，目前互联网人才市场最主要的问题在于“将多兵少”：初级职位占约43%，但高级专业人员占21%，经理占18%，总监、副总裁、企业主等决策层占18%。也就是说，高级及以上职级级别的人员比初级职位高出14个百分点。“头重脚轻”的人才结构并不利于行业发展。

　　提高用人效率成行业共识

　　互联网公司调整招聘节奏，是否与外界所说的行业泡沫即将破裂或资本寒冬有关？

　　互联网人才专业招聘网站拉勾网联合创始人鲍艾乐认为，互联网企业调整招聘结构，与行业寒冬或资本寒冬没有太大关系。从人才本身来看，只有优秀的人才才会紧俏抢手。

　　从各家互联网巨头的策略调整看，提高用人效率已成共识。这也透露出一个信号：当疯狂资本撤退的时候，企业只有靠真正的人才才能继续发展。

　　曲洋承认，眼下行业发展迅速，企业对人才的需求要重新认识，如果说以前是择优，那现在就是“优中取优”。

　　领英中国相关负责人指出，资本寒冬也许会减缓整个行业的发展步伐，但互联网前景并非黯淡。最新发布的《互联网行业人才吸引力指数》显示，互联网行业对人才的吸引力上升势头迅猛，吸引力指数更是超过金融业排名所有行业的第一位。随着用人单位对人才要求的提升，拥有计算机、工商管理、语言、经济等跨专业能力的人才更受欢迎。（记者　任翀）
