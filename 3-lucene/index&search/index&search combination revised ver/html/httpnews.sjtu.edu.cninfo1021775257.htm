上海交大举办中国城市治理论坛[图]
成立城市治理研究院 助力国家治理现代化
[发布时间]: 2015年10月26日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 沈崴奕 孙佳

10月23日至24日，中国城市治理论坛在上海交大徐汇校区隆重召开。本次论坛以“全球化背景下的城市治理与改革创新”为主题，由中国城市科学研究会、中共中央编译局比较政治与经济研究中心、上海市政府发展研究中心和上海交通大学国际与公共事务学院共同主办。 

国务院参事、中国城市科学研究会理事长仇保兴教授，中国行政管理学会执行副会长兼秘书长高小平研究员，中国政治学学会副会长、上海政治学学会会长桑玉成教授，中央编译局比较政治与经济研究中心主任何增科研究员，上海市政府发展研究中心副主任严军应邀出席。上海交通大学党委常委、组织部部长顾锋教授代表学校致欢迎词，上海交通大学国际与公共事务学院院长钟杨教授主持论坛开幕式。共有来自清华大学、中国人民大学、上海交通大学、复旦大学、同济大学、上海社科院、华东师范大学、上海财经大学、浙江大学等50余所高校的150余位领导、专家和学者出席论坛。  

城市发展面临诸多挑战专家学者云集共商对策

顾锋首先代表学校向出席“中国城市公共治理论坛”的领导和专家表示欢迎，他提到，上海交大始终以民族振兴和国家强盛为己任，主动对接国家战略需求。国务学院将城市治理作为学科发展的主要方向并成立“城市公共治理研究院”，旨在为推进我国新型城市化建设提供智力支撑和决策咨询。希望与会者能够交流思想、分享经验、形成共识，为促进我国城市发展做出更大贡献和努力。 

仇保兴教授代表中国城市科学研究会作开幕致辞，他提出，城市化速度快于政府职能转变速度，城市本身具有复杂性和脆弱性，加上信息化快速发展的叠加，这都对政府的城市治理提出了更高要求，他鼓励专家学者开展更广、更深的城市科学研究，进而使理论界和实务届共同应对城市治理的机遇和挑战。  

高小平研究员指出，城市是经济、人口和建筑要素聚集的结果。我国当前的城市治理存在法治性、公共性和社会参与亟待进一步提高的问题，深入探讨城市治理的客观规律和政府管理创新的未来模式，具有重要的现实意义。  

桑玉成教授在致辞中指出，我们应在当今国家社会的大背景下探讨城市治理问题，大背景包括两个“百年愿景”、“国家治理体系和治理能力现代化”和全面依法治国。城市治理有赖于政治、文化、技术和管理四个要素的平衡。  

何增科研究员代表中央编译局在致辞中提到，城市治理已成为推进国家治理能力现代化的重要因素之一。依托上海城市发展经验的丰富宝藏，上海交大成立城市治理研究院，期待其在未来进一步增强上海交大在城市治理领域的影响力和知名度。  

严军表示，上海在未来将崛起成为具有全球影响力和辐射力的综合性特大城市，这一过程将带来产业云集、空间结构和利益格局等全方位的调整。在加速变革的背景下，我们对城市治理的认知和研究还有待进一步加强，该中心愿同高校科研机构加强合作，共同推进城市治理领域的决策咨询研究。  

成立城市治理研究院助力国家治理现代化

顾锋和仇保兴共同为新成立的上海交通大学城市治理研究院揭牌，并殷切希冀该研究院能以响应城市未来发展需求为目标，把握当下城市治理的关键议题，致力于积极推动城市研究的多学科发展。  

钟杨教授提到，在成立“城市治理研究院”之前，上海交大民意与舆情调查研究中心已在过去若干年内完成大量有关城市居民的民意调查。四年前举行中国服务型政府指数新闻发布会，公布中国服务型政府十佳城市，颇具社会影响。钟杨表示，城市治理是当前国家治理的核心课题；在经济学、社会学、规划学、地理学等学科的城市研究热潮中，作为全国公共管理学科评估排名第七的新生学院，交大国务学院将立足于行政学、管理学、政治学的既有优势，积极推动跨学科的城市研究，进一步深化学院在城市治理领域的研究，助力新型城镇化国家战略的实践。  

城市治理研究院院长、国际与公共事务学院常务副院长吴建南教授表示，城市治理是国家治理的重要组成部分，而城市治理体系与治理能力的现代化是国家治理体系与治理能力现代化的重要突破口，迫切需要城市治理研究的进一步创新发展。吴建南进一步指出，国内城市治理研究自90年代发轫，研究热点主要包括城市发展与经济发展、城市经营与管理、新型城镇化与可持续发展、善治与治理理论、智慧城市、治理能力等；十八届三中全会以来，研究学科已在行政学及国家行政管理方面明显增加，且已经超过宏观经济管理，预示着城市治理研究进入新的阶段。未来学者与机构之间的合作将进一步加强，国家可从科研资助与学科建设层面给予进一步的支持引导，推动城市治理整体理论的研究、更加贴近国家战略。  

聚焦城市治理关键议题研判城市发展未来走向

大会首日，仇保兴教授，公共管理领域顶级国际期刊《Public Administration Review》主编James L. Perry教授，何增科研究员，中国人民大学公共管理学院常务副院长许光建教授，中国政治学会副会长、清华大学韩冬雪教授和同济大学可持续发展与管理研究所所长诸大建教授作大会主旨报告。吴建南教授主持。  

次日，美国伊利诺伊州市县管理协会主席John F. Perry、中山大学政治与公共事务管理学院何艳玲教授、南开大学周恩来政府管理学院孙涛教授、华东师范大学中国现代城市研究中心林拓教授和日本金泽大学都市工学部沈振江教授作大会主旨报告，国内外不同学科的学者专家共同探讨我国城市治理的发展走向和路径趋势。上海交大国务学院副院长章晓懿教授主持。 

论坛上，与会专家学者分别围绕“城市政府治理创新”、“区域发展与城市更新”、“城市人口与公共服务供给”和“城市公共安全与环境”四个分议题展开踊跃发言。  

论坛闭幕式由国际与公共事务学院党委书记曹友谊主持。曹友谊代表主办方再次对全体领导、专家的到会致以由衷谢意，并表示本次论坛推进了城市治理学术研究的发展，增进了城市治理领域学者之间的联系和友谊，推动了各位专家形成更为紧密的学术研究共同体。吴建南教授作大会闭幕致辞，并简要报告了中国城市治理的过去、现在和未来。  

此外，在论坛期间，中国人民大学公共管理学院行政管理学系主任孙柏瑛教授、复旦大学国际关系与公共事务学院教授、大都市治理研究中心主任唐亚林和国务学院公共经济与社会政策系主任陈映芳教授还作为特邀嘉宾做客由校团委主办、国际与公共事务学院团委承办的“2015年（首届）大学生城市治理案例挑战大赛”，为参赛学生解读城市变迁过程中的治理结构变化、区域一体化和利益博弈等议题。
