    何平会见海尔集团董事局主席兼首席执行官张瑞敏一行

    2015年09月19日 15:17:28 来源： 新华网

    原标题  新华社总编辑何平会见海尔集团董事局主席兼首席执行官张瑞敏一行

    9月19日，新华社总编辑何平在北京会见海尔集团董事局主席兼首席执行官张瑞敏、美国经济趋势基金会主席杰里米·里夫金一行。这是何平与张瑞敏、杰里米·里夫金等合影。新华社记者丁海涛摄

    新华网北京9月19日电  新华社总编辑何平19日上午会见了前来参加由经济参考报和海尔集团联合主办的“人单合一双赢探索十周年暨第二届海尔商业模式创新全球论坛”的海尔集团董事局主席兼首席执行官张瑞敏、美国经济趋势基金会主席杰里米·里夫金等。

    何平对张瑞敏和中外专家一行表示欢迎，指出，海尔的发展是中国民族工业发展的一个传奇，也是观察中国经济成功之道的一个样本。今年4月，我曾到海尔调研，印象最深的不仅是其成为全球最大的白色家电生产商，更在于敢于自我革命的“颠覆式创新”，而这需要勇气和智慧、创见与预见。创新是企业永葆生机活力的不竭源泉。此次论坛的主题就是“创新”，相信通过论坛的举办，使企业作为产品的制造者，专家作为思想的创见者，媒体作为资讯的传播者，都会从中获益，从而更好地推动中国企业以创新驱动从“中低端”迈向“中高端”，实现“中国制造”向“中国创造”的跨越。同时也希望外国专家能够从中读懂中国，进而更好地解读中国。

    美国沃顿商学院教授马歇尔·梅耶、瑞士洛桑国际管理发展学院教授比尔·费舍尔和中国社科院、复旦大学、北京大学等有关专家学者参加会见和论坛。

    9月19日，新华社总编辑何平在北京会见海尔集团董事局主席兼首席执行官张瑞敏、美国经济趋势基金会主席杰里米·里夫金一行。新华社记者丁海涛摄
