上海交通大学成立城市治理研究院聚焦城市未来发展

人民网上海10月23日电（郑薛飞腾 姜泓冰）中国城市治理论坛23日至24日在上海交通大学举办。论坛以“全球化背景下的城市治理与改革创新”为主题，邀请来自清华道学、中国人民大学、复旦大学等全国50余所高校的150余位专家和学者出席。

为积极促进中国的城市发展，培养城市研究人才，上海交通大学以本次论坛为契机，成立城市治理研究院。上海交大国际与公共事务学院院长钟杨表示，上海交大有深厚的城市研究基础。2014年，国际与公共事务学院成功立项的9项国家社科基金中，有5项聚焦城镇化、城市空间安全、政府购买服务、大城市公共服务等城市治理议题。

国际与公共事务学院常务副院长吴建南教授出任城市治理研究院院长。他介绍说，当前形势下，城市治理是国家治理的重要部分，城市治理体系的现代化也是国家治理体系现代化的重要突破口，因此，学者积极思索城市发展，与机构进行合作的必要性也渐渐凸显出来。

事实上，上海作为目前中国城市化水平最高的城市，发展至今也曾遇到许多城市治理问题。上海市政府发展研究中心副主任严军表示，上海在城市化进程中所经历的问题和解决方案，将对中国乃至世界其他发展中国家发挥重要的典型意义，专家的真知灼见将为城市的未来发展提供智力支撑。

据悉，在为期两天的论坛中，与会专家学者将分别围绕“城市政府治理创新”、“区域发展与城市更新”、“城市人口与公共服务供给”和“城市公共安全与环境”四个议题展开主题发言
