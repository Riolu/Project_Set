大学生的醒思
2009年9月29日 15:11

　　尊敬的举报中心的工作同志们：

　　你们好，很多次在报纸和电视新闻中看到关于打击淫秽色情网站的信息和新闻，我感到很欣慰。

　　但是在前一段时间看到了一条关于大学生网上贩黄而被捕的消息，我感到很震惊。同样是大学生的我，越来越感到现在大学生的法律常识的不足，在很多情况下，都是自己认为自己能力很高，学识很高，而且不管别人的死活，很多大学生都认为这样的想法是很酷很时尚的。而且相信只要能赚钱就是有能力的人，但却不知道有很多事情是做不得的。也不清楚社会是大家的，不知道自己是社会的一份子，只有大家都好起来社会才能进步。个人的经济能力提高是起不了什么作用的。所以这次的打击行动给我们大学生敲响了警钟，让我们认识到了自己的法律意识的薄弱和社会道德的缺乏，也希望举报中心能继续地对不良信息进行打击，并且能更深入到大学生之中，对大学生进行相关的教育，提高大学生的社会道德意识和法律意识。

　　最后，再次的感谢你们辛苦的工作！你们是网络的卫士！

　　祝你们身体健康！

                                                        一个即将毕业的大学生
