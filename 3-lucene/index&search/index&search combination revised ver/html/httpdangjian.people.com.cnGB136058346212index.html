　　【媒体聚焦】
守住最重要的司法关隘
----------
中央纪委日前通报，最高法副院长、党组成员奚晓明涉嫌严重违纪违法，目前正接受组织调查，引发舆论关注。作为司法系统高官，却因涉违法而遭遇覆辙，这无疑令人唏嘘：一方面，这证明了中央“高位打虎，低位除蝇”的高压反腐态势仍在继续，不存在“就此止步”一说；另一方面，尽管奚晓明具体的涉嫌违纪违法情节尚待进一步的披露，可身为执司法牛耳者却犯了法，这种角色错位也着实耐人寻味。最高法是我国最高审判机关，全面推进依法治国，必须及时把司法系统内部的蠹虫揪出来，守住这个最重要的司法关隘。


　　【舆情热点】
高温津贴不容“偷梁换柱”
----------
安监总局、卫计委、人社部、全国总工会日前下发通知，要求各地按规定发放高温津贴。与此同时，全国有27个省区市已出台“高温津贴”标准，虽然多地标准数年未涨，甚至沿用十年前的津贴额度标准，但可视为与部委通知遥相呼应，至少表明一种姿态。然而，此前许多中小企业的员工、流动性强的农民工领不到高温津贴，或是常遭打折扣。有的老板不按规定按月份发放，竟按超过35℃的高温天气具体天数发放，有的单位用绿豆汤、饮料等防暑降温物资“冲抵”，蒙混过关。而和高温津贴被“剥夺”和“偷梁换柱”相对的，是“沉默的大多数”翘盼救济。希望有关部门加大执法检查力度，使高温津贴按要求严格纳入工资总额得以兑现，敦促权益落地，真正为广大劳动者带来夏日里难得的清凉。


　　【国际视点】
希腊阵痛仍将继续
----------
经过17小时艰难谈判，希腊债务问题终于在最后一刻得到解决，至少希腊退出欧元区的可能性已被排除。谈判结束后，欧盟理事会主席图斯克称，与希腊达成协议意味着债权人将继续为希腊提供支持，而希腊必须严格满足条件；欧元集团主席戴塞尔布卢姆则表示，希腊议会应针对相关问题迅速立法，赢回信任。虽然危机告一段落，但希腊人民的痛苦尚未结束。对希腊老百姓来说，已关门一周有余的希腊银行在未来数天仍将继续歇业。而更为严峻的财政紧缩计划还会在未来缩减希腊民众的各项福利。同时，希腊总理齐普拉斯的支持率及其政党在议会中的优势都会受到巨大影响。此前已有数名激进左翼联盟关键议员反对与债权人达成协议。不难预计，齐普拉斯将面临严重信誉危机，而希腊民众将继续在财政紧缩中度日。但可以相信，这个饱受债务困扰的国家已度过最艰难的时期。


　　【抗战记忆】
民族英雄马占山
----------
马占山（下图）（1885―1950），吉林怀德县人。1903年，他因弄丢地主家一匹马，被逼落草为寇，日俄战争结束后，接受清政府收编，1925年成为张作霖麾下骑兵第二军军长。1931年11月4日，在黑龙江省嫩江桥面，他率部向日本侵略者奋起反击（史称“江桥抗战”），被认为打响了武装抗日斗争的第一枪。但随后，马占山出现在事关伪满洲国建国的“四巨头会议”上，并就任伪满洲国军政部总长兼黑龙江省长。就在人们以为他就此走上亲日路线时，他却秘密将2400万元款项、300匹战马及其它军需物资运出城外，并率军队一度攻至农安县城，威胁长春，继续抗日。1937年“七七”事变后，蒋介石任命他为东北挺进军总司令兼东北招抚，他坚持同共产党人合作抗日。1938年马占山到延安就医，中共中央举行盛大欢迎会，毛泽东致词，称赞他是始终如一、抗战到底的民族英雄。


　　【图片报道】
日本遗孤来华祭拜中国养父母
----------
近日，由54名日本遗孤组成的“东京日中友好之会感恩团”在黑龙江方正县祭拜中国养父母公墓（下图）。这次来访的遗孤都是1945年日本侵略军在仓皇撤退时，遗弃在中国东北地区并被中国养父母抚养长大的日本孤儿，如今他们已是七十多岁的老人。其中一位老人表示，“作为遗孤和历史的见证者，我会让自己成为沟通中日的桥梁，我也会把战争历史讲给身边的日本人。让生在日本的年轻人感受到战争带来的伤害，不让悲剧重演。”
