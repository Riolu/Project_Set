摇号拼人品、停车靠运气、堵车练心态……但这些都难以抵挡北京机动车数量的不断走高。这一现状除了带来交通拥堵和环境污染，最为突出的就数停车的老大难问题了，每个有车族每天都不得不去面对它。目前，在北京买辆车有多难？北京登记注册停车位数量与机动车保有量之间的缺口是否逐年缓解？停车的成本又是多少？

在朋友圈中炫富，也许不足以让人嫉妒，但在朋友圈中“炫摇号”的话，你的小伙伴们肯定会“羡慕嫉妒恨”！近年来，北京有意愿购车的人数逐年增加，摇号购车的大军越来越壮大，导致中签购车变得越发困难。

根据北京市小客车指标调控管理办公室的数据显示，2013年每月举行一次摇号，当年1月便已经有140万人申请摇号购车；2014年初改为每两个月摇号一次，截止最近一期（2015年8月）时，申请摇号人数便增加到了247万人，这相当于每139个人当中摇出一个车牌。摇号政策实行后，购车的难度正不断增加……

车位缺口在增加还是减小？

北京市曾经历汽车保有量高速增长的时期，那时相对紧缺的停车位与日益庞大的私家车数量之间便存在着巨大的缺口。

自2011年起，北京市实行小汽车指标摇号政策，私家车增长逐渐放缓，而另一方面北京也在逐渐加大停车位建设的步伐。2013年私家车保有量为426.5万辆，注册登记的停车位数量约为157万个，停车位缺口约270万，车位并不能满足停车需求。

按照北京市小客车指标计划，预测2017年私家车保有量为463.6万辆。根据北京市交通委运输管理局发布的过去十年停车位数量，推算出到2017年，北京市注册登记的停车位数量约为209万个。

通过以上预测可看出，私家车保有量增长将持续放缓，而停车位建设不断加快，停车位与私家车数量这二者间的缺口在逐步减小。

小区停车有多难？

相信有车的小伙伴们都遇见过这样一个问题，回家晚了就没有车位了，只能开车在小区附近转圈圈找车位。居住区车辆的停放问题算是车主们最头疼的问题了。通过各区户数以及各区备案停车位的数据比较，从中可看出两者之间的比例。朝阳居住区的备案停车位最多，平均3户一个车位；顺义居住区的备案停车位最少，高达14户一个车位。各区的居住区车位尚不能满足停车需求。（本项统计依据北京市交通委发布的各区居住区的备案停车位计算，可能与实际停车位数量有所出入。）
