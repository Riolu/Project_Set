北京司机多猖狂? 交警俩小时罚12辆涉牌车

　　原标题：交警俩小时罚12辆涉牌车

　　号码“8”是由“S”改过来的。

　　无牌奥迪车闯入公交车道。

　　交管部门对遮挡号牌车辆进行取证执法。何建良摄

　　私家车闯进公交车道。

　　不挂牌子、变造号牌、遮挡号牌……短短两个小时内，就有42辆车闯公交专用道，其中12辆是涉牌违法车，3名司机还因为伪造、变造号牌被处以拘留15天的处罚。昨日早高峰，交管部门在东三旗路口展开针对车辆违法进入公交专用道的专项处罚，在电子眼眼皮子底下还敢闯公交道的有相当一部分是涉牌违法车辆。

　　假牌面包“S”改成“8”

　　昨日早7时，十余名民警组成的执法小分队已在东三旗路口附近布控完毕。进城方向距离路口200米的位置停放着一辆警车和一辆执法车，两名负责观察哨的民警首先到位，其他执法队员则在路口两侧待命。随着早高峰迅速到来，一辆辆违法车辆也相继落网。

　　发现路口有民警，一些闯进公交道的司机慌了神。一辆奥迪Q5甚至想直接掉头，但刚掉到一半，观察哨上的执法车赶上前，顶住车身，逼停了奥迪车。走近一看，这辆车没挂牌，副驾驶座上摆着一副临时牌照，司机支支吾吾地说新车还没领正式牌照。随后，民警却在他的后备厢内搜出了正式牌照。“这是怎么回事，我也不知道牌照怎么在后备厢里。”这名司机的解释几乎可以用荒诞来形容。

　　不一会儿，出城方向一辆悬挂着京P82XXX牌照的面包车沿着公交车道驶来，被民警迅速扣住。经验丰富的民警远看就发现了这副车牌有异样，号码“8”感觉很别扭。凑近观察，原来这个“8”本来是字母“S”，司机将S中间断头地方的蓝色底漆刮掉，露出牌照铁板的底色，乍看上去竟可以以假乱真。民警通过查询系统查证后确认，这个号牌属于一辆已经转出的轿车，根本不是这辆小面的牌照。

　　仨司机伪造变造车牌被拘留

　　交管局涉牌专业队民警程烨告诉记者，在这条快速公交专用道上，隔不了多远就有电子眼，因此敢在这儿闯公交道的，几乎都有涉牌问题。正是仗着电子眼拍不下来，司机们才敢明目张胆地违法。在程烨身后，一长串早高峰期间违法进入公交专用车道的车辆正等待接受处理。

　　截至9时，短短两个小时内，闯入公交车道的小轿车、大客车、摩托车、面包车共有42辆，司机们排队接受了处罚。按照规定，未悬挂机动车号牌、故意遮挡机动车号牌以及不按规定安装机动车号牌的，记12分，罚款200元。执法过程中，三名司机因为使用伪造、变造的机动车号牌，被罚款5000元，并处以15天拘留。

　　单行道上逮了12辆逆行车

　　除了专项整治闯公交车道，执法队员昨日还在东三旗路口往北一条单行道设置了岗哨。程烨警官介绍，近期接到多名群众举报，在这条单行道上遮挡号牌逆行的违法行为多发，扰乱了交通秩序。果然，短短一个小时内，就有12辆逆行车辆被交警拦下，他们都不约而同地遮挡了号牌以逃避电子眼拍照。违反禁止禁令标志标线，加上遮挡号牌，这12名司机都被记15分、罚款300元。他们都必须重新学习交规，重新通过科目一考试后才能再取得驾驶资格。

　　据了解，早晚高峰期间，本市各条公交专用道以及京通快速路上的公交车道屡屡有私家车闯入，许多司机长年走同一条路，对哪儿有探头、如何遮挡都了然于心。为此，交管部门才出动民警现场查处。今后，这类查处将频繁展开。

　　今年5月，交管部门专门抽调案件侦办、号牌甄别、数据分析等方面的业务骨干，组成涉牌案件侦查专业队。交管部门表示，对于涉牌违法行为将持续保持“零容忍”的严管态势。群众一旦发现涉牌违法行为，可以立即拨打122进行举报。（本报记者 孙宏阳）
