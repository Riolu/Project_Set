香港年度汉字评选“争”“激”“辣”入围


2013年12月02日 10:02　来源：人民网　

 

据香港大公网消息，2013年即将过去，民建联回顾过去一年重大的社会事件，选出包括“争”、“激”、“辣”、“净”等为“2014年度汉字”评选活动的候选单字，由即日起开放予市民投票。经评审委员会评议后，最终结果将于明年1月正式公布。

　　“年度汉字”评选是中文文化圈国家和地区的一项热门评选活动，最早出现于日本，近年来也活跃于两岸三地。民建联自去年举办首次活动，今年再度举办，选出十个汉字包括“争、辣、激、净、稳、忧、信、融、廉、安”，作为今次的候选单字。

　　民建联副秘书长叶傲冬昨日称，去年活动由评委先选出10个单字列为候选名单，公开让市民投票，今年加入“自选”环节，先由民建联总结过去一年大事，浓缩总结为10个汉字，参与投票的市民可从中选一个汉字，或在投票表格上写上自己认为具代表性的自选汉字，增添“开放”特色。他称期望通过这项活动弘扬中国文化，并让市民表达意见与反思社会，活跃社会气氛。

　　评审委员会成员之一的浸会大学中文系副教授卢鸣东称，每个入选汉字蕴含多重含义，例如“廉”，包含了“礼义廉耻”等丰富的传统中国文化，也是香港核心思想之一，同时表达了在物价高涨的社会当中，市民祈求购物可“价廉物美”的愿望。 

　　投票日期由即日起至25日，投票人士可透过互联网、手机等方式进行投票及提名，也可透过民建联各区办事处索取及填写表格。由民建联创会主席曾钰成、民建联副主席蒋丽芸、中国语文学者何文汇、专栏作家黄毅力等学者与专家组成的评审委员会，将评议由市民提名的候选字，最后选出“2014年度汉字”。评选结果将于明年1月公布，参加活动的市民可以参与抽奖，有机会获得三星Galaxy Note 3等奖品。
