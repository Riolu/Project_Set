法律顾问

    新华网聘请北京市东卫律师事务所、北京市东易律师事务所、北京市商安律师事务所为本网法律顾问单位。

    北京市东卫律师事务所（以下简称东卫）

    东卫是经北京市司法局批准成立的大型综合性法律服务机构。东卫是在中国进一步加速法治化进程，全方位融入国际经济一体化大背景下成立的以经济、商事法律服务、刑事辩护为主的综合性律师事务所。东卫业务涉及金融证券、企业改制、并购重组、融资租赁、破产清算、人力资源、风险投资、房地产、知识产权、国际贸易与投资、刑事辩护等诸多领域。为了更好地为全球的客户提供服务，东卫与美国、欧盟、加拿大、日本、澳大利亚、新加坡、香港以及台湾等国家和地区的律师事务所建立了广泛的业务联系，可以为客户提供全方位、多样化、国际性的法律支持和服务。

    服务新华网的东卫团队为：单体禹律师为首的知识产权团队

    单体禹律师团队是一支以知识产权业务为核心的律师团队，团队目前主要涉及著作权建设、著作权维权、商标行政诉讼、商标侵权诉讼、专利行政诉讼、专利侵权诉讼、反不正当竞争以及知识产权代理等知识产权业务，每年都代理大量的诉讼和非诉讼案件或相关业务，得到了大多数客户的认可。

    单体禹律师为北京市东卫律师事务所知识产权业务主管合伙人，第八届北京市律师协会著作权法专业委员会委员，第九届北京市律师协会商标法专业委员会委员，对整个知识产权体系有宏观把握和深厚理解。

    北京市东易律师事务所（以下简称东易）

    东易创建于2002年2月，是中华人民共和国司法部批准建立的高层次、全球化、合伙制律师事务所。总部位于北京市西城区五栋大楼，办公面积1100平米。在吉林设有分所。凭借高效的管理、多国语言优势和丰富的执业经验，东易已跻身于国内领先律师事务所的行列，成为中国律师界业务能力强、服务范围广，能够为客户提供优质、高效法律服务的现代化综合性律师事务所。

    服务新华网的东易团队为：侯德隆律师为首的知识产权团队

    团队目前主要涉及著作权建设、著作权维权、商标行政诉讼、商标侵权诉讼、专利行政诉讼、专利侵权诉讼、反不正当竞争以及知识产权代理等知识产权业务，每年都代理大量的诉讼案件，仅在知识产权维权一项，近几年每年都可为客户挽回数千万元的直接经济损失，取得了一定的成绩，得到了大多数客户的认可，也为维护国内知识产权正常的市场秩序做出了一定的贡献。

    北京市商安律师事务所（以下简称商安）

    商安成立于1997年，系北京市司法局批准成立的合伙制律师事务所。“中国公众满意十佳律师服务机构”、“全国AAA级信用律师事务所”。商安作为一家综合性律师事务所，下设案件协调委员会、监察委员会、教育文化卫生委员会、诚信道德考评委员会、外事交流委员会以及知识产权法律事务部、公司证券事务部、刑事法律事务部、行政法律事务部、房产法律事务部、国土资源法律事务部和非诉讼综合业务部。国家检察官学员《中国检察官》理论研究与实践基地、西南政法大学教学实习基地、2008年度被评为“服务中小企业发展全国优秀律师事务所”、联合国南南局国际技术交易中心项目合作单位。

    服务新华网的商安团队为：刘晓陈律师为首的知识产权团队

    刘晓陈律师十年专注著作权、公司事务、新媒体版权运营，熟悉了解文化传媒行业运作流程、规律及特点 ，擅长摄影、文字、音乐、影视剧的著作权法律事务，在维权及司法实践方面有丰富经验。刘晓陈律师具有全国版权交易共同市场版权经纪人/代理人资格。 目前已为多家顾问单位提供著作权法律咨询、著作权登记、著作权代理、著作权维权等专业服务。
