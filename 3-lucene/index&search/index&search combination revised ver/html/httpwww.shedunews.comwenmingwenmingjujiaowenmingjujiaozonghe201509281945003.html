5000余种“沪产”图书亮相第25届全国图书交易博览会

2015-09-28 10:11:24    来源：上海文明网   

第25届全国图书交易博览会日前在山西太原开幕。本届书博会为期3天，设太原主会场和大同、长治、运城3个分会场。书博会期间，太原主会场将集中展示全国1000余家出版发行单位的25万多种图书，举办各类主题活动近200场。王蒙、王树增、刘慈欣、曹文轩等知名作家和学者与读者分享阅读体验。上海新闻出版业近70家单位近300人组成强大阵容亮相书博会，推出5000余种精品图书。
　　上海市新闻出版局承办的“全国图书精品展”，吸引不少读者驻足留影。据悉，上海已连续第三年承办“全国图书精品展”，本届精品展以纪念“中国人民抗日战争暨世界反法西斯战争胜利70周年”为主线，以三本交错的“大书”为主要造型，陈列了2000余种主题图书，营造充满现代气息的书香体验区。
　　上海出版展区以蓝色为主色调，上海世纪出版（集团）股份有限公司、高校出版社联合体、社科文出版社联合体、钟书书店等沪上主要出版发行单位，部分杂志社等均组队参展。一大批全面反映社会主义核心价值观、体现“十二五”期间国家和上海重点选题、纪念中国人民抗日战争暨世界反法西斯战争胜利70周年的优秀出版物已与读者见面。其中，钟书书店展区以沪上最美书店“钟书阁”的标志性装饰元素亮相，成为一道风景线。
　　书博会期间，还将举办“中华印刷之光”展览，突出山西优秀历史文化和印刷技术传承，并展示汉字信息处理技术、绿色印刷、数码印刷、立体印刷和3D打印技术等高新科技内容，让观众加深对中国传统文化和现代印刷技术的了解。此外，还将举办图书捐赠活动，让更多基层群众享受便利的阅读产品和服务。
