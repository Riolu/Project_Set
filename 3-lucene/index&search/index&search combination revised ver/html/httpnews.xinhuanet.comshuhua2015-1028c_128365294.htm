莫斯科国家大马戏团将在哈尔滨驻场演出

2015年10月28日 07:27:46 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网哈尔滨１０月２７日专电（记者何山、马迪）在刚刚过去的周末里，精彩的俄罗斯马戏表演让冰城哈尔滨的市民意犹未尽。随着哈尔滨市首家专业马戏表演场馆——会展中心马戏城投入使用，莫斯科国家大马戏团带来多场精彩演出。

    哈尔滨会展中心马戏城出资方表示，他们已与莫斯科国家大马戏团签署长期意向性协议，该团有望在哈尔滨驻场演出。

    据介绍，此次莫斯科国家大马戏团的演出包括马术、空中芭蕾、大跳板、空中呼啦圈、水上芭蕾等１６个表演。其中空中芭蕾是该团“压箱底”的节目。高难度的技巧和戏剧式的呈现方式，使得在场观众时而惊呼、时而欢笑。

    空中芭蕾表演者亚拉说：“我们去过中国的很多城市，我觉着哈尔滨的观众是最热情的。他们会给我们鼓掌，为我们呐喊，将他们的热情传递给我们演员。我有个愿望，希望在中国的演出场次能越来越多，为中国观众表演我们最好的节目。”

    哈尔滨市民王晓艾说：“演员的表演和灯光音效配合非常好，很震撼。尤其是高难度的技巧表演和芭蕾舞、民族舞相结合，真是一种力与美的完美展示。”

    据介绍，会展中心马戏城由哈尔滨工大集团出资兴建。与临时性马戏表演场所不同，马戏城内部采取３６０度环形设计，中央为圆形可移动舞台，１６００个坐席环绕舞台阶梯式分布。舞台地板收起后，下面是水上表演泳池，能满足空中、地面、水中多种表演的场地需求。

    马戏城出资方表示，他们已经与莫斯科国家大马戏团签署长期意向性协议，邀请其驻场演出，第一季演出将持续到１１月中旬。莫斯科国家大马戏团是世界规模最大的马戏团之一，在国际马戏界享有盛誉。
