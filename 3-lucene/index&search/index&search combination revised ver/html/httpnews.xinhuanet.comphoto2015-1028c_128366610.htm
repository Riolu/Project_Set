中国女队获得体操世锦赛团体亚军

2015年10月28日 08:52:58 | 来源：新华网

    10月27日，中国女队队员在比赛后合影。当日，在苏格兰格拉斯哥进行的第46届世界体操锦标赛女子团体决赛中，中国女队以176.164分的成绩获得亚军。新华社记者韩岩摄

美国赢得女团三连冠　中国连续第二次摘银

　　新华网格拉斯哥１０月２７日体育专电（记者张寒 王子江）２０１５年世界体操锦标赛２７日决出第一项奖牌归属，上届亚军中国队如愿拿到女子团体银牌，美国队成功赢取三连冠，而东道主英国队居上，力压俄罗斯队，摘得史上首枚世锦赛奖牌。

　　这是中国女子体操队连续第二次夺得世锦赛团体银牌。由于姚金男、黄慧丹因伤缺阵，中国女团本次报名６人中有多达半数的选手都是第一次参加世锦赛，这令这块银牌来得更加珍贵。

　　自２００６年首次夺冠后，中国女队已经五次在体操世锦赛的领奖台上摘得团体奖牌。此前她们在２００７年和２０１４年位列亚军，２０１０年和２０１１年连续两次摘铜。分别于奥运会年后举行的２００９年和２０１３年体操世锦赛不设团体项目。而在２００６年之前，中国女团的成绩只有上世纪８０、９０年代的两银两铜。

　　２７日的决赛中，中国女队由于仅以预赛第四名的身份晋级，出场时间总与夺冠呼声最高的美国队存在时间差，但这不影响中国“小花”们以扎实的功底、稳定的发挥征服裁判。

　　最后一项跳马结束后，率先完赛的她们以总分１７６．１６４分临时跳到榜首，而等待近２０分钟后，美国队才以两度女子全能世界冠军拜尔斯的自由体操为本场决赛画下句号，总分１８１．３３８高出中国队５分多，除高低杠外其他三项均领先于其他７支队伍，强势卫冕。

　　中国队的比赛从高低杠开始，依次进行平衡木、自由体操和跳马。首轮高低杠是中国队强项，年轻的“老将”商春松打头阵，一举拿下１５．２３３的高分，接下来上场的范忆琳和谭佳薪也成功完成各自的套路，得分均在１５分以上。

　　后来证明中国“小花”们在高低杠项目上拿到了全场最高分，比美国队多将近０．２分。美国队和俄罗斯队首轮先后在跳马上亮相，全部成功。

　　第二轮平衡木同样是中国队的强项，除第一次参加世锦赛的王妍在套路中间的第二个连续空翻时失误掉落，第一个出场的另一名世锦赛“新兵”范忆琳和第三个出场的商春松均顺利完成动作。这一轮，美国的高低杠全部成功，俄罗斯在同一项目上失误一人。

　　第三轮中国队是自由体操，商春松的结束空翻一脚踏出垫外，令一套原本干净利落、几无瑕疵的动作被强制扣分，但她仍拿到了１４．３６６分。

　　队长商春松赛后告诉记者，大家那时已经感到有把握拿银牌了，因为最后一轮的跳马比赛失误的可能性不大，而王妍、毛艺和谭佳薪也确实不负众望，在这一项共拿到４５．２３３分。

　　英国队的最后一项也是跳马，东道主在这一项上拿到她们自己全场最高的４５．０４９分，一举超过高低杠和平衡木发挥欠佳的俄罗斯，以总分１７２．３８０、高出对手仅０．４１６分的优势跃居第三名。

　　格拉斯哥世锦赛自此进入决赛赛程，接下来的５天中将依次决出男子团体、女子全能、男子全能、女子个人单项和男子个人单项的奖牌归属。
