医药行业周报

修正药业再陷负面漩涡 企业声誉修复工作不善加深危机

（10.12-10.18）

　　一周行业综述

　　监测期间医药行业药企负面新闻、移动医疗、医疗保险方面的动态受到舆论热议。成都食药监局公开对包括修正药业等企业的行政处罚，修正药业陷入负面舆情漩涡，企业“黑历史”被媒体挖掘，声誉修复工作不善进一步加深负面影响。移动医疗方面，一篇名为《论春雨医生的倒掉》的文章引起春雨医生“被倒闭”的误会，以春雨医生为代表的移动医疗前景受到热议。行业政策方面，医疗保险相关政策舆情热度极高。国务院14日印发《关于第一批取消62项中央指定地方实施行政审批事项的决定》，其中取消药店、医疗机构医保定点审批取消被舆论认为含金量最高，民营医疗机构发展迎来利好。

　　行业热点话题解读

　　修正药业再次卷入负面漩涡　企业声誉修复工作不善加深危机

　　10月9日，成都市食药监局公布一批行政处罚案件，其中修正药业因为使用不合格药包材遭到处罚。在此次事件中，修正药业使用了不合格药包材“低硼硅玻璃管制口服液体瓶”。违反了《直接接触药品的包装材料和容器管理办法》，被处以罚款。

　　自10月10日被《华西都市报》率先报道后，事件引发舆论热议，中国经济网、中国投资咨询网、《南方日报》于13日至14日相继刊文对事件进行评论和报道，其中中国经济网的文章《修正“良心药”屡登黑榜　重广告轻质量酿苦果？》的转载量最高，推动事件升温。

　　从舆论反馈看，媒体对修正药业的批评主要集中在三方面。首先，修正药业“黑历史”被挖掘和回顾，品牌信誉危机加深。《南方日报》、《潇湘晨报》、中国经济网等媒体均对修正药业2012年至2014年的负面事件进行回顾：2012年4月羚羊感冒胶囊牵涉到“毒胶囊”事件；2014年11月15日，修正药业被查出用于生产肺宁颗粒的药材返魂草部分发生霉变，且企业存在故意编造虚假检验报告等行为。两次事件影响力大，都成为当年医药行业重要负面事件。其次，媒体认为修正药业重广告轻质量。中国投资咨询网和中国经济网的文章指出，修正药业在频繁被曝出质量问题后，将著名的广告语从“良心药，放心药，选修正药，管用的药”悄悄变为“修元正本，造福苍生”，此外，修正药业近年来邀请了包括孙红雷、陈建斌等当红明星对斯达舒等产品进行宣传。中国投资咨询网评论认为，修正药业花重金做广告，却不惜使用“毒胶囊”，在药材成本上节省的行为“重广告轻质量”。第三，企业“承诺”与行动对立，表里不一。中国投资咨询网、健康时报网在报道本次事件时，特别指出修正药业董事长修涞贵曾向媒体表示：“救人一命，胜造七级浮屠。医药行业事关人的健康、人的生命，责任太重。”然而，修正药业近年来被曝光的行为严重违背了董事长的“承诺”，“讽刺性”十足。

　　修正药业声誉修复不力加深企业危机。媒体对此次事件本身着墨较少，却对修正药业的“黑历史”关注较多。究其原因，在于修正药业在此前重大负面事件后未做好声誉修复工作，留下诟病“把柄”。修正药业并未认识到声誉修复工作的重要性，声誉修复工作以实际行动推动问题解决，通过议题设置、加强主动沟通等方式积极与公众对话，能够扭转负面事件给企业带来的刻板印象。回顾修正药业此前对负面舆情的处理，可见其存在严重问题：信息不透明，未对事件后续的处理结果、整改措施进行公布，虎头蛇尾；针对两次重大负面事件爆发的根源－－供应商筛选把关不严和中药材原料管理不善没有建立解决问题的长效机制，“悄悄”改变广告语无法从根本上重修企业形象和声誉。

　　春雨医生“被倒闭”　微博文章引爆业界移动医疗发展困惑

　　10月18日，一篇题为《论春雨医生的倒掉》的评论文章在朋友圈中升温，关于春雨医生倒闭的传闻引发行业热议移动医疗盈利模式。传闻很快被证实虚假，《论春雨医生的倒掉》为微博账号“一医一世界”五月发布的一篇文章，文章从移动医疗泡沫严重、无法触及医疗核心、线下的医院和医生、没有盈利模式四个方面论述了以春雨医生为代表的移动医疗终将倒闭。春雨医生CEO张锐表示，这是网上炒作，是友商在恶搞，目前春雨医生一切运营良好。张锐还表示，春雨医生已经在线下开了150个诊所，还准备到今年底开到300家诊所，最近春雨医生还会有融资及金融等方面的重要消息宣布。

　　《论春雨医生的倒掉》一文突然广泛流传，引发包括雷锋网、创业邦、界面网、i黑马、健康界等科技类博客和关注医疗行业的媒体热议移动医疗前景。春雨医生为何会突然“躺着中枪”是此类网站热议的焦点，同时舆论普遍认为春雨医生所代表的移动医疗前景难以预测。针对春雨医生的发展前景，雷锋网的文章《春雨医生还没倒掉 但其模式真的可行吗？》分析认为春雨医生近期动作频繁，其2月发布的“私人医生干预指导下的服务电商”模式行业认可程度较低，该模式的核心为充当医生与患者、患者与设备、患者与家人之间的连接“中介”，雷锋网指出一直未拿到知名投资机构的投资说明春雨医生的模式不被认可。i黑马网的文章《从春雨医生倒闭传闻　看移动医疗行业之困》指出，连接药品、医院、医生、患者需要共同的买单方－－保险公司，未来能将这几个方面串联起来的只有保险公司，i黑马网认为春雨医生未来有可能向这个方向发展。健康界的文章《春雨医生替移动医疗挨了一枪，活该还是冤枉？》分析春雨医生突然被“黑”的主要原因在于移动医疗短时间内爆火和实际使用冷清的巨大反差，春雨医生存在的最主要问题在于医学根基过浅，缺乏与线下医生联系。“一医一世界”的文章引爆了长期积累的关于移动医疗盈利模式的困惑。

　　春雨医生背后反映的移动医疗发展困境方面，科技博客、行业网站的观点趋向两极。认为前景堪忧的一方指出，移动医疗将面临公立医院新项目审批难、公立医院数据壁垒、医疗行业信息化缺乏顶层设计导致标准不统一、对接医疗保险支付困难的局面。而认为移动医疗未来值得期待的一方则表示，互联网推动医疗行业改革的过程复杂曲折，目前仍处于“试错”阶段，中国政府在老龄化人口增加、慢性病管理任务紧迫的环境下，借助互联网的优势促进医疗改革是大趋势。

　　医保定点审批将取消 民营医疗机构发展受舆论期待

　　10月14日，经李克强总理签批，国务院发布《关于第一批取消62项中央指定地方实施行政审批事项的决定》，决定第一批取消62项中央指定地方实施的行政审批事项。在7项涉及医疗机构被取消的审批项目中，取消基本医疗保险定点零售药店以及定点医疗机构的资格审查，对于医药行业来说尤为重要，这意味着，原则上医院和药店的医保定点审批今年年底之前将在全国消失。

　　舆论对此普遍认为民营医疗机构迎来利好，发展前景光明。行业网站赛柏蓝指出如果医保额度也能与公立医院平起平坐，患者首选的将不一定是公立医院，这将会真正推动民营医院开展和公立医院的竞争，民营医院将会获得更大的发展潜能和空间，对分级诊疗的推进也将起到促进作用。此外，取消审批将杜绝原有的因审批而带来的腐败现象，有效降低医院药店隐性运营成本，间接降低百姓健康支出。《证券日报》引用行业人士的观点指出民营医疗机构会对现有的公立医疗机构产生冲击，但是对于一些存在品牌优势的公立医疗机构影响不大，例如协和医院等，他们的竞争优势不会随着民营医疗机构获得医保定点而丧失。券商对此政策也保持较高的关注度。银河证券指出，取消审批后，相当于“门禁”取消，民营医院和连锁药店在扩张时新店更容易进入医保，申请医保周期缩短，盈利时间缩短，将更有利于民营医院和连锁药店的发展。

　　此外，在地方政府具体落实政策方面，媒体指出政府相关部门职能转变，监管能力将受到考验。赛柏蓝认为，协议医保定点有可能成为民营医疗机构审批入医保取消后的主流准入医保方式，医保部门面临从审批部门变成监督部门的角色转换。健康界的文章进一步指出，协议医保定点将促使药品和医疗市场的利益重新分配，定点机构遴选标准的设计将成为地方政府改革中的矛盾焦点。

　　医保定点审批的取消，将为民营医疗机构的发展同时带来机遇和挑战。一方面，政策的实施将为民营机构发展营造更健康的竞争环境。另一方面，政策带来的地方政府管理趋向更严、机构之间竞争加剧，也成为民营医疗机构品牌形象管理能力的考验。只有那些通过管理规范获取公众信任，通过品牌鲜明、突出优势等方式展现实力的民营机构才能在市场竞争中分得一杯羹。
