船舶数字博物馆
上海交通大学船舶数字博物馆，拥有万余件馆藏，信息量大、知识面广，生动反映中西方在造船发展史上各领风骚的时代特征。她采用先进的计算机信息技术，突破建造实体船舶博物馆的时空地域和资金限制，将展示门类广博的造船科技和历史文化知识。

URL:http://shipmuseum.sjtu.edu.cn/info.asp
董浩云航运博物馆
董浩云航运博物馆由香港董氏慈善基金会和上海交通大学（创始于1896年）联合创办，设立于交大新中院。楼高二层，西式建筑，内有天井，是学校早期学生宿舍，距今已有90多年历史，充溢着历史的沧桑况味。

URL:http://www.maritimemuseum.sjtu.edu.cn/
绿色交大网
绿色交大网以上海交通大学闵行校区及其周边的种子植物作为主要收录对象，运用高清动态图片、结合文字说明，分科介绍了100余种植物。同时，网站提供了较明确的校园分布位置，便于同学们的识别与学习。

URL:http://life.sjtu.edu.cn/lab/vegetable/index.html
钱学森图书馆
钱学森是享誉海内外的杰出科学家和我国航天事业的奠基人。在他身上，充分体现了崇高的爱国主义精神，严谨的科学态度和高尚的道德情操，被誉为人民科学家。钱学森同志的杰出贡献、感人事迹和崇高品格，是我们国家和民族宝贵的精神财富。为了进一步弘扬钱学森同志爱国，创新、奉献的业绩与精神，经中央研究，决定建设钱学森图书馆。

URL:http://www.qianxslib.sjtu.edu.cn/
李政道数字资源中心
　　李政道先生是开创华人科学家获诺贝尔奖历史的物理大师，是推动中国科技教育事业发展的炎黄赤子，还是倡导科学与艺术交融的智慧使者。李政道数字资源中心网站是李政道相关文献实物及资源的最完整、最系统、最全面的收藏保存，是服务广大师生和物理学爱好者、李政道科学思想研究者的重要网络平台，同时也是李政道图书馆网络陈展和相关动态的展示发布渠道。

URL:http://www.tdleelib.info
