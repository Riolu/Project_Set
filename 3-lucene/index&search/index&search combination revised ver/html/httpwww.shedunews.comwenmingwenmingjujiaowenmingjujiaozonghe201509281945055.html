闵行探索家庭文化建设和良好家风培育有效途径

2015-09-28 10:52:44    来源：上海文明网   

闵行区以市委召开“注重家庭、注重家教、注重家风”建设工作座谈会为契机，努力探索家庭文化建设和良好家风培育的有效途径，着力推进家庭文明建设工作。

一是结合《市委宣传部、市委组织部、市精神文明办、市妇联、市教委、市文广影视局、市新闻出版局关于本市深入开展“注重家庭、注重家教、注重家风”建设工作的意见》，立足城区特点，研究拟定《闵行区关于深入开展“注重家庭、注重家教、注重家风”建设工作的实施意见》，明确职责和任务分工，形成上下联动、整体推进的工作格局。

二是围绕一个主题，即“寻找最美家庭，弘扬优良家风，践行家庭文明”，开展2015年寻找“海上最美家庭、闵行和美家庭”活动和“说家训、扬家风、传家教”家庭文明践行活动。立足全区540余个“妇女之家”，设立“最美家庭”光荣榜；结合“社区三尺讲堂”，开展讲和美家庭故事活动；发挥妇联区域妇建平台优势，把寻找“最美家庭”活动向机关、企事业单位延伸，挖掘具有行业特色的最美家庭；融合传统文化，开展“月月最美”系列活动，今年４月，以学习为主题开展最美书香家庭展示；５月，结合母亲节展示最美母亲风采。结合未成年人特点汇编《弟子规》漫画读本，向青少年传递好家风好家规，引导他们追求尊道德、讲道德、守道德的生活。

三是聚焦重点人群，启动“晒家训家规，树家风美德”主题征文活动。以党员领导干部、团员青年、社会公众人物、先进模范（劳模、三八红旗手、文明家庭等）及民主党派人士等为对象，通过征集在家庭教育中有关和谐、孝道、劝学、劝善、勤俭、廉洁、励志、修养等方面的祖训家风故事或个人亲身经历，激发群众、带动群众，发挥“正效应”，汇聚“正能量”。

闵行区将结合“三严三实”专题教育，召开家庭文明建设工作座谈会，开展党员领导干部家庭建设专题教育，推动领导干部发挥表率带头作用。结合“我们的节日”主题活动，以中秋、重阳等传统节日为契机，展示“最美家教家庭”、“最美公益家庭”、“最美孝亲家庭”等风采，弘扬家文化。结合争创全国文明城区及中共中央、国务院《关于进一步加强和改进未成年人思想道德建设的若干意见》下发十周年“回头看”活动，强化学校、家庭、社会三结合教育在推动青少年立德树人中的基础作用，引导广大市民群众修身立德、和睦家庭、见贤思齐，使核心价值观教育内化为闵行人民的精神追求、外化为自觉行动。借助区“鲁冰花”家庭教育官方微信，上海闵行官方微博、微信，闵行报社微信等新媒体力量，以及闵行报、社区报和各类宣传阵地，大力宣传家庭文化，激发人们的家国情怀以及“中国梦、家庭梦”圆梦心愿，引领全社会共同汇聚“发现美、传播爱、践行德”的正能量。
