      王海峰:利用大数据创新城市管理

      刘治彦:建新型现代城市 提升城市网络形象

      人类的现代家园必须适应新趋势，发展绿色、智慧、人文的新型现代城市。这样的城市无疑会赢得良好口碑和网络形象。 [详细]

      沈　阳：用互联网思维重塑城市网络形象

      像宁波市、深圳市一些政府app，正在把我们传统的官网迅速的搬到手机上去，互联网也可以考虑实现这些。 [详细]

      杨思松:创新服务城市 推动城市发展

      与全国很多城市一样，合肥正走在改革发展的攻坚期，也是在社会矛盾的凸显期，互联网为加强党委政府和社情民意提供了新的窗口。 [详细]

      网络文明与城市形象（网德）论坛在合肥举行

      对政府来说，要依法治网，有理有据；对企业来说，要自律有序、坚守底线；对网民来说，要心存善意，有所敬畏。 [详细]
