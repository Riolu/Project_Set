跟着候鸟看生态

天气转冷，候鸟又开始翩翩南飞。作为大自然的信使，候鸟对生态环境有着最敏锐的反馈。接下来，就让我们《跟着候鸟看生态》，一起来看看我国生态文明建设取得的成绩，共同守望我们美丽的家园，开启美丽中国的“十三五”旅程。

生态环境好 候鸟变“留鸟”

黑脸琵鹭：全球濒危物种类别之一，仅见于亚洲东部。春季3-4月到达繁殖地，10-11月离开繁殖地前往越冬地。

黑颈鹤：世界上唯一生长、繁殖在高原的鹤。繁殖于拉达克，中国西藏、青海、甘肃和四川北部一带，越冬于印度东北部，中国西藏南部、贵州、云南等地。

红嘴鸥：在世界的许多沿海港口、湖泊都可看到。繁殖于中国西北部的天山西部地区及东北部的湿地。在中国东部及北纬32度以南所有湖泊、河流及沿海地带越冬。

东方白鹳：国家一级保护动物，2009年全世界有东方白鹳野生种群为不到3000只。东方白鹳的分布繁殖地主要集中在俄罗斯远东和中国的东北，越冬地集中在中国长江流域。

白天鹅：全球易危物种。每年9月中旬开始迁徙至中国河南三门峡、山东以及南方各地越冬，第二年4月才陆续返回故乡，夏季多在中国东北、内蒙古、新疆等地繁殖。

 

环保法律、法规数量全球第一



国家级法律11个，新修订的《中华人民共和国环境保护法》2015年1月1日实施。

国家级行政法规8个，包含对环境、资源、危化品等类别的管理条例。

部门规章29个，内容涉及环境、能源、应急、危化品等类别。

国家标准17个，其中《环境空气质量标准》将于2016年1月1日起实施。
 

“绿水青山”变“金山银山”

       2001-2010年，我国环保投资规模由1107亿元增长到6654亿元，增长6倍。

　　十二五期间，我国节能环保产业投资3.4万亿元，比十一五增长62%。

　　十二五以来，国家对151个共7600多亿元不符合条件的项目环评不予审批。

　　2014年，首批实施新环境空气质量标准的74城PM2.5平均浓度同比下降11.1%。

　　百姓明显感觉到现在蓝天多了，白云多了，水也逐渐变清了。

 

跟着候鸟看生态

　　涟漪泛泛的湖面上，一只只候鸟在展翅飞翔，优美的身姿掠过湖面，构成一道靓丽的风景线。生态环境的改善，让这一切变成美好的现实，鸟儿自在迁徙，人与自然和谐共处，美丽中国的画卷在伟大的生态文明建设征程中渐渐展露容颜。
