          中国银行消费金融公司首席财务官周钧明表示，在经济新常态下，消费金融的发展更是拉动消费的重要手段。

      袁耀璋：与零售商合作是消费金融公司重要运营模式

          与零售商的合作是消费金融公司获取客户、短期内打开市场、进一步积累品牌效应、优化信贷资源的重要运营模式。

      许凌：消费金融是快消品
            京东金融消费金融事业部总经理许凌在圆桌论坛环节表示，京东集团把消费金融定位为快消品，这个快的理念最终落实在京东白条上，京东白条一分钟就能完成所有的链接，一秒钟就能完成在线支付。[详细]

            中国邮政储蓄银行消费金融部总经理王蓉晖在圆桌论坛环节透露，中国邮政储蓄银行正组建消费金融公司，预计在11月开业。[详细]

      PPF集团最高管理委员会委员梅恺威做论坛闭幕的总结发言。

      PPF集团最高管理委员会委员梅恺威：首先希望能够祝贺清华大学中国与世界经济研究中心举办这次论坛，这次论坛对于任何一位想要了解中国消费金融行业的人来说都是非常重要的，提供了非常重要的信息，而且由于在消费金融行业我们全球范围内的文献都是比较有限的，对于一个中国这样由投资驱动的国家转向消费驱动的经济体来说，我觉得这样的论坛是恰逢其时的，是非常有意义的。

      PPF集团最高管理委员会委员梅恺威：消费金融在全球经济来说，毫无疑问起到非常重要的作用，消费者能够容易的借到贷款，会使我们的全球经济变得更加便捷，更加有效，促进经济增长。信贷的可用性，可以使消费在高收入和低收入群体都得到释放。如果看到美国和欧洲的历史情况，我们就会发现消费金融是在二战以来不断的驱动着欧洲和美国的发展，之前是投资驱动的经济增长，后来是消费驱动的应该增长，消费金融增长也是非常有动力，包括不同的产品的消费的增长，以及消费者自由选择的促进，这样的经历在欧美也会得到复制。

      PPF集团最高管理委员会委员梅恺威：全球排名前48位的金融调查机构的数据显示，尽管有这么多的消费金融需求和这么多的信贷需求，全球也就是有28亿人，就是一半的成年人得不到正式的信贷服务的，他们只能从一些非正式的渠道获得贷款或者借钱，贷款的机会是非常不充分，借钱成本也是非常高昂，使得这些穷困家庭无法脱离贫困，实现富强。消费金融会很大程度上促进这个需求的满足。我想强调一下国际上的经验，在全球范围内，消费金融经常都是消费者所需要的正式的金融需求，也会带来不同产品的消费增长，这些人本来买不到这些产品。所以，消费金融在整个金融产业当中发挥非常重要的作用。

      PPF集团最高管理委员会委员梅恺威：我的一个国际经验就是创新是整个经济增长的一个驱动力，但是为了让创新发挥非常重要的作用，人们必须尝试新的产业，不断的尝试，不断的吸收经验，之后取得成功。监管和立法环境可以让我们进行不断的尝试，因为尝试可以最后对整个社会有更好的益处，促进经济增长，所以创新是整个经济增长的核心。我们从很多演讲嘉宾那里也听到很多非常重要的信息，比如消费金融，它将会是整个创新的最大驱动力，驱动中国的增长。
