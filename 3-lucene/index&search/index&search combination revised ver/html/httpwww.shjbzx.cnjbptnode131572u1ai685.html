回眸“十二五”我国互联网国际交流合作经典场景
2015年10月26日 14:11

从枕水江南的千年古镇乌镇到邕江之畔的古城南宁，从“翡翠之城”西雅图到南美最大城市圣保罗……回眸我国“十二五”时期互联网国际交流合作成果，一个个不可磨灭的经典场景跃入脑海。这些场景记录了中国从网络世界参与者到网络秩序构建者迈进的一步步脚印，记录了中国在互联网治理国际舞台上的主动与积极、胆识与魄力，以及为构建互联网强国所做出的不懈努力，记录了世界正被中国互联网发展的丰硕成果和巨大潜力嵌入鲜明“中国印记”的全过程。

首届世界互联网大会召开，开启网络空间全球交流中国主场

场景：2014年11月19日至21日，浙江，乌镇，首届世界互联网大会。大运河畔，粉墙黛瓦民居旁，来自近100个国家和地区的国际政要、国际组织代表、著名企业高管、专家学者等1000多人会聚一堂，围绕“互联互通、共享共治”主题，“华山论剑”、交流切磋、广泛合作，为国际互联网发展贡献智慧和力量。

首届世界互联网大会是中国举办的规模最大、层次最高的互联网大会，也是世界互联网领域一次盛况空前的高峰会议。

盛会期间，从习近平主席到李克强总理都对大会给予极大关注。国家主席习近平虽然正在出访，仍心系世界互联网大会，由国务院副总理马凯带来并宣读他的贺词。国务院总理李克强也在大会召开第二天会见出席互联网大会的中外代表并同他们座谈。

中国新一届领导人执政以来，积极运筹，主动谋势，“主场外交”成为中国外交的新特点，此次互联网大会的举办，让世界互联网大会成为中国“主场外交”的又一重要平台，彰显大国气魄与担当。

中国接入国际互联网到2014年已有20年，这是中国第一次尝试举办世界互联网大会。中国作为最大的发展中国家，拥有最多网民，理应为世界搭建一个具有广泛代表性的开放平台。这个平台是为中国与世界互联互通搭建的国际平台，也是为国际互联网共享共治搭建的中国平台，通过这个平台让全世界互联网巨头交流思想、探索规律、凝聚共识，让中国互联网的发展成果、中国网络企业的发展成就全景展示，让互联网成为人类的阿里巴巴宝库，努力不让它成为潘多拉的魔盒。

世界互联网大会在中国举行，显示“十二五”以来，中国在互联网领域的话语权大大提高，中国主动搭建交流平台让世界了解自己。随着中国更加全面和积极参与全球网络空间的治理,全球网络空间国际治理体系将迎来全新的发展格局。

打造双边多边互联网对话机制 构建合作共赢模式

场景：2015年9月23日，美国，西雅图，第八届中美互联网论坛。习近平主席和参加论坛的双方主要代表合影，代表中包括马云、马化腾、蒂姆·库克、扎克伯格等互联网“巨头”。有网友对照片上这些互联网大佬所经营的企业进行估值，加起来超过了2.5万亿美元，堪称史上“最贵合影”。

作为习近平主席访美的重要活动之一，第八届中美互联网论坛备受关注。本届论坛是中美互联网论坛创建8年以来规格最高的一次盛会，习近平主席出席并发表讲话。习近平主席会见出席论坛双方主要代表时说“中美都是网络大国，双方拥有重要共同利益和合作空间。双方理应在相互尊重、相互信任的基础上，就网络问题开展建设性对话，打造中美合作的亮点，让网络空间更好造福两国人民和世界人民。”习主席的讲话高屋建瓴，阐述了打造互联网“大国关系”的重要意义，展现了一个负责任大国维护网络安全、深化开放合作的信心和决心。习主席亲临现场会见企业界代表，显示出他对互联网行业的高度重视。

本届论坛也是中美互联网论坛建立以来参会企业阵容最豪华的一次，来自中美两国的数十位互联网巨头出席会议。论坛期间，来自中国和美国的互联网科技公司签署了多个合作协议。一项项协议，印证了中国国家互联网信息办公室主任鲁炜在论坛上的发言：“中美在网络空间优势互补、深度融合，互利共赢是历史的必然选择。”

中美互联网论坛从2007年至今已举办8次，其中“十二五”期间举办了四届。多年前当中国互联网协会和微软公司筹划举办中美互联网论坛时没有想到其重要性会有今天如此之高，如今的互联网论坛已经发展为探讨互联网世界潜在机遇与挑战的重要平台。

除了2011年前已创立的中美互联网论坛和中英互联网圆桌会议，“十二五”以来，中国又先后打造了新兴国家互联网圆桌会议、中韩互联网圆桌会议、中德互联网产业圆桌会议。一个个互联网交流平台的建立说明中国积极开展双边、多边互联网国际交流合作的迫切性。加强交流与合作，不仅是世界发展大势，也是互联网发展的大势。闭门造车，建设不了“网络强国”，只有相互借鉴、互相促进、互通有无，才利于国际互联网治理体系的建设。

发挥互联网在“一带一路”中的作用 开启新型国际区域合作平台

场景：2015年9月13日至14日，广西，南宁，中国-东盟信息港论坛。金秋时节，丹桂飘香，在风景如画的南宁，以“互联网+海上丝绸之路—合作·互利·共赢”为主题的中国-东盟信息港论坛举行。280多名参会代表就信息港建设的具体议题展开探讨，力图为区域内企业和民众释放“信息丝绸之路”的红利。13日，中国—东盟信息港正式揭牌，跨出信息港建设的坚实一步。

2014年9月，首届中国—东盟网络空间论坛成功举办，提出打造中国—东盟信息港，共建基础设施、信息共享、技术合作、经贸服务、人文交流五大平台，得到东盟代表的认同和积极响应。

一年后的9月13日，中国—东盟信息港论坛如约而至。一年来，五大平台建设全面推进，一大批重点合作领域初步确立，一批立足广西面向东盟的重点工程也相继落地，引领和带动着中国与东盟网络领域的全面交流和共同发展。

正如中国国家互联网信息办公室主任鲁炜在论坛开幕式上所说：“一年来的成功实践，让越来越多的人深刻认识到，中国—东盟信息港是网络互联之港、信息互通之港，是财富汇聚之港、文化交融之港，更是中国与东盟各国的合作共赢之港、成果共享之港。”

“十二五”期间在互联网国际交流合作取得的重大成果为“一带一路”的重大战略决策做出了积极贡献。2015年是国家建设“一带一路”的开局之年，在这个重要的时间点，中阿博览会网上丝绸之路论坛和中国-东盟信息港论坛先后在陆上丝绸之路重点省区宁夏和21世纪海上丝绸之路重要枢纽广西召开，显示我国充分发挥互联网在“一带一路”建设中的作用，主动将互联网、信息化融入国家“一带一路”的战略中。

同时，我国通过“一带一路”打造新型国际区域合作平台，发展与沿线国家的经济合作伙伴关系，共同打造政治互信、经济融合、文化包容的利益共同体、命运共同体和责任共同体。

中国成为全球互联网治理重要一极 发出负责任大国的声音

场景：2015年6月30日，巴西，圣保罗，全球互联网治理联盟首次会议。“近年来，中国正在积极探索符合自身实际，既推动发展又确保安全，既保障网民自由又维护网络秩序的治网之道”，中国国家互联网信息办公室主任鲁炜关于中国“治网之道”的发言在会场上引起共鸣，成为全球互联网治理联盟首次全体理事会的热点，受到委员们的高度关注。

这是鲁炜继2014年6月在互联网名称和数字地址分配机构（ICANN）伦敦会议上精彩发言后，再一次在全球互联网治理核心平台上阐述中国主张，发出中国声音。鲁炜在ICANN上发表主旨演讲，是ICANN成立16年来首次邀请非主办国官员在ICANN会议开幕式上作主旨演讲。

鲁炜在巴西的亮相也是他在2015年年初当选全球互联网治理联盟委员会成员后首次发言。2015年1月，全球互联网治理联盟经过投票选举,从来自全球的46位提名人选中选出20名委员会成员，中国两人入选。其中中国国家互联网信息办公室主任鲁炜代表中国政府，当选全球互联网治理联盟委员会成员。阿里巴巴董事局主席马云作为亚洲和大洋洲地区唯一的私营部门代表本次入选的成员。马云6月在巴西又当选为联盟理事会联合主席，他是共同主席中唯一来自亚洲的代表。这些都彰显了中国在互联网治理国际舞台上的重大突破，全球互联网治理已经无法缺少中国的声音。

一次次发言，一次次当选，说明随着实践的不断积累，中国正加速融入国际网络治理体系，利用各种平台向国外政府及组织等阐释中国互联网治理的立场和主张，并开始向国际网络空间贡献中国元素。另一方面说明，国际社会充分认可中国近年来，特别是“十二五”时期在互联网治理上所取得的突出成就，对中国的互联网治理能力充分信任，中国已成为全球互联网治理蓝图上不可或缺的重要一极。
