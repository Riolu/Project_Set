原标题：郑州16年前特大抢劫银行案告破

　　图为民警抓获犯罪嫌疑人。王洪岩　摄

　　“1999年12月5日发生的特大抢劫银行案告破。”10月27日上午,河南省郑州市公安局公开这一消息后,迅速在社会各界引起热议。

　　“案发这么长时间,公安机关怎么破的案?”“劫匪现在干什么?”《法制日报》记者采访了郑州市公安局犯罪侦查局相关人员,一一揭开这些谜底。

　　银行208万元巨款被抢

　　1999年12月5日19时左右,郑州市合作银行管城支行中药城批发市场分理处的大厅内,仍旧一片忙碌。保安在门口巡视,4名银行工作人员正在柜台内熟练地清点一天的营业款。19时20分,银行工作人员的清点整理工作进入尾声,大家心里变得轻松起来。

　　这时,突然进来3名30多岁的男子。保安立刻上前问他们干什么,3名男子没有理会,径直向柜台旁走去。此时,又有两名男子进入大厅,还没等保安开口,他们便各掏出一把手枪朝着保安的头部开枪。大厅内的人还没反应过来,保安已经倒在血泊中。正在忙碌的工作人员小白听到枪声后,随即按响了报警铃。

　　先进入大厅的几名男子拿出锤子朝柜台上面的玻璃一顿猛砸,很快砸开了两个洞。其中两人通过洞口钻入营业室内,开始实施抢劫。在不到5分钟的时间里,进入现场的5名犯罪嫌疑人抢走了3个共装有208万元巨款的款包,随后逃逸。

　　案发后,郑州市公安局随即成立1999“12・5”专案组,对此案展开侦查。

　　百名侦查员调查锤子去向

　　“案件发生后,郑州公安民警面临巨大的压力,当时穷尽一切侦查措施和手段,誓言案件不破决不收兵。”郑州市公安局分管刑侦的副局长李奎业对记者说,当年参与侦破的许多老侦查员如今已经退休,但这起案件一直是压在他们心头的一块石头。

　　“案发后,我们通过对现场留下的一段模糊视频和相关物证摸排侦查,初步确定了以物证找人、以刻画出的可疑人找人的思路,抽调大批警力查找犯罪嫌疑人使用的锤子、枪支、鞋印等物证来源,并对现场周边进行地毯式排查,寻找目击证人,以刻画嫌疑人的体貌特征及逃跑路线。”参与案件侦破的郑州市公安局犯罪侦查局局长马会强说。

　　采访中,当年负责寻找物证――锤子来源的郑州市公安局老侦查员巴西振对侦破工作记忆犹新。“其中一把锤子是山东临沂制造的,当年进入郑州市场有80把,已售出24把。为查明去向,上百名侦查员逐一调查锤子的去向,这无异于大海捞针,难度极大。可是侦查员没有放弃,连续作战不分昼夜,最终通过半年努力,查明了16把锤子的去向,仍有8把去向不明。”

　　“难度太大了,郑州的市场面向全省,后几年又陆续找到几把,可仍有一些没有找到,找到的后来也一一排除了嫌疑。”巴西振说。

　　专案组围绕现场留下的半枚带花纹鞋印,组织大批警力对全市鞋店和批发市场进行摸排,最终找到了几个与之接近的品牌鞋子,随即又派出警力远赴浙江、福建、广东等地,辗转上万公里寻找那只“鞋子”,依然没有取得重大突破。

　　但经过侦查,专案组还是初步断定几名犯罪嫌疑人为豫南口音,年龄在30岁左右。鉴于当时侦破工作的技术条件制约,案件虽几经周折,也陆续摸出了些线索,却长时间没有重大突破。

　　专案组的老侦查员到年龄退休了,年轻的侦查员继续进行侦查。

　　2003年1月5日,专案组成员、原管城公安分局局长常世忠在查证线索时累倒在岗位上,终因心源性猝死抢救无效牺牲。他曾在日记中这样写道,“案件破不掉,人民群众不满意,警察抬不起头,说啥也得把这个案子拿掉,好好出一口气,来捍卫警察荣誉”。当年3月,他被公安部追授为二级英雄模范。

　　刑侦技术发展带来转机

　　随着郑州市公安局刑事侦查技术的跨越式发展,刑侦民警每年都会把积案要案进行梳理,对相关证据和数据进行再次分析比对,以求取得突破。

　　2015年,郑州市公安局新一届党委成立,河南省公安厅副厅长、郑州市副市长、市公安局局长沈庆怀多次强调,要将各警种、各部门、各单位的科技手段、基础信息、警务保障等资源,最大限度地向侦查办案工作集中,切实提升侦查办案的整体合力,并且要求全市公安刑侦队伍通过对沉压积案的侦破工作和基础信息的采集比对,对历年来发生的重特大积案逐一研究,不留死角,不漏线索,进行强力侦破。

　　“我们今年对积案采取班子成员、中层干部、刑侦民警三级分包,采取专班上案、赴外追逃、信息采集、技术侦查、生物比对等措施,多策并举,多管齐下,集中开展重特大积案攻坚大会战,”马会强说,随着现代化刑侦技术的不断完善,一些线索和数据浮出水面。

　　“每当取出当年从案发现场提取回来的物证时,都觉得十分熟悉,可又觉得抬不起头。”巴西振说,当年专案组各种取证调查材料的卷宗,能堆满一间屋子。

　　借助现代化的刑侦技术,郑州市公安局刑侦技术人员经过几个月成百上千次反复检验和不懈努力,终于在案件现场的相关物证上找到突破口,并最终在这些检材数据中锁定了嫌疑人个体特征数据。之后的工作,就是比对和检测数据相符的群体。经过几个月努力,办案人员初步锁定在驻马店市的两个地方有和监测数据相符的群体。

　　随后,专案组根据线索,派出多个查证组在当地警方配合下,对相关乡镇村开展综合排查。

　　“在排查过程中,我们除了比对相关数据外,还走访群众了解周围人群案发前后身份及家庭状况的变化情况。”郑州市公安局犯罪侦查局副局长赵德孝说,这个排查过程异常艰辛,侦查员几乎天天围着村子转,他们穿着打扮和村民一样,在田间地头、大街小巷,一个一个地走访,既不能暴露身份,又得把每一个相关人、每一层关系都问清楚,几个月下来平均每个侦查员调查走访人员不少于两千名。

　　“排查几个月回来人也黑了,衣服也破旧了,孩子都认不出我了。”参与侦查的民警刘涛说。

　　今年10月18日,专案组结合生物科学的检材数据和情报研判、合成作战、传统侦查等手段,最终锁定石某等3名主要犯罪嫌疑人。

　　嫌疑人成资产过亿企业家

　　为防止出现意外,专案指挥部直到下达抓捕命令前夕,才把相关情况通报给每个行动小组的负责人。

　　10月21日8时许,参与抓捕的侦查员陆续到达驻马店市,与前期秘密侦查的民警汇合。负责抓捕一号嫌疑人的抓捕组,由郑州警方犯罪侦查局暴力犯罪侦查支队负责人申保成带领,他传达完指挥部的抓捕要求后,才将嫌疑人的照片等材料发到每一位参战民警手中。

　　这时候,大家才明白要抓捕的人是谁。

　　一号嫌疑人,也就是“12・5”抢劫银行案的策划者石某群,男,驻马店市人,今年53岁,当前身份是驻马店市身价不菲的地产开发商,名下拥有房地产开发公司、商贸公司、休闲农庄等多处产业,资产过亿,在当地商界有一定影响力。

　　“考虑到他们在当地的影响,指挥部决定等时机成熟时,三组同时实施抓捕,以防提前动手后惊动其他嫌疑目标。”一名现场指挥员说。

　　当天,计划中的一二三号嫌疑人全部被抓捕归案。

　　二号嫌疑人李某,46岁,驻马店人,与主犯石某群同村。据其供述,当年他们在石某群的一手策划下,实施了抢劫银行,共有5人参与作案,除石某群和余某外,还有陈某和石某群的亲弟弟石某春。

　　第三名嫌疑人陈某,正在老家打麻将时,被突然而至的民警当场抓获。

　　10月24日,最后一名嫌疑人,石某群的弟弟石某春,在其女友家也被抓获。

　　至此,震惊全国的“12・5”特大抢劫银行案5名嫌疑人全部落网。

　　十几年来没能睡一个好觉

　　据犯罪嫌疑人供述,石某群是一个比较有经济头脑的人。上世纪90年代初,他带领老家的一群人来郑州打工,主要在建筑工地上承揽一些零碎活。没几年,石某群就变成一个小有名气的包工头,手下也有百十号工人。

　　可很快石某群就不满现状,看到大老板的阔绰,再加上工程款不好要,他开始秘密计划一件事情。

　　据李某供述,当时他和余某等人都是刷墙工。1999年11月,被石某群叫在一起商量“大事”。“他当时给我们说有一个生意需要大家一起干,干好了大家都能挣一笔钱。”李某说,大家都没啥钱,也都知道石某群精明能干,就同意跟着他干。

　　其实,在石某群召集同伙之前,他已经秘密准备了三个月:从许昌购买5把改装手枪,在被抢银行周边踩点,设计周密的逃跑路线等。据石某群供述,作案前他已经掌握银行分理处的作息时间,就连作案时间都是他精心确定的。

　　案发前,石某群带着李某等4人来到郑州市合作银行管城支行中药城批发市场分理处。具体的分工是:余某和石某春每人一把枪控制保安,石某群带着李某、陈某拿锤子砸柜台玻璃成功后实施抢劫。

　　作案后,石某群等人抬着款袋,沿提前设计好的路线逃跑,最后骑着事前准备好的自行车逃回租住的房屋内。他们稍作停留就返回驻马店老家,然后由石某群带领去云南躲避风声。半个月后,他们再次返回驻马店开始分赃。李某等人分到15万元、20万元不等,其余都归石某群,这也成为他华丽转身的第一桶金。

　　一夜暴富的他们,都不敢把真实情况告诉家人,声称在云南赚了点钱。据石某群供述,他并没有急于花掉抢来的钱,而是在观察机会。2000年,他尝试在驻马店购买土地开发房地产,但并不是很成功。随后几年,房地产市场逐步升温,他相继开发了多处楼盘。

　　在16年的时间里,一起抢劫案主要犯罪嫌疑人就这样转身成为一名成功的地产商。石某群说:“多少钱能买来心安理得?十几年来没能睡一个好觉,心里没踏实过……”

　　本报郑州10月27日电　　记者　赵红旗
