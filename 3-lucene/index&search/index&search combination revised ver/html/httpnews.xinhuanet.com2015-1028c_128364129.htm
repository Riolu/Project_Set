提升服务品质 特斯拉钣喷中心布局全国市场

2015年10月28日 07:00:00 来源： 新华汽车
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网北京10月28日电（赵延心）三年前在model S正式上市之际，为了更好的服务于用户，特斯拉已经初步开展钣喷授权中心业务。当特斯拉汽车在日常遇到刮蹭或钣金损伤等现象时，用户可以及时前往钣喷中心进行维修。

    挑选优秀合作伙伴 员工经过严格考核

    有异于特斯拉一贯采用的直营模式，授权模式可以尽快开展维修服务，并派特斯拉官方员工进行驻点服务。挑选具备优秀资质的合作伙伴，确保与直营服务中心的服务水准相一致。所有授权钣喷中心的钣金技师，均在上岗前接受特斯拉官方的服务培训，只有通过严格的铝焊接等专业测试，方能持证上岗。在总部，特斯拉也在不断研发更好的维修方法，并将与合作伙伴进行资源共享。

    2014年，特斯拉已经在北京、上海、广州开展授权钣喷业务。特斯拉将在2016年加快钣喷中心授权速度。截至目前，全国市场已经有23家授权钣喷中心完成资质认证和员工培训等全部环节，另外7家授权店也即将正式成立。同时，从零部件下单到整个维修过程，特斯拉将对钣喷中心进行缜密监控，确保钣喷中心的维修服务达到最高水准。

    统一采用原厂配件 配件收费不超官方零售价

    特斯拉授权钣喷中心全部使用特斯拉原厂配件，维修工时收费按照当地工时费收取。由于零配件是特斯拉独立生产的，价格在市场具有较强竞争力，特斯拉严格限制授权钣喷中心的配件收费标准不得超过官方零售价格。
