复旦毕业生：要做“信用社交之父”

2015-02-03 15:48:12    来源：解放日报   

昨天，一场发布会在上海交大举行。沪上知名的民营创投孵化器——起点创业营宣布，对“速溶360”互联网金融平台进行千万元投资。“速溶360”由复旦大学软件学院毕业生薛梓闻创办，是一个专为在校大学生提供小额借款服务的P2P  网络平台。
　　发现大学生借贷市场空间
　　2005年毕业于复旦软件学院的小薛，毕业后进入文思创新软件外包公司。“我在那里经历了企业的大发展，包括公司在纽交所上市。”2012年，他跳槽到人人网。在这个拥有大量大学生用户的企业，他发现了在校大学生的一个需求：借钱。
　　“现在的大学生想买智能手机和平板电脑，想出远门旅行，所以需要钱，但很多人不愿问父母要。”小薛告诉记者。此外，银监会规定银行给已满18岁的学生发信用卡，必须经由父母等第二还款来源方的书面同意。这一规定让大学生办卡数量大幅减少。
　　经过市场调查，薛梓闻发现，大学生其实不缺还款来源。他们补一次课的收费在100元左右；在高校图书馆打一天工，能赚几百元。这表明，大学生只要借钱金额不高，是能通过勤工俭学偿还的。
　　找到这个市场空间后，他和两位在金融领域有成功创业经历的同伴合作创办了上海科巴度数据管理有限公司。
　　从2013年10月上线至今，速溶360网站和移动客户端的用户数已达13万，交易总额达4000万元，坏账率仅为千分之1.5，远低于大多数P2P网络借贷平台。
　　多重身份认证防范金融风险
　　公司运营总监章婕介绍，国内在校大学生（包括大专生、本科生、研究生）的学籍信息，都可在教育部的学信网上查到，这使速溶360平台能对每名借款人进行实名认证。除了学籍认证，平台还与公安系统对接，对借款人进行身份证认证，并要求用户提供手机号和电子信箱地址。尽管经过这几道认证，但仍存在掌握以上所有信息的人（如室友）冒名顶替的可能性。为此，平台还要求借款人进行视频认证——借款人须拿着自己的学生证和身份证，录一段视频上传网站。
　　完成身份认证后，学生就可以向平台上的投资人用户借款。本科生的借款最高额度是3000元，硕士研究生5000元，博士研究生7000元。借款年化利率为20%左右，由借款人选择。将利率调高，能增加借款成功的可能性。为确保金融安全，还款期限设定为6个月，正好是一个学期。发生借款人逾期不还款时，公司催收部会与学生联系；如失联1个月，再与所在学校的学生处联系。学校对大学生有监管义务，所以各校学生处了解情况后，都会把学生当前的联系方式告诉公司。通常情况下，学生接到催收电话后都会还款，谁也不想在学校档案里留下“污点”。
　　获得千万元融资后，小薛对未来显得信心满满，“我想把速溶360打造成全球首创的信用社交网站，这样我就能做‘信用社交之父’啦。”根据构想，这个网站今年将为每名借款用户设立“信用脸谱”，根据用户的还款记录、好友数量、活跃度等数据，算出他的信用指数，鼓励用户进行更多网络社交。
