    北京时间10月28日，2015-16赛季的NBA常规赛将打响。

　　下面就让我们用十个关键词，对新赛季进行一个展望：

　　1.归来

　　上赛季，杜兰特先后接受了两次脚踝手术，总共才打了27场常规赛，而雷霆也因为他的缺战，最终无缘季后赛。但今夏，杜少通过不懈努力，完成了康复。新季，杜少又恰逢合同年，因此无论从个人角度，还是球队层面，他归来后的状态都将格外引人关注。

　　同样受到伤病困扰的，还有尼克斯当家球星安东尼，上赛季，甜瓜因为膝伤早早报销。但训练营期间，他却强势归来，并展现出了不错的状态。新季，甜瓜是否能够引领球队重返季后赛，也是很多人所关心的。
