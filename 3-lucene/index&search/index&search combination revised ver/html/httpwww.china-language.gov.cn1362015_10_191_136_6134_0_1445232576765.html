教育部语用司调研“2015两岸大学生汉字文化创意大会”筹备工作

10月13日  来源：南京艺术学院

 

10月9日下午，教育部语用司司长姚喜双，江苏省教育厅语工处处长赵晓群一行就“2015两岸大学生汉字文化创意大会”筹备工作进行调研。南京艺术学院党委书记管向群，党委副书记、校长刘伟冬，副校长张承志等参加了调研座谈会。

　　座谈会上，南艺艺创处处长金捷汇报了承办工作的进展情况及下一步工作计划。校长刘伟冬表示要争取将本次活动打造成在两岸三地有影响力、号召力的汉字文化创意交流平台。校党委书记管向群表示，要制定详细宣传计划，加强对整个活动过程的动态宣传，圆满办好此次汉字文化创意大会。

　　姚喜双对南艺多年来为国家语言文字工作作出的贡献表示感谢。他强调此次两岸大学生汉字文化创意大会要加强宣传，提升影响；要以创意、展示、交流为主线，突出汉字在两岸文化认同方面的重要纽带作用。

　　南艺党校办、艺创处、戏剧影视学院等相关负责同志参加了此次座谈。
