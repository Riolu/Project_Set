美军歇兵三日坐看俄轰炸IS 辩称为避伤及平民

2015年10月28日 07:18:29 来源： 京华时报
  • 评论0
  • 打印
  • 字大
  • 字小

    根据美国国防部提供的消息，美国主导的打击“伊斯兰国”（IS)国际联盟连续三天没在叙利亚展开任何军事行动。与此同时，俄罗斯在叙利亚战场则捷报频传，在最近三天出动上百架次战机，空袭接近300个极端组织目标。

　　一些西方媒体不禁发问，美国最近在叙利亚突然没动静，是否与俄罗斯的空袭行动有关？五角大楼官员回应称：“真没关系。”

　　俄三天轰炸285个目标

　　法新社报道，23日至25日，美军主导的国际联盟没有在叙利亚开过一次火。五角大楼记录的最近一次空袭行动发生在22日，一架美军无人机轰炸了搭载“伊斯兰国”成员的车辆和藏匿弹药的地道。

　　俄罗斯方面则传来消息称，23日至25日，俄罗斯空军一共出动164架次战机，轰炸了285个“伊斯兰国”和“基地”组织在叙分支“支持阵线”目标，其中包括军火库和指挥所。

　　对于这一数字反差，美国官员显然话里有话。他们说，国际联盟在叙利亚的军事行动与俄方无关，空袭次数少恰恰表明他们的军事行动“选择更谨慎、目标更精准”。

　　五角大楼发言人杰夫·戴维斯26日说：“空袭次数本来就有多有少……我们需要查看情报，寻找可行目标，以避免伤及平民。事实很简单，我们就是最近没有发动空袭……但这并不意味着我们不打算继续空袭，事实上，我们将发动更多空袭。”

　　自2014年9月以来，美国联合60多个盟国对叙利亚境内的“伊斯兰国”极端组织采取军事行动，截至本月25日，一共发动2679次空袭。

　　自今年9月30日起，应叙利亚政府要求，俄罗斯空军开始轰炸叙利亚境内“伊斯兰国”目标，迄今为止已经摧毁超过800个目标。

　　俄罗斯寻求与美国在叙利亚军事行动中合作，遭到后者拒绝。此前俄罗斯总统普京表示，俄方有意派以总理梅德韦杰夫为首的高级代表团赴美直接对话，但被美方一口回绝。

　　政治解决叙危机徘徊不前

　　关于政治解决叙利亚危机的各种双边和多边会谈一直徘徊不前，没有实质进展。

　　普京26日与沙特阿拉伯国王萨勒曼通电话讨论叙利亚和中东地区局势。同一天，俄罗斯外长拉夫罗夫再次与美国国务卿克里通电话讨论叙利亚问题。这是俄美两国外长连续三天通电话讨论叙利亚问题。

　　法国牵头的叙利亚问题会谈27日在巴黎举行，美国国务院没有证实是否有美方代表参加。

　　23日，克里、拉夫罗夫、土耳其外长瑟纳尔勒奥卢和沙特阿拉伯外交大臣朱拜尔在奥地利首都维也纳举行会谈，但未达成实质内容。与会四方同意和所有各方协商，希望最早于本月30日召开一次更大范围的会谈。

　　俄罗斯认为，伊朗也是叙利亚问题的利益攸关方，应该加入多边会谈，但美国一直对此表示排斥。不过，美国国务院发言人约翰·柯比26日承认，美方意识到，叙利亚国内的政治进程最终还是需要伊朗参与。

　　柯比说：“他们（伊朗）是这一进程的利益相关者。他们确实与阿萨德政权有来往，与叙利亚国内势力有关系。国务卿（克里）很明白这是一个复杂的进程。这需要花费一些时间。最终问题解决需要每个人都做出一些让步。”
