景区展品“被扔钱”：有人往地牢扔 有人扔银行卡

2015年10月28日 08:55:53 来源： 中国新闻网
  • 评论0

　　中新网10月28日电(宋宇晟) 27日，广东省佛山市的祖庙博物馆中的一个铜像照片走红。在该照片中，铜像成为“功德箱”，钱币堆积成山。从图中看，大多数“捐出”的钱都是五毛和一块。而这已经不是景区展品第一次“被扔钱”了。不过，这“被扔钱”的地方和被扔的钱也不尽相同。

　　资料图：8月11日，西安，游客在楼观台景区游览时发现：文物上加装了玻璃罩后虽然保护了文物，可是却成了一些游客投掷零钞的功德箱、许愿池。图片来源：CFP视觉中国

　　不只是功德箱，玻璃罩、温泉、地牢、乌龟背上都能扔钱

　　在不少景区里，一些“神兽文物”常常被加装玻璃保护罩。这当然是为了保护文物。不过，在保护的同时，玻璃罩也给了这些“神兽”提供了接受“供奉”的机会。比如，西安景区的石牛、石狮就收到了大量一毛钱到一块钱不等的“贿赂”。

　　除了静态的“神兽”，一些还能动的动物也成了招财的吉祥物。今年9月，大连市一间海洋馆就被曝出，有游客为求好运，在景观池中丢掷钱币。而池中的养殖乌龟不仅在硬币堆内寸步难行，身上还背满了纸币。

　　如果说上面这些扔钱的行为还姑且能被看作是游人祈福，那么下面的这些就有些不好理解了。

　　例如，北京自然博物馆的恐龙骨架也收到了各种硬币、纸币。有媒体称，目测金额可达数百元。而重庆沙坪坝区白公馆的地牢和刑具上，也都有人扔钱。

　　资料图：游客在瓷房子里围观装满钱的大缸。　张道正摄

　　事实上，比起方方正正的玻璃罩，一些水井、水池、水缸更容易成为扔钱的目标。比如，天津瓷房子入门正中间的一口大瓷缸，已经被讨彩头的游客扔满了钱币，各种币值的纸币和硬币满溢。而趵突泉景区里的不少泉池底部也都有被扔进去的硬币。

　　资料图：数十年来累积在池中的垃圾、钱币和石头，降低了美国黄石公园温泉池水的温度，颜色也转变成橘、黄和绿色。

　　甚至连美国黄石公园的温泉也在劫难逃，不过这里的温泉似乎对人类的“馈赠”反应挺大。由于游客纷纷向温泉里投放硬币，过量的硬币、垃圾、落叶等使温泉变了颜色，还堵塞了通风口，减缓了水循环。温泉因此由蓝色变成黄绿色。

　　当然，如果没有水也没关系。游人总是会自行找到一些投掷的标靶。前几日，就有游人被拍到在开封府景区，向包公案前的三口铡刀的兽头嘴里扔钱。

　　资料图：漱玉泉中打捞上来的各种卡。图片来源：山东商报

　　不只是硬币，也能扔纸币、银行卡、剪发卡、校园卡、就诊卡

　　对于扔钱这种事，多数人还是偏好用硬币，不过这种习惯也会根据所处的环境有所变化。

　　作为游人祈福的对象，虽然西安石碑外部的玻璃罩没有缝隙，但还是挡不住游人的热情。不少游客把零钞卷起来，从上面留下的通气孔中塞进去。

　　除了能在市面上流通的货币，还有人显得更为有诚意。他们扔的是银行卡。趵突泉景区的漱玉泉水质清澈见底，池底的硬币、纸币也清晰可见，其中纸币以五角面值为主、硬币以一元钢镚儿居多。而工作人员还能在池中找到不少卡片，其中大部分是建行、工行、农行等银行卡或信用卡，除此之外，还有剪发卡、校园卡、就诊卡等，总共二十多张。

　　该景区的工作人员介绍，从卡片上的信息看，它们的主人很多是来自外地，“很多游客把这里当做许愿池，就把银行卡扔进去了”。

　　资料图：天津瓷房子景区工作人员清理大缸里的钱币。　张道正　摄

　　景区的应对办法：上交、捐出、兑换

　　对于这些扔出去的钱币，景区也都有一套处理办法。

　　趵突泉公园的一位工作人员就表示，针对这些“许愿币”，旺季时期景区大约半个月左右打捞一次，淡季一到两个月清理一次。“每次打捞完毕之后，还不断有游客将硬币投进水池。”打捞出的钱怎么处理？景区的工作人员表示，一般都是清理干净后上交市财政局。

　　天津瓷房子博物馆将大瓷缸里的钱币全部用于慈善事业。今年10月黄金周清理出的钱币则打包捐赠给天津市动物园，以提高动物的福利待遇。

　　资料图：老北京有一个传说，谁能将铜钱扔进白云观窝风桥下的大钱眼里，谁就能升官发财。在春节的白云观庙会上，游人们纷纷前来一试，都想为自己的前程讨个吉利。中新社图 杜洋 摄

　　而在北京的白云观，游客可以用十块钱换一堆铜板，然后乒乓开砸，即所谓的“打钱眼”。每天傍晚工作人员下梯子去捡，第二天再兑换。

　　英国著名景点白金汉宫前的喷水池里也散落着大量的硬币。据说每天都得清理。据英国媒体调查，2006年一年英国人花在扔硬币上的钱就有300万英镑，约3000万人民币。英国的人口约为6000万，折合到每个人身上约为半磅，约5块钱人民币。
