新华网招募社会责任大学生观察员

2015年10月28日 08:46:10 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网北京10月28日电（陈听雨） 近日，新华网启动了“大学生社会责任中国行”活动，目前正在全国范围内招募社会责任大学生观察员。新华网将组织大学生深入制造、能源、科技、健康、服装、金融等领域的企业，在生态文明、安全生产、慈善爱心、科技创新等方面开展企业调研。

    “大学生社会责任中国行”是新华网主办的2015年度社会责任公益盛典活动的重要组成部分之一。活动将充分发挥大学生对社会责任建设的热情，以及“奉献、客观、准确、透明”的精神，对企业履行社会责任情况进行考察。通过实地调研，让更多的大学生理解社会责任的范畴与概念，树立大学生的社会责任观念，带动社会责任的整体发展。

    据活动主办方介绍，新华网的招募工作目前已经开始，计划共招募大学生30名。希望参与的大学生可通过新华网官方网站提交申请表完成报名（网址为http://www.xinhuanet.com/tech/zt/2015CSRclass/3.htm）。组委会将根据报名大学生的专业方向、社会实践内容等基本情况进行筛选。入选者将授予“社会责任观察员”证书，参与“大学生社会责任中国行”走进企业活动。

    根据活动安排，11月开始，大学生社会责任观察员将走进企业调研。调研活动将分为四部分内容。首先，将调研企业履行社会责任的情况，包括企业有违社会责任的行为，并收集社会公众及相关机构对企业的意见和建议。其次，大学生将通过微博、微信等自媒体平台发布“企业探访纪实”。第三，大学生将根据调研情况撰写企业社会责任调研报告，并报活动组委会进行评审。最后，通过专家评审后，入围的优秀报告将在新华网举办的公益盛典活动上公布和表彰。

    据介绍，企业调研内容包括生态文明、安全生产、慈善爱心、科技创新等领域。在生态文明方面，将围绕绿色发展、循环发展、低碳发展、环境资源保护等方面，对企业资源利用方式、生产结构变化等方面进行调研；在安全生产方面，将根据2015年全国安全生产工作会议的重点要求，对企业在保障员工安全、保障社会安全等方面的责任建设进行调研；在慈善爱心方面，调研将关注企业慈善公益项目的建设与投入、对弱势群体的关注与帮助；在科技创新方面，观察员将调研企业在技术研发、科技投入、创新成果等方面展现的实力与能力。
