交大校园无线局域网使用须知

1. 校园无线局域网使用WIFI技术，

支持2.4G/5.8G频段下的802.11a/g/n制式，不支持802.11b制式(CCK)

2. 当你打开笔记本搜索无线信号时，可能检测到不止一个无线信号，交大网络信息中心提供的交大无线信号有两个，分别为：

  • SJTU
    • 通过802.1x认证的安全无线网，推荐使用。
  • SJTU-Web
    • 网页认证的无线网，安全性稍低

另外还有运营商的无线信号

  • CMCC
  • CMCC-EDU
  • ChinaUnicom

3. 目前校园无线网免费对教工和学生开放，认证采用网络信息中心提供的统一身份认证(Jaccount)平台，该平台也应用于交大邮箱等系统。

4. 无线网带宽资源有限，为了满足大多数人的正常使用，严禁通过无线网P2P文件共享，一经发现，网络信息中心有关闭其无线上网的权利。

5. 无线网有一定的开放性，所以请做好计算机安全防护，打好系统补丁、安装防病毒软件，注意保护用户名和密码等个人信息。 

关于“安全”的含义：

在传输加密的无线网络里，监听者可以获知用户在传输数据，但无法获知数据中被加密保护的信息，包括IP地址、访问链接、提交的信息、下载的内容等。

在没有加密的无线网中，无线传输报文可被轻易获取并确定原始信息，访问一些设计不良的网站甚至可能会暴露用户名、密码等所有登陆信息。若通过https连接方式，可以保护数据内容，但仍会暴露访问的站点。
