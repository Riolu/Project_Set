关于在本市教育系统开展纪念中国人民抗日战争暨世界反法西斯战争胜利70周年系列活动的通知

2015-06-26 12:25:44    来源：上海教育新闻网   

各高校，各区县教育局及各直属单位：

今年是中国人民抗日战争暨世界反法西斯战争胜利70周年。为贯彻落实《中共中央、国务院关于举行中国人民抗日战争暨世界反法西斯战争胜利70周年纪念活动及其他活动的通知》和《上海市关于纪念抗日战争暨世界反法西斯战争胜利70周年主要纪念活动的安排情况》等相关文件的要求，经研究，决定在本市教育系统开展举办“纪念中国人民抗日战争暨世界反法西斯战争胜利70周年”系列活动。现将有关事项通知如下:

一、活动主题

以“铭记历史、缅怀先烈、珍爱和平、开创未来”为主题，大力培育和践行社会主义核心价值观，准确阐释上海在抗日战争、世界反法西斯战争中的特殊历史地位和重要作用，全面客观、生动地反映抗战历史，突出时代特征、上海特点和教育特色，深入挖掘所在区域、有关学校的特殊历史资源和特色教育资源，大力弘扬爱国主义教育，在教育系统内营造浓厚的爱国主义氛围，激发广大师生的爱国情怀。

二、活动内容

根据中央和上海市的相关文件精神，主要包括“1+5”系列主题活动。在落实中央和上海市的有关要求的同时，各高校、区县教育局要结合各自实际，举办各类主题活动和多种形式的纪念活动。 

（一）认真组织好“开学第一课”

9月上旬将是整个活动的高潮。根据中央和上海市相关文件精神，大中小学要以“纪念中国人民抗日战争暨世界反法西斯战争胜利70周年”为主题，通过形势政策课、主题班会、思想品德课、教师专题学习、开学典礼、主题升旗仪式等渠道，组织好开学第一堂爱国主义教育课。各单位要根据中央的宣传口径，制作一批专题教育材料组织师生学习讨论，邀请一批相关领域专家、战争亲历者和抗日老兵走进校园讲课。同时，充分利用好在易班网上开设的活动专区，组织师生上网观看“专题播放”、“专题讲座”和历史纪实资料，宣传抗战人物、介绍抗战事迹，大力宣扬中华民族在抗日战争中所表现出来的民族精神。此前，各单位要充分利用暑假，组织学生开展好寻访红色足迹、弘扬革命传统系列活动。

（二）积极组织参加5个主题活动

1.联展活动。8月中旬，举办“抗战中的上海教育”专题展览、制作“抗战中的上海教育”专题纪录片，开展“走近边防线”上海市中学生踏访边防线追寻抗战足迹活动，挖掘上海各级各类学校在抗日战争中涌现出的英雄人物、英雄事迹，通过图片、文献介绍、实物陈列和新媒体传播等形式，大力宣传上海教育系统为抗日战争胜利作出的历史贡献。各单位要积极参与相关资料的征集工作，组织发动师生主动参与学习，接受爱国主义教育的洗礼。举办“青春在军营放飞”全国军体训练营活动，弘扬崇军尚武精神、展示大中学生时代风貌。

2.联映活动。根据教育部和市文教结合工作的要求，将成立大学生电影育人联盟，深入开展“百年树人——阳光电影行”活动，为各单位提供一批以抗日战争为背景的爱国主义影片，并配套制定相关放映电影的补贴机制，推动一批优秀的主旋律影片走进校园。同时，在易班网上开展影评等活动，让广大师生重温抗战的历史，深刻认识和理解抗战的意义。各单位要主动引进相关影片、积极组织师生观影和撰写影评。

3.联演活动。自5月份开始，结合“高雅艺术进校园”、“相约经典——大学生公益票”等文教结合项目，市教委与市文广局陆续推出一批以抗战为题材背景、适合爱国主义教育的优秀艺术作品走进校园。各单位要积极组织师生观摩，将优秀的文艺作品引进校园。同时，要鼓励学生创作相关文艺作品，深入社区开展文艺展演活动。

4.联唱活动。根据市委宣传部的统一部署，将于“七一”前夕举办2015年上海市校歌及爱国歌曲大合唱活动，作为全市抗战歌曲大联唱暨第五届上海合唱节展演活动的重要组成部分。各单位要组织广大师生以歌寄情，通过歌曲联唱的形式呈现厚重的历史场景，缅怀先烈、凝聚民族力量。

5.联讲活动。市教委与市委宣传部、团市委将共同开展上海市爱国主义教育基地抗战经典故事汇和“下一站公益”志愿讲解活动。各单位要积极引导师生发扬志愿精神，主动走进各级各类爱国主义教育基地、抗战遗址，收集和学习一批抗战故事，争做义务讲解志愿者，为广大市民和师生讲述抗战故事。 

（三）配套做好相关纪念活动

各高校、区县教育局要结合各自实际，创新形式，积极开展以“铭记历史、缅怀先烈、珍爱和平、开创未来”为主题的形式多样的宣传教育活动。市教卫工作党委、市教委将于下半年对各单位的活动项目进行评选和展示，其中优秀活动项目将报教育部和市委宣传部。

三、相关工作要求

（一）提高认识，加强领导

各级党组织、各单位要增强政治意识，认真学习习近平总书记系列重要讲话和相关文件精神，充分认识开展纪念中国人民抗日战争暨反法西斯战争胜利70周年系列活动的重要性和必要性，积极参与全市统一组织的各项活动，不断扩大宣传力度，严格遵循有关宣传口径，有序、有力、有效地推进纪念活动的开展。

（二）统筹安排，加强管理

各单位在组织开展各自纪念活动的工作中，要严格落实勤俭办会、办活动的要求，加强对活动的整体考虑和设计，严格审查活动内容和形式，尤其要加强对涉外纪念活动的管理。对参与人数多、涉及面广、影响力强、涉及敏感问题的活动，各高校要事先报市教卫工作党委审批，各区县教育局要报区委、区政府审批，并向市教卫工作党委报备，不得擅自开展。

（三）重点突出，体现特色

各高校、区县教育局在开展相关纪念活动过程中，要积极结合自身具体情况，深入挖掘特殊历史资源和特色教育资源，充分发挥纪念活动的育人作用。

（四）系统实施，做好总结

在系列活动结束后，各高校、区县教育局要对纪念活动进行系统总结，对各基层单位好的做法和优秀项目进行表彰和展示。


 


 

中共上海市教育卫生工作委员会

上海市教育委员会
