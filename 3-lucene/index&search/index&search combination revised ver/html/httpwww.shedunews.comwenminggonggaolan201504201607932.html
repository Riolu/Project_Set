2015年上海大学生艺术实践基地育人联盟研究课题指南

2015-04-20 13:54:29    来源：上海教育新闻网   

为进一步提升上海大学生艺术实践基地育人联盟学术水平，理顺机制，形成合力，现开展《2015年上海大学生艺术实践基地育人联盟研究课题》申报工作，发布指南如下：

一、申报对象

上海高校艺术教育教师、上海大学生艺术实践基地教师

二、相关事项

1. 依据本指南选题。

2. 需经所在高校文明办（或宣传部）审核，签署意见及盖章后上报。

3. 2015年5月16日前统一上报至上海大学生艺术实践基地育人联盟秘书处（纸质版快递至浦东新区蔡伦路1200号行政楼410室，电子版发送至yssjjd@163.com）

4. 重点课题资助经费8000元，一般课题资助经费3000元，在结项后统一划拨。

5. 结题时限为2016年4月30日，需提交研究报告和成果性文件。

三、选题

1. 艺术教育培育和践行社会主义核心价值观的路径研究

2. 艺术教育与社会主义核心价值观的关系研究

3. 艺术教育传承优秀传统文化的路径研究

4. 艺术教育德育功能的海外研究

5. 艺术教育发展情况的多国别研究

6. 上海高校艺术教育现状调查研究

7. 高校艺术课程的德育设计研究

8. 艺术作品的德育功能研究

9. 艺术活动设计的德育功能研究

10. 艺术育人的大中小学一体化研究

11. 艺术实践基地育人经验回顾研究

12. 艺术实践基地品牌建设回顾研究

13. 艺术实践基地发展历程回顾研究

14. 专业艺术门类的育人功能研究（门类自拟）


 


 

附件：2015年度上海大学生艺术实践基地育人联盟研究课题申请书
