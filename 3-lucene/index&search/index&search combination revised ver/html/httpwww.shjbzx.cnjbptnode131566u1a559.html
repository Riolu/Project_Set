盗窃QQ号诈骗5万元获刑
2013年12月12日 10:42

　　25岁的广西农民韦某利用木马病毒盗取某留学生的QQ号，后冒充对方与其母郑女士聊天，并借此机会骗走郑女士5万元。经查，韦某等人在网页种植木马程序，后盗取浏览该网页人员的QQ账号和密码，并对IP地址进行筛选，找出在澳大利亚的留学生QQ号码，通过查看好友备注，寻找亲友进行诈骗。韦某和古氏兄弟分工明确，有人负责聊天，有人负责接电话，有人则负责取款。

　　近日，韦某被控诈骗罪在朝阳法院受审，并当庭被判有期徒刑1年3个月，罚金4000元。
