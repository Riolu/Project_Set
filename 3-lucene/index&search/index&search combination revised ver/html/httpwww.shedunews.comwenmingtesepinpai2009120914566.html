文化建设引领 构建和谐校园

2009-12-09 14:43:29    来源：上海市复旦中学   


一、项目背景

上海市复旦中学作为百年名校，作为曾经拥有孙中山先生首席校董，拥有马相伯、严复、于佑任、邵力子、李登辉等创校先贤的“沪滨屹立东南冠”的历史名校，一百多年来，学校始终秉承创校先贤办学思想，以育才强国为宗旨，崇尚科学，弘扬人文，博学笃志，自强发展，逐渐形成了具有自身特色的教育思想、教学理念与教育追求，积淀了丰富而独特的学校文化。近年来，学校先后被评为上海市文明单位、上海市行为规范示范学校等，学校教育质量始终处在同类学校前例，享有较高的社会声誉。

今天的复旦如何在辉煌中继续前行？我们确立了“文化立校，自强育人”的办学理念，“固本”、“活化”的基本策略，其核心内容就是“文化建设引领 构建和谐校园”。


二、工作思路

㈠ 学校文化的认同与稳定的学校群体文化心理，是建设和谐校园的基础

学校文化建设引领具有以下五个特征：创造性特征——一个学校在其发展过程中人们思想活动的印记大于社会文化中的约定俗成，具有孕育先进文化的特性；指向性特征——学校文化有其特定的价值指向，即教育价值观；引领性特征——学校文化对社会文化的发展起到促进、引领功能；交融性选判性特征——社会多元文化汇集于学校，学校文化在自身构建中有一个选择、容纳和优化的特性；教育性特征——学校通过文化思想和文化行为向学生传承人类的先进文化，培育、提升学生的文化素养。一个学校之所以被命名为文明单位，更着重表明这个学校文化发展代表着健康、进取、积极的社会主流文化；表明其文化发展既是历史的也是现代的，更是指向未来的；表明这个学校文化发展所表现出的进步状态和公众对这种状态的认同。

㈡ 学校办学思想与学校精神是和谐校园建设的文化内核

复旦中学的办学思想来源于丰厚的人文沃土之中，提炼于学校师生的实践智慧之中，体现在和谐校园的建设之中；能在复旦的历史中找到它的根与魂，传承这个根与魂，是复旦作为历史名校和谐发展的不竭动力源。因此，我们坚持复旦“完人”培养的教育本色，努力朔造“文化复旦、和谐复旦、创新复旦、优质复旦”。

㈢ 学校的科学与人文相济的生态环境突出文化与和谐的学校个性特征

在复旦校园的整体布局中凸现“文明、整洁、清新、雅静、富有复旦历史内涵和现代复旦张力的文化标志。让校歌、校徽、校旗、校花等复旦文化标识，与师生的日常生活融为一体；复旦中学在创建和谐校园中，学校工作的每一个细节，使每个复旦人都能清晰地感觉到事事皆教育、处处有文化。
 
三、基本做法

㈠ 依托教师文化共同体建设，构建和谐校园

教师是建设文化复旦、和谐复旦的关键，我们着重三大共同体的建设。一是学校导师组：这是传承复旦文化、构建和谐校园的引领者。其职责就在于评教、议教、督教、咨询管理工作中传承复旦的文化与教育思想。引领青年教师。二是青年教师研究会：这是传承复旦文化、构建和谐校园的中坚者。我们把青年教师研究会办成“聚合青年教师、培养青年教师、自我发展青年教师”的平台，是传承和创新复旦文化的共同体。三是教代会：这是传承复旦文化、构建和谐校园的的推进者。教代会承载着教师群体的文化意识和行为，承载着文化发展的基本框架，它既创造了制度文化，又发展了学校文化内涵。

㈡ 提升教师品格境界，实现三个追求目标

一是人文素养的五种境界：“默默无闻，但默默有为”的事业境界；“才出于学，器出于养”的素养境界；“团结、服务、牺牲”的奉献境界；“清净莲花”的气节境界；“仁怀、大度” 的君子境界。二是师德修养的“五者”要求：追求“长者、仁者、智者、爱者、教者”的良好教师风范；三是为事、为人的五个行为追求：以善待生命之心尊重人，以宽容厚德之心接纳人，以真诚和睦之心关怀人，以通力协作之心团结人，以着眼未来之心发展人。在此基础上构建复旦学校的教师文化。

㈢ 开设“相辉文化讲坛”和“力学读书论坛”，锻造复旦文化平台

复旦“‘相辉’文化讲坛”是面向全体师生的具有复旦特色的校本讲座，是我们建设“文化复旦”的重要抓手之一。“‘力学’读书论坛”复旦文化建设和教师专业发展的一个重要平台。两年来，我们将这两项活动课程化和制度化，提升复旦的文化品位，提升教师的整体素质，提升师生的生命智慧，取得了良好的成效。

㈣ 文化导引、道德践行、培养德性，锻造复旦师生的文化品质

文化导引、道德践行、德性培养是复旦“以文化人，构建和谐校园”的重要策略。我们开设了复旦文化校本课程。课程包括《寻梦复旦园》、《走进中国文化》、《中西文化源比较研究》等，引领全体师生爱校、爱家、爱国，把自已锻造成既是文化被熏陶者，又是文化践行者，更是文化发展的创新者。
学生社团是校园文化建设中不可或缺的一部分，我们坚持使社团成为知识转化为能力的通道；又是培养学生合作精神、创新精神的重要文化载体。

㈤ 深化凝聚力工程，增强创建和谐校园的聚合力

深化凝聚力工程，是形成学校文化聚合力和张力的有力保证，也是建设和谐校园的可靠保障。在这一工作中，人是第一位的，因此，坚持以凝聚人、团结人、帮助人、关爱人、发展人，以创建和谐氛围为根本，以教研组为载体，以创文明班组为重点，以办实事为依托，以制度为保障，将常规工作做扎实，将传统的基本的事务做出新意，让教职工们始终有愉悦的心境，良好的工作氛围，更沉着，理性，自信地面对未来。


四、主要成效

㈠ 校风：德性养成、人文彰显、素质文明

建设和谐校园创建工作有力的提高了全校师生精神风貌和文明素养。我校重视师德师风建设，每年开展师德师风活动，不断提升教师的道德和文明素养，营造了全员育人的良好氛围。我们确立了“笃志：爱国爱民 服务社会；博学：尊重科学 德明才具；切问：严谨治学 追求卓越；近思：勤于思索 创造未来”的复旦育人目标。学生整体朝气蓬勃，自信而不自傲，乐观而不盲目，从学生参加的各项学校活动和社会实践中，都可以看到自强、自信、自律、谦和和精神风格，学生对学校、对复旦文化和和谐校园的认同感、归属感与荣誉感也日益增强。

㈡ 教风：严谨技谌、内涵发展、绩效明显

我们发展了学校“德教一体”的管理模式，强化教学处、德育处功能从单一的管理向“指导、培训、服务、评估”综合性管理方向发展。健全校本研修、强调复旦“五认真一加强”的传统，开展主备课制、二课三反思等研修策略，聚焦课堂，深化课改，提升教师专业发展的教学管理制度，每学期课常教学调研制度，从制度建设上保障课改落到实处。脚踏实地、严谨技谌教学风气蔚然成风。

㈢ 学风：博学笃志、切问近思、品质提升

在文化复旦、和谐校园建设中，学生进一步理解了复旦校训“博学而笃志，切问而近思”的深刻内涵，明确了复旦学生学习的“五认真一加强”和“清、熟、活”的要求，学习风气日益改善，学习效果稳步提升。我校的学生参加各项竞赛也是硕果累累，谨今年就有111名同学荣获全国、市、区各类学科竞赛奖项。


五、书记感言

复旦有百年历史，百年精华，拥有经验与文化的积淀，也有开拓与创新的精神，为此学校始终坚持以实践能力为重，坚持以文化凝聚人、团结人、帮助人、关爱人、发展人，以创建和谐氛围为根本，以“相辉文化讲坛”与“力学读书论坛”为载体，提升师生文化素养和学校文化品格，让师生始终有愉悦的心情，良好的工作、学习氛围，更沉着，理性，自信的面对未来。创建和谐校园不是一项阶段性、突击性的工作，她是伴随学校生命始终的。因此，只有持之以恒、长抓不懈，并与学生发展、教师发展紧密结合，才能真正见成效。复旦中学在这方面做了许多工作，取得了颇丰的成果。
