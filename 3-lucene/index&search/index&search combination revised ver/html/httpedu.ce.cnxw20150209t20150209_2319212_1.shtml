校园足球开展已多年 怎样开启2.0模式？(图)

2015年02月09日 09:28   来源：羊城晚报   

　　校园足球经费从何而来

　　在推广校园足球过程中，经费的缺失往往成为各个关节的障碍。在校园足球推广的前五年，国家体育总局每年只拨付5000万元作为全国校园足球经费，平摊下去无疑是杯水车薪。如今在广州，304所足球推广学校每间学校每年也仅有1.5万元专项补贴，即便省、市级的传统体育项目学校，每年也不过5万元和3万元，恐怕连一名足球专项教练的年薪都不够支付。以真光中学为例，每年的足球经费在全市排名前列，达20万元，但校方仍感到捉襟见肘。

　　解决方法：教育经费支出结构上向体育倾斜，中央财政出一部分钱用于全国性校园足球教学、研究、师资培训、聘请高水平教练以及全国性赛事。另外通过市场运作，寻找企业和社会团体资助，比如富力地产每年出资200万元，冠名广州青少年校园足球联赛。另外建议从中国体育彩票公益金中抽取部分，设立扶持校园足球的专项基金。最后，随着校园联赛日益壮大及产生品牌效应，可以参考美国等国家，出售校园足球联赛转播权，主打家庭参与和观众体验牌。

　　快乐足球如何去功利化

　　在推行校园足球的过程中有一个误区，认为推广校园足球就是单纯增加足球人口，最终服务于竞技。因此有些学校在行动上仅体现为办一个足球特长班，多招几个足球特长生来撑撑门面。但教育部门早有明示——足球进入校园体育课程，是一种普及型的通识性教育，推广的是一种运动技巧和方式。通过这项入门门槛低的运动，吸引更多的孩子爱上体育运动，从中获得健康的身体，增强人际关系的体验。

　　解决方法：小学阶段的体育乃至足球课程，首要任务是激发孩子参与体育运动的兴趣。在进行足球课程教学时，应以小场地、多球门、多触球、多射门的练习为主，辅以半游戏性质的教学赛。三年级以下的学生不必设置校级比赛，高年级学生参加比赛也不要“成绩挂帅”，要慢慢培养孩子的兴趣。

　　足球尖子的出路在何方

　　在发展校园足球过程中会涌现一批足球尖子，但他们往往会面对学业和足球的两难选择。现在体育特长生取消高考加分，但又要大力发展校园足球，这就形成了矛盾。另外，要走职业道路，势必要牺牲大量学习时间，一旦文化课跟不上，走专业足球道路又遭到淘汰的话，孩子便几乎看不到出路。

　　解决方法：孩子小时候不应走精英培训这条路，在校园多学文化，反而会令踢球更有灵性。教育更要改革，不应以分数论英雄，而是以特色人才为参考标准，独辟蹊径为足球人才升高中考大学创造上升通道。如果学生想走上专业足球之路，一是去业余体校提升能力，二是通过各种U系列的比赛，争取脱颖而出进入职业联赛梯队。最理想的情况当然是校园足球水平逐渐发展到较高水平，学生可以直接从校园进职业队、国家队，一边读书一边踢球。

　　何时有统一的教学教材

　　校园足球要大力推广的口号虽然喊得响亮，但至今并没有一套通行的针对小学的足球教材，以致各学校足球教学各自为政、良莠不齐，令校园足球难以规范和系统开展。反观足球开展得较好的日本、荷兰、德国等国家，有相当规范的足球教学教材。比如日本的足球青训大纲，U6、U8以及U10年龄段小学生校内都有统一的教材，这些教材都是邀请国外专家根据本国足球发展特点研究出来的，什么年龄段需要练习什么，细节定得十分规范细致。

　　解决方法：国家方面应该制定全国统一的学习大纲，确定在不同年级学习相应的足球技能，循序渐进。据了解，包括广州体院在内的不少院校已开始着手编写教材，一年内可能相继推出。荔湾区教育局也准备邀请一些专家编写校园足球教材。记者采访时得知，有着深厚足球底蕴的后乐园街小学已经编写并改编了一套足球教材《乐园足球》，以游戏的形式培养孩子对足球的兴趣。有关方面可以将各方面的资源收拢一处，也可参考国外优秀范本，制定符合我国特色、青少年身体特点的统一学习大纲。

　　如何调动校领导积极性

　　记者在采访过程中发现，有些学校虽然被列入304所校园足球项目推广学校，但一提校园足球话题，有些校领导避而不谈，有些直接摇头表示不打算开展。有些学校考虑的是场地缺失的问题，有些则表示本校已有其他特色项目，比如游泳、击剑等，不需要再另起炉灶搞足球。有些学校硬件很好，拥有标准足球场地，却因为领导不重视不喜欢足球，或是怕学生从事足球运动会出事，怕承担责任，导致优质资源白白浪费。有些则因为机制僵硬，教练付出努力却得不到应有的回报，导致他们的积极性被消磨掉。

　　解决方法：在发展校园足球的过程中，学校领导重视往往具有决定性的影响。在场地、经费、教练到位的情况下，学校领导需要与时俱进，摈弃陈旧观念，不应单纯以分数论英雄。另外，引导得当的话，鼓励学生参与体育运动与提高文化成绩能够相辅相成。与此同时，完善保险制度，能够减少在体育运动中产生的创伤纠纷问题。在提高教练积极性方面，荔湾教育局已经出台方案，明确规定：凡获得亚足联和中国足协颁发的各级足球教练员、裁判员证书的教师，评职称时作为重要条件之一。对优秀的布局学校给予奖励，做出突出贡献的教育部门和相关人员给予奖励。（羊城晚报记者 苏荇 陈泳强 李斯睿)
