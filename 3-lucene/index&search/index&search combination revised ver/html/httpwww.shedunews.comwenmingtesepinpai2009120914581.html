《探索·实践·跨越——2007—2008年度上海市和谐校园创建案例集》前言

2009-12-09 14:44:36       作者：李宣海

 阳春三月，凝聚着上海市教育系统文明创建工作者智慧和心血的《探索·实践·跨越——2007—2008年度上海市和谐校园创建案例集》与大家见面了

 

和谐校园建设是上海市教育系统为更好地贯彻落实《中共中央关于构建社会主义和谐社会若干重大问题的决定》和党的十七大精神，结合多年文明单位创建工作实践而推出的一项改革举措，是文明创建工作在内涵上的提升和外延上的拓展。

 

两年来，在中共上海市委、市文明办的正确领导下，上海市教育系统的和谐校园创建工作以邓小平理论、“三个代表”重要思想、科学发展观为指导，取得了可喜成绩，文明创建从以往重在形态文明发展到更为重视功能文明和素质文明；基础文明、基层文明、基地文明实现“三位一体”协调统一。同时，各级各类学校不断探索新思路，推出新举措，寻找新载体，建立新机制，涌现出了许多富有特色的工作方法，并形成了鲜明的特点。一是着力发掘创建工作的内涵深度，紧紧围绕学校中心任务开展活动，并通过加强文化建设来发展壮大学校文化软实力，切实提升创建内涵发展水平。二是不断探索创建新路，特别是在基层创建中注重载体创新，强调“以人为本”，尊重广大师生员工的创建热情。三是努力实践科学发展观，结合学校全局性、前瞻性课题的研究，使和谐校园创建工作符合学校改革发展的需求，既提高了师生文明素质，又增强了学校的核心竞争力。四是日益凸显创建工作在理论与实践结合上的跨越，在发扬思想政治教育工作优良传统的同时，借助现代科学技术知识，将先进的创建理念付诸实践。五是重视发挥学校文明创建对于社会的辐射作用，持续提高学校与社会的文明共建程度。

 

不断归纳、提炼行之有效的典型经验，加强和谐校园创建的理论和实践研究，是文明创建工作与时俱进的需要。因此，市教卫党委系统文明办在前期整理、总结的基础上，编辑出版案例集，展示和谐校园创建的成果，重点介绍开展思想教育、文化建设、基层创建、共建共享等方面的生动事例，这是一件很有意义的工作。如果这些案例能让大家有所借鉴参考，并增进相互之间的学习交流，必将有助于各单位进一步拓宽思路去开展各类体现自身特色的实践活动，使全市教育系统的和谐校园创建工作提升到更高水平。
