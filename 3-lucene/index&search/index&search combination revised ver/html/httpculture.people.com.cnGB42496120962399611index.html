唐禾香，女，京剧旦角。1976年出生。荀派传人孙毓敏弟子、中国京剧院优秀青年演员。能演绎花旦、刀马旦、青衣等不同行当的剧目。并参加多部影视剧的拍摄：《走向共和》、《五月槐花香》、《天上人间》、《七仙女》。
