本测试定名为“普通话水平测试”(PUTONGHUA SHUIPING CESHI，缩写为PSC)。 　　普通话水平测试测查应试人的普通话规范程度、熟练程度，认定其普通话水平等级，属于标准参照性考试。本大纲规定测试的内容、范围、题型及评分系统。 　　普通话水平测试以口试方式进行。
