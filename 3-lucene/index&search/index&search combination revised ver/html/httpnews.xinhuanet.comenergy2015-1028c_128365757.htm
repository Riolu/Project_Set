中国环保部启动京津冀、长三角、珠三角地区战略环评

2015年10月28日 07:45:00 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网北京１０月２７日电（记者杨维汉　崔静）中国环保部副部长潘岳２７日说，环保部根据严守空间红线、总量红线、准入红线——“三条铁线”的要求，启动对京津冀等三大地区进行战略环境影响评价。

    这是记者从环境保护部２７日在京召开的京津冀、长三角、珠三角三大地区战略环境评价项目启动会上了解到的。

    潘岳在启动会上说：“这三大地区是中国经济发展的重心所在，也是环境矛盾最凸显，公众环保需求最强的地区，是经济和环境双转型最迫切的地区，理应成为新常态下破解经济与环境两难的改革示范区。”

    “在三大地区的战略环评中，将用空间红线来约束无序开发，守住生态底线；用总量红线来调控开发的规模和强度，根据环境质量来分配控制重点行业污染物排放总量，使重点产业发展规模控制在资源环境可承载范围之内；用准入红线推动经济转型，强化产业准入源头控制，明确资源型、风险型、污染型和行业差别化准入管理要求。”潘岳分析指出。

    为了推动战略环评，环保部最近正在全力争取有关部门的支持来推动环评法、规划环评条例和建设项目环保管理条例的修订；城市建设、矿产资源开发等领域的规划环评与项目环评的联动机制正在完善中；建设项目环境保护事中事后监督管理办法、建设项目环境影响后评价管理办法、环评区域限批办法都在紧锣密鼓地制定中。

    环保部还将配合有关部门，严肃环评违法行为责任追究。潘岳说，过去规划“未评先批”、项目“未批先建”现象大量存在的一个重要原因，就是责任追究不到位。现在，环保部将依据新环保法和《党政领导干部生态环境损害责任追究办法（试行）》的相关规定制订《规划环评责任追究办法》，让违法责任人付出代价。

    据悉，环保部目前正在对河北等１６个省区市的化工园区涉危涉化重点建设项目开展环评专项检查。
