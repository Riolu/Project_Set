“交大移动”手机客户端APP正式发布公测！

2015-05-19  点击：[]

 

各位老师、同学：

 

“交大移动”手机客户端APP正式发布公测了！

 

为拓展校内移动应用建设，网络信息中心致力于打造交大私有云平台环境下的手机客户端品牌“交大移动”，为校内师生提供丰富多样的移动应用。当前主要功能如下：

 

1、新闻聚合

2、消息即时通讯

3、应用市场，当前主要应用有：校内通讯录、无线临时账号申请、高清电视（限校内）、掌上一卡通。

 

   各位老师和同学可以直接扫描下述二维码或通过手机浏览器访问http://m.sjtu.edu.cn，点击下载安装试用，用jaccount认证登陆。特别提示：“掌上一卡通”可以直接为校园卡充值和充宿舍电费哦！  

 

   “交大移动”是一个开放的移动应用平台，校内各部门、各位师生都可共同参与。网络信息中心立足于交大统一身份认证和开放的数据接口，鼓励和吸引校内开发者特别是有兴趣的学生共同参与，创造更多有价值有意思的应用。网络信息中心将尽力为各应用开发者提供技术、平台和数据支持，共同打造交大专属的移动校园生态环境。  

 

任何问题可反馈至service@sjtu.edu.cn。
