台军购自美国的阿帕奇“水土不服”生锈

2015年10月28日 07:31:42 来源： 新华台湾
  • 评论0

台湾部分阿帕奇直升机出现生锈情况。 网上图片

　　花费逾140亿 30架仅8架能飞

　　据香港《文汇报》报道，台湾陆军航空特战指挥部指挥官黄国明27日证实，今年3月起在维修时发现部分阿帕奇直升机（AH-64E）尾旋翼齿轮箱生锈，经各种补救措施后，情况并未改善，因此9月全面停飞特检，机队成军训练目前按计划执行，没有落后，安全性检查预估11月底完成。再交由美方成立的项目小组评估，找出真正原因。目前更盛传台湾30架阿帕奇直升机，因“水土不服”仅8架能飞。

　　综合报道，台湾陆军航特部指挥官黄国明少将27日在“国防部”例行记者会上表示，今年3月起陆续发现部分AH-64E尾齿轮箱轻微生锈，立刻向美方反映。由于AH-64E齿轮箱使用全新材质，不是传统铝合金，美方非常重视，因此9月起，就协请美方逐架进行全面性检查。预计11月底可完成检测。

　　疑与潮湿含盐量高有关

　　对于媒体质疑新加坡与日本同为向美国采购阿帕奇直升机，为何没有生锈情况发生，黄国明表示，日本与新加坡使用的都是AH-64D型机，尾齿轮箱是传统铝合金材质，台湾采购的是AH-64E型机，尾齿轮箱为铝镁合金材质，两款机型材质有所不同，美方也相当重视此事，建议实施加强防锈以及缩短定检间隔的时间，但是情况依旧存在。

　　至于是否因为台湾气候环境过于潮湿、含盐量高导致飞机生锈，黄国明表示，美方已成立项目小组做评估；关于后续更换零件美方是否需负担费用，黄国明说：“等项目小组评估完后才会有完整报告。”

　　台陆军向美国采购的30架AH-64E阿帕奇直升机，含后勤、人员训练等，总价新台币593亿元(约142亿港元)。采购的30架最新型AH-64E阿帕奇攻击直升机，可谓一波三折。先是2014年初交付的第二批飞机在美国上船海运后，波音原厂就发布技术命令，要求对主旋翼齿轮箱进行全面检查；由于在海上无法进行相关检修，因此飞机于2014年1月2日抵达高雄之后，须先经陆路运送到归仁机场，进行相关检查后才放飞。其后一架阿帕奇直升机在同年4月25日进行飞行训练时，坠落桃园龙潭民宅损毁。如今又出现尾旋翼齿轮箱生锈的问题，未能解决。

　　遭爆料9架机严重锈蚀

　　台湾一间电视台前日引述一名航特部退役军官爆料指出，剩下的29架阿帕奇直升机，有9架的后发动机齿轮严重锈蚀，若造成后发动机失效，恐怕会失速打转而停飞；还有12架飞机因备用材料（零件）不足，只能待料停机；目前只有8架能飞，妥善率不到3成。

　　对此，黄国明强调，由于阿帕奇未成军，并没有妥善率的问题，目前陆航的阿帕奇可以飞行的数量，绝对满足飞行训练所需。陆军航特部表示，阿帕奇全机队9月开始进行逐一检查，预计11月底能够结束。
