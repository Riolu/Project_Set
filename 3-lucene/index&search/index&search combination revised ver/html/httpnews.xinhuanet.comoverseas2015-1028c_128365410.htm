美女博士街头卖炒饭 浙大回应：是韩国留学生

　　微博上流传的“美女博士”炒饭照片

　　女孩第一次发图文微博澄清

　　她真的是浙大在读博士生，还是位韩国留学生

　　今天上午，一张"美女博士’街头卖炒饭”的照片在微博引发关注。照片中一位姑娘身穿“博士服”，却在炒着“铁板炒饭”。傍晚，记者从浙大国际教育学院方面确认，这姑娘确实是浙大在读博士生，今年刚入学，而且还是个韩国留学生。

　　发带研究生证照片的微博

　　想澄清却招来不少质疑声

　　美女穿“博士服”炒饭的照片今天上午在微博流传。记者通过微博找到了照片女主角的微博账号“@珍珠女神48号”。

　　今天中午11点多，“@珍珠女神48号”发微博澄清“博士服照片”，称图片是自己发在微信朋友圈“自黑”的，朋友把图片发到微博让她“被误解有点委屈”。姑娘还在微博中澄清自己“目前确实是博士在读，因为读的是食品所以有时候会自创一些奇怪的菜式借用老板摊位卖”。

　　除了文字，这条微博还附上了浙江大学的研究生证照片，成了网友质疑的目标。一小时内这条微博就有200多条评论，很多人都认为姑娘借着浙大的名头进行炒作，也有自称浙大食品科学专业的网友表示，开学时间和学生证上的发证时间不符合，质疑微博中的研究生证是买来的。

　　而下午4点多，“@珍珠女神48号”的这条图文微博被删除，半小时后又发布了一条没有图片微博再次澄清。

　　记者查找了一下，关于“@珍珠女神48号”的照片，早在10月20日就有微博营销号在流传发布了。

　　姑娘确系浙大博士生

　　老师说自称博士不正确

　　记者联系上了姑娘研究生证上所写的浙江大学生物系统工程与食品科学学院，学院研究生教育科的冯老师告诉记者，今天早上已经关注到了微博上的消息，也大致了解了一些情况，她建议记者去咨询一下浙大国际教育学院，“这应该是一个留学生”。

　　之后，记者又联系上了浙大国际教育学院，一位负责研究生事务的朱老师告诉记者，已经有辅导员在了解此事。随后，记者从浙大宣传部获得确认，这姑娘确实是浙大的学生，也确实是位留学生，来自韩国，就读于食品科学专业，学制3年半。

　　“作为留学生，她是属于国际教育学院的学生，但她的学习，都是在生物系统工程与食品科学学院的，应该是属于两个学院共管。”朱老师告诉

　　尽管学校没有规定学生不能在业余时间借别人摊位 练手，但生物系统工程与食品科学学院的一位专业教授还是表示，姑娘这样做并不符合卫生要求。另外，穿着类似博士服样子的服装来招徕顾客，也是对博士服的不 尊敬，“更何况，她现在才刚刚入学，只能叫博士生，等到完成全部课业、科研和论文学分，并通过博士论文答辩之后，才能被授予博士学位，那时才能叫博士，那 时候再穿上博士服，才是庄严正式的”。
