纪念习仲勋同志诞辰102周年

从20句精彩语录读懂开国元勋习仲勋一生

2015年10月15日08:16   来源：人民网－中国共产党新闻网

编者按：10月15日是杰出的无产阶级革命家习仲勋同志诞辰102周年纪念日。在中国近现代史上，特别是30年改革开放中，习仲勋是一位重要人物。14岁时，习仲勋参加学生运动，21岁时当选陕甘边苏维埃主席。他一生与两个特区紧密相关，一个是他参与创建的陕甘边特区，一个是他力主建立的广东经济特区。在改革开放的浪潮中，习仲勋一直站在改革潮头，为广东要权，为改革开放“杀出一条血路”。在晚年，他留守深圳12年，以实际行动支持改革开放。纵观习仲勋一生，除了为党和国家做出卓越贡献，更以其正直人品、开明作风，为后来者提供表率。我们摘选习仲勋一生20句精彩语录，与大家一起缅怀这位可敬的开国元勋，学习他的高尚品格和政治智慧。

1、毛主席不到陕北，陕北根据地就完了。毛主席晚到四天，就没有刘志丹和我们了。要不是毛主席“刀下留人”，我早已不在人世。

――1935年10月，中央红军长征胜利抵达陕北，毛泽东解救出被批斗的刘志丹、习仲勋。

2、一、把屁股端端地坐在老百姓的这一面。二、不当“官”和“老爷”。三、走出“衙门”，深入乡村。

――1944年秋，在绥德地区召开的司法会议上，习仲勋发表了《贯彻司法工作的正确方向》讲话。

3、一切工作都要在民族团结基础上采取“稳进慎重”方针进行。争取各民族上层人士，争取宗教方面人士，然后去发动，不可颠倒过来。

――1945年，习仲勋被任命为中共中央西北局书记后提出的解决民族问题的新思路。

4、在革命队伍内不该有自己不劳动而要人家供养的观念，那是剥削阶级的东西。

――1946年3月8日习仲勋在延安纪念三八国际妇女节大会上的讲话。
