培训班学员感言:给培训班的几个意见
2009年9月29日 15:36

　　通过培训，让我很直观地了解到编辑工作不仅仅是CTRL C+CTRL V，它承担着发布、传播信息的责任，是网络信息传播的重要环节之一。培训过程中，老师们用种种案例为我们分析编辑工作的重要性，以及容易出现的纰漏，为我们日后的工作提供了丰富的借鉴。有以下几点意见：

　　1、调整课程设置，使非新闻专业的学员可以接受新的理论知识，新闻专业的学员也可巩固。同时在课程的先后顺序上也可以由浅入深，从抽象的理论知识到具体的业务培训。

　　2、拓展培训形式，在传统的讲座、讨论模式基础上，新增参观、实际操作甚至是分组完成项目或课题研究的培训形式。

　　3、增进学员沟通，由于大家来自不同单位，建议在培训前先通过一些游戏让大家相互认识、熟悉。

　　4、更改结业考试形式，作为业务培训，可考虑结合业务特点进行主观认识考核，例如实际网络操作和平时上课参与融合程度考核相结合。
