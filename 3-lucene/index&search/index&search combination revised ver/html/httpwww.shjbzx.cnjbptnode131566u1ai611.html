三大网站上缴罚款32万 陌陌公开道歉
2015年5月20日 12:00

　　根据全国扫黄打非办公室部署，今年3月至9月在全国开展“扫黄打非·净网2015”专项行动。迄今为止，已相继通报了两批共八起案件。网络传播淫秽色情信息危害社会，严重影响未成年人健康成长，必须予以严厉打击。本报自今日起，推出“净网行动·2015”专栏，持续深入报道净网行动进展和成效。

　　制作传播淫秽色情信息危害社会，影响恶劣。自全国开展“净网行动2015”以来，迄今已公布两批共八起案件，公开曝光传播淫秽色情信息的网站，处以罚款，并提出整改要求，责令整改。今天，《法制日报》记者就此次行动中查处的百度、陌陌、人人网的后续整改情况，采访了北京文化执法总队网络执法队队长沈睿。

　　沈睿介绍，北京文化执法总队根据百度、陌陌、人人网的违法情况，分别处以罚款百度21万元、陌陌6万元、人人网5万元。对这三家网站的罚款已全部处罚到位，目前三网站已如数缴纳罚款。

　　沈睿表示，百度、陌陌、人人网都按照要求作了整改。三家网站都按执法总队要求，将审定过的淫秽色情关键词加入字库。执法总队要求，对淫秽色情内容要严加监管，不放过任何一个漏洞，需要完善的地方继续完善。

　　陌陌是否已按要求向社会公开道歉?对此，沈睿称，陌陌已将向社会公众的道歉信放在了官网。

　　记者联系了陌陌网公关部相关负责人。该负责人称，陌陌网对处罚非常重视，主动交了罚款，进行了全面整改。关闭了淫秽色情信息群组，删除了大量含有色情淫秽内容的文字、图片、视频信息;“换妻、一夜情、同城约炮”等含有明显淫秽色情内容的群组一个不留全部关闭。

　　据介绍，有一些微博，专门制作转发淫秽色情内容，对这些微博，陌陌网毫不留情，一律关闭。专门写作、制作、转发、传播淫秽色情内容的博主均被清理。今后，陌陌将定期对全部群组进行筛查，及时删除涉色情涉淫秽的内容。

　　记者在陌陌吧上看到了陌陌楼主的一篇微博，题目是《并肩行动，共建晴朗社交空间》。文中写道：“媒体报道了陌陌平台存在不良信息的问题。陌陌公司已经在第一时间成立专项清理小组，解散、封禁、关闭所有涉事群组和账号，并立即全面自查整改，彻底清理其他可能存在的不良内容。”

　　陌陌表示，“未来，陌陌将进一步完善内容监管体系，通过和业界领先团队合作，加大人工巡查力度，第一时间过滤不良内容。同时，陌陌还将联合主管部门开展专项行动，携手打击不良信息，创建良好的移动社交环境。”

　　该负责人称，陌陌感谢公众和媒体的监督，开通了24小时举报专线。举报信息将实时录入后台，由审核团队24小时处理。同时，陌陌还专门开辟栏目，提示网民“如何举报不良色情信息”。

　　陌陌对错误的坦诚和对营造晴朗网络空间的积极态度，也受到网民的肯定和赞赏。网民“迷迷迷奇异果”表示，“一个优秀的APP是需要不断改进的，陌陌正是如此。”网民“繁华已消逝”表示，“任重而道远。”

　　沈睿告诉记者，新浪就传播淫秽色情信息案再次向公众致歉，并表示继续接受媒体和社会监督。4月27日，新浪读书两签约作者乔某某、谭某某涉嫌制作、传播淫秽物品牟利罪案在海淀法院开庭审理。

　　去年4月，北京市扫黄打非办查明，“新浪网读书频道”向公众提供《全村女人的梦中情人：极品小村医》、《美少妇的不伦之恋：小镇情人》等20部互联网作品在线阅读服务。上述20部互联网作品经鉴定为宣扬淫秽、危害社会公德、妨害未成年人身心健康的互联网作品，上述作品自发布之日起点击量高达9000万余次，涉案违法经营额50.8万元，属严重违法情节。

　　北京文化执法总队责令新浪阅读公司立即改正违法行为，并处罚款508万元;国家新闻出版广电总局依法吊销新浪阅读公司《互联网出版许可证》。新浪阅读公司签约作者乔某福、谭某祚等3人因涉嫌犯罪被移送司法处理。

　　沈睿表示，新浪签约作者涉嫌制作传播淫秽物品罪案法院尚未判决，但今后打击淫秽色情的力度不会减弱。根据全国扫黄打非办的要求，对网络“扫黄打非”，将加大清理未备案网站力度，进一步研究开发运用技术手段和打击措施。有条件的网络运营企业都要建立相对统一的信息安全中心，并做到“四个统一”：统一领导、统一标准、统一研发、统一联防。　□记者张红兵

　　净网亦须老虎苍蝇一起打

　　短评

　　淫秽色情信息的网络传播已成为社会公害。淫秽色情网站，不法分子无良网站正是看中了它投入少利润高的特点，大肆制作传播淫秽色情信息和视频。

　　国家有关部门对网上传播淫秽色情及低俗信息一直保持高压态势，但随着互联网应用的不断创新，尤其是移动互联网的蓬勃发展，当前网上淫秽色情有回潮势头。不仅有无良无业的不法之徒在网上制作传播，而且一些大的门户网站也放松管理，不加自律，任凭低俗信息充斥。一些视频播放软件及手机客户端、网络硬盘、网络资源下载工具等成为淫秽色情及低俗信息滋生地，引起人民群众尤其是青少年学生家长的强烈不满。

　　对色情淫秽信息和网站的打击，政府、公民、青少年家长和社会组织应该携起手来，布下天罗地网，不遗余力地施以重拳，不给他们任何喘息机会。

　　净网行动是一场持久战，除了每年开展的专项治理活动，更应该保持其常态化治理。按照部署，净网行动要求“治标治本一起抓、网上网下一起查、老虎苍蝇一起打”，行政查处和刑事打击相结合。通过净网行动，彻底铲除制售淫秽色情出版物的源头、窝点，查办一批传播淫秽色情信息大案要案，推动网络空间更加清朗，出版物市场更加干净，整体社会文化环境更加健康良好。

　　净网行动即净化网络环境专项行动。这项行动开始于2011年。当年8月24日起至11月底，公安部宣布在全国范围内部署开展以清理整治制作贩卖枪支爆炸物品违法信息为重点的“净网行动”。2013年3月5日，全国扫黄打非办公室宣布，从3月上旬至5月底在全国范围内启动“净网行动”，以整治网络文学、网络游戏、视听节目网站等为重点，开展网络淫秽色情信息专项治理。分别于2013年1月下旬及7月中旬，国家互联网信息办两次组织开展净化网络环境专项行动。

　　2014年4月中旬至11月，全国扫黄打非办公室、国家互联网信息办公室、工业和信息化部、公安部决定在全国范围内统一开展打击网上淫秽色情信息“扫黄打非·净网2014”专项行动。国信办依法查处淫秽色情网站110家，关闭相关频道、栏目250个，关闭微博客、博客、微信、论坛等各类账号3300多个，关停广告链接7000多个，删除涉黄信息20余万条。

　　2015年，国家扫黄打非办公室从3月上旬至5月底，全国“扫黄打非”办公室部署了在全国范围内开展网络淫秽色情信息专项治理“净网”行动，以整治网络文学、网络游戏、视听节目网站等为重点，抓源头、打基础、切断利益链，网上与网下治理相结合。在本次行动中，全国“扫黄打非”办公室特别强调，打击开办淫秽色情网站和传播网络淫秽色情信息的违法犯罪行为是这次“净网”行动的重点任务，要充分利用技术防范。4月2日和9日公布了两批八起案件。百度、网易、陌陌、人人网等均因传播淫秽色情信息受到查处。
