贵州研讨语言有声数据库建设暨世居民族母语文化陈列方案

来源：中国民族报  发布日期：2013-05-10

　　

为贯彻落实《国家中长期语言文字事业改革和发展规划纲要（2012C2020年）》提出的“积极建设中国语言资源有声数据库和少数民族语言文字数据库”要求，日前，贵州省少数民族语言文字办公室、贵州省语言文字工作委员会办公室、贵州民族文化宫、贵州民族研究院和贵州民族大学文学院召开了专家座谈会。

 

　　会议围绕中国语言资源有声数据库贵州库建设和贵州世居民族陈列馆母语文化陈列方案进行了研讨，并决定：将数据库存储点初步选在贵州民族大学文学院、贵州民族博物馆和贵州省电教馆，资源整合相互备份，以保证数据保存的可靠性；先期启动一些濒危语言和较容易收集数据的语种进行探索，为后期各民族数据的收集打下基础、积累经验；共同推动汉语方言和少数民族语言文字数据库建设，确保建设一个完整的贵州母语数据库；将贵州省各民族语言文字数据库建设同世居民族陈列规划有机结合，统筹推进“库、馆”两项工作的开展。
