两岸文教界代表座谈会举行


来源： 教育部网站 

  

　　2月19日，两岸文教界代表座谈会在教育部举行。教育部副部长郝平出席会议，相关司局、部分高校和中央台办有关负责人陪同参加。作为中国国民党荣誉主席连战及台湾各界人士访问团成员，台湾教育大学系统总校长吴清基等8名台湾文教界代表参加座谈。

 

　　郝平首先代表袁贵仁部长对来访的台湾代表表示欢迎。他指出，18日中共中央总书记习近平在京会见连战荣誉主席及随访的台湾各界人士，就推动两岸关系和平发展发表重要讲话，使我们深受鼓舞并更加坚定了实现中华民族复兴和共圆中国梦的信心。近年来，在两岸关系和平发展的大背景下，两岸教育交流呈现了“大交流、大合作、大发展”的良好态势，许多方面实现了新突破。郝平对台湾方面就推动两岸教育领域交流合作所作的努力表示感谢，希望双方秉持“两岸一家亲”的理念，抓住两岸和平发展的有利时机，顺势而为，齐心协力，共同推动两岸教育领域交流合作取得更多成果。

 

　　吴清基先生就习总书记会见连战先生时发表的重要讲话给予了极高的评价，并对教育部积极推动两岸教育领域交流合作各项事宜表示感谢。他希望双方共同努力推动两岸教育交流合作更好发展。

 

　　此外，双方还就推动互设教育交流服务机构、商签教育交流协议、增进两岸青少年交流、开展校际合作交流、共同开展人才培养、课程合作等问题广泛深入地交换了意见。
