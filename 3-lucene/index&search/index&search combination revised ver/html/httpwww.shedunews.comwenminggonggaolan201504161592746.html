2015-04-16 13:51:46    来源：上海教育新闻网   

亲爱的同学：

您好！

我们的校园是我们的家！以科技服务师生，用创新引领未来，不仅是学校的目标，相信也是所有师生的共同心愿。为了解同学对于校内服务现状及“智慧校园”前景最真实的感受和期望，我们将对在校大学生进行匿名问卷调查。为了把我们共同的家园建设得更好，衷心希望得到您的配合。我们将以不记名方式进行调查，调查资料仅作研究之用，并对调查资料进行保密，请您放心！
