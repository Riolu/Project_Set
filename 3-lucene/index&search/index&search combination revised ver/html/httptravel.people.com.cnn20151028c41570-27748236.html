长假过后 短途游乡村游受“银发族”青睐

2015年10月28日08:39    来源：杭州日报    手机看新闻

长假过后，“上班族”、“学生族”恢复了两点一线的生活，“银发族”却活跃了起来。昨日，记者从杭州的旅游市场了解到，受黄金周过后秋高气爽、景区人少、节后旅游价格走低等因素影响，50岁以上中老年人出行大幅增长，并成为节后错峰出行主力，其中不少人更是以旅游的名义举办聚会、联络感情。

据浙江省光大国际旅行社了解到，近期报名的跟团游的客人中，将近7成都是中老年人，由于国内的线路近期价格跌幅大，他们更愿意错峰出行。相比短途旅游，长时间飞行可能带来疲惫感和身体不适、语言不通导致沟通不畅等问题，中老年群体尤其青睐那些旅行时间短、行程轻松、节奏舒缓的短途游或周边乡村游，以至于当前国内线路的游客更多一些，北京、重庆-三峡、成都-九寨沟、海南、西安、张家界、黄山、昆明-丽江、千岛湖等成为近期的热门路线。

就在日前，“游客报不合理低价团将受处”的消息一出，各方讨论甚嚣尘上。光大国际旅行社方面表示，当前旅游市场报价可谓公开透明，近年来中老年人对于旅行项目的选择上也日趋理性，由子女陪同出游的家庭游增多，而子女为父母挑选购买旅游套餐的情况较为普遍，盲目选择低价套餐的中老年人越来越少了，深度游、品质游不仅受到年轻一族欢迎， “银发族”也热衷品质游，他们同样偏爱有情调、无购物点的旅游行程。

携程旅行网跟团游订单数据显示，近期专门针对“银发族”出游而推出了近300条“爸妈放心游”产品，近期预订火爆程度可见一斑，比上季度增长一倍，报名者中的老年人比例超过70%，父母结伴出行占7成，子女陪同老人出行占3成左右

相较于在校的学生和白领上班族而言，老年游客有更多的闲暇时间来预订和安排自己的行程。以60岁及以上的老年人，携程旅行网发布的一份《国内老年人旅行行为分析报告》指出，老年游客预订出游的平均提前周期要早上10天，预订国内游大约会选择提前25天左右，预订出境游则会提前60天-75天，而老年游客平均每年的出游天数要更比年轻人多出15天以上。令人意外的是，老年游客中已出现一批热衷自驾游体验的达人，他们通常会选择在淡季出游，设定较长的旅游时间，出发地与目的地间的驾车不会超过4小时，以合理保存体力。

另外，那份报告指出，北京、上海、杭州、大连、苏州、洛阳、桂林、无锡、深圳、南京为老年游客群体最爱去的十大热门目的地，其中四成的游客更倾向选择于火车出行，只有三成的比例选择飞机出行。在旅行的过程中，他们越来越倾向慢旅慢游的理念，外出旅行逐渐成为中老年人联络感情、定期聚会的一种特定方式。（见习记者 张雅丽）
