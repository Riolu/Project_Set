香港２０１５油画大展开幕

2015年10月27日 22:47:34 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

　　新华网香港１０月２７日电（记者张晶）尖沙咀码头的黄昏、印度西姆拉、圣若瑟教堂、东方仕女图、小盆栽与檀香扇……２７日，香港２０１５年油画大展在香港中央图书馆开幕，近２００幅精彩作品纷纷亮相。

　　该展览由香港油画研究会主办。自从２０１２年香港首届油画大赛举办以来，今年已是第三届。此次展览报名参加者共有２２０多位，作品达上千件，经过精心挑选，入选作品共有１８０多幅，来自８０余位画家。展览将持续至１１月９日。

　　香港油画研究会主席林鸣岗表示，今年展览从作品题材丰富性到技巧成熟度来说，都比前两届要更好。

　　他说，油画艺术表现力极为丰富，是任何一种艺术媒介无法取代的，而且展览、收藏都极为方便，是最容易与大众达成沟通共鸣的艺术文化品牌。

　　他表示，中国文化艺术发展一日千里，油画艺术的水准也越来越高，香港喜欢油画艺术的人群也越来越多，作品品质也普遍提高，可以说是正走入“寻常百姓家”。

　　在展览中，不难看出很多画作展现了香港本土生活的场景，比如描绘中环嘉咸街的热闹街市、烟霞笼罩下的维多利亚港湾、大澳的渔民生活……

　　陈中枢已经是第二年参加油画展比赛，此次他的作品《往昔》被选中参展。画中描绘了已有６０多年历史“何希记造遮”店。这是香港一间制造和修理雨伞的老字号，从１９４７年开业开始的６０余年间，何洪希先生一生造出至少１０万把雨伞。

　　“老一代香港人吃苦耐劳、自力更生，能够一直坚持自己的事业，我想画出他们的故事，希望这种香港精神能够一直传承下去。”他说。

　　记者了解到，为促进与内地油画界的交流和合作，此次展览主办方还特意邀请到来自中国美术学院、广州美术学院等院校的７位有丰富创作经验的油画艺术家，他们带来了１４幅精选作品。

　　林鸣岗告诉记者，这是第一次邀请内地油画艺术家带原作前来香港，之前都是通过印刷品的形式展出，毕竟画作运输起来有诸多不便。此次机会难得，香港的油画爱好者可以近距离欣赏他们的作品，仔细学习、研究、交流，希望大家通过比较和借鉴能进一步提高自身的艺术水准。

　　中国美术学院教授翁诞宪是第一次来参加香港油画大展，他的最大感受是“很接地气，表现了香港油画家对生活的细腻感受。”

　　他说，这个展览把香港大部分的油画画家都动员起来了，无论从表现形式还是从题材选择上来说，都很有香港特色。展览的作品来自老中青三代，年轻人热衷创新，色彩也很活泼跳跃，年长者的作品却更娴熟、有韵味，“能看出他们都对油画创作很热爱很投入。”
