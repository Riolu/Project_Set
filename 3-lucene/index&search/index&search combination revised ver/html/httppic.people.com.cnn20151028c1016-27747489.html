郑州一居民区现“悬空楼阁” 六根钢管架起一间房

【查看原图】
搭建悬空楼阁的分别为居民楼的第二、第三、第五和第六层。
1234下一页
来源：人民网-图片频道  2015年10月28日07:58

2015年10月27日，河南郑州市岗杜街1号院内，一幢6层楼拐角处被人用钢管和彩板在墙与墙之间搭建起了书层“悬空楼阁”。搭建悬空楼阁的分别为居民楼的第二、第三、第五和第六层。记者查看发现，用于楼阁底部承重的材料为6根方形的钢管，钢管被横架在两面山墙之间，钢管上被铺上木板被当做地板，悬空楼阁的侧边还开有用于采光的窗户。有居民称，社区内的房子建于1983前后，因居住面积太小，才会有人突发奇想，在山墙之间穿洞，架设房屋，目的是腾出些吃饭的空间，平时并不在悬空屋里住人。据社区管理方称，这些悬空房屋，是在10月初搭建的，属违章建筑，存在严重安全隐患，郑州市城乡规划局工作人员现场查看后，也下达了行政执法文书，责令搭建者尽快拆除。（ic/东方IC）
