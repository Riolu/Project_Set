港媒：陆资恐惧症恐使台湾加速边缘化

2015年10月28日 09:32:54 来源： 新华台湾
  • 评论0
  • 打印
  • 字大
  • 字小

　　据《香港商报》报道，对于陆资赴台投资成效不佳，台湾政大“国发所”所长童振源曾点出五大因素，包含两岸政治冲突与安全、政策变动风险、市场规模、开放幅度不足、配套措施不足。他认为，对陆企来说政治风险大但市场规模不大，且投资过程繁琐冗长，未必愿意跨海赴台投资。

　　当然，今天台湾仍有相当的财富积累，自认资金充裕、“只缺市场不缺钱”。但要正视的是，区域经济融合是大趋势，台湾与韩国产业及贸易结构高度相似，近年全球经济遇冷，各国各地区都在创造更大的外部机会，这方面韩国已占尽先机。

　　韩促开放台反行之

　　其实，台湾认清自身困境的前因后果并不难：内需市场不振，资金回报率低，外资大幅流出，“钱不能生钱”，经济活力消退。以民间资本为主导的投资转向房地产，炒高房价又引发民怨。在相似的困境下，韩国秉持积极开放的策略，台湾却是反其道而行，经济势必加速陷入内缩困局。

　　中国社会科学院台湾研究所研究员王建民曾撰文指出，台湾患上严重的“陆资恐惧症”与“陆资多虑症”，这种病态的“抗拒陆资”与“防范陆资”心理，这种失去自信的恐惧与恐慌，只会阻挠台湾经济发展，只会加速台湾的衰落。

　　台湾的困境要靠台湾自己想通，改弦易辙才能回到正轨，尤其就大陆而言，台湾2016“大选”在即，政治变量甚多，为避免不确定风险，大陆对台政策自然转趋谨慎保守。若台湾“防拒陆资”格局不破，不要许多时日，台湾就将在区域一体化的进程中自动退居边缘，或许，届时要伤脑筋的就不是“台湾让不让陆资来”，而是“陆资带不带台湾玩”。
