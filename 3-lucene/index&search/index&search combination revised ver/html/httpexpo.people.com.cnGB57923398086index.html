由文化部、农业部、安徽省人民政府联合主办，人民网、安徽省音乐家协会承办的第五届中国农民歌会“我最喜爱的十首乡村歌曲”推选活动。活动将从改革开放以来经典乡村题材音乐作品中，遴选出部分优秀乡村歌曲供群众、专家推选。这些歌曲响彻乡村田园，传唱祖国大地，为推动农村改革发展，激励农民蓬勃向上产生了深刻影响。

投票原则：每天每个网友在“我最喜爱的十首乡村歌曲”30首乡村歌曲中最多推选出最喜爱的10首歌进行投票，每首歌曲不能多次投票。（排名不分先后）
