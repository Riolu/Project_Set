朱巍：法律、技术与网络诚信

中国政法大学传播法研究中心副主任朱巍做客“你诚我信·大家谈”系列访谈，从法律角度为广大网民解释如何应对和遏制互联网时代的谣言及其传播。

访谈摘录

  • 网络谣言是谣言传播的特殊形态

    网络谣言是通过网络技术方式传播谣言的一种特殊形态。

    2015-10-16

  • 阻止谣言传播 网站责任重大

    网站需要对消息源进行核实，并及时删除谣言，把权威消息向全网发布，让权威消息变成主流的声音。

    2015-10-16

  • 网民要做谣言的戳穿者

    互联网时代，每个人都是网上发言的主角。普通网民不应当仅仅是谣言的接收者，更要做谣言的戳穿者。

    2015-10-16

  • 大数据时代也要注意隐私保护

    网站应该把如何使用网民隐私和网民说清楚。同时各路媒体应把大数据、隐私权和公民权益之间的关系向网民普及，这样网民才能更好的维护自己的权利。

    2015-10-16

本期嘉宾

  • 朱巍

    中国政法大学传播法中心副主任

工作经历：

毕业于中国人民大学法学院，获法学博士学位。2010年至今，担任中国政法大学新闻与传播学院副教授、硕士生导师。

精彩辑录

  • 网络谣言就是通过网络技术方式把谣言传播的一种特殊形态。
  • 网络服务提供者要更加诚信。
  • 网民在不造谣、不传谣的基础上用自己的专业知识做谣言的戳穿者。
  • 网民不仅要懂法律常识，也要懂技术常识，才能更好地维护自己合法权益。
