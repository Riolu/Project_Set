和谐师生关系 共创和谐校园

2009-12-09 14:36:47    来源：上海市长宁区江苏路第五小学   


一、项目背景

㈠ 立项原因

现在70后、80后的青年教师在小学教师队伍中所占比例正呈上升趋势，这些青年教师由于从小在家里是娇生惯养的对象，所以她们普遍存在着一些不足，如对学生关心不够、依赖性较强、工作中不能吃苦耐劳等。另一方面，小学生日益表现出了独立自主意识增强、思想活跃开放的特点。因此，如何针对当前小学生群体的新特点来构建新型的师生关系，显得尤为必要。

㈡ 工作基础

上海市江苏路第五小学自1984年以来，一直旗帜鲜明地坚持“和谐教育”理念，致力于建立民主、平等的师生关系。1995年，学校成立“学校和谐教育机制的理论与实践的研究”课题，以科研带动学校教育改革，改善师生关系。2001年以来，学校开展“创建和谐师生关系的实践研究”，“蹲下来和孩子对话”逐渐成为全体教师的共识和追求 。


二、工作思路

㈠ 指导思想

师生关系问题是一个贴近实践、充满个别化而且极为具体、多变的现实问题。必须采用实证研究方法，通过一线的跟踪调查、实地观察、个别访谈，掌握第一手资料，在此基础上归纳整理总结，才可能真实地反映师生关系并有助于切实改善师生关系。

㈡ 基本思考
和谐师生关系是指教师和学生在教育教学过程中，遵循教育规律和师生双方身心发展规律，处理好影响师生关系形成和发展的各有关因素，充分调动师生交往的各种积极要素，使师生关系向积极健康，生动活泼的方向发展，避免和消除因不协调产生的内耗，形成师生间民主与平等、尊重与信任、理解与合作的良好关系，促进师生协调、主动、活泼、充分地发展。


三、基本做法

通过实践经验概括，我们从师生交互作用的视角，归纳出有利于师生关系和谐发展的主要因素包括四方面，并就这四方面开展了大量实践活动。四大方面的实践研究如下：

㈠ 加深师生相互认知的操作方法

1、“今天怎样做教师”系列活动。学校举办“今天怎样做教师”征文比赛和演讲比赛；组织教师对“今天的教师如何看学生、如何爱学生”等话题开展座谈讨论。通过这一系列活动，使教师不断产生积极的角色认同。教师们纷纷提出：教师应当成为学生的“朋友”、“伙伴”、“对话者”、“伴奏者”、“引导者”、“咨询者”等等。

2、“赞我心中的好老师”学生来信推荐活动。在开展这项活动期间，学校共收到学生及家长来信276封。 来信中述说了一个个生动的爱生故事、一件件成功的教学事例、一幕幕感人的师生交往场景。教师们没想到自己认为微不足道的举动，却会给孩子留下深刻的印象。小学生尽管年龄小，但他们也是感觉敏锐、有独立想法的个体。教师的为人、责任心和爱心、学识和教学能力水平、对学生的态度等，都会在平日的言行举止中传达给学生。教师付出的爱心和奉献，会得到学生的真情回报。

3、“今天我做小老师”活动。在部分年级部分学科中，让学生尝试做一次教师。这一活动充分调动了学生的学习积极性，课前充分准备，课上承担领读、讲解等任务，课堂反响特别好，学生们兴趣盎然，发言踊跃，听得格外认真。“今天我做小老师”活动更让学生切身体会教师工作的甜酸苦辣，加深学生对教师工作的理解。

㈡ 拓展师生交往时空的系列活动

1、围绕“充分利用和拓展师生交往时空”的主题，开展备课、说课、听课、评课活动。在教研组活动时，改变以往仅重视教材教法研究的偏向，把“改善师生课堂交往”问题也纳入到教学研讨中，达成“留一点时间关注学生”的共识。教师们在备课、说课、听课、评课时，不仅备教材、说教材、评教学，而且关注学生的起点、关注学生兴趣的激发、关注学生在课堂上的多动脑思考与多动手实践、关注课堂上学生与教师间的交流情况。

2、让学生参与教研活动。为了使教学活动能更符合学生的兴趣特点，并让学生进一步发挥学习的主动性，我校推出一项新举措——让学生参与教研活动。即让学生们把各自在学习上遇到的困难和解决方法写下来，投到专门设立的“教研组信箱”，教师则在教研组会议上对学生们的来信进行讨论，并请提出意见或建议的学生共同参加会议，与他们一起商量解决办法。现在，在各年级各学科的教研组会议上，常常能看到学生们的身影，不少学生提出的好建议被教师采用。

3、教师拓宽关注各类学生的视野。提倡教师在课堂上与学生进行平等对话式的双向交流与互动。让教师鼓励学生“敢于指出老师教学中的不足之处”、“敢于向老师提出意见”、“敢于就某一个知识点的教学与老师共同探讨”。教师们在课堂的提问、答复、要求、评价等师生交往中，尽量做到给学生干部、一般学生、后进生以同等的交往机会。

4．发动广大教师积极参与学生的课外活动。在课外活动中，教师巧妙地帮助学生把课堂中所学到的知识用于生活和娱乐中，使学生们亲身体验到“学以致用”的乐趣。在课外活动中，教师和学生之间更象一种伙伴关系，教师在课外活动中表现出的教学能力之外的其它能力特长，对学生具有极大的吸引力和感召力。通过课外活动中的师生之间平等的“互通有无”，教师有了更多的机会从各个侧面更进一步地了解学生：了解学生的兴趣、需求、认知特点，了解更多的学生个体。学生们比从前更乐意亲近老师、更“服”老师了。双方由此产生的好感还迁移到教学、德育等其它工作中去。

㈢ 改进师生交往方式与方法的操作方法

1、设立多种形式的座谈会，畅通师生对话渠道。学校召开少代会的同时，由校长、教导、总务等行政领导直接与学生代表对话，鼓励学生从学校教育、教学、生活服务、教师教风等方面反映情况、提出意见并“出谋划策”。教导处每学期召开各类座谈会，听取学生意见，了解教师师德、师爱、教学、学生负担等情况，并及时将情况在教工会上作反馈，及时解决出现的问题。

2、开展“给老师写封信”、“老师给学生回信”活动，拉近师生的心理距离。“写信、回信”活动具有特殊功能，有助于教师和学生的情感保持私下亲切交流，有利于拉近师生的心理距离。这种特殊的交往形式让不少师生感到“既新鲜又亲切”。学生来信中，有的写自己的喜怒哀乐，有的把曾经隐瞒的秘密告诉老师。学生通过写信使老师更深入地了解自己的学习、生活和想法，教师通过回信使得学生更能感受到教师的亲切与平易近人。

3、开辟“学生心语”、“教师心语”板报专栏，加深师生情感交流。对于不少学生尤其是高年级的学生来说，他们的表达方式往往比较含蓄。每个班级教室后面的“学生心语”专栏为他们提供了窗口。“学生心语” 让那些需要爱和关怀但又不擅表达的学生、让那些受了委屈但又不擅于当面解释的学生也逐渐愿意敞开心扉，向老师诉说心里话；“教师心语”的美好祝愿、积极鼓励、甚至善意的批评，通常也能得到学生们的积极回应。“心语栏”成为师生情感沟通的桥梁，成为师生都乐意光顾的地方，成为“江五”创建和谐师生关系的又一个新窗口。


四、主要成效

和谐教育不是一蹴而就的，而是在师生长期相处、交往、相互磨合的过程中逐步形成的。江五小学全校师生经过多年坚持不懈的合作共建，形成了闻名全国的“蹲下来和孩子说话”以及“对话式教学”等教育品牌。这些教育品牌的形成和提升，一是有专业基础好、事业心强、有自己的教育理想并坚持躬身实践的学校领导的管理，二是有大批献身教育并在教育实践中形成自己独特性和创造性的教师实践，三是学校在教育实践中善于总结和反思学校办学特色的形成过程，并且在创建的过程中与时惧进，不断创新。

五、书记感言

学校是社会的一个子系统，更是一个育人的场所，因此，如何构建校园的和谐就显得尤为重要。回顾“江五”的办学历程，我们很清晰地可以看到：和谐教育是学校办学的高起点，即“校高则立”；和谐教育又是学校办学的鲜明特色，即“校特则兴”；和谐教育更是学校办学的审美追求，即“校谐则和”。和谐教育铸就了学校精神文明建设的原动力，促进了学校不断的和谐发展，形成了闻名全国的“蹲下来和孩子说话”以及“对话式教学”等教育品牌，形成了学校和谐的环境，奋进的精神，务实的作风，求新的思维，铸就了学校精神文明建设的原动力，促进了学校不断的和谐发展。
