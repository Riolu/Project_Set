旅游文明行为指南及公约

  • 中国公民出境旅游文明行为指南
    中国公民，出境旅游；注重礼仪，保持尊严。
    讲究卫生，爱护环境；衣着得体，请勿喧哗。
    尊老爱幼，助人为乐；女士优先，礼貌谦让。
    出行办事，遵守时间；排队有序，不越黄线。
    文明住宿，不损用品；安静用餐，请勿浪费。
    健康娱乐，有益身心；赌博色情，坚决拒绝。
    参观游览，遵守规定；习俗禁忌，切勿冒犯。
    遇有疑难，咨询领馆；文明出行，一路平安。
    中国公民国内旅游文明行为公约
    1.维护环境卫生。不随地吐痰和口香糖，不乱扔废弃物，不在禁烟场所吸烟。
    2.遵守公共秩序。不喧哗吵闹，排队遵守秩序，不并行挡道，不在公众场所高声交谈。
    3.保护生态环境。不踩踏绿地，不摘折花木和果实，不追捉、投打、乱喂动物。
    4.保护文物古迹。不在文物古迹上涂刻，不攀爬触摸文物，拍照摄像遵守规定。
    5．爱惜公共设施。不污损客房用品，不损坏公用设施，不贪占小便宜，节约用水用电，用餐不浪费。
    6.尊重别人权利。不强行和外宾合影，不对着别人打喷嚏，不长期占用公共设施，尊重服务人员的劳动，尊重各民族宗教习俗。
    7.讲究以礼待人。衣着整洁得体，不在公共场所袒胸赤膊；礼让老幼病残，礼让女士；不讲粗话。
    8.提倡健康娱乐。抵制封建迷信活动，拒绝黄、赌、毒。

专家声音

【中国旅游研究院院长戴斌】
    我觉得不文明行为不能打上中国的标签，因为旅游是从自己的惯常环境到了另外的地方，人在自己的惯常环境会受很多约束，一般离开惯常环境自我约束力会减弱，这是正常心理。

【中国社会科学院旅游研究中心副主任戴学峰】
    造成出境游不文明现象的根本原因是国人传统优良品质“温良恭俭让”的缺失。在出境游发展初级阶段，由于国人不熟悉国际行为规范，造成不文明行为的发生，这是一个普遍现象。

【中国社科院旅游研究中心特约研究员刘思敏】
    不应该笼统地说中国游客就是不文明，这只是部分人的行为。虽然没有必要上纲上线，只是不文明，并不违法，但是问题的确必须引起我们的重视，值得我们游客以此为鉴。
