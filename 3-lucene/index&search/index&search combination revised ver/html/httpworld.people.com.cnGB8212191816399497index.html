论坛简介

    “北京-东京论坛”创立于2005年，是由国务院新闻办直接倡导推动并支持的中日两国民间交流年度性论坛，每年在北京和东京轮流举办。前十届由中国日报社与日本非盈利法人机构――言论NPO于2005年共同主办，自2015年起由中国外文局与NPO共同主办。
