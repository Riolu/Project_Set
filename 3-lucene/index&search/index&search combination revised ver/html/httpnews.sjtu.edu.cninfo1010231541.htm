医学院党委中心组（扩大）举行大学治理专题学习[图]
[发布时间]: 2014年05月01日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 思源

　　2014年4月30日下午，医学院党委中心组（扩大）学习在黄浦校区懿德楼101会议室举行。上海交通大学校长、中国科学院院士张杰应邀作题为《大学治理：以人为本的制度激励》的专题报告。医学院党政领导、附属单位党政负责人、中心组成员、机关部处负责人等共70余人参加了学习。会议由校党委副书记、医学院党委书记孙大麟主持。

　　张杰从大学与社会的关系谈起，指出大学不仅仅是知识的坚守者、传承者和创造者，更是人类思想、精神和道德的制高点，是整个社会良心、公平和正义的最后堡垒。研究型大学的本质就在于把一群极具创新思维的教师和一群极具创新潜质的学生聚集在一起，让他们的创造力互相激励，产生使学生终身受益的创新能力和智慧。

　　张杰指出，大学之于社会的根本生命力在于创新，当前中国经济社会发展正面临向创新驱动的转型，大学理应也能够成为中国经济社会发展的创新引擎和源泉。他提出，大学应当建成卓越的创新人才成长体系、卓越的科学技术创新体系和卓越的思想文化创新体系。而为了实现创造卓越的三个体系，建设成为世界一流大学，在大学治理中必须坚持以人为本的制度激励。大学最重要的组成是教师、学生、管理人员这三个群体。相应地，要从三个不同的视角去激励他们，要让教师有尊严感、学生有自豪感、管理人员有成就感。

　　张杰结合上海交通大学的积极探索与不断实践，深入阐述了以人为本的制度激励在治校过程中的实际应用。学校建立学术荣誉体系，重塑学术追求的大学灵魂；提出“三位一体”的育人理念，实施卓越人才系列培养计划，推动研究生培养改革，不断提升国际化水平；全方位建设“人文关怀”的大爱校园，通过文化为生活提供梦想。目前，学校三大体系初步形成，办学规模、学科建设、师资队伍、人才培养、科学研究、社会声誉等关键办学指标取得了跨越式发展。

　　面向未来，学校将进一步完善以制度激励为主线的大学治理，着力推进三大体系的进一步完善，发展模式上实现行政主导向学术主导的转变，管理模式上实现“校办院”向“院办校”的转变，激励方式上实现学校主导发展向师生自我实现的转变，致力于走出一条具有中国特色的发展之路，建成一所大师云集、人才辈出、科技成果和人文思想交相辉映，在国家富强、民族复兴和人类文明进步的进程中贡献卓著的一流大学。
