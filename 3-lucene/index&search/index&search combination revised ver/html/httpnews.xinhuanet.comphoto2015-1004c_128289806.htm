【大美中国】云绕雪山别样美

2015年10月04日 14:37:23 | 来源：新华网

    这是9月29日拍摄的云雾缭绕的玉龙雪山。玉龙雪山位于云南省丽江市玉龙纳西族自治县境内，海拔5596米，以险、奇、美、秀著称，是国家重点风景名胜区、云南省级自然保护区，山顶终年积雪，山腰常有云雾，远远望去，宛如一条玉龙腾空而得名。新华社记者 陆波岸 摄
