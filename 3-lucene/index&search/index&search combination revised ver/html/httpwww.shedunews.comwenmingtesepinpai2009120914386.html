励志宣讲 诠释自强

2009-12-09 14:29:05    来源：上海市久隆模范中学   

 

久隆模范中学的励志宣讲团，它是由久隆学生自发组建的，是在学校领导、社会各界的关心下慢慢成长起来的。宣讲团的成员们是由初、高中在读学生和毕业生志愿者组成，他们都有相近的生活背景——贫困，都拥有相同的财富——自强。志愿者们用演讲、交流等方式讲述自己在家庭困危面前的顽强故事，给人以精神上的激励，一个个真实的故事感动并影响着一批又一批的同龄人。
宣讲团成立于2005年6月，三年多以来，足迹已遍及了4个区县、十多个学校、街道和社区。央视《东方时空》栏目、《新闻晚报》、《新闻晨报》、《新民晚报》、《解放日报》、《至爱》等多家媒体、杂志都对他们的活动做了追踪报道，在社会上反响热烈。
【活动背景】
久隆模范中学是一所特殊的学校，在这里就读的学生全都来自贫困家庭，他们依靠低保维持着平时的日常生活，但他们同时又是非常幸福的，在久隆这个温暖的大家庭中他们享受着“五个免”——免学费、书杂费、校服费、午餐费、活动费，这为他们原本不富裕的家庭经济无疑是雪中送炭，不但解决了中学期间几乎所有的学习开销，为高中乃至大学积累教育经费，更重要的是在久隆他们受到了优质的教育，使他们能够通过读书成才最终改变现在的命运。可以说久隆的开办凝聚着党和政府对社会弱势群体的关爱，更体现了社会主义的优越性。
走进久隆的校园“滴水之恩，涌泉相报”八个大字赫然醒目，它们不仅镌刻在石碑之上更烙印在每个久隆团队员的心中。如何让久隆学生在感受爱的同时懂得传递爱、回报爱，成了久隆团队工作的特色与关键所在，开展回报社会的社会实践活动是最生动有效的活动载体。
【活动思路】
一、励志宣讲活动设计的创意
1、久隆的学生来自贫困家庭，他们不断接受着来自社会的关爱与支持，雪中送炭的温暖使他们产生巨大的感恩社会、回报社会、服务社会的使命感。
2、生活的逆境磨砺了久隆学生自强不息、勤俭节约的优秀品质，与大多数同龄人相比他们显得更懂事更成熟。
3、励志宣讲团的成立能最大限度的发挥了久隆学生的优势，实现他们回报社会的愿望。将他们身上的那股自强不息的韧劲感染许多身边的同龄人，通过演讲的方式让更多的同学懂得艰苦奋斗和自强不息，起到感人育人的效果。
二、励志宣讲活动的目标定位
1、面向社区和小学至高中阶段在校学生及其家长，使广大学生感受久隆精神，接受生动的艰苦奋斗教育；为家长提供教育子女的素材和实例。
2、对于久隆学生自己而言，同样也是良好的自我教育内容。通过学生之间的相互激励，使他们勇敢面对生活中的挫折，笑对人生。
3、培养一批具有自强故事的演讲者，通过宣讲活动锻炼他们多方面的素质和能力，在同学中树立榜样形象，起到模范带头作用。
【活动步骤】
1、制定活动计划，并聘请指导老师
2、由个支部中队推荐具有自强不息事迹的学生人选（也可毛遂自荐）。
3、撰写演讲稿
4、团委审稿（真实性、文学性）
5、组织“自强之星”评选（A、演讲比赛 B、风采展示版面）
6、接收获奖人选成为励志宣讲团成员并进行演讲技能培训
7、联系活动场所
8、利用节假日深入社区学校开展励志宣讲活动
9、评价与反馈
A、社区、学校、听众的反馈
B、指导老师、团长对团员的采用优、良、待提高的等级制进行综合评价
C、宣讲团成员的表现在德育评定、成绩手册中得以体现，是评优、推荐的重要标准之一
【活动过程】
第一次宣讲意义非凡，我们的第一站是位于杨浦区的国和二村居委。小小的一间房间，却简直是达到了爆棚的效果。那天，社区的居民知道有这样一个演讲，家长一大早就带着孩子来到了会场。宣讲团成员的背后都有着不幸的故事，父母下岗、亡故，但他们的脸上却始终带着坚毅和淡定。当同龄人听到团长孙悦礼把母亲几年来往返于单位和家庭间穿破的鞋子，放在床头激励自己的故事，很多孩子的眼眶湿润了。宣讲结束后，“擦干眼泪，振作起来”、“微笑着面对挫折”、“自立是块金子，在困难与挫折的熔炉中散发光芒”、“我是生活的强者”这些铿锵有力的话语回荡在演讲厅。爸爸们动容，妈妈们哽咽……，他们唯一想到的是向成员们要联系电话，希望能保持联系，帮助他们的孩子，为他们的孩子树立榜样。《新闻晚报》对此在头版作了报道：“小小少年遇贫弥坚，大大梦想宣讲爆棚”的标题赫然醒目，他们正是用自己的实际行动在传递关爱，诠释自强，为共建和谐社会而努力。
紧接着，通过一位青少年研究所专家的介绍，宣讲团联系上了浦东的联洋社区的“彩虹中心”。那儿的孩子都是富家子弟，平时不愁吃穿，但缺乏艰苦奋斗精神，在得知团长的意图后，他们很感兴趣，想通过他们的亲身经历来让这些孩子也懂得“勤俭、勤劳”这一古训。尽管那天烈日当头，在路上就花了2个多小时，但成员们都无半句怨言。
当听完了演讲，“彩虹中心”的主任王老师十分感动，希望所有的成员都能够成为他们的志愿者，并把演讲团命名为“彩虹励志演讲团”，“不经历风雨，怎能见彩虹”，希望那些无法走出受挫阴影的同学，在听了他们的演讲后，能够重新树立起信心。
几次成功地宣讲，增强了成员们的信心和决心，在无论是富家子弟聚集的浦东联洋社区的“彩虹中心”还是面向贫困学生的专场，无论是与家长、孩子的家庭圆桌还是向祖国母亲生日的献礼——“红心向党青春行”主题活动，无论是传授学习经验还是与社区的孩子、家长一起到敬老院，共同体会帮助他人的乐趣。久隆学子自立自强的精神深深感染着到场的每一个人。
【活动站点】
浦东新区联洋彩虹中心、杨浦区妇联、上海市档案馆、风华初级中学、金杨中学、北站街道、彭五居委会、浦东新区公安局、新中初级中学、当代中学、实验学校、志存育海中学、杨浦区国和二村居委等地。
【所获荣誉】
2005年上海市中等学校系统团组织暑期工作“优秀项目奖”
2005年度闸北区基层团组织优秀工作项目团建创新三等奖
2006年度闸北区基层团组织优秀工作项目开放式特色活动项目一等奖
2007年闸北区中学生明星社团
【领导感言】

金钱和物质的支援，可以使受援者的生活境况好转，而精神的激励，则可以改变他们的人生。久隆模范中学学生的励志宣讲就是一种精神激励，把自己真实的成长经历，通过演讲的方式告诉身边的同龄人，使他们懂得艰苦奋斗和自强不息的重要，既起到了感人育人的积极效果，也是对励志宣讲团成员们逆境奋起的自我肯定。久隆模范中学根据学校的实际情况，结合学生的特点，富有创意地设计并很好地运作了这个品牌教育项目。通过一种贴近生活、贴近心灵的教育方式，宣传“化贫困为动力，传爱心暖他人”的自强、奉献精神，有效地激发了学生感恩社会、传递爱心、回报社会之心，得到了社会各界的广泛肯定和褒扬。
