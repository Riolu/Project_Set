女教师遇害案续：文化部将严查网吧接纳未成年人

原标题：女教师遇害案续：文化部将严查网吧接纳未成年人

　　10月18日，湖南省邵东县3名中小学生涉嫌杀害当地一名小学女教师(本报10月21日、23日报道)，案件持续引发关注。今日，记者从邵东县县委宣传部获悉，当地启动追责程序，对县文广新局局长李秋兵立案调查，对县文化市场综合执法局局长李正端免职并立案调查。

　　10月18日19时许，邵东县公安局刑侦大队接警称，廉桥镇新廉小学教师李某(女，52岁)在该校宿舍楼遇害。民警现场勘查与走访排查后发现，廉桥镇3名少年刘某(13岁，初三学生)、赵某(12岁，初二学生)、孙某(11岁，小学六年级学生)，有重大作案嫌疑。

　　19日，刘某等3名嫌疑人被抓获。随后，当地官方通报称，为遇害女教师李某启动“因工殉职”报批程序，刘某等3名未成年嫌疑人被移送工读学校。

　　今日，邵东县县委宣传部相关负责人提供的一份《邵东县“10・18”教师遇害案相关责任人被追责》通报显示：26日下午，邵东县委常委会研究同意，决定启动追责程序，由县纪委对县文广新局局长李秋兵进行立案调查，对县文化市场综合执法局局长李正端给予免职并立案调查。同时，邵东县开展为期3个月的网吧专项整治行动，涉嫌违规接纳未成年人的3家上网服务营业场所被责令停业，并立案调查。

　　中国青年报记者在当地走访发现，此次案件中的刘某等3名少年，均严重沉迷网络游戏。根据已公布的信息，案发后，3名嫌疑人拿着抢劫来的钱，继续到网吧游戏。警方的通报亦显示，案发次日14时许，在县城一网吧内，刘某、赵某直到被抓，还在打游戏。

　　邵东县“10・18”教师遇害案，引发文化部高度重视。文化部立即启动文化市场突发事件应急预案，部署要求湖南省、邵阳市和邵东县文化行政部门及文化市场综合执法机构迅速调查。下一步，文化部将以县(区)和城乡结合部的上网服务营业场所为重点，紧急部署开展专项整治行动，要求各地严查严管违规接纳未成年人行为。文化部同时公布了全国文化市场统一举报电话：12318。(记者朱娟娟)
