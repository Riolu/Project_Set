微信交通安全课首次进校园

2015-03-31 11:52:31    来源：上海文明网   

3月30日是全国中小学生安全教育宣传日，申城也推出了第一个通过微信向中小学生进行交通安全的授课新模式。
　　9点，在新时代小学大华校区课堂里，一节特殊的安全教育课在这里进行着。这次的主题为《看见与被看见》，投影仪上，一只模拟手机正在向学生们展示着来自于微信的交通安全防范知识。
　　“过马路时，你看见对面开过来的车子的司机了，那么，开车的司机也一定能看见你吗？”浦东交警余晖将这一问题抛给认真听讲的学生们。“不一定哦。”在听了学生们的答案后，余晖又给大家分析起来。
　　可以说，中小学生安全通过路口的最重要因素，就是必须具备一定的“被看见”意识，而不是自己看见车里驾驶员了，就判断对方也一定会看见自己。要让车辆驾驶员看见自己其实并不难，首先要做到遵守交通法规，其次，在过马路时，熟记安全口诀：左看右看，再左看。而这就是《看见与被看见》所要讲述的部分交通安全知识点。

微信课内容。

像这样通过微信将交通安全知识送进课堂，可以说是申城第一次。据浦东交警介绍，希望以此为突破口，改变长期以来中小学校没有统一交通安全教育课件和相应考核标准的窘境。根据浦东公安分局与新区教育（体育）局的协商，自今年3月起，浦东交警负责制作中小学生识险避险知识微信课件，定期更新，暑假期间将专门组织全区中小学生开展识险避险知识竞赛活动，活动纳入中小学生《成长手册》，由家长签名确认后交班主任，同时鼓励高年级中学生有手机的也主动关注。可以说，如此一来，这个微信也成为全区中小学生交通安全教育的移动课堂，成为全区40余万中小学生家长交通安全知识的“自修室”。
