国内最大火车票代购网被罚150万
2014年1月26日 09:59

　　国家发改委昨日发布信息，针对部分民航、铁路客票销售代理代购企业趁春运客票紧张之机乱涨价、乱收费问题，国家发展改革委价监局组织有关省市价格主管部门立即查处，部分交通运输客票代理企业乱涨价受到严厉处罚。其中，上海迅途票务代理有限公司因强制服务并收费的价格违法行为被罚150万元。该公司的铁友网是国内最早、最大的火车票代购网站，而且是携程网火车票代购唯一合作方。该公司相关负责人表示，罚款已经上缴并从去年后半年开始就对强制服务进行了改正。

　　记者了解到，近年来携程、京东、去哪儿等都曾推出过各类火车票代购业务，且搭售保险一度成为普遍现象，而现在多数网站已经变强制搭售为可选项目。

　　被罚网站已认罚并改正

　　据国家发改委透露，上海迅途票务代理有限公司在代购销售铁路客票时，在规定的政府定价之外，强制加收旅客10元至20元不等的保险费，属于强制服务并收费的价格违法行为。对此，价格主管部门已依法予以罚款150万元。

　　记者昨日从上海价格主管部门了解到，确实已经对该公司进行了处罚。迅途票务代理有限公司的代购网站为铁友旅行网，根据其官方网站介绍，铁友网前身是久久票务网，成立于2009年，独创全国铁路旅行一站式服务业务，是国内领先的铁路旅行服务专业平台。并称网站注册用户突破1200万，月度服务用户2亿次。

　　记者昨日辗转联系到了该公司相关负责人。该负责人表示，其公司在火车票销售上是网络代购，主要平台是铁友网，是携程火车票代购频道的唯一合作方。去年6月份上海价格主管部门就来进行了相关调查，在去年11月份进行了处罚，现在该公司已经认罚。在去年6月份价格主管部门调查后，铁友网的车票销售方式已经进行了改正，把原来必须购买的保险变成了可选择性购买。

　　搭售保险曾是普遍现象

　　其实，早在2009年，网络代购车票就开始出现。当年支付宝和当时的久久票务网合作，开始网络代购业务。旅客只需查询所需的火车班次，选择火车票类型和购票数量后，就可以直接通过支付宝付费。网站会在2小时内通知用户是否代购成功，代购成功的会在48小时内送票上门，而代购失败的则会在规定时间内全额退款。此后，赶火车网等一些网站也加入网络代购行列。按照相关规定，转让车票时价格超过车票原价格5元以上的，就有倒票嫌疑。所以代购网站从开始就基本不收取或收取很少的代购费，主要盈利方式是采取送票上门的服务方式，并收取相对较高的快递费。而后来，收费方式再次发生变化，开始普遍搭售保险。

 

　　记者去年上半年曾在尝试铁友网购票，点击进入预订页面后，用户可自由选择出发城市、到达城市、出发日期及车次，订票流程与12306网站相似，票价也一致。不同之处是，这个网站可24小时向用户提供订票服务，且每张车票必须搭售一份20元的交通意外保险。记者还曾通过一些其他代购网站尝试购票，发现均会有这个保险搭售。业内人士表示，这是代购火车票的盈利点所在，相比之下，即使在火车站窗口购票，铁路代销的旅行意外险也只需5元左右。

　　记者昨日再次登陆铁友网、去哪儿网等代购网站，发现代购仍在进行，但已经不再强制购买保险，但是购票时会有显示“购买保险可成为VIP，VIP客户可优先处理订单。”但是，和12306相比，这些网站无法保证百分百有票。如携程网上有火车票业务，其票源就来自合作方铁友网，用户在线提交订票需求后，铁友网将人工帮助用户代购，因此无法做到即时出票，也无法保证百分之百的成功率。
