上海教育报刊总社

上海教育报刊总社（www.sepg.net.cn）成立于2001年1月，是中共上海市教育卫生工作委员会和上海市教育委员会的直属单位，是上海市专业从事教育新闻宣传和学生素质教育的主要媒体机构，是宣传和服务上海教育改革发展的"主阵地、主渠道、主力军"之一。

上海教育报刊总社目前拥有《东方教育时报》《少年日报》《上海中学生报》等3种报纸，和《上海教育》《现代教学》《上海托幼》《成才与就业》《当代学生》《好儿童画报》《康复》《教育参考》等8种杂志，以及上海教育新闻网、《上海教育手机报》和《东方教育视线》（校园视频推送系统）等3种数字新媒体。此外，还与上海文新报业集团合办《新读写》杂志，并设有《中国教育报》上海记者站。

上海教育报刊总社长期坚持 "育人为先、读者为本"，高度重视报刊内容质量建设。《东方教育时报》《少年日报》《上海中学生报》《上海教育》《上海托幼》《好儿童画报》《当代学生》《新读写》等先后被国家新闻出版总署等组织评为"全国优秀少儿报刊金奖"、"双效期刊"、"中文核心期刊"；在上海市新闻出版局实施的全面编校质量检查中，上海教育报刊总社的3种报纸均被评为"优秀"，8种杂志被评为"优秀"或"良好"，其中，《东方教育时报》在全上海97种报纸中编校质量排名第一。

上海教育报刊总社始终坚持"植根教育、服务教育"，围绕报刊主业，积极拓展会展、咨询、培训、活动、出版等现代教育服务业，逐步培育出上海教育博览会、世界头脑奥林匹克创新活动、亲子嘉年华、上海教育新闻人物评选、鲁迅青少年文学奖、中学生古诗文阅读大赛等30余种教育服务项目，成为教育传媒延伸服务、联系受众、扩大影响的重要载体。

上海教育报刊总社下设新空气文化中心、集文电脑排版中心、一众印务中心、学友广告公司、新时代广告公司、小记者培训学校、中学生业余学校、读者服务部、音像制作室等经济实体，努力为教育、为社会提供更丰富更优质的服务，履行新闻媒体应有的职责。

上海教育报刊总社坚持科学发展、深化内涵建设、转变发展方式，努力建设一流的、现代的教育传媒集团，为上海办好人民满意的教育，率先实现教育现代化、率先基本建成学习型社会，不断作出新的更大的贡献。
