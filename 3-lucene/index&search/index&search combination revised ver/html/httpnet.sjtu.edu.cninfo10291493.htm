如何在 Foxmail 中设置与邮件服务器的安全连接

2011-12-28  点击：[]

1、打开Foxmail，点击菜单“工具”，然后选择“帐号管理”（图1）。

                                                      图1

 

2、帐号管理的界面上，点击右侧“服务器”的选项，确认接收和发送邮件服务器地址为mail.sjtu.edu.cn，并且确认“发送服务器需要身份验证”的勾打上（图2），然后点击“高级”。

图2

 

3、高级设置“服务器”选项里，POP3和SMTP的“此服务器要求安全连接”两个选项都打上勾，下拉框选择“TLSV1”（图3），点击“确定”。

图3

 

4、点击“确定”，完成。
