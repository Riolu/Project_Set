“赋诗索贿”的官员是缺德还是有才？

何浩民

2015年08月24日14:07   来源：人民网－中国共产党新闻网

河南省纪委日前披露周口政法委原书记朱家臣贪污受贿案详情。据披露，发短信索贿是朱家臣惯用伎俩。逢年过节，他都通过短信给一些基层干部“赋诗”一首，目的是“提醒”一下。如果对方没反应，就再发一首诗，直到人家主动来“看”他。

说起索贿理由，可谓是千奇百怪、闻所未闻，有的打“孝子牌”，以“为母买药”为借口，向企业索贿；有的打“雅好牌”，让他人四处“淘宝”，投其所好；还有的打“同情牌”，以“被情妇勒索”为由，向下属索贿。大凡官员贪腐，总有自己的一套说辞，但无论这理由多么冠冕堂皇，实质都是权力敲诈，都是贪欲作怪。

权力是个好东西，特别在某些官员手上，其“威力”得以超常发挥。权力不仅能让自己“高高在上”，还能成为生财工具，谋取利益。表面上看，“赋诗一首”是涵养高、有文化的表现，而背地里却是借诗为名向他人索贿，不仅玷污诗词的意蕴，更辱没了公权的尊严。

在部分人看来，类似“雅贿”、“赋诗索贿”等手段，把腐败行为变得“温文尔雅”，“友好”地让对方领会其中深意，既能“恰到好处”地委婉“提醒”他人，让自己看起来不是那么“坏”；同时又具有一定的隐蔽性、欺骗性，不易被纪检部门察觉，难以被人抓着把柄。殊不知，任何伪装都无法掩盖腐败的本质，“赋诗索贿”只不过是掩耳盗铃、欲盖弥彰，只会一步一步把自己送上了贪污腐败的犯罪深渊。

对于“有才”官员来说，缺乏“官德”的危害更大。干部的文化素质和自身修养，首先应该体现在为群众服务的实际行动中，舞文弄墨、附庸风雅式的贪腐行为，不仅不会给自己脸上贴金，相反只会成为笑柄，最终等待自己将是党纪国法的严厉制裁。
