原标题：香港专业人士（北京）协会成立青年事务委员会

　　中新网北京5月23日电 (记者 邢利宇 杨)成立4年有余、会员300余人覆盖20多个行业界别的“香港专业人士(北京)协会”，23日迎来其旗下颇具活力的新成员――青年事务委员会。

　　全国人大常委会副秘书长、香港基本法委员会主任李飞发来贺信。

　　贺信中说，青年是国家和香港的未来，在以往工作基础上，协会进一步成立青年事务委员会，必将为在京香港青年深入认识祖国和内地，提升自我专业发展、密切相互友谊等提供更好的服务平台，发挥更大作用。

　　协会荣誉顾问、全国港澳研究会会长陈佐洱出席当日青委会成立仪式并致辞。

　　陈佐洱说，青年是香港的未来。当前，香港正在坚定不移向前推进政治体制民主化，“民主发展是大趋势”。香港特别行政区政府根据全国人大常委会“8・31”决定提出的2017年行政长官普选方案，可以使香港实现一人一票普选行政长官，“这是民主一个很大的发展，很了不起。”

　　陈佐洱表示，希望这一方案在立法会顺利通过，在座青年2017年就可以投票选举特首，行使人生中参与政治民主的神圣权力。

　　陈佐洱还介绍了国家经济社会发展的相关情况。他指出，香港的经济成长和国家发展相伴随。从来都是互利共赢，而不是此消彼长。在座香港专业人士多年在香港、内地打拼的经历都证明了这一点。“发展民主也要把握好与发展经济之间的平衡。”陈佐洱强调。

　　香港专业人士(北京)协会会长冯国佑在接受中新社记者专访时表示，香港回归祖国，两地联系日益密切，来京学习与发展的香港青年持续增长。该协会本着“团结互助，安居乐业，融入北京，贡献祖国”的宗旨，长期支持爱国爱港的青年事务。

　　他指出，此次成立青年事务委员会目标是希望通过举办不同类型的活动，团结香港青年，并给在京香港青年和港生打造认识祖国的平台，希望他们将来成为京港两地交流互动的桥梁。

　　据介绍，该协会将提供职业和专业引导的相关机会，扶助港青学业、就业发展，并通过不同政府单位帮助解决他们所遇到的问题。

　　目前，该协会青年事务委员会已有会员100余人，干事20余位。(完)
