“一小撮”我的小伙伴在国际上几近“声名狼藉” ，所有小伙伴儿的人格一起被质疑了。
专家说，《文明旅游倡议书》、《出境游行为指南》神马的都没用，国内不排队凭啥国外就要排？
中秋节、“十一”长假就要到了，在随之而来的旅游高峰中，小伙伴们能管好自己吗？
好行为不分国内外，习惯成自然；从我做起，从现在做起，国人文明出游将成为世界更美好的风景。
尊重他人才能赢得他人的尊重，不要让“一小撮”变成了“中国人”。
