“2015年两岸大学生汉字书法艺术交流夏令营”顺利举行


来源：教育部语用司
　　

    8月3―7日，“2015年两岸大学生汉字书法艺术交流夏令营”在台湾艺术大学顺利举行。来自中央美术学院、北京语言大学、四川大学、中国美术学院、绍兴文理学院，以及台湾艺术大学、台湾师范大学、台南大学、长荣大学、华梵大学、文化大学、东华大学等12所两岸高校的近百名大学生欢聚一堂，通过书法讲座、创作、交流、展览等多种方式交流书法技艺，增进互相了解，共同传承弘扬中华优秀传统文化。

　　教育部语用司副司长、中国语文现代化学会副会长彭兴颀，台湾中华文化总会秘书长杨渡，台湾艺术大学美术学院院长林进忠分别在开营式上致辞。此次夏令营为期5天，“汉字源生”讲座、台湾体验书法讲座、赴台北故宫博物院参观“钟鼎彝铭――汉字源流展”等活动一一举行。此外，在墨香飘逸的两岸学生作品展间，同学们还进行了篆隶书法扇面创作、行草书法斗方创作、笔会交流和集体长卷创作等活动。夏令营各项活动得到了两岸师生的高度评价。

　　两岸大学生汉字书法艺术交流夏令营是贯彻落实习近平总书记关于加强两岸青年文化交流的重要讲话精神、落实《第五届两岸经贸文化论坛共同建议》及两岸中华语文工具书合作编纂工作会谈精神的重要成果。自2014年起，由国家语委牵头的中华语文工具书大陆编委会和台湾中华文化总会轮流举办，首届活动驻营王羲之故里浙江绍兴。活动旨在以汉字书法为纽带，促进两岸语言文字沟通交流的常态化，加强两岸青少年的互通互信和理解认同，共同增进对中华优秀传统文化热爱和传承的自觉意识和使命感。
