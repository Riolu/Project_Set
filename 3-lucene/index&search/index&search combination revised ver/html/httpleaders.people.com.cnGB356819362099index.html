鄂前旗致力于工业崛起 着力构筑循环经济产业格局

    近年来，在鄂前旗旗委、旗政府的正确领导下，全旗各族干部群众高举科学发展观旗帜，抢抓国家扩内需、保增长的历史性机遇，坚持“工业崛起”战略不动摇，全面加快上海庙能源化工基地建设，着力构筑以煤炭为基础、煤电为支撑、煤化工为主导的循环经济产业格局，工业经济快速发展。

    每户拥有一处5000亩左右草牧场、一处140亩饲草料基地、一套耕种播收一体化农机具、一处标准化养殖棚圈、一套大型喷灌设备，加一套新型住宅。
