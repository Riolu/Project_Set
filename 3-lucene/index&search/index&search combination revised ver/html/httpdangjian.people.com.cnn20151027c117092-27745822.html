组图：走进谷文昌故居

2015年10月27日15:25   来源：人民网－中国共产党新闻网

谷文昌故居 （人民网记者 王金雪）

中国共产党新闻网林州10月27日电 （记者王金雪）今年是“四有”书记谷文昌同志诞辰100周年，中共安阳市委、人民网联合举办“学习谷文昌 践行‘三严三实’做‘四有’好干部”纪念活动。今天上午，100余位参会人员一同走进谷文昌故居，认真学习谷文昌的先进事迹。

谷文昌故居位于林州市石板岩镇郭家庄村南湾自然村，1915年10月谷文昌出生于南湾村一户贫农家庭，在此居住长大。
