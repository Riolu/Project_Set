    三年前一款社交K歌手机应用“唱吧”一经推出用户猛增：五天创造了App Store的第一名、十天圈到一百万用户、八十天圈了一千万用户，创始人兼CEO陈华告诉记者如今唱吧用户突破3亿，“可见传播速度有多么吓人。”
    2012年电视节目《中国好声音》席卷全国，而在互联网上，普通大众通过手机k歌软件唱吧也圆了自己的明星梦，他们带来的“中国互联网好声音”也获得万千网友点赞；2014年唱吧获得“新浪科技2013风云榜-年度创业公司之社交娱乐应用奖”。
    陈华的目标远不止这些。
   “唱吧除了有线上唱吧直播间、游戏等，在线下有唱吧麦颂KTV，还有校园巡演、校园歌手大赛等各种活动”，陈华坦言唱吧未来还是聚焦围绕听歌唱歌业务，“围绕听歌唱歌、追星这样的生态做事情，未来可以玩的花样非常多。”
