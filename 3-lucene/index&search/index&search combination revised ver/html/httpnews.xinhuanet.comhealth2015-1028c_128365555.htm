化万圣节妆 当心油彩汞超标还难洗

    其实，想要过一个精彩的万圣节，不一定要靠妆容“吸睛”，还可以选择穿着特色衣服、制作南瓜灯或是佩戴面具等方式。

    万圣节马上就要到了，化一个另类妆容去参加“大趴”成为很多潮人和小朋友们的选择，万圣节面部油彩也随之热销。然而，当五颜六色的油彩涂在脸上时，你是否想过它会伤及皮肤甚至危害健康？

    此前有媒体报道，面部油彩或存在潜在危险，此类产品往往含有潜在的不安全成分，例如铬、镍、铅和钴。

    事实真的如此吗？近日，记者分别在小商品市场、文具店及网上购买了5种万圣节面部油彩（选择其中9个送检），其中，网购的三种面部油彩均未注明成分、使用方法、生产日期、有效期限、产地等信息，但在产品正面显著标明“无毒”字样。

    检测结果发现，9个样品中，5个样品重金属汞或砷超过国家《化妆品卫生规范》标准，且重金属钡元素含量较大。

    专家表示，如果只是短暂使用重金属超标的油彩，不至于造成吸收蓄积和中毒，但对敏感肌肤会有刺激作用，建议使用正规品牌化妆品替代油彩达到效果。

    实验地点：自然大学实验室

    实验样品：9个不同颜色万圣节面部油彩样品，分别通过文具店、小商品市场及网店购买

    实验1

    4样品汞元素超标 最多54倍

    实验人员分别将9个万圣节油彩样品编号，使用重金属检测仪器对准样品，静置30秒左右，当仪器显示屏读数稳定后开始读数。

    中国检验检疫科学研究院相关负责人介绍，面部油彩中重金属元素含量可参照国家《化妆品卫生规范》中对重金属限量的规定，其中，汞、铅和砷的限量标准分别为1mg/kg、40mg/kg、10mg/kg。

    参照这一国标，共有4个样品的汞含量超标，其中一个红色的蜡状油彩汞含量为55mg/kg，超标54倍。另外三个汞含量分别超标7倍、23倍、44倍。还有一个样品的砷含量超标，为12mg/kg。

    此外，9个样品中7个测出重金属钡，最多的一个达6948mg/kg，虽然《化妆品卫生规范》中未对钡含量做出规定，但这些数值也较高。

    实验2

    油彩顽固 洗面奶卸妆油都洗不掉

    对于面部油彩的清洗方法，多个油彩的包装标明“用肥皂即可轻松洗掉”，或是“去除颜色请先以面纸擦拭再以肥皂或洗脸清洗剂清洗干净”。

    记者分别将颜料涂抹在手上，伴随着手上鲜艳色彩的是一股刺鼻气味，其中3个味道较为强烈。

    接着，记者使用香皂、洗面奶、卸妆油反复多次揉搓，并用清水冲洗后，皮肤中仍留存大量颜料。

    记者询问多个卖家，他们均回应产品用清水加香皂即可清除。如果仍无法洗涤干净，用温水即可，并坚称自己的产品为水性而非油性产品，不存在洗不掉的情况。

    但是，记者用温水洗，手上仍然留有印记。

    ■ 追访

    为颜色鲜亮添加重金属色粉

    为什么涂在脸上的油彩会出现汞、砷等重金属超标的情况？广东省检验检疫中心化学检测工程师田勇解释，油彩中有不同的颜色，为了保证各种颜色的鲜亮性，一些厂家可能会添加重金属色粉。

    记者致电生产厂家，多位负责人表示他们绝未用重金属等有毒成分，并且使用的是环保色粉。田勇表示，环保色粉代表其重金属未超标，但仍含有一定量重金属。

    “重金属含量不同的色粉混合在一起也可能导致其总量超标，”他说，“此外，制作和混合过程中，不同色粉发生反应，最终可能导致重金属超标”。

    重金属化妆品可引发汞中毒

    中国农业大学食品科学与营养工程学院副教授朱毅说，化妆品中含有汞、砷、镉等物质，首先可能造成色素沉着、黑斑。如果重金属经皮肤进入身体，一个月左右可能出现汞中毒等临床症状。汞中毒症状有精神和神经症状，同时肾脏功能也会受到损害，会表现出脾气亢奋、抑郁、植物神经功能紊乱、肾功能综合征等临床症状。

    虽然面部油彩大多在万圣节当天使用，暴露时间短暂，朱毅表示，“不至于造成吸收蓄积和中毒，但对敏感皮肤可能有刺激作用”。

    钡含量多可致儿童皮肤红肿

    实验结果显示，7个油彩中测出重金属钡，且含量较多。朱毅表示，钡元素如此“凶猛”，有可能是添加用于增加色彩度的硫酸钡，使油彩显得更为鲜艳。田勇也表示，相对于汞、砷等重金属，钡在生活中较为常见，一方面可能由于环境污染，另一方面，某些原材料加工工艺中也需要添加重金属。

    虽然金属钡不在《化妆品卫生规范》的限量标准中，但朱毅表示，钡含量较多可能会导致敏感细嫩皮肤红肿瘙痒，尤其是儿童。

    ■ 小贴士

    1.面部油彩洗不掉怎么办？

    中国农业大学食品科学与营养工程学院副教授朱毅说，这种油彩已经不是水性和油性的问题，更像是画画用的颜料，会有很多黏稠的物质洗不掉。她表示，如果用化妆油也洗不掉，可以用颗粒状洗面奶加入红糖进行长期搓洗。

    2.万圣节怎样化妆最安全？

    朱毅建议，化万圣节妆最好不要涂抹这样的面部油彩。如果想要达到效果，可以选择大品牌的化妆品，例如口红、胭脂、眼影等，也能达到相似的效果。

    （记者信娜）
