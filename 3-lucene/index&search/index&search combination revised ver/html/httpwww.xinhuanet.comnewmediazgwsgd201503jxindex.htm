“中国网事网络感动人物评选”活动由新华通讯社发起并主办，新华网、新华社“中国网事”栏目承办，迄今已成功举办五届。

“幸福傻子”叶德奕（福建省中共周宁县委宣传部推荐）

他有80多个娃，十多位“父亲”，是20多名贫困学子的大哥，十余年不变的电话号码是小县城的服务热线，他是别人口中的“傻子”。[详细]

“轮椅天使”丁玉坤（河北省石家庄市网信办推荐）

一台轮椅，一抹微笑，漂染的头发映衬着精致的妆容……从“吃低保”到帮助“吃低保”的贫困残疾人，坚强的信念撑起了她的微笑人生。[详细]

“大爱妈妈”冉存英（湖北省宜昌市委宣传部推荐）

这位平凡得不能再平凡的母亲，正用一种不平凡的母爱，擦亮一群孩子灰色的天空，从一个孩子的母亲，变成一百多个孩子的“母亲”。[详细]

“留守儿童学校女校长”汪付珍（中国网事推荐）

女校长汪付珍在贵州麻山腹地坚守近３０个年头，除了上课，汪付珍还要做饭、洗衣……就像妈妈一样把“母爱”给了边远落后山区的留守儿童。[详细]

“身患顽疾”坚守工作岗位的袁野（湖北省咸宁市委宣传部推荐）

一位平凡的银行客户经理，一位勇抗肝癌的斗士，他用乐观、坚毅、敬业和奉献，谱写了一曲银行人自立自强、爱行敬业、服务客户的感人赞歌。[详细]

“阳光指引者”王金云（中国网事推荐）

１２年来的公益行动：劝服５３名在逃人员投案自首，帮助１７０多名出狱失足青年找到工作，无一人重新犯罪，６００多人受到感化，放弃犯罪念头。[详细]

防范微视频中“表情帝”汤彦杰（江苏省常州市公安局推荐）

“我上着QQ、摇着微信，突然就有人说我中宝马了先要上税……记住！要你转账都是大忽悠！” 一位江南民警，操着东北口音，11部微视频创造了无数神话。[详细]

“逆行消防员”魏振远（中国网事推荐）

入伍关头他毅然转身奔赴火场，回到了他“消防农民工”的岗位，在天津“8·12”事故现场救援。错过入伍，他只想永远守着兄弟们战斗过的地方。[详细]

“双拐企业家”王铁继（中国网事推荐）

“我要让残疾人的出行半径由一公里变成一万公里”，为了这个梦想，江西“双拐企业家”王铁继五年来义务帮助上千残疾人考取驾照。[详细]

“好男人”张忠祥（中国网事推荐）

妻子患病两天一透析，智障哥哥、失明母亲要人常年照顾，父亲、妹妹相继意外离世。20余年，张忠祥独自撑起多“病”的家，用行动诠释了男人的担当。[详细]
