曦潮书店第二季“seacheers”书香主题阅读月

时间：2015年04月21日 10:40 发布人： 浏览次数：[]

[责任编辑]李敏妍 [信息提供]曦潮书店  

 

“方寸世界·历史新生”

 

　　方寸　凝练　历史精华
　　世界　浩渺　新生坦荡  

 

　　值此交大119周年校庆之际，以及迎接4.23世界读书日，曦潮策划了“方寸世界·历史新生”书香主题阅读月系列活动。用珍藏书香共同回忆交大跨越三个世纪的厚重历史；并借助阅读，让时光在这浩淼世界迎接新生。  

 

丰富活动具体安排如下：

　　继2014年12月第一季“书香暖冬——手工艺的温度”，2015年4月第二季“方寸世界.历史新生”之后，曦潮书店每年都将推出两季“seacheers”主题书香阅读月活动。推荐优质书籍，策划优质且常态化的活动，搭建文化平台，提升交大、华师大及周边社区的书香氛围，用行动实践曦潮“构建人文生活常态”的愿景。

　曦潮书店简介：

 

　　曦潮书店，典摘交大校友，北大校长蒋梦麟先生之《西潮》，谓“理想，希望和意志”。  

 

　　构建“人文生活常态”，是我们的愿景。  

 

　　2014年4月8日试营业，初生至今，刚满一年。现为交大，华师大，及紫竹科技园区之唯一实体书店，
曾被母校寄予“曦潮”是一面镜子，映衬着百年交大的人文底蕴；“曦潮”是一颗火种，引燃了交大人的激情与梦想；“曦潮”更是一个磁场，凝结着交大人的人文力量。”的厚望（——摘自  张杰校长《让人文之光照亮未来——在2014届本科生毕业典礼上的演讲》），也曾被媒体报道为《隐居交大，做上海最好的独立书店》。  

 

　　书店三位创始人均为交大毕业，毕业后分别在各地从事着专利法，中学教师，和设计师的工作。时隔十余年回到母校。情怀满满，经验浅浅。
