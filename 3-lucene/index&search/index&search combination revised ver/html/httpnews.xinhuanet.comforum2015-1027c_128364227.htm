网民眼中的“十二五”，稳稳的都是幸福

2015年10月27日 18:39:47 来源： 发展论坛
  • 评论0
  • 分享

  • 字大
  • 字小

    “十二五”时期是我国发展很不平凡的五年，也是改革开放伟大历史进程中具有里程碑意义的五年。今年是“十二五”的收官之年，新华网发展论坛推出【大型征文】让幸福多飞一会，我与“十二五”的这几年，汇集民声民意，广大网民在网上发帖、跟帖，用“稳定”“福气”“舒服”“骄傲”等字眼表达对过去这五年自己生活满满的幸福感，以及对“十三五”全面建设小康社会美好的展望。　　

　　收入增加，钱包鼓起来

　　这五年，中国稳居世界第二大经济体，经济增速领跑全球主要经济体。国家富强了，老百姓的钱包也渐渐丰满起来。

　　网民“幸运丁丁”说，这五年生活的变化太大了，总感觉像是在做梦一样。收入提高了，钱袋子鼓了不少，来一场说走就走的旅行，已经是每年和家人的约定。

　　网民“勤奋的刘小朵”说，这几年收入分配改革加速，央企高管降薪，但企业退休人员养老金十一连增，农村教师迎来涨薪潮，收入差距持续扩大的趋势得到遏制。城乡之间居民收入差距也不断缩小，钱包鼓了，时常去看一场电影，买自己喜欢的商品，也不在是奢望。

　　铁腕治污，生活更舒服

　　老百姓的钱包日益鼓起来以后，更深切体会到，美好生活不仅来自物质财富的增加，还包括环境的改善。

　　网民“纳西哥2015”说，这两年切身感受就是环境质量真的有所改善，而且很明显！以前从来不在外头唱歌，唱歌不等张嘴就呛得慌，现在出来感觉这空气很新鲜。

　　网民“爱抹茶的小白”说，“十二五”以来，很多地方开始摒弃“带血的、有毒的GDP”：南京关停了数百家“三高两低”企业，河北也综合整治了重污染小企业8347家。生态环境保护犹如一场接力赛，站在“十二五”和“十三五”这两个五年规划的交接线上，回望过去，党和政府坚决向污染宣战，生态环境保护取得明显成效；展望未来，让天蓝地绿水净成为常态，任重道远。

　　就业稳定，阵脚不乱

　　就业是民生之本，没有稳定的就业，发展注定不可持续；没有高质量就业，百姓注定不会幸福。

　　网民“胖纸一枚”说，“十二五”期间新增就业千万人，必须点个赞。“十二五”时期，是就业和社会保障制度改革力度最大、发展最迅速的时期。党始终把人民过上美好幸福生活作为治国理政的核心理念，我骄傲！

　　网民“贴心小棉袄暖男”说，国家出台实施了一系列促进“大众创业、万众创新”的政策措施，我作为一个刚刚毕业的大学生，站在“互联网+”的风口上，年轻是最大的本钱，二十岁的时候有属于自己名字的小公司，算完成了一个小目标。下一个二十年，我会更加拼命。

　　网民“水雲豪侠”表示，其实推动就业也离不开就业环境的持续改善，“十二五”期间国家下大力气缩小不同行业的社会保障等相关差距，为劳动力的顺畅流动铺平了道路。

　　医改发力，百姓好福气

　　网民认为“十二五”期间尤其是党的十八大以来，因病致贫、因病返贫的现象正在逐渐改观，老百姓看病慢慢地有了底气。

　　网友“北极星”说，不管你是城里人，还是农村人，总有一款医疗保障可供你选择。“十二五”医改来，老百姓得到了更多的实惠，看病也有保障。报销比例高了，药物价格低了，卫生室敞亮了……医改对基层老百姓绝对是一剂强心针、一场“及时雨”。

　　网民“明月自在怀”跟帖说，十二五以来，互联网与益民服务的融合日益加深。利用云计算、大数据搭建的公共信息平台，提供长期跟踪、预测预警的个性化健康管理服务，有效了解决老人的健康管理问题；而在线预约诊疗、诊疗报告查询、网上社保办理、跨地区医保结算等一系列应用改善了“看病难”、“看病贵”问题。

　　网民“水眸如是”表示，我国是一个人口的大国，国情不一，民情复杂，所以政府在推进医改上也克服了很多困难，取得现在的成就的确不容易。为祖国加油鼓劲，一定要坚持改革的决心，在“十三五”期间将医改进行到底。

　　教育改革深化，学生幸福成长

　　教育事关千家万户，涉及亿万群众，也关乎祖国的未来。

　　网民“原野风霜324”说，做为家长，我最关心孩子的教育问题。“十二五”期间，“入园难”得到缓解，教育资源公平统筹大力推进，2015年，高考录取率最低省份与全国平均水平的差距从2010年的15.3个百分点缩小至5个百分点以内……教育事业在这五年里取得了显著成就。

　　网民“恋恋初秋”说，国家扩大了向贫困地区的定向招生，让农村的孩子也能考上好大学，用知识改变命运，教师队伍建设以及教育投入都取得了良好的效果。在提高教育水平和改善教育公平上确实让你忍不住拍手叫好，中国给力！
