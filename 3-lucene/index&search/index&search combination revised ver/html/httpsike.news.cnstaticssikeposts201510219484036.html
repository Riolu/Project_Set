李彦宏：做让这个世界变得不一样的事情

思·锐享

为读者提供最有价值的观点

发表于  10/28 06:30   约5分钟

　　近日，百度董事长兼首席执行官李彦宏现身复旦大学，以“发现改变世界的力量”为主题，发表演讲并与现场学生进行交流。


　　我们这些一代又一代的计算机工作者，一直都有一个理想，就是什么时候可以让计算机像人一样智能。过去这几年，这方面应该是发展得非常迅速，一是计算能力提升上去了，另外一个原因是计算成本下降得非常快。这就使得以前我们觉得不能做、不可能做到的事，现在却变得可能了。

　　现在，随着计算机资源越来越强大，越来越丰富，计算机资源的价格越来越低，这些东西在短时间内就能够做到了。

　　其实，我已经有了我一生都花不完的钱了，完全没有必要纯粹为了钱去做很多事情。但是，如果因为你做了什么事情，让这个世界变得不一样了，那我死了以后很多很多人会想起我，这是多么美妙的一件事情。

　　所谓的“美国梦”，其实就是一个中产阶级的梦，有房、有车、有美满的家庭，朝九晚五。但这些东西真的是我想要的吗？其实不是的，原因大概如下：第一，我学计算机；第二，我工作的地方是在硅谷。硅谷有一帮人有点不食人间烟火，他追逐的不是好的生活条件、大的房子，他追逐的是改变世界。他们这些人真的是有理想的，他觉得用他的努力可以让这个世界变得不一样。我在硅谷待了几年之后，我自己也变成了那个样子，我自己觉得挣多少钱不重要。

　　其实，现在的社会还有很多不平等，如果你出生在高端和富豪的家庭，你可能有很多普通人不能够获得的东西。即使是在同样的家庭背景之下，母语是汉语和母语是英语的人也是不平等的。我在大学的时候，大概花了三分之一的时间学英语，因为那时候我想出国，需要拿到奖学金才能拿到签证，这就意味着我不仅要专业好，还要英语好，托福要很高分。所以我当时花了很多精力去学英语。当时，我想如果我生在美国，母语是英语就不用花那么多时间学英语了。其实，在这个方面上，世界各个国家、各个种族、各个年龄层面、各种性别、各种家庭背景的人，这些人都还是非常不平等的。

　　这个社会还有很多很多不平等的事情，但是，如果我们从一个比较正面的角度来看这个问题，这些不平等实际上给我们带来了机会。如果你能够把不平等变得更加平等，你就给更多的人创造了机会，你给更多的人创造机会，你就给这个社会带来了价值。如果你用这个去创业你就成功了。所以不平等现象是正常的，而且会给你带来很多机会。

　　我在美国读书的时候，感觉在学校里研究的那些东西虽然技术含量很高，但是其实我真的不知道它有什么用处。我觉得学术界研究的很多东西，很可能五年十年内都看不到实用的场景。

　　我觉得工业界研究的东西和学术界研究的东西不一样，工业界研究的东西有非常清晰的想法，就是这个东西做出来可以产生什么样的价值，市场为什么需要这个。我觉得要想让我做这个事情，我一定要想为什么，解决这个事情之后到底能给我们带来什么样的改变。

　　后来，我到硅谷之后发现真正能够带来改变的东西都是这样子的：它一定是开始有一个事儿说我想要干什么，这个东西不是去做研究，而是说它能够对市场产生一种影响，或者说市场对这个是有需求或者有潜在需求的。你首先要搞清楚，市场或者这个世界需要这个东西，你再去做，你才能做出来好的技术，这样才有可能产生影响力。此外，你还真的需要下功夫，要获得相应的资质，太笨了也不行。

　　我在读中学的时候读到列宁说的，世界一旦有了需求，就比十所大学更能够对世界产生推动力。当市场有需求的时候，会有很多人、很多资金涌入到这个领域，充分的人员和资金的加入才会使技术出现。

　　今年，总理开始讲“互联网+”，互联网可以加任何一个垂直行业，包括加医疗，当然医疗是一个很大的行业。每一个垂直行业都有它独特的地方，但是通常互联网带来的最大改变就是效率上的改变。而互联网+医疗的改变，确确实实改变了医疗行业的效率。

　　中国大概有200万左右的医生，可是我们一旦需要去医院看病就会首先考虑去公立医院。上海也好、北京也好，三甲医院都是人满为患，大家排队都要花很长的时间。我们百度想做的事情，就是把合适的病人送给合适的医生。这样，医疗资源和病人之间就会得到更加合理的匹配，效率就会更高。

　　（作者：李彦宏，百度公司创始人、董事长兼首席执行官）
