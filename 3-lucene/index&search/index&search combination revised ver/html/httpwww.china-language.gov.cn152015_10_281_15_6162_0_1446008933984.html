云南省举办2015年云南省中小学“经典诵读”教育教学研讨会

云南省语委办

    为进一步贯彻落实党的十八届三中全会和《国家中长期语言文字事业改革和发展规划纲要（2012―2020）》精神，传承祖国文化精髓、弘扬民族文化传统，进一步推动由教育部、国家语委和中央文明办主办的“中华诵•经典诵读行动”的深入开展，不断提升云南中小学校经典诵读教育的整体素质，培养经典诵读教师的中坚力量，云南省语委办委托中华书局经典教育推广中心和昆明乐之者教育咨询有限公司于2015年10月17日至18日在云南大学科学馆举办了2015年云南省中小学“经典诵读”教育教学研讨会。来自全省14个州市的237名语委工作人员、语言文字规范化示范校校长和中小学一线语文教师参加了研讨。

    本次研讨内容丰富：

    ――云南省语委办金程主任从行政管理角度，介绍了经典诵读活动开展的背景，云南省经典诵读活动情况，经典诵读活动的必要性和经典诵读写的要求。

    ――中华书局经典教育推广中心祝安顺主任从历史发展角度，梳理了优秀传统文化教育的历史和现状，结合中、高考改革风向标，以《论语•学而篇》为载体，在文本解读中阐述经典诵读活动的意义：文化有力量，文化有信仰，语言里藏着思维方式，传统文化是中华民族的文化基因。

    ――语文特级教师、“素读”经典课程创始人陈琴老师以《楚辞•渔父》一课为示范，通过吟诵、歌诀乐读法，在有气韵的诵读氛围中，漫谈《传统文化中的书声气质》，诵读经典，涵养性情。

    ――北师大文学院李山教授以文化为切入点，具体阐述现代教育中的文化缺失。讲述经典诵读的意义，思考训诂学、音韵学、文字学在基础教育中如何渗透。通过文字的千年演变，文学、文化的传承，明确了教师在传递传统文化中必须担当的责任。

    ――云南人民广播电台播音指导朱滨老师，从有声语言表达的角度出发为大家上了一堂生动的普通话朗读课。他指出：有声语言表达的关键就是理解稿件，具体感受，行之于声，及于受众。对于作品层次的把握就是学会理清层次，深入理解作品，传递真情实感，动人心腹。

    ――台湾第一女子高级中学国文教师易理玉女士为大家带来了一堂生动的国学课：慎重交友。以“失落的一角”的故事为引入，让学生在活动和讨论中理解传统文化里的深刻道理。教学效果令人拍案叫绝。

    研讨收到了良好的效果。参加研讨的很多老师表示：两天的讲座，有如徜徉在艺术的海洋。专家们用通俗的语言阐释了汉字、汉语中蕴含的线条艺术、文化信息、音乐色彩。令汉语言文学的魅力充满会场，如花香一般沁人心脾。
