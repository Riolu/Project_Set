培训班学员感言:归属感
2009年9月29日 15:35

　　从事互联网工作已经五年了，应该说直到获悉上海市府新闻办网络处组织了这个培训班并推出相关资格认证，才真正对这个行业有了归属感，因为原先从事的传统媒体行业都会有相关的资格标识，如记者证、职称等等，而网络由于是新兴行业，什么都没有，这始终让我们在这个领域的从业人员产生一种没有根的感觉，但现在不同了，随着培训认证体制的建立，不仅让网络行业逐步规范起来，逐步提高从业人员整体素质，而且网络编辑也开始有了资历积累，有了行业经验的标识。虽然培训仅有短短的八天，但在这其间，通过这些知名教授、业内资深人士的授课，提高了我们的业务能力，更重要的是，让我们看到了互联网发展的巨大前景，使我们对网络事业充满信心。

　　作为网络工作者，我们大多数编辑通常每天都是一人面对一台电脑过八小时，不要说与外界，与单位内部人员的沟通也是很少的，但此次培训给了大家一个很好的拓展同行圈子的途径，新闻办也帮大家整理了通讯录，来自各单位的同行们可以通过MSN进一步熟悉，相互交流。我们希望在培训结束后，大家能继续保持联络，互通有无，对我们的工作，对我们的生活都是一种丰富。

　　最后衷心感谢上海市府新闻办网络处，和网络处所有的工作人员，是他们的努力和争取才让我们先于其它城市进入网络从业人员培训与认证体制，让我们的事业有了稳步向前发展的基石。

　                 　新华电信宽频网  宋菁菁
