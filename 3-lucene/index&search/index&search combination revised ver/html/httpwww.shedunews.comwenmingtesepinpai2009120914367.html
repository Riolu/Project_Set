开展志愿服务 推进和谐校园建设

2009-12-09 14:27:31    来源：闸北区青少年活动中心   

 

【项目背景】

志愿服务是人类文明发展到一定阶段的产物，它在我国现代化建设的进程中具有举足轻重的地位。在教育系统中，区教育党工委一直将开展志愿服务，发挥学校对社区的服务功能作为精神文明建设的重要推动力。随着时代的发展，校外教育机构在场地、师资、辅导队伍等方面的诸多优势，呈现出的良性发展趋势，社会化、规范化、专业化的发展道路，对志愿服务的开展起着积极的推动作用。
【工作思路】
我们认为“教育发展教师为本，教师素质师德为本”。教师是落实以德治国方略、推进素质教育的实施者，是学生增长知识和思想进步的指导者和引路人，是实现科教兴国战略的生力军。中心以邓小平理论、“三个代表” 重要思想、科学发展观和十七大精神等为指导，以《中小学教师职业道德规范》为依据，以“崇尚师德、争做楷模，树立形象”为主题，以志愿服务建设为抓手，通过志愿服务活动的开展，不断提高教师的思想道德素质，以志愿服务促中心的和谐校园建设。
【基本做法】
中心将志愿服务作为师德实践的有效载体，通过大会宣传、交流展示、学习研讨、典型引路和实践活动等多种形式，积极开展动员部署工作，并要求教师结合中心特色和自身工作实际，设计活动载体，创新活动形式，建立师德实践档案，树立师德实践典型，突出师德教育的实效性。中心还定期开设师德讲坛，以志愿服务为切入点，总结师德学习与师德实践过程中的感人事例和宝贵经验，评选、表彰师德学习与实践活动中的先进典型。
一、落实行动，做好一个“建”字
建立机制是根本。中心志愿者队伍建设，由党支部牵头，中心书记、主任分别担任组长，各部门部长为骨干，成立了志愿服务中心组。以“奉献、友爱、互助、进步”的志愿精神为指导，确立了“完善自我、情暖人心、服务社会”的活动宗旨，采取定期（每学期）和不定期（主题活动）相结合的活动制度。由此，初步构建了活动中心的志愿服务机制。
建设队伍是关键。根据中心部门分工明确、教师各有所长的特点，同时也针对社区服务对象的特性，我们在两个层面上建立了两支志愿者队伍：一支是由19名党员教师组成的党员志愿者团队，另一支则是科教、艺教、群方等以部门为基点组成的特色志愿者团队。
建好基地是保障。我们认识到，中心的志愿服务，要立足校园，走进社区，服务社会。因此，除了中心作为志愿者服务本部外，我们还与临汾街道积极联系，将汾西路261弄居委会等作为志愿服务的基地。
二、志愿服务，突出一个“活”字
两支志愿者团队各有所长，这使实践中的志愿服务活动个体与集体交融、常规与特色互动，一系列的活动在实效显著中更显得灵动、活泼，生气勃勃。
服务类型，灵活多样。有个体社工型：以党员志愿者与名师工作室的志愿者为代表，个体式地为服务对象进行课程、兴趣小组活动的帮助指导。学习助理型：以图书馆的志愿者为代表，立足于与学生、家长进行学习方法、资料搜索等交流沟通。设备辅导型：以科教部的志愿者为代表，进行设施设备的维修、管理、补助、装配、使用指导等。特需支援型：以艺教部、群方部的志愿者为代表，为社区提供有特定需求的艺术指导、活动策划、组织实施等服务。
服务内容，活泼生动。中心党员志愿者们在支部的组织下与汾西路261弄居委会开展了双结对活动，为创建和谐小区共同努力。党员志愿者们与小区内两户困难家庭结对，为了使结对工作不局限于每年节假日的问候，还特意制作了“结对联系卡”，一一注明支部负责人和志愿者的联系方式，使服务对象遇到急事难事时有了“保险”。曹建玉老师坚持一周一个电话，及时了解情况。当她得知13岁的小翁酷爱英语和书法时，便主动联系，让他免费参加中心的剑桥英语班和市民族文化培训班。刘耀媖老师打电话到结对家庭时，得知他们正为女儿如何填写报考大学志愿犯愁时，就主动上网帮他们查找资料，并为他们出谋划策，解决了实际困难。在中秋节到来之际，志愿者又提着月饼和水果为他们送去节日的问候和祝福。同时，志愿者们还对与中心结对的20位贫困学生，或给予个性特长上的指导，或让他们免费参加“水乡同里绘家园——民族文化考察活动”、“相信自己，创造奇迹——主题教育活动”等社会实践活动。科教部志愿者与临汾街道联合举办“百万家庭网上行”活动，为社区居民提供了学习技能的场所；艺教部志愿者为居委合唱队、社区舞蹈队编排出了一个个精彩的节目；群方部志愿者坚持以丰富多彩的群众活动，丰富居民们的文化生活：“科普进社区”、“军民鱼水情”、“少儿书画赈灾义卖”、“七彩纳凉晚会”等一项项主题鲜明、内容精彩的活动，为构建和谐社区做出了贡献；图书馆的志愿者建立了面向社区家长和孩子的心理辅导站，开展了“亲青服务热线”、“姐妹们走进图书馆”、“青少年团体心理训练营”、“忘年交读书会”等各类活动。从07年开始，中心的党员志愿者们还利用暑期休息时间，为全区青少年提供“党员志愿争章”的争章指导服务，受到大家的欢迎。有时间做志愿者，有需求找志愿者。中心的志愿服务物质与精神并进，可谓内容丰富、活泼生动。
服务品牌，活力无穷。我们在思考，中心的志愿服务在发挥中心教育、文化、服务等多项辐射功能的同时，也应创立自己的志愿服务品牌。
05年6月起，由中心党支部、汾西路261弄党总支、区教师进修学院现代教育技术中心党支部等联合举办的“居民网上学习圈”正式启动。投资十余万元在中心图书馆建立了“社区居民休闲廊”，配置了计算机和上网设备，成为社区各层次、各年龄居民网上学习、休闲娱乐的好处所。中心的志愿者们为社区青少年和居民提供网络操作技能培训，积极探索信息技术与社区教育相融合的方法和途径，为促进青少年和居民网上同学同乐，共同提高，创设了有利条件。其中党员志愿者们还冒着酷暑和严寒，在临汾街道汾西路261弄多次举行“信息技术进社区”——便民咨询活动。活动包括硬件维修，软件咨询和上网解答等等……“居民网上学习圈”成为志愿者们一展身手的大舞台，也成为中心努力打造的志愿服务品牌。
如果说，“居民网上学习圈”体现了志愿者的团队精神，那么由区拔尖人才胡文亮老师作为志愿者开展的“社区书法资源共享圈”建设，则是中心充分发挥名师工作室效应，发挥优质教育资源共享最大化理念指导下积极打造的又一志愿服务品牌。这一项探索性的工作不仅已取得了一定成效，而且还受到了区委书记姚海同的赞赏与认可。
【主要成效】
提高教师队伍整体素质是学校工作的永恒主题，也是和谐校园建设的核心工作。中心始终坚持教师的师德建设，要求教师通过志愿服务的开展，将师德实践与自己的理想信念、行为道德和知识能力等完美结合，从而不断强化敬业精神、奉献精神和创新精神。志愿者服务活动的开展，不仅提高了中心全体教职工的思想素质，促进了他们教育观念的转变，增强了他们的敬业精神，还极大带动了其他工作的开展，中心蝉联六届“上海市文明单位”，中心艺教部两次被评为“区优秀师德群体标兵”，去年又获“区先进集体”称号。2008年，“党员志愿服务”、“开展志愿服务 推进师德建设”分别被区教育党工委授予党建特色项目和师德建设项目优秀奖。
【领导点评】
  闸北区青少年活动中心的志愿者队伍自成立至今，多次开展了有规模的志愿者服务活动，得到了家庭、学校、社区的广泛认可与好评。志愿者服务是教师在实践中践行师德、完善自我的课堂,也是动员教师参与和谐校园建设的载体。通过这个项目，活动中心创建了志愿服务的品牌项目，打造了“居民网上学习圈”和“党员志愿考章”特色品牌，探索了志愿服务的组织机制，创建了制度化的工作体系和网络化的发展模式，优化了志愿服务的队伍建设，培养了志愿者的现代服务理念、现代服务素质及现代成功素质。他们提出的用爱心、诚心、热心、耐心和恒心，用奉献、友爱、互助、进步的精神践行“诚心服务，优质教育进千家；竭诚奉献，心心相印连万户”的活动理念，在和谐校园建设中发挥了积极的推动作用。
