第三届湖北省农村中小学教师普通话大赛

颁奖典礼在武汉举行


湖北省人民政府副省长、省语委主任郭生练、教育部语用司司长姚喜双为一等奖获得者颁发证书

    8月28日，由湖北省语委、省教育厅主办的第三届湖北省农村中小学教师普通话大赛颁奖典礼在武汉举行。湖北省政府副省长、省语委主任郭生练，教育部语言文字应用管理司司长姚喜双，湖北省委副秘书长岳勇，湖北省政府副秘书长黄国雄，湖北省人大常委、教科文卫委员会副主任委员张继年，湖北省委组织部副巡视员谢安仁，湖北省委高校工委书记、教育厅党组书记蔡民族，湖北省教育厅厅长陈安丽，湖北省广电局副局长曾婕，湖北省委高校工委副书记、省教育厅副厅长严学军，湖北省教育厅副厅长欧阳建平、黄俭、徐金山、徐雁冰、省教育厅纪检组长王安怀、省教育厅副巡视员涂桂辛、刘怀俊等出席闭幕式并为获奖单位和选手颁奖。


湖北省政府副省长、省语委主任郭生练出席颁奖礼

教育部语用司司长姚喜双讲话

姚喜双司长首先代表教育部副部长、国家语委主任李卫红同志对湖北省农村中小学教师普通话大赛的成功举办表示衷心的祝贺！他在颁奖典礼中指出，党的十七届六中全会提出“大力推广和规范使用国家通用语言文字，科学保护各民族语言文字”，今年的推普周的主题就是“大力推广和规范使用国家通用语言文字”，新时期新形势，推普的重点既要抓好四大领域，又要抓好农村的推普工作，广大农村地区已经成为我们今后推普的重点，农村学校和教师在推普中具有重要作用。湖北省委、省政府高度重视语言文字工作，特别是农村的语言文字工作和农村教师的推普工作。作为湖北省第15届推广普通话宣传周重点活动之一，拉开了推普周活动的序幕，此次农村中小学教师普通话大赛举办得很成功。选手虽然都是来自农村的教师，但是他们的普通话水平和语言艺术的表现力，都体现出了较高的水平，尤其是今天获奖的参赛选手，他们的普通话水平和艺术表现力可以和专业播音员、主持人相媲美。他们的这种对大赛的真情投入，对语言表达艺术的忘我追求，对推普事业的热爱，都给我们留下了深刻印象。有的比赛选手怀着孕，生着病，有的带着喂奶的孩子参加比赛，充分体现了他们的敬业精神，也体现了文化自觉和文化自信。他还指出，近几年湖北省的语言文字工作取得了可喜成绩，二类城市语言文字工作评估任务完成时间居全国前列，三类城市语言文字工作评估势头良好；普通话水平测试质量和规模居全国前列；推广普通话宣传周活动特色鲜明、亮点突出、成效显著。他希望湖北省要进一步发挥农村教师在推广普及普通话工作中的示范和表率作用，做好农村语言文字工作，全面推进素质教育。

湖北省委高校工委书记、教育厅党组书记蔡民族出席颁奖礼

 

湖北省教育厅厅长陈安丽出席颁奖礼并致辞

陈安丽厅长在颁奖典礼致辞中指出，这次全省农村中小学教师普通话大赛，是对我省农村中小学教师普通话水平的一次检阅，是我省加强农村教师队伍建设的一项重要措施。全省农村中小学教师普通话大赛已经成功举办了三届，形成了一个推广国家通用语言文字、展示农村教师风采、传承优秀文化的具有品牌效应的重要平台。希望广大农村教师能够成为农村地区出色的推普员，辐射带动农村学生热爱国家通用语言文字，让每一个农村学生能够拥有一颗健康文明的心灵，说一口标准的普通话，写一手漂亮的规范汉字。希望这项活动能够形成制度，长期坚持下去，为促进社会主义文化大发展、大繁荣，为建设文明湖北、幸福湖北，为提高我省农村教育质量、促进教育公平再作新贡献。

此次大赛以“提升语言文字能力，传承优秀文化传统”为主题，历时4个多月，共有17个市（州）、神农架林区近百个县市区的4万名农村教师参与了此次活动。
