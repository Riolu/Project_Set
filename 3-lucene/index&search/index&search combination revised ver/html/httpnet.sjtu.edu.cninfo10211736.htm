webmail中的邮件怎么没有了

2015-04-30  点击：[]

一、mail收发方式

大致分为两种：  

1、通过浏览器（internetexplorer、firefox、chrome等）访问mail系统（也就是webmail了，比如学校的webmail地址是http://mail.sjtu.edu.cn）；

2、使用邮件客户端软件（outlook、foxmail等）；  

 

二、webmail中邮件突然没有的可能原因有：

1、使用邮件客户端收取邮件，同时在邮件客户端中没有设置“在服务器上保留邮件”；  

2、在webmail中设置了“自动转发”，同时又设置了服务器上“不保留邮件”；  

 

三、相应设置（以下文档中部分素材来自互联网，清晰度不高，或许和实际操作有出入，同时图片中的部分信息也可能实际不符，请忽略之）

1、邮件客户端设置，在服务器上保留邮件副本。

不同的邮件客户端，甚至同一个邮件客户端的不同版本，设置操作上都不尽相同。下面就一些常见的邮件客户端设置，略作说明。  

Outlook 2007

选择菜单”工具“，然后选择”帐户设置“  

 

在“帐户设置”对话框中，选中需要更改的帐户，然后点击”更改“按钮  

 

在打开的”更改电子邮件帐户“对话框中，点击”其他设置“按钮  

 

在打开的额”internet电子邮件设置“对话框中，选择”高级“选项卡，勾选其中的”在服务器上保留邮件的副本“。  

 

 

Outlook 2013

依次选择“文件”——“信息”——“账户设置”  

 

进入“帐户设置”，选择对应的邮件帐户，点击“更改”  

后续步骤和outlook2007的接近。  

进入“POP和IMAP帐户设置”  

然后选择“其他设置”，进入“Internet电子邮件设置”  

最后勾选“在服务器上保留邮件的副本”就可以了。  

 

 

Foxmail 6

选择菜单“邮箱”，选择“修改邮箱帐户属性”  

 

在“邮箱帐户设置”中，选择“接受邮件”  

在右侧，勾选“在邮件服务器上保留备份”，即可。  

 

 

Foxmail 7.0

主界面选择“工具”-“账号管理”，左侧选择POP帐号，右侧选择“服务器”，点击“保留备份”即可设置收取邮件后是否在服务器端保留备份。  

 

 

Foxmail 7.1

点击右上角“三条线按钮”进入菜单=》账号管理=》高级=》查看服务器备份是否选择了保留天数或不保留邮件备份，如果不希望删除请改为“全部的”。  

 

 

2、Webmail系统中，如果设置了“自动转发”，同时又勾选了“本机内不保留邮件副本”选项的话，也会导致webmail中没有邮件（都被转发了）

进入webmail系统后，选择“首选项”，然后选择“邮箱”。  

看看是否有选择“本机内不保留邮件副本”——如果你希望在webmail中还是要查阅收到的邮件，那么千万不能选中这个选项。
