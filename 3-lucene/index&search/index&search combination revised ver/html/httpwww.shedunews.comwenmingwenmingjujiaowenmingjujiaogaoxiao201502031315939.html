沪上大学生寒假有份特殊作业：给家乡写一首“情诗”

2015-02-03 15:18:21    来源：上海文明网   

近日，多所高校学生已开始寒假生活。记者从上海对外经贸大学、东华大学等高校获悉，不少大学生今年领到了一道特殊的寒假作业——写“情诗”，这些诗歌都有同一个对象——家乡。虽然大部分学生此前缺乏诗歌写作训练，但他们作品中表现的家乡之情也很充沛、细腻。
　　上海对外经贸大学商务信息学院的这道作业题就叫“但愿人长久，千里家乡情——‘写给家乡的三行情诗’”。这几天，已有学生陆续在微博上提交了作业，他们不仅写诗，还给诗配上了家乡的美丽照片。
　　2014级统计2班王瑞臻同学来自四川攀枝花，他写的诗是“年幼，我踮起脚去触摸冬日里的温暖/长大，我背着包去寻找四季中的繁华/而你，家乡，你在等我，而我却还未归来”，并配上了一幅题为“冬日阳光花城”的家乡漂亮图片。因为文字深情款款，这条微博很快被当地媒体官微转发。
　　家在本市奉贤的唐翊婧同学写了一首颇具田园风情的诗:“抓田里的一撮稻米/取河里的一瓢水/就是你不一样的味道。”松江人李雨洁同学的诗也充满暖意:“曾经，在你的臂弯里快乐地玩耍/现在，望着你那熟悉的背影，回味曾经的点滴/将来，会用最感激的爱意去守护你。”来自杭州的周庄蝶同学则写了一首藏头诗:“浙里水绕吾梦/江边人盼我还/美不过此乡。”
　　无独有偶，东华大学也布置了一道类似的作业——寻找社会主义核心价值观的印象符号。该校团委教师介绍，所谓“核心价值观的印象符号”，可以是“童年儿歌”的歌词，可以是诗词，可以是经典语录，也可以是成语俚语。不少同学选择了自己摘录或编写诗歌。其中不少作品也和家乡有关，如一名来自浙江绍兴的同学写了首赞美家乡的诗:“曾经的你，是鱼米之乡，也是桥乡、水乡/现在的你，灯红酒绿，火树银花/不管你的前世今生/我爱你的心从未改变。”
　　上海对外经贸大学团委书记陈颖辉表示，高校倡导学生关注家乡的变化，通过新媒体平台展示自己的诗作，抒发对家乡的热爱与赞美，这样的寒假作业特别有意义，“不少学生是‘宅一族’，如果没有聚会，寒假期间他们大多会‘宅’在家里。给他们布置‘情诗’作业，就是想让他们走出家门，感受家乡的自然风景和人文环境，再用诗歌和照片来表达心声。从初期交上来的‘作业’看，我们初步达到了这个目的，有同学告诉我们，自己第一次发现原来家乡也很美丽很可爱，以后会更多地体验和观察家乡。”
