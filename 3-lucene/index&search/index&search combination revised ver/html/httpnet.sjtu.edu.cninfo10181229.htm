关于进一步做好我校域名管理和网站备案工作的通知

2010-06-25 佚名 点击：[]

各分党委、党总支、直属党支部：

各院（系）、部、处、直属单位：

为加强校园网站域名管理、登记备案及内容建设工作，营造安全、和谐文明的互联网环境，创建平安世博。根据工业和信息化部< <关于进一步落实网站备案信息真实性核验工作方案(试行)>>的相关要求，我校将加强对中共上海交通大学委员会< <关于做好我校网站备案及管理工作的通知>>[沪交委潘字（2005）66号]精神的贯彻落实，现将相关工作通知如下:

1.各单位应尽快成立由分管领导牵头负责的专门工作队伍，集中精力，认真组织实施由本单位主管、主办或承办的网站登记备案工作。及时取消失效、过期或不用的域名，对有特殊需要而申请保留的域名或没有具体网站的域名也应主动在备案系统内登记。相关备案工作要求在2010年7月10日前完成。网络信息中心将对逾期未备案的域名和网站暂停服务。各单位可以登录http://net.sjtu.edu.cn/beian（用jaccount帐号，即E-mail帐号进行身份认证）, 查询现有未备案的域名。

2.新申请域名的用户需填写域名申请表（申请表可到网络信息中心申领或到其网站上下载），由各分党委（总支）书记签字，并加盖分党委（总支）公章，递交网络信息中心用户服务部，并在一周内进行域名备案登记，不备案的域名将不能被使用。用户应在建成相关网站后再申请域名（不建网站的除外）。

3.域名备案登记采用在线方式填报。登陆http://net.sjtu.edu.cn/beian，用jaccount帐号进行身份认证，在线填报备案信息。如申请域名备案的网站前期已在工信部进行过备案，还须注明已备案ICP登记号。

4.域名备案由网络宣传管理办公室、保密办公室和信息化办公室共同对相关网站内容(含栏目)进行合法性审核。网站内容一经审核通过，将由信息化办公室批准并下发网站备案编号：沪交ICP备***，同时在备案网站上公布。没有通过审核的网站域名将被取消，没有具体网站的域名，原则上只作登记不批备案号，也不得用于提供网站服务。

5.备案工作完成后，各网站应将网站备案编号刊登在网站首页显著位置，便于学校的核查和管理。

6.学校将定期对已经备案的网站内容进行审查，审查不通过的网站需进行整改。整改后仍不符合要求的网站，将暂停其域名使用或取消域名；对因管理不善而造成不良影响的网站，学校可根据实际情况暂停其互联网接入服务。

7.用户一经网站备案登记，即承诺执行《互联网信息服务管理办法》(国务院令第292号)、《非经营性互联网信息服务备案管理办法》(信息产业部令第33号)等相关规定，详情请参阅http://net.sjtu.edu.cn/ReadDoc/rule.jsp。

8.各单位在备案工作中如遇到相关问题，或有任何意见和建议，烦请及时联系有关部门。
