    新华社社长蔡名照会见埃菲社社长

    2015年09月21日 18:09:49 来源： 新华网

    9月21日，新华社社长蔡名照在北京会见西班牙埃菲社社长何塞·安东尼奥·贝拉。 新华社记者 姚大伟 摄

    新华网北京９月２１日电（记者谭晶晶）新华社社长蔡名照２１日在北京会见了西班牙埃菲社社长何塞·安东尼奥·贝拉。双方就通讯社业务转型发展和加深相互合作进行了交流。

    蔡名照说，面对互联网迅速发展带来的挑战，传统通讯社应积极探索变革，实现数字化转型，以更好地占领新闻市场。新华社在新媒体领域进行了有益尝试，致力于向网络媒体用户提供更丰富的新闻信息产品。

    他表示，新华社和埃菲社自１９８９年签署新闻合作协议以来，合作非常顺畅。目前两社正就互购新闻协议进行磋商，希望今后双方合作能向更广领域、更深层次拓展。

    贝拉说，埃菲社与新华社长期保持着密切合作。在新的媒体发展环境下，埃菲社希望继续加强与新华社的交流与合作，借鉴新华社在新媒体报道方面的有益经验。
