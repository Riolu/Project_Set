王老师谈留学那些事儿

第五篇：留学怎么学？选课自由强度大

王晓霞

今年暑期的赴美考察，我受邀在女儿所在的加州大学洛杉矶分校（UCLA）访问。美国大学的学期制主要有早学期制、学期制、学季制、三学期制等四种，美国大学采用最多的是学期制，其次是学季制。学期制即将九个月（除掉3个月的暑期）的学年共分成两个学期，耶鲁大学就是典型代表；学季制即将一年分为春、夏、秋、冬四季，每一季一学期，加州的大学大多采用学季制。因此，以下的观点主要以学季制的UCLA为例做一个阐释与分享，并以大四在读女儿的专业变化予以佐证。

美国大学最大的特点之一，就是专业选择的高自由度与课程选择的多样性和通识性。

女儿在UCLA的专业选择如下：

大一，主修经济学，辅修德语；大二，从经济学转修计算机与语言学，加修第二专业德语文学；大三，辅修音乐产业；大四，加修音乐表演。

最终，女儿将获得语言学、德语文学双学位，计算机和音乐产业辅修学位。

作为人文学位获得者的女儿，大学四年，不仅选修了与“两个主修+两个辅修”学位相关的专业课程，还选修了许多通识课程，如，从生理学和社会学的角度看待性、世界文化和宗教的荣与辱、心理学入门、太阳系和星球、宇宙的本质等，由上观之，美国大学专业选择的自由度以及课程选择的多样性与通识性可见一斑。

美国大学最大的特点之二，就是课程学习的高强度与自主性和互动性。

UCLA每学季有10周左右，去掉期中考周、期末考周，上半学季、下半学季分别只有4周左右的学习时间。第一周试听课，确定选修课程；第二周开始准备专题报告以及应对老师布置的繁重阅读与写作练习；第三周开始进入期中考准备状态，第四周则全力备考，第五周期中考结束；之后，进入到下半学季，除了第六周在等待考试分数中适当放松一下心情，其余几周完全和上半学季一样。

由此可见，以学季制为代表的UCLA课程学习的高强度。

而我亲身经历的暑期（暑季）课程，分为两期，每期6周时间，比秋季、冬季、春季学季还要短。

第一期，我虽然选了发展心理学导论一门课，但是，这门课分三个班，由三位不同的老师主讲，征得老师的同意后，我分别去听了三位老师主讲的同一门课。总体感觉，无论是中美老师还是美国不同老师对于这门课程的理论和实验研究的把握都很到位，差距不大。虽然第一期三位老师主讲的同一门课程，在内容上大同小异，但前后顺序以及实际案例各不相同。

第二期，我选了教育社会学、学前教育学、亚裔美国人（Asia American）三门不同的课。 老师的讲义是要上传的，选了该课的学生要紧跟教授的进度走，因此，要提前在选课系统进行网上注册并下载、预习讲义，查找相关资料，进行自主学习。

一般课程开始的第一堂课，老师会把这门课的主要内容和基本要求以及上课方式给大家做一个简单介绍，

还会让学生自由组合成学习小组，给定学习任务后，由小组代表上报小组成员、选题以及汇报时间。从第三周开始就依序有小组汇报；一般四五人一小组，一次一个小组汇报一个专题，汇报程序是：首先，小组成员上台轮讲一遍；然后，台上同学与台下同学互动答疑；最后，老师小结。接下来，老师再讲新课。为准备课堂汇报，课下小组同学就要认真分工、寻找材料、研究讨论、形成报告、最终在课堂上展示、汇报，既分工又合作，共同完成专题学习汇报，集体接受全班同学乃至老师的答问。一般一门课程要与同学合作完成至少一个专题汇报，参与多个专题的互动交流，这就是美国的课程学习，老师根据学生的平时表现给出平时成绩，大概占课程总成绩的30%左右。

王晓霞：副校长、副研究员、博士、文中母亲

中国教育科学研究院课程教学研究中心副研究员，厦门实验中学科研副校长，教育学博士。

主要研究领域：语文教育、国学教育。2002年到中央教科所工作至今，现任中国教育科学研究院课程教学研究中心副研究员、中国教育科学研究院厦门实验学校（厦门实验中学）科研副校长。

宋黛霆：留学生、文中女儿

美国加州大学洛杉矶分校大四学生，毕业于人大附中。自小热爱音乐和旅行，曾背着吉他独自前往欧洲和当地乐队合作演出、街头表演等等。
