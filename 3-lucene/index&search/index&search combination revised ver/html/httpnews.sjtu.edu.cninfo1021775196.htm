2015中国青年创业领袖论坛在上海交通大学举行[图]
[发布时间]: 2015年10月26日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 孙佳

10月25日，“中国梦 创未来 2015中国青年创业领袖论坛”在上海交通大学闵行校区李政道图书馆报告厅举行，大米科技CEO杨健,行业领军企业“饿了么”创始人、交大校友张旭豪，高维资本创始合伙人、高维创业岛创始人兼CEO郭鑫等青年创业领袖与数百名交大学子共聚一堂，聚焦青年创业热点问题，分享创业经验体会，为青年学子创新创业答疑解惑。  

中国青年创业领袖论坛由上海交通大学发起并创办。上海交通大学党委副书记朱健，成都电子科技大学党委副书记申小蓉，辽宁省大学生就业局副局长王宪明，上海地产闵虹集团副总经理邱彤，上海市教委学生处副处长李长治等来自政府、高校、企业的嘉宾与广大创客一起，共同探讨新环境下的双创发展和沉淀多年的创业感悟。论坛由上海交通大学EMBA校友、上海广播电视台电视新闻中心制片人、著名主持人刘江贤和上海交通大学凯原法学院学生边宇共同主持。   

杨健：“创业或人生都首先要想明白三个问题”

嘉宾杨健现为大米科技CEO，曾任达沃斯世界经济论坛“全球杰出青年社区”执委，全国大学生创业标兵，团中央十七大代表，中国青年创业国际计划（YBC）导师，有大学研究中心主任、学术期刊主编、国企总经理、地级市政府副秘书长和自主创业等多重职业经历。经历丰富的杨健讲述了自己从连“.Com”都不知何意的“门外汉”成长为“大家电脑有问题都来找”的“准专家”的过程，讲述了自己因为要“用一年不长的时间，做一件一生难忘的事情”从而去青海支教的历程；讲述了“用半年时间把第一个公司‘做黄’”的历程，也讲述了最后“24岁研究生创业成为百万富翁”挖到第一桶金的“传奇经历”。   

阅历丰富的杨健经历了几次成功的创业后，此次创业将目光投向了与食品安全密切相关的领域——大米，他创立大米科技公司，“用科技武装大米”，利用双向溯源系统，实现大米生产、流通和消费的全环节监控，让消费者吃到正宗的五常大米。杨健团队还同时开发出智能米桶和智能电饭煲，智能米桶能够定量出米、除虫防潮，具备温度、湿度、米量实时监测等功能，通过使用大米科技手机APP，不但可以监测大米的剩余量并直接手机订购，还能随时随地远程遥控做饭，实现到家即可吃饭。借助“互联网+”思维，大米科技正在从线上到线下打通从农田到餐桌的全供应链，可以说杨健不是在卖大米，而是创造了一种全新的大米交易模式。   

杨健深有体会地告诉大家，创业和梦想和人生一样，首先都要想明白三个问题：“你想做什么？”“你能做什么？”“你做了什么？”他用自己在每一个人生关键时刻的经历和思考告诉大家：知道自己想做什么确定人生目标，根据自己能做什么从而选择通往目标的路径，然后每一段时间都回头总结一下自己做了什么，“创业者的人生就应该如此”。   

人生苦短，无暇平庸。杨健诙谐地表示：“大学生最大的优势是可以不断去尝试”，所有的经历甚至失败经历都可以成为人生的宝贵经验。他鼓励同学们“找到激情和梦想，然后把它们付诸实施”，“不知道怎么办的时候先做了再说”，“经营好自己的人生之路”。   

张旭豪：“大学生创业和经商有区别”

张旭豪是行业领军企业“饿了么”网上订餐的联合创始人兼CEO，张旭豪2009年在交大校园内开始创业，如今员工达4000多人，订餐平台覆盖全国超过250个城市，拥有20万家餐厅（品牌馆商家4000家左右）及2000万用户，公司现估值已达30 亿美金。张旭豪本人曾获“上海大学生建功立业单项奖”、中国创业创新界最高奖“年度创业家”奖等多项荣誉。但他对交大学子们讲述的创业体验，却是从在学校周围店铺做兼职接单和外送讲起。   

张旭豪回顾了自己和同寝室的四位室友一起“在寝室开始的创业”，饿了么的NAPOS是全国第一个专门针对餐厅商户后台管理而研发的软件。最开始，餐厅老板们在没有足够利好预期的情境下，并不愿意轻易尝试。为了让餐馆老板们了解并接受这套网络后台管理系统，饿了么的团队曾经一家一家地免费试装，甚至邀请他们来公司参观，给他们讲述饿了么对于外卖O2O的设想和公司的理念。事实上，不少中小型餐厅第一次安装电脑，第一次通过互联网的方式做生意，都是缘起于和饿了么平台的合作。   

“现在没有一个行业是一个人就可以做的，所以团队很重要。”张旭豪在创业过程中，团队也在不断变化。最开始一起创业的四个室友，在需要“休学创业”的时候已经有不同的选择，在大学毕业之际也有各自的侧重，张旭豪从中得出的结论是：团队一定要成长，个人要和团队一起成长，共同进步。   

今年“饿了么”以公司名义向母校上海交通大学捐赠500万元成立创业基金，鼓励更多交大学子投身创新创业，实现人生价值。但同时张旭豪也告诉学弟学妹“不要执着于总结创业成功的经验”，他认为所谓成功，其中个人因素只有30%，更多的是大势所趋；同时也不是“要赚多少钱才算成功”，而是要真正对社会发展有意义的事情。张旭豪认为，现在创业条件和氛围已经都很好，大家反而要心态好，不要执着于“所谓成功”，因为“成功永远是小概率事件”，“追求成功不如追求做有意义的事。”   

张旭豪认为，大学生创业最重要的不是赚多少钱，而是通过创新的方式，通过新的手段、新的商业模式、新的技术给社会创造价值，从而能够大大提高效率、降低成本，“这是大学生创业的根本，我们不能完全为了钱”。   

郭鑫：“互联网不是万能的”

如今似乎所有的项目都愿意戴上“互联网+”的帽子，但郭鑫却告诉大家：互联网不是万能的。   

郭鑫是高维资本创始合伙人、高维创业岛创始人兼CEO，他曾受邀于北京航天城与习近平总书记座谈青年创业，随汪洋副总理出访德国，被美国彭博社评为“全球30岁以下杰出创业家”，先后获得过“中国大学生年度人物”等荣誉，是“第19届中国青年五四奖章”的唯一在校大学生获奖者。   

郭鑫大二时创办注册的农产品跨境电子商务平台诚信通，把退耕还林地区农民生产的林下产品经由这一平台销售到国外，帮助农民增收。在不到一年时间里将全国18个省的一百多个县都接入到他创建的农贸对接平台上，与全世界近30个国家互联互通，使国内一千万农民受益。一年后被一家知名企业以上亿元的市值估值并购。获得人生第一桶金后，他选择做一名天使投资人，并打造了创新公寓项目，申请入住的创业者，需要通过郭鑫团队的审核，符合条件的入住者，不仅享受设备完善的居住条件、低廉的房租，还会得到团队的帮助。目前，北京创新公寓已经引进了外部的投资机构，变成了一个公众的创业平台。2014年３月，郭鑫又和合伙团队创办了“闯先生网站”，为青年创业者提供一站式服务，从而实现“让创业如此简单”的使命和愿景。去年年底，郭鑫又发起成立了高维“创业岛”，对于各类创业项目进行深度评估，挖掘其中的潜力项目进行资源注入和模式打磨，然后引入拥有实战经验的企业家担任创业者的“副驾驶”。   

“失败乃成功之母”——成功的郭鑫一直在讲述创业初期的“败笔”：最初做的网站只有两个人浏览，还是自己团队；最初和别人谈设想总是被质疑；最初连公司都没有，营业额已经过亿还没有赚到钱；刚刚赚到钱就面临被合并的局面……郭鑫告诉大家：任何创业都是击败竞争对手然后活着，所以没有研究清楚之前不要轻言用互联网颠覆，“互联网只是一种好的应用模式，但不是万能的”，所以创业是“轻易不要开始，想好不要回头”。   

纵观郭鑫的创业路，这位90后四川小伙已经从一名创业的独行侠一步步变成了制造创业的“生产商”，开始为更多的志同道合者打造更为便捷强大的创业通道。   

“大学最主要的是学到快速学习的能力”

“天下武功，唯快不破”。“大学最重要的是学到学习特别是快速学习的能力”。除了畅谈如何创业，三位嘉宾也非常愿意和大学生们分享更多的人生体验。  

大学生到底应不应该休学去创业？相信这也许是很多学子都很关心的话题，对此，创业领袖都没有给出直接的答案，但他们的经历无疑就是最好的回答。用嘉宾观察员的话来说就是，几位嘉宾有几个共同之处：他们洞察了社会，了解社会需要什么；洞察和掌握了技术，知道如何用技术去颠覆原有的模式；他们还洞察了自己，知道自己能做什么，有了这样的洞察力，自然知道在每一个时间阶段应该做什么了。三位创业者都认为：大学阶段非常重要，除了学到知识，还学到了学习特别是快速学习的能力，这让自己在走上社会后的创业中，能够快速掌握响应的技术和能力。   

“没有人能随随便便成功，如何克服创业过程中的负面情绪？”对此，虽然年仅23岁，却已在汶川地震中经历了生死考验的郭鑫告诉大家，自己最喜欢的书是《创业维艰》，因为书中遇到的所有困难自己都遇到过，“无论遇到什么问题，和当初在地震中活下来比都不算什么”。他建议同学们要“经常归零”，遇到挫折之际就想想初衷，把所有挫折都当作学费，不要着急做决定。   

“大学的创业教育能教给学生什么？”张旭豪认为：大学的创业教育更多的应该是知识和服务，学校可以教给大家更多的知识，比如财务知识、商业知识、基本商业技能等，“把基本功夯实也重要”。   

“散播一片创客心，引领万千追梦人”。改革开放带来了中国30余年来的快速发展，那么未来快速发展的契机是什么？也许这就是“大众创业、万众创新”的最重要意义。引导、鼓励和培育更多大学生放飞创业梦想，或许正是时代赋予高校创业教育的历史责任。
