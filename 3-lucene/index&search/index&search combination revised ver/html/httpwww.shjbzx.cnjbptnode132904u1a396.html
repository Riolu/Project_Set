培训班学员感言:创造、责任、理想
2009年9月29日 15:34

　　经过这一个月八天的培训，经历了各位老师精彩的讲演，受益匪浅。

　　第一，我觉得自己的大局观得到了提升。作为网络编辑来说，已经不仅仅是一名搬运工。如果要提升必须要创造东西，可以是对现有资源的整合来创造，可以是增加原创的稿件。这样才能摆脱机械性的劳动，提升自己。

　　第二，我觉得对于自己的责任感有了加强。网络传播速度快幅面广，每次作出的稿件影响的范围很广。所以每次按下发布按键时，都要对稿件的内容负责。

　　第三，要坚持树立理想。树立理想从小时候就讲起，但年龄长大，经历世事，我们常常会说是岁月磨平了我们的棱角，也磨掉了我们的理想。我觉得这样是要不得的，理想是自我发展的原动力。我们要相信什么，要坚持什么，要致力于什么。这些问题将伴随我们一生。

　　                             上海热线  马焱
