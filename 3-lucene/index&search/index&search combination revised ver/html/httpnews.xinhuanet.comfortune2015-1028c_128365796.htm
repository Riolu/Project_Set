A股再现深V反转 “十三五”概念股砸出黄金坑

2015年10月28日 07:42:33 来源： 证券日报
  • 评论0
  • 打印
  • 字大
  • 字小

　　■主持人 张晓峰 吴 珊

　　两因素支撑深V反转

　　《证券日报》记者：沪指昨日上演V型反转好戏，早盘低开后跳水，一度大跌近百点，下探至3332.62点后企稳回升，最终在创业板的带领下，沪指展开持续回升，最终小涨0.14%，报3434.34点。请问，是什么原因导致A股盘中V型反转，投资者应该如何判断？

　　天信投顾：在央行“双降”释放7000亿元左右流动性和十八届五中全会召开的背景下，昨日A股出现盘中V型反转，短期走势值得期待。据最新数据显示，上周四（10月22日），沪深两市的融资余额达到9983.05亿元，逼近万亿元大关，而当日的融资融券余额则成功突破万亿元大关。实际上，融资余额已连续4日维持在9800亿元左右的高位。这说明市场信心仍然处于恢复和提振阶段，未来股指或出现大幅震荡格局，但是暴跌走势出现的可能性是微乎其微的。

　　中证投资：上周压制股指4个月之久的60日均线被突破，反弹行情已经基本确立。不过，大盘刚刚脱离均线密集交汇区，越往上走，面对的压力越大，反弹之路并不平坦。但行情尚未走完，盘中市场的波动幅度会放大，强势的基调不会变。这主要是由于资金利率下行与风险偏好上升的双支撑。一方面，央行“双降”后，场内流动性充沛，无风险利率下行的趋势不改，在资产配置荒背景下，股市将显著受益流动性外溢影响；另一方面，伴随着场外配资清理基本结束、汇率逐渐稳定，市场中的不确定性因素已经大大减少，投资者的风险偏好正在缓步上行。昨日盘中快速跳水但并未演变成大幅杀跌，折射出资金情绪已经有所恢复。

　　有望构筑新的震荡平台

　　《证券日报》记者：沪深两市股指昨日低开低走，沪指盘中跳水跌近3%，临近尾盘强势翻红。从技术面来看，股指将如何运行？

　　巨丰投顾：目前沪指3400点一带套牢盘众多，央行双降利好将缩短市场在这一位置的震荡周期，后市依旧向好。昨日股指快速杀跌并击穿3350点，午后再上攻3450点，行情颇为极端。预计大盘将在3300点（3200点至3450点）一带构筑新的震荡平台。

　　天信投资：总体来看，股指从10月16日进入3400点压力区间以来，近两周的时间均维持在3400点一线横盘震荡，同时伴随着成交量的放大。市场的横盘走势即将进入尾声。而昨日市场上演了惊人的百点大逆转，更加确认短期横盘震荡走势的结束。

　　大摩投资：从技术面来看，沪指重回3400点关口，创业板指则创出反弹新高，从分时量能可以看出市场在探底反弹的过程中不断有资金进场抄底。由于指数上方面临较重的套牢盘压力，指数每向上一步都需要消化解套盘和获利盘，因此就需要有反复的换手，未来预计像早盘这种快速跳水将会频繁发生。

　　关注“十三五”概念股

　　《证券日报》记者：昨日大盘再现深V反转，在经过盘中短时间的暴跌后，创业板表现强势，题材热点也再度活跃，指数积蓄了一定动力，航母、卫星导航和健康中国概念板块涨幅居前，后市有哪些热点值得持续关注？

　　中方信富：最近，受十八届五中全会正式开幕的影响，湘邮科技、同花顺等软件服务概念股以及九安医疗、乐金健康等健康中国概念股表现活跃，考虑到软件服务、环境保护、新能源锂电池、智能机器、健康中国等方向有望成为“十三五”规划相关的重要领域，故上述热点在经过盘中的剧烈波动后，仍有望反复活跃。

　　齐鲁证券：短期市场仍将维持宽幅震荡走势。从相对较长的时间段来看，随着市场情绪逐渐转暖以及资金成本维持低位，市场将进行一轮估值修复行情，蓝筹股将表现突出，建议关注以券商为代表的大金融板块和估值修复行情。
