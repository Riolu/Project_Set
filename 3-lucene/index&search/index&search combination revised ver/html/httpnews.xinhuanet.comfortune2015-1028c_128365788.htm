国信总裁陈鸿桥自杀被指因抑郁 家属称无遗书

2015年10月28日 07:41:29 来源： 北京青年报
  • 评论0

　　对于陈鸿桥的“突然死亡”，外界充满了猜测。第一大疑点就是他和已被免职、目前正在接受调查的证监会前主席助理张育军之间的关系。公开资料显示，张育军与陈鸿桥两人曾在深交所有过多年的交集。第二大疑点则是，在已经落马的张育军所主导的本次救市中，陈鸿桥和他所领导的国信证券到底扮演了何种角色。

　　本报讯（记者 朱开云）昨日上午，国信证券总裁陈鸿桥遗体告别仪式在深圳举行。10月23日，国信证券总裁陈鸿桥突然身亡的消息，震惊了整个证券市场。适逢金融业反腐进入“深水区”之际，陈的身亡不免让人浮想联翩。昨日，在遗体告别仪式上，万科集团总裁郁亮透露陈鸿桥自杀实因严重抑郁症，而非流言所述。

　　上周五收盘后爆出陈鸿桥身亡消息

　　10月23日股市收盘之后，传出了国信证券总裁陈鸿桥在阳明山庄家中自缢身亡的消息。网上对其死因一时传言四起，被边控、留有“请勿扰妻儿”五字遗书等流传甚广。

　　公开资料显示，陈鸿桥1994年起担任深圳证券结算有限公司副总经理，1997年任深交所资金交收部总监，2001年任深交所创业企业培训中心主任，2003年起任深交所副总经理。2014年6月，陈鸿桥接棒胡继之正式出任国信证券总裁。2014年12月29日，国信证券正式登陆A股。

　　家属再次澄清没有遗书

　　对于陈鸿桥的“突然死亡”，外界充满了猜测。第一大疑点就是他和已被免职、目前正在接受调查的证监会前主席助理张育军之间的关系。公开资料显示，张育军与陈鸿桥两人曾在深交所有过多年的交集。第二大疑点则是，在已经落马的张育军所主导的本次救市中，陈鸿桥和他所领导的国信证券到底扮演了何种角色。

　　昨日，陈鸿桥家属现场再次澄清并未发现遗书，国信证券也发文表示这些传言严重伤害家属和公司感情，保留追究造谣传谣者责任的权利。

　　告别仪式上，万科集团总裁郁亮透露陈鸿桥自杀实因严重抑郁症。他说，陈鸿桥是北大的优秀校友，是公益事业带头人，是深圳资本市场的建设者，是金融、管理领域的专家，更是一名低调的实干家。“虽英年早逝，却是立德、立功、立言的一生。”

　　国信证券称陈鸿桥积极落实监管要求

　　昨日，北京青年报记者联系国信证券方面，办公室相关工作人员表示，目前，公司各项业务和日常管理工作均正常开展。昨日，国信证券在公司网站上发布的公告称，陈鸿桥任职公司期间，始终以党和国家的方针政策为指引，积极落实监管机构的监管要求，全力做好资本市场稳定、建设工作。陈鸿桥先生对工作精益求精、勤勉敬业，对同事谦和友善、平易近人，为公司的经营和发展做出了重要贡献。
