中日民间联手索要古石遭拒 市民：古石在日本存在是耻辱

2015年10月28日 07:03:41 来源： 环球时报
  • 评论0
  • 打印
  • 字大
  • 字小

　　27日，中国南京民间抗日战争博物馆与南京市民代表团前往日本宫崎县厅，递交书面申请，要求日方归还战争时期掠夺的部分用作“和平塔”基石的古石。对此，虽然日本爱好和平的市民团体认为应该归还给中国，但是日本宫崎县首次明确表态拒绝归还。

　　据共同社27日报道，中国南京民间抗日战争博物馆馆长吴先斌当天向日方递交了书面申请。宫崎县城市规划课课长森山福一接受了申请，但他表示拒绝归还。森山福一说：“和平塔在战后为县内外的人们所熟知，无法拆除。今后会继续维持现状妥善保存。”据《读卖新闻》报道，该县城市规划课负责人称“没有证据表明塔上基础部分是掠夺的，中国方面申请书上的内容也无法确认。”吴馆长对此表示，打算不断登门直到日方改变错误认识。

　　报道称，博物馆要求归还的是刻有中国传统神兽“麒麟”等图案的三块古石。馆方在与宫崎县日中交流团体策划战后70年纪念活动的过程中了解到这些古石的存在，并希望收回。

　　与官方的态度不同，宫崎县市民对中方博物馆的行动表示支持。日本《新闻邮刊》11月6日号刊称，目前，日中市民团体联合起来，要求日本归还有历史文物价值的基石。日本“宫崎县日中友好运动恳谈会”对中国民间的这一要求给予全面支持，该组织曾最早代表中国方面向宫崎县厅转达了索还的意向。“这样的东西在日本存在对日本也是一个耻辱。”该组织代表称，“日本和平塔上还有孙文墓的部分石材。这对于中国是侵略的象征。我们和南京的市民团体交流了约3年，在对话中，自然而然地开始共同向宫崎县索要被掠石材。” （林梦叶）
