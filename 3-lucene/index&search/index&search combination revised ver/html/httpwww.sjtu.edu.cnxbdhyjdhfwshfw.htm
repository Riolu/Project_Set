地方合作

　　上海交通大学地方合作办公室是学校与地方进行联络与合作交流的职能部门，主要工作职责是：

  • 根据学校总体发展目标和地方社会经济发展的实际需要，制定学校与地方合作的规划；
  • 构建服务社会运行机制，组织、协调学校各学院、职能部门为地方经济社会发展提供成果转化和产学研结合、人才培养和交流、战略规划和咨询等方面的服务；
  • 负责学校与地方合作协议的签订和实施；
  • 对学校与地方开展的合作项目进行跟踪、协调、评估和考...

出版社

秉承一个世纪前中国出版业的先驱——张元济立下的出版理念：以出版扶持教育为己任，发源于中国最早成立的近代大学出版机构——南洋公学译书院的上海交大出版社艰苦创业、探索改革，已成为建设精神文明和物质文明的坚强阵地、出版高校教材和学术专著的坚实基地、反映学校教学质量、学术水平、科研成果和人才培养的重要窗口。

其他服务

　　非学历教育：为体现我校服务社会的职能，充分利用办学资源，为构建学习型社会和全民终生教育体系作贡献。我校成立了非学历教育培训管理领导小组，负责学校各项非学历教育培训的管理工作，领导小组下设非学历教育培训管理办公室，对学校的教育培训进行归口管理。管理办公室管理范围包括举办研究生层次的培训教育、外国留学生的培训教育、继续教育、远程教育及其各类培训、进修、研修（讨）等各种形式的教育培训。

　　上海交通大学分析测试中心：是一个具备向校内外科学研究和品质鉴定提供公正、权威测试数据能力的重要机构。主要面向物理、化工、材料、纳米、环境、生农医药等众多学科，从事材料微结构分析、有机和无机化合物的定性和定量分析、材料性能测定、计量检定、材料质量综合评定、环境质量检测和药物分析等工作。部分重要仪器已实现24小时服务。
