十八届五中全会前瞻之四

学者观察：“十三五”规划将有哪些特点值得我们关注？

2015年10月27日10:17   来源：人民网－中国共产党新闻网

中国共产党新闻网北京10月27日电 （万鹏）党的十八届五中全会26日召开，会议将审议通过国民经济和社会发展第十三个五年规划。“十三五”规划具有什么特殊重要的意义，其规划的总体布局、具体内容、战略举措和实施保障有哪些值得我们关注？国家行政学院研究员胡敏在接受记者采访时指出，“十三五”规划是实现全面建成小康社会宏伟目标的最后一个五年规划，也是充分体现新一届党中央体现治国理政思想方略的重要规划。他认为，这个五年规划将体现四个方面明显特点，将成为我国经济社会发展关键时期的一个具体行动纲领。

特点一：针对性，为了实现全面建成小康社会目标

“从党的十六提出全面建设小康社会目标到十八大提出到2020年全面建成小康社会的‘两个翻一番’的具体任务，保持国民经济持续稳定发展将成为全面建成小康社会的重要着力点。”胡敏向记者谈到，要取得实现全面建成小康社会的新胜利，规划将更加注重针对性。他谈到，未来五年，发展仍将会被强调为第一要务，经济增长不仅需要稳定的增长速度并保持在合理区间，而且要进一步强调有质量、有效益、环境友好型的增长。而促进稳增长的一切规划、政策和措施，必须针对有利于加快产业结构转型升级、有利于激发一切创新驱动要素的活力、有利于促进区域经济协同协调发展与相互支撑、有利于增进全体人民福祉、实现普惠性的民生改善，由此体现小康社会的全面性。胡敏认为，因此规划中，促进四化并举、推进新型城镇化和农业现代化、促进区域经济协调发展、切实解决精准扶贫的章节有望重点着墨。

特点二：创新性，充分适应我国经济社会步入新常态

“当前我国已经进入经济增长由高速转向中高速、产业由中低端迈向中高端水平的新常态。这一新常态不仅表现为今后一个时期经济社会发展的基本转态，而且经济结构、体制结构、社会结构都将适应新常态下错综复杂的国内外环境和条件发生根本改变。”胡敏认为，要充分适应我国经济社会步入新常态，“十三五”规划将更加注重创新性。他向记者谈到，随着进入新常态，我国经济的发展内涵、改革取向、增长方式、增长动力和发展政策都要切实转变。体现在新的规划中，将更加强调以拉动消费作为经济增长的主动力；依靠创新驱动促进产业转型升级特别是形成全方位创新驱动战略，广泛引入新产业、新业态、新组织、新市场和新模式如新能源汽车、智能产业、互联网+，等等；更加强调用市场的方式推进一批以重大工程和社会基本公共服务为重点的基础设施项目；在推进生态文明制度建设中更加强调人口、资源、环境的匹配并制定过去可行但新常态下必须加以改革调整的新政策，比如人口政策、资源环境使用政策、社会保障政策、收入分配政策等。

特点三：整体性，集中展示“四个全面”的战略布局新构想

“四个全面”战略布局完整地展示了新一届中央领导集体治国理政的总体框架，使得当前和今后一个时期党和国家各项工作、重点领域、主攻方向有了遵循。胡敏认为，为了集中展示“四个全面”的战略布局新构想，此次规划将更加注重整体性。“十三五”规划将贯穿这个战略布局的主线并进行具体勾画和落实并具体体现在政治、经济、文化、社会、生态文明五位一体的总体架构中。就此，整个规划和政策措施会充分彰显发展的精神、改革的精神、法治的精神和体现党总揽全局、协调各方、统筹推进、保障落实的工作方针。

特点四：奠基性，推进国家治理能力和水平新提升

党的十八届三中全会提出完善和发展中国特色社会主义制度、推进国家治理体系和治理能力现代化的全面深化改革的总目标，其改革的核心是要正确处理政府和市场的关系，发挥市场在资源配置中的决定性作用和更好地发挥政府的作用，形成科学的宏观调控体系和有效的政府治理。党的执政能力、政府的科学执政、依法执政、民主执政，都将通过未来我国经济社会发展的一系列规划的决策、制定和执行之中。最后，胡敏指出，要推进国家治理能力和水平新提升，规划将更加注重奠基性。“十三五”规划的出台已经经过了多轮充分调研、讨论和论证，集中了各方面智慧，今后的落实执行以及成效，是对党和国家面向未来、面向世界的治国理政能力的具体检验，也为我们最终能够走向现代化文明国家奠定基础。
