  • 西藏自治区成立50周年庆祝大会

    今年是西藏自治区成立50周年。9月8日上午10时，西藏自治区成立50周年庆祝大会在拉萨市布达拉宫广场举行。

  • 西藏庆祝大会举行 观众进场

    今年是西藏自治区成立50周年。9月8日上午10时，西藏自治区成立50周年庆祝大会在拉萨市布达拉宫广场举行。

  • 高原圣殿布达拉宫

    布达拉宫，座落于海拔3700米的西藏自治区拉萨市中心的红山上，因其建造的悠久历史,建筑所表现出来的民族审美特征,以及对研究藏民族社会历史、文化、宗教所具有的特殊价值，而成为举世闻名的名胜古迹。

  • 大昭寺，朝圣者的天堂

    始建于吐蕃王朝鼎盛时期的大昭寺，座落于拉萨市中心，按照藏传佛教的理想世界“坛城”模样建造。因其供奉着释迦牟尼12岁等身像，大昭寺成为西藏举行重大佛事活动的中心，在藏传佛教中有着至高无上的地位。

  • 在纳木错与自己的灵魂对话

    纳木错是西藏著名的“三大圣湖”之一，藏语意为“天湖”，其秀美的自然风光和独特的民族风情吸引大批游客来此游览。

  • 西藏拉萨市的环保“正能量”

    近年来，西藏拉萨市公交营运投入了新能源公交车。传统出租车全部完成了油改气，纯电动出租车已经开始试点营运。500多辆设置在拉萨市街头的公共自行车为市民绿色出行带来了便利。在居民生活方面，充分利用太阳能热水器、太阳灶等设备。拉萨市节能减排的种种措施，为造就拉萨市的好环境增添了“正能量”。

  • 西藏非遗保护成绩回顾

    藏北当雄县职业技术学校的学生扎西在画唐卡。唐卡一词源于藏文音译，意为“佛祖的卷轴画”，内容以表现藏传佛教为主，题材涉及历史、文化、医学等诸多领域，至今已有1300多年的历史，被誉为“藏族的百科全书”。

  • 西藏是世界环境最好的地区之一

    西藏是国家重要的生态安全屏障，对亚洲乃至世界都有着重要意义。西藏在发展过程中，始终遵循经济规律、社会规律和自然规律，不以牺牲自然环境为代价，注重经济、社会、生态的和谐统一，走可持续发展之路。

  • 澜沧江畔的西藏千年古盐田

    位于西藏东南部与滇北高原毗连处的昌都芒康县盐井古盐田，是至今仍保持最原始手工晒盐方式的盐田。据史料记载，早在唐朝时期这里就开始晒盐。目前该处有盐田3000多块，从事盐业生产的居民有320多户。

雪域脊梁

阴法唐
有这样一位老人，他虽已94岁高龄，仍精神矍铄；各种有关西藏的重大活动、纪念活动，他几乎从来没有落下过；在西藏前后27年的工作和生活，早已让这里成为了他挂念的家。他就是西藏自治区党委原第一书记阴法唐。

叶星生
1961年，13岁的叶星生随十八军父母进藏。如今，他从事西藏艺术创作、研究、收藏已达半个世纪，为民间文化遗产的抢救保护做出了突出贡献。他是西藏发展、变化的参与者，也是见证人。叶星生说，这是他与西藏的缘分。

“国旗老阿妈”次仁曲珍
1965年，西藏自治区成立。国庆节那天，老阿妈和村民们自发地在家门前挂起了鲜艳的五星红旗。从那以后，日复一日、年复一年，老阿妈一直重复着这一件事，每一次，她都那么认真、专注，国旗一次又一次地在她的手中升起。

贡布
贡布，藏族，西藏日喀则地区聂拉木县人。登山运动员，曾荣获“新中国体育开拓者”称号、国家体育运动荣誉奖章。1956年参加中国人民解放军。1959年被选入中国登山队。曾当选为西藏体育总会主席，全国人大第三、四届代表大会代表。

扎拉
突然袭来的强烈地震可以摧毁房屋，但却震不垮一个基层党支部的核心凝聚力；无情的灾难可以夺走人的宝贵生命，但却夺不走共产党员的神圣职责。扎拉和他领导的羊易村党支部成为灾区人民抗震救灾、重建家园的坚强柱石。

孔繁森
孔繁森出生于1944年7月，是山东聊城人。1979年，国家要从内地抽调一批干部到西藏工作，时任地委宣传部副部长的孔繁森主动报名，并写下了“是七尺男儿生能舍己、作千秋鬼雄死不还乡”的条幅。

李素芝
在西藏山南地区曲松县，次仁旺姆把最美好的祝福，献给她不是亲人的亲人，“他是个医生，救我的命，就是个菩萨嘛！”这个“菩萨”，叫李素芝，西藏军区副司令员兼西藏军区总医院院长。

郭毅力
38年军旅生涯，56个生命年轮，他把汗水与心血洒在了雪域高原；从士兵到将军，他把忠诚与智慧献给了党和人民；为了西藏的繁荣稳定，他不惜用生命守候着这一片高天厚土……

西藏大事记

　　1951年5月23日　中央人民政府和西藏地方政府在北京签订《关于和平解放西藏办法的协议》（即十七条协议），西藏实现了和平解放。
　　1965年9月1-9日　西藏自治区第一届人民代表大会第一次会议在拉萨举行，西藏自治区正式成立，掀开了西藏历史的新篇章，昔日的农奴和奴隶成了新社会的主人。
　　1995年11月29日　国务院特准坚赞诺布继任为第十一世班禅额尔德尼。
　　2009年1月19日，西藏自治区九届人大二次会议决定，把每年3月28日设立为“西藏百万农奴解放纪念日”。
