培训班学员感言:精彩回忆 值得回味
2009年9月29日 15:35

　　我来自一家外资背景的网站，之前对网站的策划、经营、管理不是特别熟悉。经过四个周末的培训，感触良多。短短八天里，既提升了专业知识，也了解了政府的管理政策，同时，也和兄弟网站建立了行业联系。

　　培训中，有很多的精彩片断，略捡取若干，以作回味，^_^

　　1、新闻学院黄瑚副院长的缜密

　　2、市新闻办张晓宇处长的大气

　　3、市网宣办谢海光副主任的激情演绎

　　4、张涛甫博士的飞扬文采

　　5、纪华强教授的海量信息

　　6、张海鹰教授深入浅出的讲解

　　7、殷晓蓉教授的独特研究视角

　　8、东方网高国营总监的武功秘籍，现实指导

　　9、程士安教授的最新前沿研究

　　10、新闻办华宾和小徐两位同志的细致周到，备受感动

　　这些都令我们回到了幸福的学生时代，再一次体会到学习的愉悦，在我们享受精神食粮的同时也令复旦的蚊子有机会同享物质食粮。^_^

　　这次培训，也激发了我强烈的求知欲和危机感。因为学习，才知道自己还很无知，学海无涯，需不断努力。因为学习，发现了个人进步的方向，某些同学很可能确立了自己进一步深造的志向。因为同行交流，让我们意识到，优秀还离我们很远，明确了网站的前进目标。

　　本次培训中所产生的师生情谊和同行友谊，将一直伴随着我们今后的工作，更我们创造了相互提携、合作共赢的机会。

　　                       家长汇网  潘江雪
