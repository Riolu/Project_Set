10月27日，买家在观看灯饰。

    2015年10月27日，由香港贸易发展局主办的香港国际秋季灯饰展一连4天在香港会议展览中心举行，该展览一直是亚洲灯饰业界的瞩目盛事，今年已踏入第17届。新华社发（伍永健摄）
