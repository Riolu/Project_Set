中国国际时装周盛大启幕

    在为期9天的时间里，将有90余家中外品牌和机构、73位中外设计师和135位设计新秀携最新作品亮相本季时装周，在北京饭店、751D・PARK、国贸大酒店等场地举办66场时装和相关时尚产品发布，以及“真维斯杯”休闲装、“乔丹杯”运动装备、“锦霓杯”人造皮草时装、“迪尚”时装设计大赛成果奖评选等4项赛事总决赛和1所院校毕业作品发布。发布品牌涵盖了女装、男装、婚纱礼服、运动装、休闲装、牛仔装、童装、亲子装、人造皮草、鞋类和彩妆造型设计等多个品类。本季时装周期间，还将举办DHUB设计汇、新闻发布、沙龙论坛、专题讲座、颁奖典礼等20余项专业活动。[详细]

设计师刘薇 在面料上画出秘密花园

    刘薇将源于东西方文化交流所发生碰撞而产生的多年积淀和对当今世界流行时尚气息的理解；精神的自由，超然的唯美，民族的融合；浪漫奔放而又复古典雅，质朴脱俗而又神秘高贵；崇尚柔性的唯美、弹性的自由，将国际最新潮流时尚与民俗色彩重组创新，将柔性的优雅和弹性的浪漫、时尚与科技邂逅的唯美自由融为一体。[详细]
