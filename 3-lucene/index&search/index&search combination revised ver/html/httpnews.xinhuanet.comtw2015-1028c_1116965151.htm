国台办：盼两岸双方坚持“九二共识”续推和平发展

2015年10月28日 13:13:11 来源： 新华网
  • 评论0

　　国务院台湾事务办公室28日上午举行例行新闻发布会。图为国台办新闻发言人安峰山。 中国台湾网 尹赛楠 摄

　　新华网北京１０月２８日电（记者齐湘辉　赵博）国务院台办发言人安峰山２８日在例行新闻发布会上应询时就广州“张夏会”表示，国台办与台湾陆委会建立常态化联系沟通机制的重要基础和必要条件都是“九二共识”，希望两岸双方在坚持“九二共识”的基础上，继续推进两岸关系和平发展。

　　新华社记者在发布会上问：国务院台办主任张志军与台湾陆委会主委夏立言本月１４日在广州会面，达成多项共识。此次会面临近台湾大选，两岸关系重新面临道路选择的重要节点，请问发言人对此有何评价？

　　安峰山回答说，国台办与台湾陆委会在坚持“九二共识”的基础上，建立起了常态化的联系沟通机制。两部门的负责人互访及会面沟通，是这一机制的一个重要内容。我们要强调的是，两部门建立常态化联系沟通机制的重要基础是“九二共识”，它的必要条件也是“九二共识”，因此我们希望两岸双方在坚持“九二共识”的基础上，继续推进两岸关系和平发展。

　　有记者问：中国国民党以朱立伦替换洪秀柱参选下届台湾地区领导人，请问国台办对朱立伦的两岸政策有何看法？安峰山回答说，我们不介入、不评论台湾的选举。

　　他同时表示，就两岸关系而言，从２００５年，特别是２００８年以来，国共两党在坚持“九二共识”、反对“台独”的共同政治基础上，推动两党关系和两岸关系取得了重要积极的进展。当前两岸关系再次处于一个重要的时间节点，希望两党继续在这一共同的政治基础上，巩固和增进政治互信，坚持两岸关系和平发展的正确方向，为维护台海局势的和平稳定，造福两岸民众，继续不懈努力。

　　在回答有关问题时，安峰山表示，大陆方面对民进党的政策是明确、一贯的。２００８年以来，两岸关系之所以能够取得和平发展的良好局面和一系列重要成果，关键在于双方均坚持“九二共识”、反对“台独”的共同政治基础，而这一基础的核心在于认同大陆和台湾同属一个中国，两岸不是国与国的关系。只要认同这一点，台湾任何政党和团体同大陆打交道都不存在障碍。

　　针对有记者提及近期岛内有传言称大陆有关部门将在台湾选举期间减少赴台大陆游客数量，安峰山回答说，因为当前台湾局势处于一个特殊敏感的时期，社会气氛高度政治化，大陆组团社会考虑他们自身的利益，赴台游客也会考虑旅游环境和他们的自身权益。从以往情况看，这些情况对大陆游客赴台旅游的意愿会有不同程度的影响。我们也希望大陆居民赴台旅游健康有序的发展环境能够得到维护，大陆游客的权益也能够得到重视和保障。
