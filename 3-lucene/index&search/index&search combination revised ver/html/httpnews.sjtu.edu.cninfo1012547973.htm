[交大记忆]徐光宪学长：交大教会了我怎样去做研究[图]
[发布时间]: 2015年04月10日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 孙佳

[编者按]交通大学建校至今，始终以振兴中华、造福人类为己任，在跨越三个世纪的风雨历程中谱就了薪火传承、思源致远的历史篇章，也汇成学校文化和精神的宝贵财富。今日的交大源于那些被铭记至今和默默奉献的昨天，而无论震撼或平凡，都值得记取和思索。走进交大记忆的长廊，回瞰那些通往此刻的真实和感慨，注目历史，为了更好地迈向未来。

徐光宪学长是上海交通大学化学系著名校友、中国科学院院士，物理化学家、无机化学家。他是2008年度国家最高科学技术奖得主，被誉为“稀土之父”。2009年，徐光宪学长与钱学森学长、张光斗学长、吴文俊学长一起获得上海交通大学首届“杰出校友终身成就奖”，并在113周年庆祝大会上受到隆重表彰。位于徐汇校区的钱学森图书馆承扬钱学森精神，已广为人知，钱学长的品格和贡献深入人心。而人们或许不太熟悉，在闵行校园，也有光斗路、文俊路、光宪路，与学森路一起见证着学校的新发展。“交大记忆”在此连续推出三期，缅怀张光斗学长，致敬吴文俊学长、徐光宪学长。       

徐光宪学长生于1920年，浙江绍兴人，1944年毕业于交通大学化学系。1951年获美国哥伦比亚大学物理化学博士，同年回国任教。他在量子化学、萃取化学、配位化学、稀土化学、串级萃取理论及稀土分离工艺中的应用等研究领域作出了重大贡献。在交通大学2009年新年联谊会上，刚刚获颁国家最高科学技术奖的徐光宪学长谦虚地表示：“我有今天的成绩应该感谢母校的培养，培养我在求学上严谨的态度和勤奋的作风，这些一直支撑着我在科学生涯中获得今天的成绩。”       

徐光宪学长回忆说，1940年他考取交大时，抗日战争已经开始，学校被迫迁出徐家汇的校舍，上课只能借震旦大学的教室用，物理实验在中华学艺社实验室做，化学实验借已关闭的文化墨厂的地方做。在那样的艰苦的条件下，交大的校风一直非常严格，“由数学系主任胡敦复教授讲微积分，理学院院长裘维裕教授教物理（学二年），张我裁教授教普通化学，都是最强的教授教基础课，所以，基础课学得很扎实，至今感觉得益匪浅”。       

徐光宪所读化学系属于理学院，作为一专业，不细分无机、分析等方向，但各方向知识都有所涉略，课程非常繁重。化学系当时就开有化工课程，这是原化学主任徐名材求实业思想在教学中的体现。在徐名材的主持下，化学系的课程不搞纯科学研究，对于化学工程问题也特别重视。工程化学和与化工有关的工程应用课，约占全部课程的二分之一。在四年的学习生涯中，徐光宪学长一直保持着全班第一的成绩。       

“交通大学培养了我独立工作的能力，教会了我怎样去做研究”，说起做毕业论文的过程，徐光宪学长颇有感慨。导师顾冀东教授给出的论文题目极具前沿性，除必要的指导和帮助外，都要求自己勤学勤思勤实践。徐光宪学长的毕业论文《用萘为原料以气相催化氧化法合成制备邻苯二甲酸酐》实验中要用的铁质冷凝管也是他自己设计再到铁匠铺去订做的。       

大学毕业后，徐光宪学长给顾冀东教授担任了一年助教，课余做完了Noyes and Sherriel的Chemical Principles（《化学原理》）的全部498道习题。113周年校庆时，徐光宪学长把这498道习题作业的复印件赠予母校，勉励交大学子。他特别谈到，这些习题不是放在每一章的后面，而是穿插在主要内容之中，往往讲一小段后就是一个习题，能够很好地引导学生自己去发现并推导一个定律，形成严密的逻辑思维，这对他影响很大，使他终身受益。       

1948年，徐光宪学长申请进入哥伦比亚大学研究生院，由于在暑假班选修试读中的成绩优异，校方不仅允许他入学，还推荐他获得校聘助教奖学金。1951年，徐光宪学长获哥伦比亚大学博士学位，随即返回祖国任教。面对祖国的需要，他三次改变研究方向，其中许多领域正如他谈及放射化学时所说的，“其实放射化学等不是我的所学，也是边学边教，靠的是有那么一点基础，这样，也为国家培养了第一批放射化学专业人才和研究生。”       

“我的研究方向改变了好几回，之所以每个领域都有一些贡献，我觉得是交大培养了我独立思考的能力，能自己去解决问题。后来，面对国家的需要，多次改变自己的研究方向，都能很好地适应。”2009年4月12日，徐光宪学长回母校为年轻学子们作题为《科研创新方法学探索》的报告，他说，“回顾走过的路，我常感到学无止境，但深信打好数理基础，才能更有效地为祖国繁荣、科技兴国作贡献。”坚实的基础，使徐光宪学长在变化的专业与工作中一次又一次迅速起步，次次从头开始，次次有所建树。       

（摘编自《思源·往事》一书中“徐光宪：我的498道化学习题”，原文编者为上海交大党史校史研究室漆姚敏，该书由上海交大出版社于2012年出版）
