上海召开重点网站举报工作专题会议
2015年7月16日 10:16

    继全国百家网站去年在京签署《积极开展举报工作承诺书》后，按照中国互联网违法和不良信息举报中心近日通知精神和工作部署，根据上海市网信办领导批示精神，上海市互联网违法和不良信息举报中心于7月14日召开上海重点网站举报工作专题会议。会议布置任务，明确职责，夯实上海市互联网违法和不良信息举报工作的信息处置和报送工作相关机制。

    上海互联网新闻研究中心主任冯卫、上海市互联网违法和不良信息举报中心副主任陆雷，东方网、解放网、东方财富网、携程网、阅文集团、世纪佳缘等列入中国互联网违法和不良信息举报中心统计名录的上海市重点网站，均派出网站举报中心负责人与会，网研中心、举报中心工作人员也参加了会议。

    会议传达了中国互联网违法和不良信息举报中心《关于进一步规范举报情况统计报送工作的通知》精神，强调中央网信办对网络举报工作高度重视，做好网络举报工作，有助于发挥网民在网络治理中的参与和监督作用，同时也是网站实现行业自律、承担社会责任、维护网民权益的有效途径。上海重点网站对网络举报工作非常重视，网站对举报部门的支持力度大，取得了较好的成效，为其他网站做好相关工作起到了表率作用。会议要求上海市网站要高度重视网络举报工作，坚决守住“七条底线”，齐心协力营造积极健康的网络环境，打造网站和网民的和谐关系。

    与会的各重点网站举报中心负责人相继表示，网站将继续积极履行社会承诺，设立举报部门，公布24小时举报电话，及时处置公众举报；积极受理网民举报的违反法律法规底线、社会主义制度底线、国家利益底线、公民合法权益底线、社会公共秩序底线、道德风尚底线和信息真实性底线等违法和不良信息；及时核实举报信息，对违法和不良信息做到及时删除，并向举报人反馈。对恶意散布违法和不良信息的用户，坚决采取处置措施，并向有关管理部门报告；欢迎网民监督，对网站的举报工作提出意见和建议。

    会议就上海市重点网站举报工作及统计报送工作达成共识，一是明确举报电话必须由工作人员人工接听并登记，及时做好举报内容核实、处置和反馈；二是明确每月24日前向市互联网违法和不良信息举报中心报送上月20日至当月21日的统计数据和处置案例；三是明确保证举报内容数据真实可靠，不遗漏不瞒报。
