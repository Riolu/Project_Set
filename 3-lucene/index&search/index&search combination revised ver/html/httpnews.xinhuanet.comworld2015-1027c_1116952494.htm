王毅出席中日韩研讨会并致辞

2015年10月27日 13:38:13 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

　　新华网北京１０月２７日电（记者王慧慧、伍岳）外交部长王毅２７日出席中国公共外交协会在北京举办的以“沟通对话、合作共赢”为主题的中日韩研讨会并致辞。

　　王毅表示，李克强总理将应邀出席第六次中日韩领导人会议并对韩国进行正式访问。中日韩领导人会议时隔三年再次召开，国际社会予以高度关注，期待这次姗姗来迟的会议能够克服干扰，取得积极成果，推动中日韩合作重回正常轨道，开辟出新的前景。

　　王毅表示，朋友可以选择，但邻居不能选择。中日韩三国几千年来比邻而居，友好与合作是三国人民始终不变的共同追求。十多年来，中日韩合作为促进三国各自发展、促进东亚合作作出了积极贡献。但遗憾的是，过去几年，中日韩合作因历史问题而受到严重干扰，这种局面完全不符合三国人民的共同利益，也是我们不愿看到的。中日韩三国是亚洲重要国家和东亚主要经济体，肩负着维护地区和平稳定与发展繁荣的重大责任。三国应排除干扰，携手共进，共同朝着睦邻互信、全面合作、互惠互利、共同发展的方向前进，努力实现２０２０年建成东亚经济共同体的既定目标。

　　王毅指出，要实现这一目标，我们应当认真思考中日韩合作的正确方向，重点处理好正视历史与面向未来的关系、政治互信与务实合作的关系、三国合作与东亚合作的关系、传统合作与新兴合作的关系以及政府合作与民间交往的关系。

　　王毅强调，首先，正视历史是开创未来的前提。历史问题处理好了，三国合作就能得以推进，反之则必然停滞。希望日方真心诚意地反省过去所犯的错误，痛痛快快地与那段不光彩的历史决裂，以焕然一新的面貌与中韩两国人民携起手来，推动三国合作重回正轨，走上健康、稳定和可持续的发展道路。第二，政治互信是务实合作的基础。我们应当不断加强三国之间各个层次的对话与交往，就共同关心的重要问题随时沟通协调，不断增进相互信任，为三国合作提供坚实的支撑和保障。第三，三国合作是东亚发展的动力。中日韩合作应同１０＋１、１０＋３、东亚峰会等地区其他机制相互补充，协调发展，共同致力于东亚经济一体化和亚洲的整体振兴。三国应加强国内宏观政策的深度沟通，促进地区国家经济的融合发展，共同维护好地区经济增长和金融稳定。第四，传统合作与新兴合作应齐头并进。我们应本着互利共赢精神，早日完成建设中日韩自贸区这一进程，加快推进ＲＣＥＰ谈判，共同迈向亚太自贸区的宏伟目标。积极拓展新兴领域合作，考虑以东南亚地区为主要方向，探讨开展国际产能合作。第五，政府合作与民间交往应相互促进。当前，中日韩民间交流空前活跃，我们要顺应三国人民的意愿，继续扩大人员往来，筑牢三国友好的社会基础，为三国合作提供强大民意支撑。

　　王毅最后表示，中日韩合作离不开三国社会各界的广泛参与，希望本次研讨会集思广益、畅所欲言、献计献策，为推进中日韩合作作出贡献。外交部支持中国公共外交协会明年举办中日韩公共外交论坛，相信这将有助于促进三国交往的良性互动，为促进三国合作发挥建设性作用。

　　中国公共外交协会副会长吕凤鼎、前驻日本大使徐敦信、前驻韩国大使张庭延、日本前驻华大使宫本雄二、韩国前驻华大使李揆亨、中日韩三国合作秘书处秘书长杨厚兰、韩国国立外交院外交安保研究所所长、首任中日韩三国合作秘书处秘书长申凤吉，以及来自中日韩三国的专家学者、媒体代表等１５０余人出席研讨会，围绕“立足中日韩合作，促东北亚繁荣”、“以心相交，成其久远”等议题展开研讨。
