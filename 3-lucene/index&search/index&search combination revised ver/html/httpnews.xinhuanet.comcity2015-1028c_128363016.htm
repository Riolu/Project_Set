吉林高铁航空竞争激烈 9元机票再现

2015年10月28日 07:10:57 来源： 新文化报
  • 评论0
  • 打印
  • 字大
  • 字小

　　随着“东北最美高铁”的开通，长春到延吉乘坐高铁只需两个多小时，而且车隔差不多就半个点，这让“吃货”的天堂———延边一下子火了。

　　“东北最美高铁”的热乎劲还没散去，南航吉林分公司又对外发出消息———延吉、长春开通“空中巴士”，机票价格仅260元。紧接着，九元航空宣布，将于10月29日正式执飞长春到长白山航线，最低票价仅为9元。

　　航空公司在这时开通“空中巴士”和9元票价的航班，一场“空铁大战”难道即将上演？

　　长珲城铁“公交化”运营

　　“长珲城际铁路自开通以来，客流量持续上升，在10月1日当天，客流量出现最高峰，当天发送旅客1.4万余人次。”长春站工作人员向新文化记者介绍，从 9月30日至10月7日，长春站共发送旅客87万余人次，仅高铁列车就发送旅客近50万余人次，而长珲城际铁路在此期间，共发送旅客9.2万余人次。

　　“如今长春站发送长珲城际铁路的旅客人数平均每天有6000余人。”工作人员表示，随着这条线路的逐渐成熟，客流不断增长是必然趋势，而在今后的节假日中，在运输旅客方面也将发挥越来越重要的作用。

　　据了解，长春到延吉，乘坐长珲城际铁路的成本相对不高，二等座仅需115.5元，一等座138.5元，从早上5点多到晚上9点，车隔也就半个小时，基本实现了铁路的“公交化”运营，稳坐长春至延吉的旅客运输第一宝座。

　　“空中巴士”每天两个航班

　　“我们开通‘空中巴士’是正常的市场需求。”前不久，新文化记者从南航吉林分公司获悉，该公司正式对外公布，延吉—长春—延吉航线推出特价机票，单程260元，往返520元，为广大旅客搭建便捷的“空中巴士”。

　　据了解，“空中巴士”每天两个航班，延吉—长春航班起飞时刻为9时50分、16时10分，长春—延吉航班起飞时刻为14时25分、22时15分，全程飞行时间50分钟。

　　“这些航班经长春可以中转到杭州、长沙、西安、桂林、三亚、大阪、名古屋等国内、国际各大城市。”工作人员表示，这几个航班并不是单独运行于长春、延吉之间，而多是在长春、延吉中转的航班。

　　“开通以来旅客随到随售，每天销售量保持10张以上。”工作人员解释说，这260元为公布的票价，是不含机场建设等相关费用的。单独看这260元的机 票，似乎航空公司在做亏本买卖，而从航空市场来看，这也是吸引客流的一种方式，因此与刚刚开通的高铁并不冲突，而是让旅客出行多一种选择。

　　每个航班放10张以上9元票

　　日前，九元航空宣布，将于10月29日正式执飞长春到长白山的航线，最低票价仅为9元，这将是长春至长白山之间的首条低成本航线。

　　截至目前，九元航空已在长春开通往返广州、南京、满洲里和长白山等航线，票价均为9元起。“从10月29日至11月3日，新航线的每个航班均放出了10张以上的9元机票。”九元航空的工作人员表示。

　　新航线将由全新波音737-800型飞机执飞，逢周二、四、六、日执飞，当日往返。去程航班号为AQ1045，13：30从长春起飞，14：20抵达长白山；返程航班号为AQ1046，17：00在长白山起飞，17：50抵达长春。

　　客运优势全无 全线停运

　　在这场客运市场竞争中，公路客运可以用完败来形容。

　　“自从高铁开通之后，我们公路客运的优势全无，如今长春到延吉、敦化、珲春的大客全部停运了。”昨日，长春公路客运集团有限公司营运处处长张海波表示， 自从长珲城际铁路开通之后，公路客运受到极大冲击，长春至延吉的11台、长春至敦化的9台、长春至珲春的3台大客车全线停运。

　　“高铁对我们影响太大了，我们现在只好把更多的力量放在高铁覆盖不到的区域。”张海波表示，如今公司的线路受影响非常大，只好继续做通化、白山、农安、松原等方向的线路工作，而这些方向暂时还没有受到高铁的影响。
