Windows XP 使用校园安全无线网

 

基于Windows XP Service Pack 3 制作

先确认本计算机的时间、日期（注意时区）都与标准时间一致，差异不应超过5分钟，尽量使用网络对时服务来确认时间。

打开无线网络硬件开关和软件开关，选择察看无线网络连接

选择右下角[修改高级设置]
