[群众路线特稿]交大学指委深入实践党的群众路线 响应学生需求建一站式服务中心[图]
[发布时间]: 2014年02月24日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 郑浩 思源

自2013年7月上海交大启动党的群众路线教育实践活动以来，学指委明确学生工作系统践行党的群众路线是中央的要求，更是以学生为中心改进工作作风、全面提升工作水平的时机。学指委以各种形式广泛深入听取学生的意见，通过座谈、问卷共收集同学提出的数百条意见与建议，将相关问题指定具体负责人，制定整改时间表。

在调研中，学指委发现很多同学都提出了目前全校学生服务各种手续办理的地点太零散的问题。经过梳理学生反馈和实地调研，在校期间各项学生事务、手续的办理分散在8个楼栋20余个科室，每个楼栋办公地点相对分散，学生往返奔波，十分不方便，而且时常不知道具体的事务到底应该在哪里办理，导致空跑好几个地方。

基于“以学生为中心”的工作理念，在深入学生开展充分调研的基础上，学指委参照国际一流大学学生服务模式，建设了550平方米拥有24个办事窗口的学生一站式服务中心，并作为学指委党的群众路线教育实践活动中“即知即改”整改工作的重要内容。在设计、建设过程中，充分考虑大学生的性格特点和现实需求，色调欢快明亮，布局开放通透，中英双语指引，符合当代大学生的审美观点，也体现了交大国际化、开放性形象。学生服务区域面积和教师办公区域的面积比约为10：1，充分体现了为学生服务的理念。从服务大厅启用至今，目前可以为学生提供就业鉴证、学生活动审批、奖助学金、勤工助学近20项业务办理。同时也可作为保送生、暑期交流生等小范围群体入学报到的办理处。目前学指委正在积极联络其他涉及学生服务的机关部处，研究进一步整合全校学生服务窗口。除了人工窗口之外，学指委还将建设网上学生服务平台，研发自助式服务设施，打破空间局限，整合更多的学生服务功能，为学生提供更贴心的“一站式服务”。

一站式服务中心全景

目前学生一站式服务中心已累计服务学生超过1万人次，深受学生的好评。不仅中央电视台新闻联播、东方卫视看东方栏目等报道了大厅情况。还吸引了各级领导、兄弟高校前来调研12次。

上海市市委副书记李希同志到一站式服务中心调研

《新闻联播》栏目播出一站式服务中心

除了建设一站式服务中心之外，学指委以党的群众路线教育实践活动为契机，进一步落实“以学生为中心”的育人理念，对于自身存在的“四风问题”进行了深入的整改，及时解决并回应学生关心的问题，尤其是对一些校内学生普遍关心的热点问题，进行及时的反馈和持续的跟进。通过党的群众路线教育实践活动的开展，学指委一方面进一步强化“以学生为中心”的工作理念，改进工作作风，另一方面在全校支持下，合力推进重点整改工作。下一阶段，学指委将继续围绕“改进作风促发展，凝心聚力创一流”主线，进一步对接大学生成长的切实需求，改进工作作风，创新工作载体，固化工作制度，加快建设完善服务学生全面成长成才的学生工作体系。
