“八七会议上”的毛泽东和邓小平：毛泽东任政治局候补委员 与邓小平首次见面（组图）

2015年10月26日09:11   来源：人民网－中国共产党新闻网

位于湖北省武汉市汉口鄱阳街139号的八七会议会址。

编者按：1927年8月7日，在共产国际的帮助下，中共中央在汉口原俄租界三教街41号(今湖北省武汉市汉口鄱阳街139号)召开中央紧急会议，即“八七会议”。八七会议是在中国革命的危急关头召开的，会议正式确定了实行土地革命和武装起义的方针，把领导农民进行秋收起义作为当前党的最主要任务，指明了今后革命斗争的正确方向，为挽救党和革命作出了巨大贡献。中国革命从此开始由大革命失败到土地革命战争兴起的历史性转变。在这次会议上，邓小平与毛泽东是首次见面，毛泽东被选举为中央政治局候补委员。

会议选举出临时中央政治局 毛泽东、周恩来任候补委员

由于白色恐怖，形势紧迫，会议只开了一天就结束。出席会议的有部分中央委员、候补中央委员及中央机关、共青团中央、地方代表共22人。共产国际驻中国代表罗明纳兹等参加了会议。会议由瞿秋白、李维汉主持。经过讨论，会议选举出临时中央政治局正式委员9人，包括苏兆征、向忠发、瞿秋白、罗亦农、顾顺章、王荷波、李维汉、彭湃、任弼时。候补委员7人，包括邓中夏、周恩来、毛泽东、彭公达、张太雷、张国焘、李立三。8月9日，临时中央政治局举行了第一次会议，选举瞿秋白、李维汉、苏兆征为中央政治局常委。【详细】

会议地址选在俄租界 邓小平负责筹备并参加

中共中央在当时形势下，最后将会址定在汉口原俄租界三教街41号（现鄱阳街139号）二楼。这里是共产国际代表拉祖莫夫夫妇的住所。这幢楼房是英国人1920年修建的公寓，前后有楼梯，后门通小巷，屋顶凉台与邻居凉台相通，发生情况，便于撤离；一楼是印度人开的百货商店，周围居住的大多是富有的外国人，不大引人注意。作为会议唯一的工作人员，中共中央秘书邓小平第一个来到会场，接待代表，安排食宿，负责安全。为防万一，女主人拉祖莫娃在门口望风，代表们的午餐只是就地吃面包。会议结束后，代表们分批撤离。邓小平完成善后工作，最后一个离开。【详细】

会议因形势紧迫只开一天 毛泽东与邓小平首次见面

会议当天，代表们分批到场，有的是按通知自己找来，有的由交通员带来；有的从前门进，有的从后门入。8月的武汉，火炉一般，但为了保密，代表们进入会场就不能出去，甚至连门也不能开。邓小平作为党中央秘书，筹备并参加了八七会议。紧急会议只开了一天，他却在那里整整呆了六天，他所做的工作为会议顺利召开和圆满结束起了重要作用。八七会议是邓小平参加的第一次中央级别的重要会议。在这次会议上，毛泽东与邓小平首次见面。【详细】

毛泽东批评陈独秀的右倾错误 强调“须知政权是由枪杆子中取得的”

关于军事工作，毛泽东尖锐地指出：“从前我们骂中山专做军事运动，我们则恰恰相反，不做军事运动专做民众运动。”他着重强调：“以后要非常注意军事，须知政权是由枪杆子中取得的。”这是一个对中国革命有着极其重要意义的论断。这个论断是从大革命失败的血的教训中取得的，它指出了中国革命的特点，实际上提出了以军事斗争作为党的工作重心的问题。他建议“此次会议应重视此问题，新政治局常委要更加坚强起来注意此问题”。【详细】
