    【要闻一】王毅出席中日韩研讨会并致辞 

    外交部长王毅２７日出席中国公共外交协会在北京举办的以“沟通对话、合作共赢”为主题的中日韩研讨会并致辞。王毅表示，李克强总理将应邀出席第六次中日韩领导人会议并对韩国进行正式访问。中日韩领导人会议时隔三年再次召开，国际社会予以高度关注，期待这次姗姗来迟的会议能够克服干扰，取得积极成果，推动中日韩合作重回正常轨道，开辟出新的前景。>>

    【要闻二】外交部副部长张业遂召见美国驻华大使博卡斯

    外交部副部长张业遂27日召见美国驻华大使博卡斯，就美国军舰进入中国南沙群岛有关岛礁邻近海域提出严正交涉和强烈抗议。>>

    【要闻三】外交部：将根据地震灾情向阿富汗、巴基斯坦提供援助

    针对２６日发生在阿富汗兴都库什地区的７．８级地震，外交部发言人陆慷２７日在例行记者会上表示，中国政府将根据灾情和阿富汗、巴基斯坦两国需求，向其提供救灾援助，中国红十字会也将分别向两国提供紧急现汇援助。>>

    【要闻四】欧盟向希腊追加590万欧元 应对难民危机

    欧盟“迷你”峰会达成“17点”行动纲要之后，欧盟委员会10月26日立即采取行动，向难民危机的“前沿”国家希腊追加590万欧元以应对形势紧张的难民危机。据悉，这些援助主要来源于难民、移民及融入基金。欧盟方面期望通过这笔金援帮助希腊政府落实临时庇护场所、转移安置难民，以期在未来4个月的时间里，满足至少六万难民的合理需求。>>

    【要闻五】冲绳知事叫停决定“失效” 防卫省重启美军新基地建设    

    日本国土交通大臣石井启一２７日宣布，终止冲绳县知事翁长雄志叫停美军新基地施工行政决定的效力。防卫省将重启边野古填海造地，推进普天间基地搬迁计划。石井启一当天在例行记者会上称，若施工长期停滞，“普天间机场搬迁项目将难以为继，普天间基地附近居民将继续面临危险”。>>

    【要闻六】普京与沙特国王讨论叙利亚和中东地区局势

    俄罗斯总统普京２６日与沙特阿拉伯国王萨勒曼通电话讨论叙利亚和中东地区局势。普京与萨勒曼还讨论了中东地区局势，双方对近期以色列和巴勒斯坦局势持续恶化表达了关切。萨勒曼在电话中高度评价俄罗斯一直以来在推进中东和平进程方面发挥的积极作用。>>

    【要闻七】伊朗表示希望加入金砖银行

    伊朗塔斯尼姆通讯社26日援引该国政府经济官员哈扎伊的话称，伊朗希望加入金砖国家新开发银行。据悉，哈扎伊是在一场伊朗与巴西的经济会谈中做出上述表示的。>>

    【要闻八】欧盟委员会主席容克撰文：英国留在欧盟更好

    外媒报道，欧盟委员会主席容克27日撰文称，英国留在欧盟更好；他准备聆听英国对欧盟改革的要求。英国首相卡梅伦曾表示，他下个月将向布鲁塞尔发出一份书面的改革建议清单，如果欧盟进行这些改革，他将建议选民们在英国是否保留欧盟成员国地位公投中支持留在欧盟；他将在两年内举行这一公投。>>

    【要闻九】日本向吉尔吉斯提供多项援助 涉政治军事多领域

    据日媒报道，正在造访吉尔吉斯的日本首相安倍晋三本月26日与该国总统阿坦巴耶夫会晤。据悉，双方主要就以下等事项达成协议：日本向吉尔吉斯提供选举设备，协助议会民主制在该国扎根；日本向吉尔吉斯提供约120亿日元的日元贷款，用于修缮国际干线公路，完善交通运输领域的基础设施。>>

    【要闻十】世卫组织称吃加工肉确实致癌 质疑声音涌现

    位于法国里昂的世界卫生组织下属国际癌症研究机构２６日发布报告，把火腿、香肠等加工肉制品列为致癌物，同时说食用红肉也“可能”致癌。这是国际癌症研究机构首次对肉类进行评估。这一机构没提出具体的政策建议，但“建议限制肉类的摄入”，尤其是加工肉制品。对于这份报告，一些肉类生产商予以抨击，部分独立的专业人士则提醒在解读数字时应谨慎。>>
