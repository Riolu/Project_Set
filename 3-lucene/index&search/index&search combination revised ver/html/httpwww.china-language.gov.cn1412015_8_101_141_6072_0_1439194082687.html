国家五部委联合调研甘肃省国家通用语言文字和少数民族语文使用状况

 

2015-05-13      来源： 甘肃省民委

 

　　为切实抓好党中央、国务院关于推进民族地区各民族干部学习国家通用语言文字和少数民族语言文字重要决策部署的贯彻落实，5月4日至8日，由国家民委教育科技司副司长周晓梅带领的中央组织部、中央统战部、教育部、国家人力资源和社会保障部、国家民委等五部委联合调研组，对甘肃各民族干部学习国家通用语言文字和少数民族语言文字情况进行调研。

　　5月4日，甘肃省民委党组书记、主任沙拜次力主持召开“国家五部委专题调研工作甘肃汇报座谈会”，甘肃省民委少语古籍处、甘肃省委组织部干部教育处、甘肃省委统战部民族宗教处、甘肃省教育厅语管处、甘肃省公务员局综合处负责同志先后在座谈会上对全省民族语文工作情况、藏汉双语干部培训情况、国家通用语言文字使用情况、少数民族考生在公务员招录过程中享受的优惠政策等情况给调研组进行了专题汇报，并提出了相关意见和建议。

　　沙拜次力指出，目前，甘肃省民族语文工作方面还存在一些问题，如认识不到位、民族语文立法缺失、体制机制建设不完善、缺乏专项经费投入等问题，希望国家民委制定出台相关政策，切实从顶层设计方面解决民族语文面临的问题困难。

　　5月5日至8日，调研组一行先后前往甘南藏族自治州、阿克塞哈萨克族自治县、肃北蒙古族自治县进行实地调研，通过开展干部语言文字能力抽测、个别访谈以及问卷调查等方式，详细了解基层各族干部使用“双语”的情况，并听取了相关部门工作情况汇报。
