内蒙古举办民族学校蒙汉两种语言文字诵读/书写中华经典比赛活动


内蒙古自治区汉语委办

8月25～26日，内蒙古自治区第三届民族学校蒙汉两种语言文字诵读中华经典比赛总赛区决赛在呼和浩特举行。经过复赛选拔，来自全区各盟市、部分高校117名选手参加诵读决赛，并有320余份作品参加了书写决赛。

内蒙古是边疆少数民族地区，通用蒙汉两种语言文字。建设和谐的语言生活，必须认真处理好国家通用语言文字与少数民族语言文字使用之间的关系。蒙古语言文字作为自治区的通用语言文字之一，是行使民族自治权和文化传承的重要工具；国家通用语言文字，是国内各族人民交际和文化传承的重要工具。内蒙古自治区民族学校蒙汉两种语言文字诵读/书写中华经典比赛活动，是在我区贯彻党的十七大提出的“弘扬民族优秀文化，建设中华民族共有精神家园”和党的十八大提出的“建设优秀传统文化传承体系，弘扬中华优秀传统文化。推广和规范使用国家通用语言文字，繁荣发展少数民族文化事业”精神，以及全面落实《中共中央、国务院关于加强和改进新形势下民族工作的意见》关于“推广国家通用语言文字，确保少数民族学生基本掌握和使用国家通用语言文字，不断提高少数民族语言文字教育水平”要求的重大举措，每两年开展一次。为扩大参与面，历届比赛都采取从班到校、旗县、盟市、自治区层层组织，逐级选拔，并设小学生组、初中生组、高中生组、中小学教师组、高校学生组、高校教师组等6个组别。

推广普通话，推行规范汉字，要和教育结合，要和文化结合，要与地方实际结合。在我区开展使用蒙汉两种语言文字诵读/书写经典比赛活动，在开展中华经典诵读活动方面，我们认为找到了语言文字工作与民族文化、地方特色的结合点。我们希望通过这样比赛活动，进一步增强广大民族学校师生热爱我们蒙古语言文字和国家通用语言文字的法律意识和规范意识，养成良好的语言观和书写习惯，从不同文化经典中吸取营养，不断提高两种语言文字应用能力。这也是在我区全面实施素质教育的需要，是全区各级各类民族学校贯彻落实《内蒙古自治区蒙古语言文字工作条例》和《国家通用语言文字法》，大力宣传民族语言文字政策和国家通用语言文字政策，进一步促进蒙古语言文字的繁荣发展和国家通用语言文字的推广普及，弘扬中华民族优秀传统文化，加强民族团结教育，提高全区民族学校师生运用蒙汉两种语言文字能力的一项重要措施；是在新的形势下传承我们中华民族文明、推进语言文字工作与文化大发展、促进民族学校师生学习掌握蒙汉两种语言文字的重要举措，对建设文化大区意义十分重要。

我们将继续高度重视对民族学校师生的语言文字规范教育，进一步加强蒙汉两种语言文字诵读、书写活动的研究和实践，不断丰富活动形式和载体，为在我区推进素质教育服务。
