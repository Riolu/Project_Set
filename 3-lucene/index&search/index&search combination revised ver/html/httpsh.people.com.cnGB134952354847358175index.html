【品牌故事】中国铅笔“中华” 一个时代传奇

    今天，提及“中华”铅笔，似乎无人不知。但，又有多少人知道“中华”品牌从何而来？回首来时路，“中华”曲折跌宕，每一步都印记深深。
    20世纪30年代，中国民族工业举步维艰。今天看起来制造易如反掌的铅笔，在当时风雨飘摇的中华大地却难于登天。
    1932年香港九龙由国人投资将英商经营的铅笔厂改建为大华铅笔厂，乃是中国第一家铅笔厂。而真正完全由国人创建的铅笔厂却在三年之后才诞生。
    “铅笔在各种文具用品中占重要位置而与小学生关系尤切。虽九龙已有我国唯一之大华铅笔厂创设，然持此一厂，抵制外货，力当未逮，此种工业实大有提倡发扬之必要”――1933年，年轻人吴羹梅在日本铅笔制造企业技术学成归国，满腔实业救国热情......

品牌观察

【品牌观察】做到极致 “小产业”即是“大产业”

                            “中华”难复制 整合传承
    “中华”的注册成功是时代铸就，不可复制。因为上个世纪90年代后，要想用“中华”二字做商标必须得北京审批。现在，以“中华”注册商标的也就只有牙膏、铅笔、烟三种类别。
                               做到最极致 升级创新
    长长的流水线上，各式各样的铅笔从眼前鱼贯穿行。有的是让人记忆犹新的那种绿色、有的是充满时尚的斑斓，但他们所共有的一个特点就是共属“中华”。
                            人生第一笔 逐梦世界第一笔
    书写工具逐渐电子化的时代，你还在用铅笔书写吗？技术的进步往往会带来人类一种方式的消亡。书写类铅笔会在这个行列内吗？因铅笔而生的“中华”也逃脱不了消失的宿命吗？

品牌大事记

【品牌大事记】

    1934年7月7日，在上海八仙桥青年会召开第一次发起人会议，议定厂名为――中国铅笔厂股份有限公司，资本额银币5万元。
    1935年10月8日，中国铅笔厂股份有限公司开幕纪念日，对外正式营业。
    1936年12月21日，公司定名为中国标准国货铅笔厂股份有限公司。
    1942年9月，中国铅笔厂股份有限公司投资100万元发起创办中和化工公司，生产铅笔漆供应本公司。
    1942年12月22日，在重庆交通银行举行改组后的第一次股东大会，决定更名为“中国标准铅笔厂股份有限公司”，资本额法币800万元，任命吴羹梅为公司总经理。
