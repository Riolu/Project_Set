韩国拟推动立法严惩保健品造假

　　原标题：“白首乌事件”曝保健品市场违法乱象 韩国拟推动立法严惩保健品造假

    在韩国,随着生活水平的提高,国民对于保健、养生方面的投入越来越大,保健品市场已成为商家巨大的赢利空间。然而今年4月,韩国消费者协会一份有关白首乌保健食品造假的报告却犹如一颗惊雷,炸醒了多数消费者的“养生梦”,在韩国保健品市场引起不小的打假风波。韩国国会保健福祉委员会、韩国食品医药安全处(简称食药处)、韩国消费者协会(简称消协)等党、政及民间团体围绕保健品安全问题均呼吁加强法律监管。10月15日,韩国执政党新国家党与食药处召开党政合议会,共同商讨综合对策,拟加快推进立法,试图恢复国民对保健品的信心。

　　白首乌事件令消费者不安

　　4月22日,韩国消协发布调查结果,在韩国保健品市场中流通的32种白首乌保健品,经过DNA检测方法检测,只有3种产品真正使用了白首乌原料,其中12种产品只用了异叶牛皮消,9种产品使用的是白首乌和异叶牛皮消的混合物,其他8种产品虽标识使用白首乌原料,但未检测出白首乌成分。

　　在韩国,白首乌作为一种保健药材被编入食品法典,以此为原料可以加工成用于改善更年期综合症、提高免疫力的保健品,具有3000亿韩元市场规模。而异叶牛皮消虽与白首乌同属萝藦科植物,根茎外观与白首乌相似,但其不但不属于药材,而且还可能造成肝中毒、神经衰弱等副作用。

　　“一石激起千层浪”,韩国食药处为确认市面上流通的以白首乌为原料的保健品中混用异叶牛皮消的情况,遂对以白首乌为原料生产保健品的256家食品制造加工企业和44个保健品制造企业进行全面检查。但是,向6家保健品制造企业提供“白首乌复合提取物”原料的Naturalendo科技公司却对消协提出异议,认为消协单方面、随意提出的调查结果不科学,并且搬出2月韩国食药处的检测报告来佐证,即“经过DNA分离与增幅反应(PCR),没有发现使用假原料”。韩国食药处只好再次进行调查取证,其结果显示,Naturalendo科技公司的产品中检测出异叶牛皮消成分。

　　5月6日,韩国食药处处长金胜熙在国会接受保健福祉委员会议员质询时也表示,Naturalendo科技公司生产的“白首乌宫”保健品含有不可食用的异叶牛皮消成分。

　　这一事件引起消费者不安。消费者普遍认为,购买到假产品并不可怕,可怕的是使用了违禁原料的产品竟然在市面上堂而皇之地流通,可见食药处的监管形同虚设。正当韩国各界围绕保健品市场应如何加强监管展开激烈讨论但还未形成统一意见之时,9月17日,韩国农畜产营农合作法人又被媒体爆出制售假红参保健食品。

　　法律缺失消协有力使不出

　　据韩国媒体报道,随着保健品市场的不断壮大,一些不法商家为追求利益更大化,有意降低产品成本,甚至以次充好,制造出一些“有问题”的产品,这些产品充斥着韩国保健品市场。

　　韩国食药处公布的一份有关“保健品副作用事例申告现状”统计资料显示,保健品副作用申告事例2006年为16件,但是到2014年却暴增为1733件。截至今年6月,已接收到332件申告。在2014年向食药处申告的保健品副作用案件中,有301件与白首乌有关。

　　韩国消协4月22日公布了白首乌保健品存在的问题并向售卖假冒白首乌产品的GS家庭购物、家&购物、现代家庭购物等6家电视家庭购物公司提出给受害消费者赔偿的方案建议。但是这6家公司对于消协的建议均表现得不屑一顾,并拒绝赔偿。

　　据了解,韩国消协目前面对的尴尬现实是,其不具有执法权限,对于给消费者造成欺骗、伤害的案件,只具有建议权,不具备强制执法权。按照韩国现行《消费者基本法》,消协职权只限于发布消费者安全警报、与产品安全相关的事实公告及向制售危害产品企业提出整改建议等。消协没有权利要求违法企业提供相关资料,也不具备代表消费者对违法企业提出集体诉讼的权利。

　　韩国法律规定,当发生特定企业侵害消费者权益的情况时,只有在大韩商工会议所、中小企业协作中央会、中央行政机关等登记注册过的非营利民间团体才具有提起集体诉讼的权利,消协并不包括在内。韩国消协的弱势地位注定了其羸弱的打假力度,始终处于有力使不出的尴尬状态。无奈,消协只能把售卖“假白首乌”产品的家庭购物公司与生产企业在其网站中推广的虚假、夸张广告等不当情况以及搜集的相关资料一并提交给韩国食药处。但令人失望的是,食药处却行动迟缓,消协6月3日将有关白首乌宣传广告失实部分的调查材料提交食药处,至今4个月过去,食药处却迟迟拿不出解决方案。

　　9月14日,在韩国国会进行的国政监查会上,消协对食药处的迟缓行动表示不满。但食药处则表示,对于消协提出的相关材料及处罚请求,食药处已成立中央调查团,需对涉事企业与公司进行调查取证,目前正处于取证阶段。

　　尽快立法严惩不法造假者

　　随着“白首乌事件”不断发酵,韩国社会各界呼吁政府完善相关法律,彻查保健品市场违法乱象,严惩危害消费者权益的不法造假者。6月19日,新政治民主联合议员闵炳梪提出《消费者基本法》修正案,核心就是赋予消协提起集体诉讼的权利及为消协能够对危害消费者权益的行为实施调查提供法律保障。

　　韩国会立法调查处8月5日表示,消费者购买保健品的目的是为了有效改善身体状况,在现行的保健品认证管理体系上,应制订有效提高信赖度和客观性的政策方案。立法处指出,目前食药处在对保健品的安全与功能性认证方面,过度依赖生产企业提交的研究认证材料。“白首乌事件”暴露出来的问题就是单纯依靠企业提交临床报告缺失了审查认证的客观性。

　　为恢复消费者对保健品市场的信心,韩国食药处与执政党新国家党10月15日召开了党政合议会,就对使用违禁原料生产保健品的不法企业加大处罚力度、在对产品保健功能认证时提高国民参与度等方面达成一致意见。

　　韩国国会保健福祉委员会执政党干事李明淑议员在会后表示,对保健品市场的安全管理,不是单纯地完善法规制度,而是要把重点放在严格落实执行上。她强调,首先对使用违禁原料制造产品的不法企业应加大处罚力度,将处罚标准从现行的有期徒刑5年并处罚5000万韩元,调整为有期徒刑10年并罚款1亿韩元。

　　她表示,从强化原料及产品标准检测角度,把问题多的企业全部拉入“黑名单”的做法执行起来较为困难,但会采取禁止其销售、停业整顿等紧急应对措施。另外,针对企业一次性获得产品认证后一劳永逸的弊病,今后将调整为以5年为一个周期进行重新认证的做法。

　　李明淑议员强调,对于达成的以上意见,执政党将尽快推动立法或做出相应预算,使这些措施尽早发挥效力,整顿保健品市场秩序,恢复消费者信心。

　　□　本报驻首尔记者　单士磊
