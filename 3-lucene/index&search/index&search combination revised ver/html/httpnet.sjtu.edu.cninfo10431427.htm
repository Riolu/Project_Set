校园网 / 常见问题

2010-12-03  点击：[]

问：交大校园网是宽带网吗？带宽是多少？
答：交大校园网是国内最早建立万兆校园网的高校之一，目前是以太网架构为主覆盖了所有校区，学校各主要楼宇实现千兆接入，大部分楼宇实现桌面百兆接入。交大校园网通过中国教育和科研计算机网（CERNET）实现到internet的连接。
更多信息可访问：http://net.sjtu.edu.cn

问：网络不通，网络中心的问题还是我的问题？
答：如果发现不能正常上网，在致电网络信息中心之前，请用户协助先确认以下步骤
第一步：先查看网卡的灯是否亮的，“本地连接”是否连接正常
第二步：确认自己的IP地址、掩码、网关、DNS是否设置正确，如果不清楚自己使用的IP地址，请致电网络信息中心
第三步：利用ping命令（“开始”-->“所有程序”-->“附件”-->“命令提示符”，输入ping 202.120.2.101），检测网络是否是通的，不能ping通网关，重启计算机或使用的小路由器，尝试网络是否恢复
第四步：做过以上操作，仍不能上网，请致电网络信息中心。
注意：学生宿舍网络故障报修请联系本楼网管员或联系学生网络信息管理部。

问：外网访问很慢，怎么回事？
答：校园网属于中国教育网，互联网网站一般都是在电信网或国际网上，而教育网与电信网、国际网之间存在着互联瓶颈，因此访问电信网站的时候会慢，这属于正常的。我们也一直在努力通过各种技术手段和渠道进行网络改造或者升级，尽量缓解访问电信网、国际网速度问题。如果你访问校内或者教育网其他学校资源，速度应该是很快的。
你可以试一下校内的网络测速http://powerneting.comic.sjtu.edu.cn/ 

问：听说使用国际网有限制，是这样吗？
答：国际带宽属于紧缺资源，为保护大多数人的正常使用，防止个别用户长期占用有限的国际带宽资源，我们将给予大量下载国际网资源的IP进行警示，甚至暂时封禁。我们鼓励使用网页浏览国际网资源，不鼓励大量下载，正常的浏览网页和少量下载是不会有什么限制的。

问：某某地方为什么没有无线网？
答：无线网作为有线网络的补充，目前覆盖了教学楼等公共热点区域，考虑到无线使用率，并不是所有楼栋都会无线覆盖（有些是各院系自行覆盖）。不同的教室，信号强度可能也不一样，因此也不是每个教室都会收到很好的信号，如果信号不好，可以换个位置换个教室试一下。
无线网覆盖范围及相关配置等信息，可以访问http://wlan.sjtu.edu.cn 获取。
