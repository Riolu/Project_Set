<h1><a href='/n/2015/1012/c369519-27688442.html' target="_blank"><P align=center><STRONG><FONT size=5>关注“中国城市学年会・2015”</FONT></STRONG></P></a>
</h1>
<p class="lh22 blue">国内首家以“城市学”冠名的智库――杭州国际城市学研究中心，以探索城市化的中国模式和破解“城市病”为己任，每年围绕流动人口、教育、交通、环境、住房、土地、医疗、文化遗产八大主题所涉及的城市病开展优秀研究成果征集，以“系统综合”的研究方法汇聚国内外城市治理的理论实践前沿，每年在征集作品的基础上邀请国内权威专家开展评选，以“集成创新”的研究方法总结和展望中国城市化进程中的成功经验和重点难点，每年召开一届“中国城市学年会”。自2011年起，中国城市学年会至今已成功举办四届，在国内外城市管理者、理论研究者以及关心关注城市发展的各界有识之士的努力之下，正逐步成为中国城市学领域国际化、专业化、高端化、品牌化的学术盛会。
