“心有梦想自飞翔”学校少年宫展示活动见证成长

2015-05-11 10:29:18    来源：上海文明网   

日前，宝山区江湾中心校举办“心有梦想自飞翔”学校少年宫展示活动。区文明办、区教育局、高境镇政府等分管领导和区内各学校少年宫负责人、指导老师代表、学校家委会代表等一起参与了本次活动，。
    “梦想家”少年宫于2013年9月正式开放运行，开设了版画、合唱、网球、中国象棋、舞蹈、中国结艺等24个活动项目，共27个活动班，少年宫的成立为学生展现自我、发展特长提供了很好的舞台。此次展示主题为“心有梦想自飞翔”，大家参观了版画、中国结艺、书法等社团的学生优秀作品展，现场观摩了意大利语、网球、车模等学生社团的现场活动，观看了微视频《心中有梦自飞翔》，少年宫的学员们把学到的舞蹈、乐器、合唱等本领进行了汇报演出，共同见证“梦想家”少年宫的成长与收获。
