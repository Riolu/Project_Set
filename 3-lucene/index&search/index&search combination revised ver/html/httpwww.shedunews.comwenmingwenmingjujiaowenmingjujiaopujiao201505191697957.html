“红色印记”青少年党史普及教育活动启动

2015-05-19 10:33:45    来源：上海文明网   

围绕反法西斯战争、中国抗日战争胜利70周年暨《义勇军进行曲》诞生80周年这个主题，国歌从这里唱响一一2015年“红色印记”上海市青少年党史普及教育系列活动启动仪式日前在上海国歌展示馆举行，并向在场的学生发放红领巾红色寻访护照以及书籍《国歌的诞生》。
    活动现场，围绕“你所不知道的国歌故事”为主题，从国歌知识专家、国旗班军人、国歌志愿者学生三个不同的年龄层次、不同的角度阐述了他们与国歌之间的羁绊以及故事，让在场的青少年了解到许多鲜为人知的国歌故事。仪式最后还向在场青少年发放红领巾红色寻访护照，将上海党史教育基地的内容以手册寻访的形式融入到青少年课外素质教育中，以形式多祥的活动培养青少年对党史的兴趣，激发他们的爱国热情，并扩大“红色印记”系列活动在社会上的影响力”。
　　据悉，“红色印记”上海市青少年党史普及教育系列活动自2011年开始实施，充分发挥党史“以史鉴今、资政育人”的作用，在市委党史研究室、团市委、市党史学会、等多家单位指导下，面向全市青少年开展的党史知识普及教育活动。
