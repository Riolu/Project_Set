2015年上半年《地方领导留言板》为民解决5万项诉求

2015年07月13日09:30        手机看新闻

    2015年上半年，人民网《地方领导留言板》网民留言及办理各项指标继续走高，公开回应网友留言的市委书记市长增加了20位，县委书记增加了50位。目前，省市县三级的地区覆盖率分别为96%、76%和57%。

    上半年，栏目共收到网友留言近8万条，省市县三级领导作出的回应超过5万项，回复率为70%。各地留言办理单位平均每个工作日对近300项网友诉求作出答复。其中河南、四川、安徽、甘肃、陕西五省在全国留言办理工作中位居前列。

    上半年，多省地方领导以各种形式对进一步做好网民留言办理工作做出指示。今年1月，吉林省召开网友留言督办专题工作会，专题研究如何进一步加强和改进人民网网友留言督办工作。吉林省委书记巴音朝鲁今年以来先后两次批示人民网网友留言办理工作。

    现任辽宁省委书记李希担任省长期间，在辽宁省政府办公厅《关于人民网省长留言办理情况的报告》上批示：要求“不断总结升华，在办理好‘个案’的同时，注意制度层面的梳理、思考和制定，以形成长效机制”。

    “网上投诉问题解决得好，日常来信来访量就会大幅下降”，天津市委代书记、天津市市长黄兴国今年5月批示指出，天津市各区县、各部门“网民诉求落实工作做得比较好，从一个侧面反映了我们的工作作风有很大的转变。”

(责编：许涵霖、赵岍)
