武汉轨道交通举办收缴违禁物品陈列展

【查看原图】
2015年10月27日，武汉轨道交通举办收缴违禁物品陈列展。
1234下一页
来源：人民网-图片频道  2015年10月28日08:13

目前，武汉市轨道交通开通运营1号线、2号线、4号线76个车站共配置X光安检机103台、安检门109扇、手持金属探测仪180把。截止2015年9月，共查获各类违禁品62390件，其中仿真手枪子弹等38件，管制刀具100余把，一般刀具30621把，斧头、砍刀、锯子等4280件，发胶、喷雾剂等易燃易爆物品约2580件，警用军用击打器等物品580件，催泪器58件。（视觉中国）
