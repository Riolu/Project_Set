把意见箱放厕所让正义之举憋屈

　　一家之言

　　让正义者勇敢，需要职能部门把胸脯挺起来。你都挺不起来，意见箱放哪儿效果都一样。

　　菏泽市纪委第三巡查组进驻市城管局之后，为保护举报人隐私而出高招——把意见箱，摆在了这家单位的厕所里，原因是这地儿“能避开单位监控摄像头或太过明显的地方，以保护举报人的隐私”。

　　当地纪委也算是用心良苦。工作人员告诉媒体，意见箱设置不当，不仅不利于保护举报人，而且会导致没人愿意参与举报。理是这个理儿，但既然监控这么可怕，那么，这意见箱也是避得开室内却避不开室外。谁进过厕所，想查的总能查得到。

　　意见箱进驻厕所，菏泽市纪委的这一招，显然是从有效性去解读的，体现的是“求意见若渴”的诚意。但窃以为，人民有意见需要跑厕所里提，是典型的正不压邪。这种防止打击报复的技术含量，是在回避矛盾冲突，是无可奈何的妙招，也是本末倒置的滥招。

　　实践证明，举报投诉不畅通，是与对举报人员的保护乏力有关联的。职能部门的问题，不能及时反馈到纪检部门的耳朵里，其中重要的一条原因就是举报人怕打击报复。它导致的结果是，在一些地方、一些部门，正义之举反而沦为弱势的“过街老鼠”。前几天中纪委通报的山西某县公安局，把县委和信访局附近的2个摄像头对准省委巡视组设立的2个举报箱，便是如此。

　　党的十八大以来，信访渠道已经多样化，其中最有效的一招便是各级纪检部门开通了网络举报路径。如果正常的举报渠道畅通，并且有举必应，那么，真正弱势的不是举报者，而是那些涉嫌违法乱纪的官员。让正义者勇敢起来，首先需要维护公平正义的职能部门自己把正义的胸脯挺起来。你都挺不起来，意见箱放哪儿效果都一样。这时候，人们憋的不只是尿，憋的是屈。

　　□刘雪松（媒体人）
