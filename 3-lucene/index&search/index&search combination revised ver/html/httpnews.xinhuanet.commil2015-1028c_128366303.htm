武警官兵实战化现场会 拐弯枪无人机特战车亮相
 
2015年10月28日 08:24:06 | 责任编辑: 王瑶 | 来源： 中国军网   

　　“红军”队员正在参与“红蓝对抗”

　　射击场上，弹无虚发；战术演练，瞄准实战；小组对抗，铁血碰撞……2015年10月24日至25日，武警西藏边防总队在西藏自治区日喀则市举行警务实战化现场会，全面检验部队实战化训练成果。期间，边防总队十余个支队级单位数百名参演官兵全程参与“红蓝对抗”、“战术演练”、“武器运用”、“边境捕歼”等课目演练，全面探索新形势下高原边防保卫工作的新战法、新训法。

　　今年以来，武警西藏边防部队针对西藏边境地区点多、线长、面广，稳边控边压力巨大的实际情况，大力推进部队实战化训练，不断提升部队在高原特殊环境下遂行急、难、险、重任务的能力和水平，探索并推出了一批新的战术战法，使训练与实战无缝对接，切实提高了部队管边控边能力。（胡俊浩、许超、郑永波摄影报道）
