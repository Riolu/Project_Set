“全国党建云平台”微信公号正式上线

2015年07月01日15:54   来源：人民网－中国共产党新闻网

中国共产党新闻网北京7月1日电 7月1日，在中国共产党成立94周年之际，中国共产党新闻网推出的“全国党建云平台”微信公众平台正式上线。微信用户只需搜索微信公众号yuncpc（或直接手机扫一扫文末二维码）即可关注。

全国党建云平台是通过运用互联网和移动新媒体等技术手段实现党建宣传的数字化、信息化、智能化，为基层各级党组织在网上搭建党建信息宣传平台、典型经验展示窗口、基层党组织服务平台，从而更好地推动基层党建工作的创新发展。截至2015年7月1日，已有1314家各级各类党组织入驻全国党建云平台PC端，今日，随着微信同名账号“全国党建云平台”正式上线，全国党建云平台将进一步加快建设步伐，形成网上党建集聚宣传第一方阵。

“全国党建云平台”微信公众平台将利用互联网和移动互联网终端的优势，力争把平台建成传播党的知识、交流党建信息、推进党务公开、提供党员服务的网络交互平台。平台除发布最新党建资讯外，还将提供一系列的服务性功能，包括专题学习、在线测试、资料查询、优秀平台展示等功能。今后，全国党建云平台用户以及广大网友如有基层党建、党务工作及其他相关问题，还可以通过平台即时与工作人员进行交流互动。
