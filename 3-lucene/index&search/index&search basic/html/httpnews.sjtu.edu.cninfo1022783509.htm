交大电院获上海国际马拉松赛志愿服务突出贡献奖[图]
[发布时间]: 2015年11月02日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 洪纳娜 孙佳

11月1日，2015年上海国际马拉松赛志愿者裁判员动员及培训大会在东方体育中心召开。上海市体育局局长黄永平、东浩兰生集团副总裁唐贵发、上海市体育总会副主席陈一平出席大会。近六百名志愿者及裁判员代表参加大会，其中包括来自上海交通大学的四百名志愿者。上海交通大学电子信息与电气工程学院作为上海国际马拉松十五年来的志愿者组织方，在此次大会上获颁上海国际马拉松志愿服务突出贡献奖。  

黄永平作动员讲话。黄永平首先指出“上马”的意义重大，“上马”不仅是国际田联金标马拉松、在田径界和跑友中有重大的影响力，而且对于上海的意义在于它是给人幸福感最强的全城嘉年华。他还指出，由于马拉松运动具有一定的安全风险，因此志愿者要将跑友的安全作为工作的重中之重。最后，他表示志愿者工作是志愿者实现人生价值的载体，提出志愿者工作要“追求价值”，并衷心祝愿所有志愿者在志愿工作中得到实实在在的收获的提高。  

唐贵发向上海交通大学电子信息与电气工程学院颁发了志愿服务突出贡献奖，以表彰交大志愿者及上海交大电院在十五年的志愿服务中为“上马”作出的突出贡献。随后，唐贵发向上海交通大学志愿者代表授予了“上马”志愿者团队旗帜，以勉励今年即将奔赴前线的志愿者们继续战斗。   

陈一平向上海田径协会颁发了裁判工作突出贡献奖。上海田径协会自1996年来一直承担裁判工作。田径协会为“上马”贡献了老中青三代的专业裁判，而裁判员们始终秉承一丝不苟的光荣传统，在每个环节都做到极致。  

来自交大电院的邵华青代表志愿者发言。她回顾了“上马”二十年来的辉煌历程及“上马”志愿者十五年来的辛勤奉献。她表示，志愿者们将会以最科学的方法、最饱满的热情投入到志愿服务中，让“上马”在它二十岁生日之际再添新彩、再创辉煌。最后，她代表全体志愿者郑重宣誓：“我们定将用我们的决心、毅力，行动、汗水，为第20届上海国际马拉松赛的圆满成功，奉献全部！”   

吴玮代表赛事裁判员发言。他指出，“上马”如今取得的成绩离不开历届主管单位、领导和老同志的付出，并提议全场以热烈掌声表达对领导和老同志们的感谢。他号召，全体中青年裁判一定继承老一辈裁判的光荣传统，一丝不苟、开拓创新，将“上马”打造成国际顶尖赛事。  

动员大会结束后，志愿者及裁判们登上大巴车各自赶往工作地点熟悉赛事路线及工作流程，为即将到来的赛事服务工作做足准备。   

二十年上马辉煌，十五载志愿同舟。2015年，交大“上马”志愿者会以更加饱满的热情，更加坚定的行动服务“上马”，继往开来，再创辉煌。   

上海交大志愿者至今已连续服务“上马”赛事十五年，上海交大电院青年志愿者服务队承担了赛事志愿者的招募与组织工作。近年来，每届“上马”都有来自交大的两千余名志愿者奔赴赛场，为赛事后勤、运动员安全保障、能量补给、成绩记录等工作提供了大量帮助和支持，保障了赛事有序开展。
