电院首届国际青年学者论坛聚焦“智慧城市”[图]
[发布时间]: 2015年10月30日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 洪纳娜 孙佳

10月29日上午，以“智慧城市”为主题，首届“国际青年学者论坛”在上海交通大学电子信息与电气工程学院3-200报告厅拉开帷幕，来自海内外三十余名信息技术领域的青年学者出席论坛。 

本次论坛以“智慧城市”为主题，涉及了与信息科技相关的所有学科，不仅包括基础的实验研究，也包括实际的社会应用。论坛旨在为世界各地的青年学者提供一个交流的平台。众多青年学者可在论坛中交流思想，互动合作，相互启发，不断地在高速发展的信息科技领域迸发出新的灵感。 

电子信息与电气工程学院院长毛军发出席并致辞。毛军发向来自世界各地的青年学者和特邀嘉宾表示诚挚欢迎，并介绍了本次论坛的主题，指出本次论坛对电院、对交大的重大意义。副院长刘辉介绍了交大和电院的发展历史、专业方向、教学与科研成绩等多方面。嘉宾华盛顿大学电气子工程系主任Radha Poovendran教授和上海交大计算机系系主任过敏意教授做论坛特邀报告。Radha Poovendran教授讨论了Smart Connected Communities。过敏意教授从大数据的角度谈论了“智慧城市”。 

下午，围绕智能交流、大数据安全模式、微电子技术应用、媒体与绿色能源四个大主题，来自微软亚洲研究院，广东虚拟现实科技公司等科研机构的研究人员，哈佛大学，新加坡南洋理工大学等著名学府的学者展开了讨论交流。由张峰牵头的关于5G通讯、无人车载天线组网、节能无限通讯设计与可靠加密技术等方面的讨论，让通讯领域的研究有了更广泛的合作。计算机系主导的关于高效计算技术、尺度化数据分析、虚拟现实、医学图像处理、神经网络图像处理目标识别等方向的研讨，与生活息息相关。关于多能源系统操作分析，纳米电子技术存储创新等方面的绿色能源研究，做到了发展高新技术与环境友好的有机结合。  

在此次大会上，来自全球各地的学者交流沟通，青年学者得以零距离交流接触，深化各个领域的友好合作，促进了“智慧城市”的研究发展，同时也为开展更广泛的国际交流活动奠定良好基础，促进了先进科学文化在全球范围内的流动，为全人类信息技术事业的进步和发展助力。 

10月30日，各嘉宾及青年学者游览了交大美丽的校园，参观电院相关实验室。
