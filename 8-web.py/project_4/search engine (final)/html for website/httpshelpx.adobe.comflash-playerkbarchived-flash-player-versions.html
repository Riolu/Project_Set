Disable auto update

Flash Player is normally set to auto update mode based on your selection of the "Install Updates Automatically When Possible (Recommended)" check box during installation. This method is recommended.

If you have to install only an old version and don't require auto update (due to system restrictions, version conflicts, and so on), do not select the check box during installation. Instead select the "Notify me when updates are available" check box and click Done.
