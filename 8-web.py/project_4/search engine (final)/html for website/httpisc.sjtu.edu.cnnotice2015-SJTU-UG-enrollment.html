2015上海交通大学本科留学生4月份录取名单公布

 

各位同学：

2015年上海交通大学本科留学生4月份录取名单现已公布。请大家凭报名号（准考证号，20150开头）到以下名单中进行查询：

2015上海交通大学本科留学生4月份录取名单

 

另外，有以下几点希望大家能够注意：

  1. 1. 已被录取的学生请按照预录取通知书要求在规定时间内预交部分学费并确认就读。

  2. 2. 未在名单内的学生请5月7日前登陆系统查询状态，可提交HSK证书 (4级以上) 报考国际教育学院汉语言本科专业 (只需在原申请表中将学习项目的第二志愿改为汉语言，然后提交报名表即可)。

  3. 3. 如不想报名汉语言，可选择申请我校本科预科项目（文科类），具体内容请参考：http://isc.sjtu.edu.cn/UG_foundation_admission.html

  4. 4. 第一志愿为国际教育学院的学生以及中国政府奖学金候选学生不在本名单之内；

  5. 5. 已被录取学生如还没有邮寄原始纸质材料，请尽快将材料邮寄至留学生发展中心。

地址： 上海市华山路1954号桃李苑1011室

 

恭喜所有被我校录取的学生！欢迎加入上海交通大学！
