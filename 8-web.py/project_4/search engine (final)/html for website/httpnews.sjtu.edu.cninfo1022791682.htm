上海交大多项高科技成果亮相第17届工博会[图]
[发布时间]: 2015年11月05日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 曹杰

11月3日上午，第17届中国国际工业博览会在国家会展中心（上海）拉开帷幕。本届工博会以“创新、智能、绿色”为主题，设有工业自动化、节能与新能源汽车、机器人和航空航天技术等9大专业展区，超过2200家参展商参展。   

上海交通大学15项高科技成果参加此次博览会。在学校有关院系的积极支持下，科研院精心组织了参展项目，内容涵盖了先进装备制造、机器人、医疗设备、新材料、临床医学等多个领域。不少参展项目获得过国家科技进步奖，具有很强的创新性和工业应用价值。   

11月4日下午，上海市政协主席吴志明，上海市经济与信息化委员会主任李耀新，上海交通大学校长、中科院院士张杰，上海市教委副主任袁雯，上海交大副校长梅宏，在上海交大科研院常务副院长关新平的陪同下，参观交大展台，并听取工作人员的项目介绍，对交大的参展技术成果和参展工作给予充分肯定。   

交大展出的科研成果格外吸引眼球，展台一直人气火爆。交大的参展项目，包括机械与动力工程学院彭颖红老师的“折叠式微通道管辊弯成形制备工艺及装备”，电子信息与电气工程学院管海兵老师的“虚拟机运行支撑关键技术与应用”，附属第六人民医院的“脑动脉瘤及相关血管无创成像和微创治疗新技术”，电子信息与电气工程学院唐厚君老师的“大功率数控精细等离子切割机”和“电动汽车边走边充蓄电技术”，电子信息与电气工程学院苏剑波老师的“纳豆服务机器人”，电子信息与电气工程学院刘兴钊老师的“地基多功能雷达系统的应用”，化学化工学院王仕峰老师的“高性能沥青路面新材料及制备技术”，机械与动力工程学院胡鸣若老师的“伤口治疗仪”，电子信息与电气工程学院杨明老师的“面向心力衰竭辅助治疗的人工心脏辅助装置”，电信学院王勇老师的“新型有源电力滤波器与静止无功发生器”，电信学院张卫平老师的“指甲尺度扑翼微飞行器”，化学化工学院孟庆华老师的“基于光固化的全色系多功能3D打印材料”，附属第九人民医院的“眼眶外科修复材料和关键技术的研发和应用”，附属新华医院的“眼眶外科修复材料和关键技术的研发和应用”等项目，吸引了很多观众前来观摩问询和洽谈。   

“中国国际工业博览会”创办于1999年，由国家发改委、商务部、工信部、科技部、中国科学院、中国工程院、中国国际贸易促进会和上海市人民政府共同主办，中国机械工业联合会协办，已成为中国工业经济领域规模最大、功能最全、水平最高的展览盛会之一。据悉，本届工博会将持续到11月7日。
