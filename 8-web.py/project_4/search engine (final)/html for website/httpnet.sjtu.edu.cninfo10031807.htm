虚拟主机变更通知

2015-08-03  点击：[]

虚拟主机托管用户，你们好！
因为网络信息中心内部服务器近期有调整，或许会给虚拟主机的管理和使用带来一些问题。
如果你们在使用过程中发现异常，请详细记录下，并通过邮件告诉我们。

联系我们：
如果您是虚拟主机申请的“负责人员”、“联系人员"，那么请查阅虚拟主机申请时的邮件往来，当时，我们是有邮件发给你们的，请联系当时给你们发邮件的网络信息中心工作人员；
如果您已经找不到当初的联系邮件，那么请发邮件到：services@sjtu.edu.cn；

特别说明：
原则上，我们只会跟虚拟主机申请时表单上的”负责人员“和”联系人员“，联系和反馈虚拟主机信息。
如果您不在上述人员范围内，但目前又确实是在参与虚拟主机管理和使用的，那么请联系虚拟主机申请表单上的”负责人员“和”联系人员“，让他们做身份审核和确认。

上一条：中国高等教育学会教育信息化分会网络信息安全工作组在我校成立 下一条：网络信息中心2015年暑假工作安排
