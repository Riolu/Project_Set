关于微软停止对WindowsXP、Office2003软件服务支持的公告

2014-03-04  点击：[]

微软Windows XP、Office 2003支持服务终止的影响

　　微软已宣布将于2014年4月8日停止对Windows XP，Office 2003等软件的支持。届时微软不会再针对新的漏洞提供任何补丁修复，硬件厂商也不会针对XP提供新的驱动更新。
　　虽然运行Windows XP的电脑过了4月8日仍然能够工作，但面对层出不穷且变化多端的网络安全威胁，Windows XP系统将极有可能会被病毒、黑客入侵，进而造成资料泄露或系统错乱。已走过十多年的Windows XP系统也无法更好的支持很多新的应用，我们建议用户尽快由Windows XP系统过渡到Windows 7或Windows 8/8.1的版本。

 

 我电脑的Windows XP能升级到Windows 7、8或8.1么？
　　Windows 7及更新的系统对硬件要求比Windows XP有所提高，如果想在现有电脑上升级系统，硬件配置最好不低于2006年购买的主流配置。具体限制：
　　CPU：主频最低1GHz（建议至少配置双核CPU）；如安装64位操作系统会有更多的限制；
　　内存：大小最低要求为1G（建议配置2G或更多的内存）；
　　磁盘：最低16G空间，建议保留100G以上的空间，磁盘利用率最好不超过75%；
　　图形卡：有WDDM驱动以支持DirectX 9
　　驱动：很多电脑厂商没有为旧的设备准备新系统的驱动程序，强行使用旧驱动程序可能影响新系统的使用和稳定性。
　　
　　Windows 8.1 并非设计用于安装在运行Windows XP的旧电脑上，因此在考虑升级之前，建议运行Windows 8 升级助手，检查电脑是否达到最低要求。


我的电脑能使用校内微软正版服务么？
　　校内微软正版服务适用于学校公共服务场所使用的用学校的资金购置的PC机，公共场所包括办公室、实验室、教室、公共机房。另外学校教职工可以在自己的家中的一台机器上安装提供的软件。使用本服务的电脑需出厂预装微软操作系统。
　　出厂时购买了预装微软操作系统许可的电脑可将现有系统升级到指定的企业版本，而裸机购买（出厂时未预装windows操作系统）的一般不能安装交大的微软正版操作系统。
　　可以通过检查台式电脑机箱、笔记本底部的微软正版标签来识别电脑购买时是否预装微软系统。
　　更多信息，参考 http://mvls.sjtu.edu.cn 网站。

 

我应该升级到哪个操作系统？
　　Windows 7更接近Windows XP的界面，熟悉起来更快；但新装的Windows 7 （已集成Service Pack 1）还有近百个系统补丁需要安装，安装补丁会耗费时间并占用磁盘空间。
　　Windows 8/8.1界面变化较大，熟悉需要一些时间，取消开始菜单可能会改变用户使用习惯；花时间熟悉后新系统的操作还是很便捷的；新系统开机快，需要安装的补丁也较少；建议对系统熟悉的用户使用。

 


我应该选择32位还是64位操作系统？
　　64位系统是趋势，大部分32位程序都可以正常在64位系统里运行，64位程序无法在32位系统中运行，64位系统内存和磁盘占用都会稍多一些，4G或更多内存的机器最好选用64位系统。
　　如果需要使用特殊硬件如加密卡、旧式打印机等，请先确认这些设备有新系统的32位或64位驱动。

 


升级操作系统会影响我的现有程序么？
　　Windows 7/8/8.1不支持从Windows XP无缝升级，用户需要使用安装介质（光盘、U盘等）全新安装。所有用户的文件和之前安装的程序都不会自动迁移，用户需要手工重新安装程序的对应新版本，并复制自己的文件到新系统。

　　微软提供了Windows 轻松传送工具来减少用户迁移系统的工作量。

 


网络中心的服务会有什么变化？
　　随着Windows XP的服务终止，网络中心可能会逐步终止对旧系统的服务支持。这可能包括停止针对旧系统的网页适应性测试，停止相关技术支持，删除相关下载内容，推出、改用更安全但旧系统无法匹配的服务等等。

 


Windows Server 2003会受到影响么？
　　根据微软产品生命周期网站给出的信息，Windows Server 2003扩展支持周期至2015年7月14日，建议用户尽早准备更新系统。
