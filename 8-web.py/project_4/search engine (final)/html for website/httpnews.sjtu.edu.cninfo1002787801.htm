上海交通大学南明光纤互联网实验室揭牌[图]
[发布时间]: 2015年11月06日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 洪纳娜 孙佳

11月6日，上海交通大学南明光纤互联网实验室揭牌仪式在上海交大闵行校区举行。上海交通大学电子信息与电气工程学院党委书记苏跃增、中国光纤网络系统集团有限公司（以下简称“中国光纤”）执行董事孟欲晓为联合实验室揭牌并致辞。上海交通大学科学技术发展研究院常务副院长关新平、上海交通大学先进产业技术研究院党支部书记罗哲，奉贤区政府相关领导、上海市和闵行区“千人计划”科技专窗相关领导，复旦大学合作伙伴和南明光纤代表出席仪式。揭牌仪式由上海南明光纤技术有限公司（以下简称“南明光纤”）董事长夏霓主持。上海交通大学致远讲席教授、中组部“千人计划”特聘专家、区域光纤与新型光通信系统国家重点实验室主任何祖源作总结发言，并展望了联合实验室的未来发展。  

联合实验室的成立得到各级政府领导的高度重视。光纤技术的普及和发展已经深刻影响到国民经济和人们的日常生活。光传感和光互连的应用价值将无法估量，联合实验室开发的技术和产品也将助力发展智慧城市。 

苏跃增代表学校对联合实验室的建成表示祝贺。他表示，上海交大与南明光纤及其母公司中国光纤建立了成功的校企合作典范。资本投入、科研合作、技术共享等都给双方带来了新的发展机遇。联合实验室的成立，将进一步推动和深化上海交大与行业内各高科技企业的合作，促进光互连领域的长足发展。 

孟欲晓感谢上海交通大学和何祖源教授团队在光互连技术产业化等方面所作出的贡献。作为母公司的执行董事，他表示将继续积极推动上海交大和南明光纤在技术和人才等方面的合作，促进双方合作共赢的长远发展。 

何祖源教授介绍了联合实验室的筹备和发展情况。对于实验室重点研究的光互连技术产业化，他表示，要实现“互联网+”必须大幅度提升网络容量，提高网络速度，而这其中的关键支撑技术就是光互连。何祖源教授还提到，光传感和光互连技术是智慧城市的光纤“神经网”。他表示，智慧城市的概念不仅是人们常说的智能家居、智能交通等，还包括运用信息和通信技术感测、分析、整合城市运行核心系统的各项关键信息，对各种需求做出智能响应，而这些都离不开光传感和光互连技术的发展。 

苏跃增和孟欲晓共同为联合实验室揭牌。 

揭牌仪式后，领导和嘉宾们参观了联合实验室。经过近一年筹备，实验室已经初具规模：新增实验室面积200平方米；购置高速全光采样示波器、3D显微成像和表面测量系统、100G有源光缆评估系统、平面光波导制备系统等大型设备；搭建五个光互连技术实验平台和一个评估测试平台。实验室现有固定教学科研人员5人，在读博士和硕士研究生共29人。  

联合实验室的科研团队是一支从海外归来的队伍，多年来在一起从事科研工作的经历使他们彼此相互了解、配合默契。在研究路线和风格上，他们坚持“既要高大上，又要接地气”，也就是既开展真正面向国际学术前沿、具有国际先进水平的研究，又紧紧抓住成果转化环节，推进新技术的产业化应用。来到交大后，这支队伍不仅在科研上攻坚克难，取得了一系列创新性科研成果，而且真正实现了科研与产业相结合。  

此次与上海交大合作的南明光纤是国内光通信行业的一匹黑马，由中国光纤的全资子公司河北四方通信设备有限公司与何祖源教授团队于2014年合资成立。南明光纤主要从事特种光纤及光纤器件等多种光通信和光传感产品的研发、生产和销售。南明光纤已投资1000万元用于联合实验室的项目建设。未来三年，南明光纤还将继续投入约1500万元的科研经费。  

夏霓表示，南明光纤的目标是成为行业领先的技术型公司，成为资本与技术结合的典范。未来，南明光纤将瞄准B2B市场，利用自身背靠上市公司的优势，充分发掘技术、资本及渠道资源，打造上海“智”造型科创企业。
