中国高等教育学会教育信息化分会网络信息安全工作组在我校成立

2015-10-13  点击：[]

       2015年10月10日，中国高等教育学会教育信息化分会“网络信息安全工作组”成立大会暨高校网络信息安全研讨会在上海交通大学徐汇校区召开。上海交通大学张安胜副校长、教育部教育管理信息中心网络运行处安宏处长、上海市委网信办杨海军副总工程师、上海市教育信息中心王明政主任、中国高等教育学会教育信息化分会蒋东兴理事长、种连荣副理事长等领导出席大会并在会上讲话或做主题演讲。

   张安胜副校长在致辞中指出，互联网已经融入了国家发展、社会进步、人民生活的各个方面，深刻改变了人们的生产和生活方式，高校受到的影响也越来越深，面临的网络信息安全挑战越来越大。及时成立网络信息安全工作组，通过汇聚全国高校网络信息安全的力量，发挥交流、协作、共享的优势，共同应对高校网络信息安全的威胁，其意义非常重大。上海交通大学有强大的网络信息安全学科，网络信息中心在网络运行、安全监测、攻防演练、安全审计、数据分析等方面也有一定经验。我们愿意和兄弟高校分享成果和信息，相互切磋，相互学习，共同为教育行业的网络信息安全作出贡献。 

   据悉，网络信息安全工作组由我校网络信息中心顾一众主任任组长，将联合全体成员从调查研究、技术测评、联合防御、示范推广等方面开展工作，为全国高校提供网络信息安全制度规范、信息交流、技术协作、培训演练等服务。 

   本次会议吸引了来自高校、企业的网络信息安全专家以及全国100多所高校的200多位相关成员参加，就高校网络安全的制度建设、等级保护、威胁监测、技术防范和信息共享等热点问题，分享了最新的研究成果和行业动态。
