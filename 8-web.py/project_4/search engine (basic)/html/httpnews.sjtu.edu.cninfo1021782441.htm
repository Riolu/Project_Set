上海市市长咨询会议企业家走进上海交通大学[图]
[发布时间]: 2015年11月01日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: snew

2015年10月31日下午，作为上海市市长企业家咨询会议（IBLAC）今年新设的会外活动环节，“IBLAC企业家走进上海交通大学”论坛活动在上海交通大学徐汇校区安泰经济与管理学院报告厅举行，众位IBLAC成员企业家代表与交大师生代表一起共话创新人才与创新精神。上海市市长杨雄，WPP首席执行官苏铭天爵士等10位IBLAC企业家，上海市人大常委会副主任、上海交通大学党委书记姜斯宪、校长张杰、党委常务副书记郭新立，上海市政府和有关委办局代表，IBLAC企业代表和交大师生200余人共同参加了活动。活动由安泰经济与管理学院院长周林主持。     

杨雄在致辞中指出，市长咨询会议创立27年来，国际企业家献计献策，为上海发展贡献了宝贵的智慧和力量。创新是城市发展和社会进步的不竭动力。上海一直非常重视创新，特别是面对全球新一轮科技革命和产业变革的重要机遇，上海正积极推进以科技创新为核心的全面创新，着力重构城市发展动力，激发全社会创新创造活力，加快向具有全球影响力的科技创新中心进军。杨雄表示，创新事业需要创新精神，呼唤创新人才。大学是培育创新精神的摇篮，是培养创新人才的基地。今年市长咨询会议创新会外活动形式，增加了国际企业家走进校园活动，邀请市长咨询会议部分成员和嘉宾与交大师生共话“创新人才”与“创新精神”，让大家在面对面的互动交流中，竞相迸发智慧火花。既充分体现国际企业家对大学生创新创业的关心和支持，也进一步增强市长咨询会议的公众参与度和社会影响力。     

姜斯宪在致辞中对上海市市长国际企业家咨询会议的首次会外活动选择上海交通大学表示感谢和欢迎。姜斯宪表示，即将迎来120岁生日的上海交通大学，是我国历史最悠久的现代高等学府之一，上海交通大学始终坚持面向世界科技前沿、面向国家重大战略需求和未来产业发展，致力于推动人类健康和引领未来产业发展，致力于创新人才培养，秉持一流大学与世界城市共生互动的办学理念，上海交通大学一定能够为上海建设具有全球影响力的科技创新中心做出重要的贡献。创新人才如何培养，创新生态如何营造，是全社会都非常关心的课题，希望此次活动成为企业家与大学生沟通对话的崭新平台，取得丰硕成果。     

本届IBLAC主席、WPP首席执行官苏铭天爵士在致辞中表示，非常高兴和大家探讨创新精神。苏铭天表示，这次IBLAC来到上海交通大学，相信我们也是和上海以及中国未来的领导人做交流。他表示希望论坛能够激发在座各位的思想，共同探讨如何更好地培养创新人才。希望能够营造一种创新的精神、创新的文化、协同的精神，并且能够真正地付诸实施。     

创新人才应该如何培养？创新精神该如何培育？与会嘉宾围绕“创新人才的需求与培养”“创新精神与创新文化”主题进行了论坛交流，大家分享企业的成功和发展，畅谈对人才及创新的感悟和体会。普华永道全球主席戴瑞礼先生应邀担任论坛主持。     

ABB集团总裁兼首席执行官史毕福博士表示，ABB集团有124年的历史，没有创新就无法生存，创新是ABB的基因，也是ABB的标志，公司每年大概要花15亿美元投入到研发，全体的员工都需要能够抗起创新的大旗，每天不断创新，走在前面。     

安永全球主席兼首席执行官马克•温伯格先生表示，创新可以获得奖励。事实上经常会有一些好的想法，但难的是如何把好的想法转化成价值，如何落实和执行。他认为：重要的是去学习，而不仅仅关注当下的事情，学习就是如何让多元化的团队团结、能够倾听其他团队成员的点子、彼此合作、最终能够协同创新。     

贝卡尔特集团董事长狄镐夫先生认为，合作非常重要，一个团队中并不是一部分人负责创新，另一部分负责和客户合作。公司每年都鼓励人们选择好的想法，“如果你的点子成功通过过滤，会获得资金支持”，有很好的创新成果，会获得认可和嘉奖，已经形成一种激励机制。狄镐夫强调，创新是很重要，失败也是很重要，在鼓励创新的同时也需要包容失败。     

上海交通大学校长张杰在论坛中介绍了上海交大在人才培养及用科技创新造福社会取得的成绩，强调中国的大学不仅应该具有科学精神，还需具有人文精神；没有人文情怀引导的创新，对人类社会不会有太大意义。上海交通大学强调的是创新精神与人文情怀的融合。学校在创新人才的培养中，特别重视培养学生能力建设：培养学生批判性思维的能力、提出问题并解决问题的能力、知识整合的能力、沟通的能力。经过多年努力，交大人才培养已经走到了新的阶段。     

美国国际集团总裁暨首席执行官韩沛德认为，保险行业已经生存数百年而没有大的变化，但在当今社会也要拥抱科技、拥抱创新。韩沛德比较注重“同理心”，也就是“换位思考”，认为公司要从客户的角度去思考。同样道理，创新来自于人们之间的合作，合作者可能是不同的，都有不同的视角，学习的背景也都不一样，双方要有很好的理解才能有创意的火花。如果对待不同的背景的人也能有“同理心”，就能有更多的文化。     

蒂森克虏伯集团董事会主席兼首席执行官海里希•赫辛根博士表示，企业如果无法持续创新就有可能消亡，不管技术还是服务，一旦停止了创新的步伐，就有可能“逆水行舟，不进则退”。我们要把创新的无形“有形化”：CEO很重视创新，团队的人也会重视；我们还要打造平台，在内部集聚人气，与外部加强交流。海里希•赫辛根认为，最大的挑战就是合作的能力，最值得称奇的成果，往往是不同的领域擦出的火花。应该营造让大家都可以献计献策的文化。     

赢创工业集团董事长英凯师认为，创新是非常重要的。很多杰出的工作，不是在一个学科方面，或者仅仅是在工程方面的单独发展，而是IT、医学、化学、工业等领域共同的整合。对于一个大型的项目来说，可能需要各个方面的专家，必须进行跨学科的协作，应该营造起这样的文化。     

论坛嘉宾还与在场交大师生进行了互动交流，回答现场提问，并共同围绕创新人才、创新文化、创新发展、企业发展等话题进行深入交流。   

论坛开始前，部分IBLAC成员企业家参观了上海交大校史馆。     

上海市市长国际企业家咨询会议1989年由时任上海市市长的朱镕基同志创办，此后每年秋天举行的市长咨询会议，针对上海经济发展中的1-2个突出问题进行专题讨论，为上海市市长提供咨询和建议。目前已拥有16个国家的43位成员和7位名誉成员，行业分布几乎涵盖了所有领域。作为会议今年首次增设的全新环节，“IBLAC企业家走进上海交通大学”活动，紧扣 “上海建设具有全球影响力的科创中心”的主题，聚焦“创新人才”与“创新精神”，分享企业家的智慧和观点，旨在构建企业家与高校师生互动交流的平台，在创新氛围中实现校企之间的良性互动，激发更多创新智慧与实践。
