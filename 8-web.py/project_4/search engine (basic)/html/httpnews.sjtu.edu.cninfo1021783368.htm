上海交通大学2015年学生暑期实践活动总结大会召开[图]
[发布时间]: 2015年11月02日
[推荐新闻]     [我要纠错] 字号：[大 中 小]
[责任编辑]: 殷自强 思源

10月30日下午，“行万里路，知中国情”上海交通大学2015年学生暑期实践活动总结大会在闵行校区陈瑞球楼报告厅召开。校党委副书记朱健，总装备部驻交大选培办主任耿梅娟，组织部、宣传部、学指委、教务处、研究生院、国际合作与交流处、发展联络处、校友总会办公室、地方合作办公室、档案馆、团委等相关部处和学院负责人，以及参与暑期实践活动的师生代表参加大会。大会由校团委书记方曦主持。  

学指委秘书长林立涛介绍了2015年暑期实践活动总体情况。今年暑期实践互动“同学踊跃、组织有序、成果丰硕”，共有537支团队，5811人次赴全国参与实践活动，有327位教师参与指导。林立涛从注重顶层设计和项目培育、切实提高实践质量、加强安全保障、完善支持体系、注重总结凝练、充分发挥优秀团队和个人的积极影响等方面进行总结，并指出引导学生在校期间多次参与社会实践，推进社会实践常态化建设等后续工作重点。  

医学院董艺蕾作为实践学生代表分享“山山圆梦”团队前往偏远地区支教留守儿童的感悟，收获和课堂学习不一样的成长。环境科学与工程学院王欣泽作为教师代表，分享连续6年悉心指导云南洱海污染调研与治理实践团的切身经验与感受。  

机械与动力工程学院“智慧青春行 工业强国梦——基于‘中国制造2025’的智能工厂发展现状及前景展望”团队，船舶海洋与建筑工程学院的“120周年校庆文化设施修缮”团队与“雪域天珠 西藏江孜县建筑遗产保存状况调查与风险评估”团队，电子信息与电气工程学院的“‘人潮肆意，何所适从’——地铁站内人群分流优化”团队，选培办指导的“重走抗战十大战役”国防生团队等分别展示了实践成果。既有追溯历史，又有面向未来，既躬身实践，又学以致用，既立足校园，又关注社会，展现了交大学子在实践中的所思所想与社会担当。   

大会为获奖先进个人、优秀指导老师、十佳项目团队、优秀组织颁奖，以表彰他们在实践中的突出表现。  

朱健作总结讲话，肯定了老师和同学的努力和表现，向圆满完成暑期社会实践活动的团队表示祝贺。朱健指出，新形势下的社会实践及实践育人工作要紧紧围绕立德育人这一主题，从“争做社会主义核心价值观的代言人”、“加强创新创业实践”、“深入开展社会调查活动”三方面对大家提出了更高的期望。朱健寄语大家不忘初心，不负光阴，在来年能有更加优秀的表现。  

暑期实践活动总结大会象征着2015年暑期实践活动画上圆满句号。交大学子将青春投洒在祖国的山川大地，用激情书写属于自己的热血篇章，期待更多的交大学子投入实践之中，不负青春年华。
